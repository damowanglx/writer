## 一、当前热门方向（番茄平台）

### 🔥 第一梯队：规则怪谈 × 无限流（绝对霸主）

| 指标 | 数据 |
|------|------|
| 代表作品 | 《十日终焉》(296万字/9.9分/400万在读)、《诡舍》、《规则怪谈：从山坟开始》(单日+10.9万在读) |
| 流量特征 | "夜间流量王"——夜间阅读转化率远超其他类型 |
| 核心读者 | 18-35岁男性为主，偏好智斗+解谜+反转 |

**爆款公式**：
- 规则触发(0-300字) → 危机觉醒(300-800字) → 首个小爽点(800-1500字) → 章尾新钩子
- 主角智商全程在线，利用规则漏洞反杀
- 每500字至少1个小悬念，每章结尾"悬崖急刹车"

### 🔥 第二梯队：悬疑灵异 × 民俗风水（复兴赛道）

| 代表作品 | 《我以狐仙镇百鬼》(41万在读)、《誓不成仙》(55万在读) |
|----------|------|
| 流量特征 | 起点重启灵异分类，作者+读者双回流番茄 |

**爆款要素**：特殊职业主角(殡葬师/守林人/749局)、融入真实民俗/风水知识、靠氛围不靠血腥

### 🔥 第三梯队：诡秘暗黑修仙 × 克系恐怖（新蓝海）

| 代表作品 | 《诡道神话》《我不可能是妖魔》《前方高档小区，伪人止步》 |
|----------|------|
| 优势 | 竞争较小、差异化强、读者新鲜感高 |

**核心设定**：吞鬼修仙、序列觉醒、克苏鲁污染、都市高武+恐怖撕裂感

## 二、番茄平台爆款共性公式

### 开篇铁律
```
0-300字   → 黄金冲突钩子（生死/背叛/死亡威胁）
300-800字  → 金手指/能力激活
800-1500字 → 第一次爽点小爆发
1500-章尾  → 新悬念钩子/更大危机
```

> **前三行没冲突 → 80%读者直接划走**

### 章节节奏
- 每章2100-2300字，日更4000+
- 每500字至少1个小悬念
- 每章1-2次情绪峰值
- 情绪曲线：虐→爽→虐→大爽
- 前3章必须出"大钩子"（规则触发/觉醒/上交国家）
- 第5章必须有第一个大转折

### 数据门槛
| 指标 | 标准 |
|------|------|
| 首章读完率 | ≥40% |
| 完读率 | ≥65% |
| 收藏:阅读比 | >1:8 |

## 三、读者审美疲劳 / 避雷指南

### ❌ 已经被读者吐槽到烂的套路

1. **副本场景高度同质化**：村庄、学校、寺庙、医院四大件反复出现
2. **恐怖描写模板化**：只会用"头皮发麻""浑身发冷""惊出一身冷汗"
3. **设定大搬运**：《诡舍》被指与《死亡万花筒》《地狱公寓》核心设定高度重合
4. **主角降智**：为了制造恐怖而让角色做不合逻辑的事
5. **解释太多**：鬼一出现就科普"因为三年前死过人"——不解释才是高级恐怖
6. **圣母主角**：番茄读者极度反感
7. **慢热铺垫**：前10章还在介绍世界观 → 必扑

### ✅ 读者正在追捧的差异化方向

1. **冷门职业主角**：殡葬师、文物修复师、凶宅试睡员、749局成员、守林人
2. **非传统规则怪谈**：迪化流融合、群像叙事、诡戏场/污染区设定
3. **中式恐怖 + 玄幻融合**：《誓不成仙》的九尾狐+规则怪谈+反套路修仙
4. **上交国家/民族元素**：加分项，也利于安全审核
5. **智斗博弈**：主角用智商碾压，读者代入感强

## 五、对本项目的战略建议

### 推荐方向（按优先级）

| 优先级 | 方向 | 理由 |
|--------|------|------|
| ⭐⭐⭐ | **规则怪谈 × 无限流 × 冷门职业** | 流量最大、黑马不断、签约通过率高 |
| ⭐⭐⭐ | **悬疑灵异 × 民俗风水 × 上交国家** | 复兴赛道、起点作者读者回流中 |
| ⭐⭐ | **惊悚游戏 × 749/收容物 × 脑洞** | 差异化强、短剧改编潜力大 |

### 书名公式（番茄风格）
```
《题材/背景：核心卖点/金手指》
例：《规则怪谈：从山坟开始》《诡异邻居来自夸夸群》
```

### 字数规划（50万字 / 5卷）
```
每卷约10万字，约45-50章（每章2100-2300字）
总计约225-240章
```
