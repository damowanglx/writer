# 第二卷「病区」目录

> 色调：灰暗 → 暗黄
> 支配感官：听觉（回声、呼吸、敲击）

| 011 | 回声 | 陆沉在负二病区昏倒被救，左前臂B超发现规则片状异物。老周笔记本中1940年照片被划掉脸的人——左前臂有同一位置的疤痕。照片背面写着"第七任：周德海。第八任：陆沉。" | 陆沉的名字是谁写上去的？谁是第一任？ |
| 012 | 活着 | 陆沉第二次下负二病区，通过洞口进入地下病区——七张铁床围成半圆，七个人形在被子下呼吸。闪光灯拍照后呼吸全停。铁门方向有人说"别拍"。那人左胸口袋上插着老周的笔——老周下去了。 | 一号柜把手上出现第九道刻痕。新刻。三小时内。柜子里有过活人。 |
| 013 | 名单 | 陆沉在档案室查了三天——1975至2025年，每年七个"体表标记异常"入柜，奇数月1-7号柜轮流。三十年周期缺10个人=儿童。陆沉找到退休老护士方秀兰。 | 方秀兰说"那不是什么病区。那是一份协议。每一代院长上任时都会签。" |
| 013B | 缺口 | 陆沉深入挖掘被跳过的儿童病例。发现陈副院长1975年后拒绝送孩子下去的原因——他六岁病逝的女儿。儿科病房夜间的呼吸声、监护仪的呜咽、玩具掉在地上的声音。 | "太年轻了。"陈远志在病历边缘的批注。一页被撕掉的死亡登记。 |
| 014 | 协议 | 方秀兰讲述协议起源——1973年太平间扩建挖出岩洞，韩工程师下去后申请调离。陈副院长的父亲陈国良签了第一份协议——每月送一个体温尚存者下去，让井保持稳定。1975年负二病区37个病人同时出现标记。 | "被推进去的人里面——有多少是被推进去的时候心还在跳的？" |
| 015 | 七号柜 | 赵勇在七号柜里装死二十八小时，被陆沉发现。赵勇说推他进太平间的是自己手术的主刀医生。赵勇逃离医院前留下便签"下一批：7月1日"。陆沉在镜中看到左臂皮下浮现一个字——"开"。 | 赵勇逃离。冷藏柜不仅是保存尸体的——是定位系统。 |
| 016 | 装死的人 | 陆沉去赵勇的修车铺，赵勇讲述冷藏柜里看到的"活死人"——比他更早被推进来的，肌肤冷但心还在跳，不是在等死，是在排队。赵勇说"他们不是等死——是在排队。"陆沉锁定陈副院长。 | "主刀医生——他不觉得自己在做坏事。那种人比知道自己做坏事的人更危险。" |
| 017 | 手术刀 | 陆沉调查陈副院长履历：三十七年手术生涯，手术死亡率比同行高0.3%。陈副院长值班日"体表标记"尸体出现概率是平时的三倍。陆沉去办公室对质——看到办公桌上1975年2月的照片，陈远志站在负二病区门口。陈远志没有否认。 | 陆沉约陈副院长对质。他来了。他承认了。但他说——"我在救人。" |
| 017B | 幸存 | 陆沉找到一名被跳过没送下去的幸存者——一个建筑工人，三年前出现体表标记但康复出院。他讲述了标记出现时听到的低频嗡鸣——像地底深处有一台巨大的机器在运转。他说"陈医生救了我。" | 幸存者手臂上消退的标记印痕——在特定光线下还能看到。他每晚仍能听到那个声音。 |
| 018 | 三十年前 | 陈远志的坦白——全书最重要的对话场景。1975年春节前，负二病区三十七个病人同时出现皮下印记，指向地下。他父亲陈国良自杀。孟小满——八岁的孩子——自己走进井底做了温度锚点。"下面要的不是尸体。是活着的人体上带着的温度。" | "第一个被送下去的人是谁？"陈远志没有回答。暗处传来童谣声——"叔叔——你下来了吗？" |
| 019 | 保温 | 陆沉做实验：调高温度→尸体没动；调低→动得更剧烈。发现冷藏柜是井的定位系统——温度骤降的柜子=被选上的信号。他换柜——把七号柜尸体换到八号，自己躺进七号柜。凌晨2:14，温度骤降，柜门被拉开——门外的人白大褂，口罩下面没有脸——光滑的皮肤。 | 他被推上推车，推进电梯井——门开后不是电梯，是向下的坡道。坡道尽头有光。 |
| 019B | 换柜前 | 陆沉换柜前的最后准备——身体上刻意降低体温、练习屏息；心理上反复梳理所有线索、录下遗言。他坐在黑暗的值班室里，听着太平间压缩机的声音，最后一次翻看老周的笔记本。 | "如果我回不来——至少有人知道发生了什么。"手机录音的最后一行。 |
| 020 | 下去 | 陆沉被推到圆形大厅——岩层中劈出的空间，石桌、人脂油灯、名册——从1937年到2025年，最新一笔"陆沉"。墙壁嵌满冷藏柜门，找到孟小满的柜门——把手被频繁打开磨得锃亮。蝴蝶发卡上写着"他在下面。别下来。带一件不属于这个时代的东西。他怕新的。"石棺在地下更深处。沈孝先——第一任院长——在井底等了七十年。 | 石棺盖板移动了。标记退潮。但井记住了他。一号柜把手上——出现了第十道刻痕。 |
