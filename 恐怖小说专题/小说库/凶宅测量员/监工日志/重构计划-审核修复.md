## 第一卷章节详细规格

### 第001章：试睡员（~3万字）
**开场方式**：陈渡已在湖畔别墅内，午夜将至。他正在用卷尺量客厅尺寸。温度骤降。镜子里出现了不该出现的东西。

**必须包含**：
- [ ] 前500字内出现规则：午夜后不能照镜子
- [ ] 陈渡的卷尺仪式（量门宽→写"正常"→这个动作的情感含义通过动作而非叙述传达）
- [ ] 至少一次温度/气味/触觉的感官恐怖描写
- [ ] 陈渡第一次在镜中看到林瑶——不是直白描述，是"镜子里多出来的角落"
- [ ] 章末卡点：陈渡发现镜面上有从内侧往外刮的字迹——"不要照镜子"——字迹是新的

**写作要求**：
- 文笔：冷峻克制，短句为主。不用"似乎""仿佛""可能"
- 对话：极简，不解释，陈渡话少
- 恐怖手法：日常物的异化（镜子照出不对的东西）
- 字数：25000-35000字

### 第002章：穿镜（~3万字）
**开场方式**：陈渡在地下室找到了IT男留下的12面镜子围成的正十二边形，圈中央是破镜框——刻着"林瑶 2003年6月14日"。

**必须包含**：
- [ ] 陈渡与周姐/老余的对话——获取林瑶失踪案的线索
- [ ] 老孙（镜匠）传授镜子修复原理
- [ ] 陈渡修复碎镜，校准银层反射方向
- [ ] 午夜穿镜的过程——高张力的动作描写
- [ ] 进入镜廊后的视觉奇观描写
- [ ] 章末卡点：石阶之下有东西在翻身

### 第003章：带回（~3万字）
**开场方式**：陈渡在镜廊深处，与林瑶面对面。石阶下的东西在呼吸。

**必须包含**：
- [ ] 陈渡走石阶——每级的厚度、磨损、温度变化
- [ ] 台阶底部：一张桌子、一本册子、一把空椅子
- [ ] 林瑶被救出镜子——不是被拉，是她自己走出来
- [ ] 爽点：陈渡用卷尺量了门宽——1.2米，之后把尺放在桌子上，不是留下，是证明"有人来过，量清楚了"
- [ ] 林瑶出镜后第一句话不是"我自由了"——保持原文"沙发换了"
- [ ] 章末/卷末卡点：周姐打电话，新活来了——纺织厂宿舍。陈渡说："带一个人。"

## 写作规范（所有写手必须遵守）

### 文笔规范
1. **禁用词**：似乎、仿佛、好像、或许、大概、应该、可能
2. **禁用句式**：她感到一阵……、他内心涌起……、一种说不清的……
3. **感官锚点**：每个恐怖场景至少包含2种感官描写（温度+触觉优先）
4. **句子长度**：动作场景≤25字/句；描写场景≤40字/句
5. **对话标签**：用动作替代"说"——"他把豆浆杯放下。'湖畔别墅。'"

### 恐怖手法规范
1. **不用jump scare**——不用突然出现+尖叫
2. **用日常物的异化**——镜子反出不对的东西、温度不正常的跌落、被子里多了什么
3. **用信息不对称**——读者和陈渡同时发现异常，没人提前解释
4. **用规则的不可违抗性**——违反规则的人会怎样？让后果可见但不过度渲染

### 爽点规范
1. 每卷至少1个"用卷尺巧妙破局"的高光时刻
2. 陈渡的聪明体现在：观察细节→发现规律→用规则反制规则
3. 不是"陈渡很厉害所以破解了"——是"陈渡注意到了别人没注意到的细节"

### 卡点规范
1. 每章结尾必须有一个未解问题或即将发生的危险
2. 卡点类型轮换：信息揭示型 / 危险逼近型 / 规则矛盾型 / 人物情感型
