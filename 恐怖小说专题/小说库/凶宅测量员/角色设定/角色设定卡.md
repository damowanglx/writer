# 《凶宅测量员》角色设定

## 主角：陈渡

- **年龄**：28岁
- **前职业**：建筑师
- **现职业**：凶宅试睡员（从业3年）
- **创伤**：两年前工地事故，搭档徐岩在他面前从脚手架坠落。他没能抓住徐岩的手。之后无法继续做建筑——每次看到脚手架手会抖。
- **遗物**：徐岩留给他的钢卷尺。尺子末端有徐岩用记号笔写的"量清楚"。
- **性格**：话少、擅长找规律、在恐惧中出奇冷静。有点冷幽默但不常说出口。
- **习惯**：每进一栋凶宅先拿着卷尺量房间尺寸——既是专业习惯，也是强迫性的仪式感——量完之后他会知道"自己站在哪里"。
- **成长线**：从"替地产商写报告"→"替死者完成遗愿"→"揭开规则背后的真相"

## 搭档：徐岩

- **状态**：已故（两年前工地事故）
- **与陈渡关系**：大学室友→建筑事务所搭档→挚友
- **影响**：卷尺是徐岩的，尺上的字也是。陈渡每次拉开卷尺，尺子回弹的声音是他给自己做的心理锚点。

## 关键人物

- **林瑶**：湖畔别墅的前房主女儿，16岁，死于三年前。她的残魂被困在别墅的镜子里。是陈渡接到的第一个"不是凶宅报告，是求救信号"。
- **周姐**：地产中介，给陈渡介绍凶宅业务的女人。35岁，精明但义气，相信陈渡是她见过的"唯一真的能试出东西"的试睡员。
- **老余**：退休刑警，现在在城郊开一家24小时便利店。偶尔帮陈渡查凶宅的历史档案。看陈渡的时候眼神总像在看一个案发现场。
- **规则之源**：不是人。是这座城市所有凶宅规则的源头——被遗忘在城建档案馆地下的东西。
