# 《凶宅测量员》——规则怪谈恐怖悬疑小说大纲

## 一、基本信息

| 项目 | 内容 |
|------|------|
| **书名** | 《凶宅测量员》 |
| **类型** | 规则怪谈 / 都市恐怖 / 悬疑解谜 |
| **总字数** | 约 45 万字 |
| **卷数** | 5 卷 |
| **目标读者** | 20-35岁，偏好规则解谜+都市恐怖 |

## 二、核心概念

**「规则」**：每一栋凶宅都有自己的规则。规则是死者留下的执念与建筑本身的历史凝结而成的约束力。违反规则的人会被宅子"留下"。

**「试睡员」**：主角陈渡，职业凶宅试睡员——被地产商雇佣，在挂牌出售的凶宅里住满规定天数，出具"安全报告"。他在这个灰色行业里干了三年，以为自己什么都知道。直到接了湖畔别墅的单子。

## 三、五卷结构

| 卷 | 凶宅 | 核心规则 | 隐藏真相 |
|----|------|----------|----------|
| 第一卷 | 湖畔别墅 | 午夜后不能照镜子 | 前房主女儿被囚禁在镜中 |
| 第二卷 | 纺织厂宿舍 | 每夜凌晨3点必须关灯 | 1987年火灾，47名女工被困 |
| 第三卷 | 地下防空洞 | 不能数台阶 | 战时避难所，有人在黑暗中永远迷路 |
| 第四卷 | 孤儿院旧址 | 不能回应孩子的呼唤 | 被遗弃的孩子们在等一个从未到来的领养人 |
| 第五卷 | 规则之源 | 所有规则在此交汇 | 陈渡发现自己与规则的起源有关 |

## 四、主角

陈渡，28岁，职业凶宅试睡员。前建筑师，因一次工地事故失去搭档后转行。冷静、话少、擅长在恐惧中找规律。他的搭档死后留给他一把卷尺——他每次进凶宅都会先量房间尺寸。
