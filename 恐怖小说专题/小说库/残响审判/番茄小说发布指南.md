## 一、作品简介（可直接复制使用）

### 版本一：标准简介（推荐，约280字）

> 前犯罪侧写师姜树收到一封匿名信，里面只有一张照片——他失踪三年的未婚妻林小雨，站在一座废弃精神病院的天台上。
>
> 他追踪坐标来到城郊，推开了地下室的门。
>
> 然后世界崩塌了。
>
> 他坠入了一个叫"裂隙"的地方——生与死之间的禁忌维度。每一个带着巨大执念死去的人，意识会在这里凝结成"回响"，反复重演生前最痛苦的记忆片段，形成一座座恐怖副本。
>
> 沉默教室里，43名学生的灵魂被永远定格在火灾前的最后一课。血月馆中，一栋会移动的法式别墅每隔七年就会吞噬一批新的调查者。被淹没的古城在镜子的另一面无限循环，47,382名居民的乡愁凝成一座永远走不出去的水下迷宫。
>
> 而所有线索都指向同一个真相——林小雨还活着。她在裂隙的最深处，用自己撑着一道门。
>
> 他说：别来。
> 她说：别来。
> 他们都在等对方先走。
> 但姜树从来不是听话的人。

## 二、番茄小说发布全流程

### 2.1 发布前准备

| 事项 | 具体要求 |
|------|---------|
| **书名** | 《残响审判》——世界观核心机制，有力量感。副标题可用《银哨》 |
| **笔名** | 一次性设定，不可修改。建议用简洁好记的2-4字中文名 |
| **实名认证** | 本人身份证+人脸核验，必须≥18周岁 |
| **封面** | 必须有。淘宝5-20元可搞定，建议暗黑色调+月光+一道门的视觉元素 |
| **题材分类** | 男频 → 悬疑灵异 / 或 男频 → 科幻无限流 |
| **标签** | 无限流、悬疑、恐怖、烧脑、爽文、解谜、心理恐怖 |

### 2.2 关键数据节点

```
2万字 ──→ 申请签约（机审，1天内出结果，全书仅3次机会）
8万字 ──→ 进入7天验证期（小流量测试，核心看追更率≥35%）
10万字 ──→ 正式首秀推流（约持续21天）
20万字 ──→ 可申请书名测试
```

### 2.3 更新策略

- **日更4000-6000字**。本书每章约7000-9000字，建议拆分为2-3个小节发布以适配手机阅读节奏
- **定时发布**：固定时间段（如每天中午12点/晚上8点），培养读者追更习惯
- **严禁断更**：断更是番茄最大的红线，断一天流量就可能腰斩。发布前至少存稿15-20万字
- **存稿建议**：全书已写完，建议每天只发1-2小节（3000-4000字），保持稳定连载节奏，给推荐算法充分时间

### 2.4 本书分段建议

每章原长约7000-9000字，在番茄上建议拆分为A/B两小节：

| 原章节 | 拆分方式 |
|--------|---------|
| 第001章 失踪的侧写师 | 001A 照片（~4000字） + 001B 地下室（~4000字） |
| 第004章 沉默教室·黄昏 | 004A 43个学生（~4000字） + 004B 黑板上的规则（~4000字） |

这样53章可以拆成约100小节，每天发1-2小节，能稳定连载2-3个月。

## 三、避坑指南

### 3.1 AI审核红线（最重要！）

> ⚠️ **2025年9月起番茄强制执行AI内容标识办法。纯AI生成作品一经发现，永久封禁身份证，不可再注册。**

这本书是你自己参与创作并修改润色的，但不代表没有风险。建议：
- 番茄系统会检测文本的"AI痕迹特征"（机械化句式、过度工整的结构等）
- 你已完成去AI味的修改，大幅降低了风险
- 发布前可以人工再过一遍，把残留的生硬句式改掉
- **不要声明使用了AI辅助**，否则会被打标签限流

### 3.2 内容红线

- 本书是恐怖悬疑，注意**不要过度描写血腥、肢解等具体画面**。心理恐怖、氛围恐怖是安全的
- 副本故事涉及师生关系、校园霸凌等敏感话题时注意尺度
- 不要涉及真实政治事件、真实历史人物
- 不要宣扬封建迷信——把"回响""裂隙"定位为科幻设定，而非灵异

### 3.3 书名/封面避坑

- 签约后20万字前不能修改书名，想清楚再定
- 封面不能有擦边球、血腥、恐怖元素（平台审核会拒）
- 书名不建议太像已有热门作品，避免被判定蹭流量

### 3.4 运营避坑

- ❌ 首秀期间不要爆更（会打乱推荐算法）
- ❌ 首秀期间不要大改正文、换封面、重写简介
- ❌ 不要刷数据（番茄反作弊极严，一抓就死）
- ❌ 不要多号登录同一个作者号
- ❌ 签约后不要一稿多投（独家授权5年）

## 四、番茄推荐机制速览

番茄是**纯算法推荐平台**，没有人工编辑推荐。流量分配的四个核心指标：

| 指标 | 说明 | 本书优势 |
|------|------|---------|
| **追更率** | 读到最新章的人数占比 | 每章末尾有爆点/悬念，天然追更友好 |
| **完读率** | 读超过一定比例的人数 | 剧情紧凑、层层反转，完读率会较高 |
| **互动率** | 评论、点赞、催更 | 悬疑解谜类天然引发讨论 |
| **留存率** | 新读者是否继续读下一章 | 开篇300字就进入悬念（照片+失踪），非常快 |

## 五、本书在番茄的定位分析

**优势：**
- 无限流题材在番茄2026年新增了标签支持，属于上升赛道
- 悬疑+爽文的组合在男频有稳定受众
- 50万字体量适中，不长不短，完读率友好
- 已完本，不存在断更风险

**潜在风险：**
- 恐怖类天生比爽文/甜宠流量小，预期收益要放平
- 成人向/烧脑风格与番茄主流的"快节奏爽文"调性有偏差
- 心理恐怖可能不如直接爽文来得"爽"，开篇需要足够吸引人

**应对策略：**
- 首秀前务必把第一卷前5章（开篇副本"沉默教室"）写得节奏极快
- 简介走"爽文风格"，强调爆点和悬念
- 开篇不要铺垫太多世界观，直接进入恐怖场景
