# 主角人设卡：姜树

## 基本信息
- **姓名**：姜树
- **年龄**：31岁（故事开始时）
- **职业/身份**：前省厅犯罪侧写师 → 裂隙聆听者 → 静默大厅管理员（终局）
- **一句话描述**：一个用三年时间等一个失踪的人，然后用一辈子去听她没有说完的话。

---

## 创伤内核
- **核心创伤**：未婚妻林小雨三年前失踪，生死不明。不是"死亡"——是"不知道"。这种不确定性比确定的死亡更折磨人。
- **行为体现**：
  - 左手无名指上有一道自己用手术刀片划的疤（林小雨失踪第三个月刻的，本意是"留个东西提醒自己别忘"）
  - 反复确认林小雨的笔迹——"北"字撇很短，"来"字最后两笔连成圆
  - 见到烟灰色风衣时有生理反应（林小雨失踪那天穿的衣服）
  - 从不摘脖子上的银哨子——十年如一日
- **触发崩溃的场景**：期待落空。以为找到了林小雨，实际不是。

---

## 性格矩阵
| 层面 | 内容 |
|------|------|
| **表面性格** | 安静、理性、话少、观察力极强 |
| **真实性格（压力下）** | 极度固执、情感深沉但不表达、在关键时刻会做出"不理性"的选择 |
| **性格缺陷** | 过度独立——"一个人去"是他的本能反应；不擅长接受帮助 |
| **性格优势** | 三年侧写训练出的冷静——"情绪是最不可靠的证据，但他可以让身体不受情绪干扰" |

---

## 特殊技能
| 维度 | 内容 |
|------|------|
| **技能** | 犯罪侧写（行为分析、现场重建、笔迹识别）、高共鸣能力（最终达到7级） |
| **技能来源** | 省厅重案组多年侧写训练 + 裂隙中的副本历练 |
| **技能局限** | 侧写对超自然现象无效；共鸣越深被反噬风险越大；过度依赖自己的判断 |

## 恐惧体系
| 层级 | 恐惧内容 |
|------|----------|
| **本能恐惧** | 无特殊本能恐惧——他的恐惧都是情感层面 |
| **情感恐惧** | 再次失去林小雨；辜负信任他的人；变成和反派一样的人 |
| **存在恐惧** | "如果她不是失踪而是主动离开，那她爱我吗？" ——这是他在第001章的核心恐惧 |

---

## 成长弧光
```
起点：一个执着于过去的寻找者——他的全部驱动力是"找到林小雨"
转折：进入裂隙后，从"寻找一个人"变成"理解一个世界"
终点：从寻找者变成守护者——不再只是找林小雨，而是和她一起守护裂隙
```

## 识别特征
| 类型 | 内容 |
|------|------|
| **身体标记** | 左手无名指上手术刀片划的疤（后被锚点银环盖住）；脖子上从不摘的银哨子 |
| **行为习惯** | 进任何建筑先数窗户（侧写职业病）；用手摩挲哨子上的摩斯电码（SOS刻痕） |
| **特殊物品** | 银哨子（林小雨送的生日礼物，后揭示为姜树七岁时在父亲火灾现场捡到的） |
| **口头禅/沉默** | 几乎不说"再见"；在最紧张时反而最安静 |

## 在故事中的功能
- **主线驱动**：寻找林小雨 → 发现裂隙真相 → 对抗白馆组织 → 成为裂隙守护者
- **特殊能力作用**：侧写能力帮助分析副本规则和死者执念；高共鸣让他能深入残响但风险更大
- **与反派关联**：父亲姜正阳与白馆的纠葛；他本人是白馆制造人工副本的目标（后门豁免权对人工副本无效）
