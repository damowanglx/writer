# 恐怖实体设定卡：五卷副本全档案

---

## 副本一：沉默教室（第一卷·前半）

| 维度 | 内容 |
|------|------|
| **类型** | 残响副本（规则型 + 心理恐怖混合） |
| **形成原因** | 小学教师张玉兰的执念——她的学生在她的课堂上被带走，她没能保护他们 |
| **核心机制** | 教室规则：不能说话（失声者主题）、黑板上的粉笔字自动书写、违反规则会被教室"吸收" |
| **恐惧触发层级** | 心理恐惧（教育的暴力）+ 社会恐惧（集体沉默） |
| **破解方式** | 理解张玉兰的执念——她不是在惩罚学生，是在试图用沉默"保护"他们 |
| **隐喻** | 集体沉默的暴力；教育系统对个体的碾压 |

## 副本二：血婚纱（第一卷·后半）

| 维度 | 内容 |
|------|------|
| **类型** | 活体副本（能主动吸引聆听者、利用弱点引诱） |
| **难度** | 中等 |
| **通关记录** | 3次成功 / 11次失败 |
| **核心机制** | 海边教堂中的血婚纱——复仇与爱的两种执念缠绕在一起 |
| **已知存活者** | 林小雨（共鸣五级）、顾衍（五级）、第三人 |
| **关键信息** | 林小雨在此副本中留下了某样东西——通关后她整个人变了 |
| **恐惧触发** | 爱情本身可以成为最深的恐怖（婚纱=婚姻承诺 vs 血的复仇） |

## 副本三：血月馆（第二卷）

| 维度 | 内容 |
|------|------|
| **类型** | 移动副本 + 反向共鸣副本 |
| **背景** | 1920年代法式三层别墅，沈家七口+十二名仆人葬身于1943年血月之夜大火 |
| **核心机制** | 七个夜晚规则：每晚解锁一个新房间、每晚一人被迫阅读沈家日记（共鸣触发）、每晚午夜一人变成馆中物品、第七夜前找到真相并熄灭核心执念 |
| **最大恐怖** | **反向共鸣**——副本也会共鸣你。它在你记忆中找到最脆弱的地方，将你的恐惧变成副本内部的物理实体 |
| **特殊限制** | 通关过的人不能再触发核心共鸣（副本不对通关者说话）；未解锁区域的房门绝对不能开 |
| **K-00石像** | 地下室中的石像，能展示"谎话层"——不是真实，是观看者最恐惧的版本。林小雨在被反噬前反向穿入石像核心层嵌入自我记忆 |
| **隐喻** | 秘密的代价；战争时期的叛国与家族秘密；看见真相的同时被真相改变 |

## 副本四：镜面都市（第三卷）

| 维度 | 内容 |
|------|------|
| **类型** | 大型空间异常副本 |
| **核心机制** | 镜像世界——每个进入者面对自己的倒影/另一面；47382份乡愁的数据化呈现 |
| **恐惧触发** | 存在恐惧——"我是谁？如果镜子里的那个人不是我，那我是什么？" |
| **关键节点** | 湖心亭（连接深渊之门的入口）、倒生柏（裂隙植物，根部渗透空间） |
| **破解路径** | 接受自己的另一面而非对抗 |
| **隐喻** | 身份认同危机；城市化中的乡愁丧失；无数个体的故事被淹没 |

## 副本五：父亲的回响（第四卷·前半）

| 维度 | 内容 |
|------|------|
| **类型** | 个人残响（与主角直接关联） |
| **核心** | 姜正阳（姜树之父）的残响——1999年在白馆自焚嵌入后门 |
| **关键信息** | 遗书包含：裂隙起源、第一残响本质、白馆历史、后门豁免权运作方式 |
| **时间线揭示** | 甦人文明12万年前建裂隙 → 7万年前第一残响诞生 → 1.2万年前初代织命者 → 1920年代日本人发现K-01 → 1990年代白馆成立 |
| **恐惧触发** | 存在恐惧——父亲的爱是一种后门程序，自己的命运在出生前就已被设定 |
| **破解** | 理解父亲的选择不是控制而是保护 |

## 副本六：晨曦副本（第五卷·终章）

| 维度 | 内容 |
|------|------|
| **类型** | **人工副本**（非自然形成——白馆制造） |
| **核心恐怖** | 这是人为制造的残响——白馆试图用科技复制裂隙的能力 |
| **关键危险** | 姜树的后门豁免权对自然裂隙有效但对人造副本无效——在这里他和普通聆听者一样 |
| **破解路径** | 顾衍从林小雨戒指烙痕中提取的反干扰频段编码 → 定位人工副本入口 → **抢先踏入而非被动等待** |
| **终局意义** | 从"被副本选择"到"主动选择副本"——主角完成了从猎物到猎人的转变 |
| **隐喻** | 科技滥用的恐怖；当人类试图扮演神时的后果 |

## 终极实体：第一残响 & 白色平原

| 维度 | 内容 |
|------|------|
| **身份** | 裂隙中第一个残响——七万年前诞生，是所有残响的源头 |
| **形态** | 塔的形式存在，塔尖有黑日（后被老赵的灯罩盖着）；与白色平原相连 |
| **与主角关系** | 姜正阳的后门与第一残响有关；白馆试图捕获它 |
| **终局状态** | 被修复——每天给镜面都市倒生柏幼苗浇水；"塔尖的黑日亮度和一只守夜桌上的阅读灯相当" |
| **象征** | 不是神也不是魔鬼——是最古老的悲伤。被倾听后，从威胁变成了守护者 |
