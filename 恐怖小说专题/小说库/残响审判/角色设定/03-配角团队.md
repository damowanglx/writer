# 配角团队设定卡

---

## 林小雨 — 失踪的未婚妻 / 高等级聆听者

| 维度 | 内容 |
|------|------|
| **年龄** | 29岁（失踪时26岁） |
| **外表** | 烟灰色风衣，短发，字迹"北"字撇很短"来"字最后两笔连成圆 |
| **身份** | 姜树的未婚妻 → 裂隙高等级聆听者（共鸣五级通关血婚纱、血月馆） |
| **性格** | 比姜树更主动、更有行动力。温柔但极度坚韧。会说"你这个人太安静了，偶尔吹一下，让我知道你在哪" |
| **功能** | 麦高芬——她是姜树进入裂隙的原因。但她不是被动等待救援的人——她一直在铺路 |
| **关键行为** | 在血月馆中用自己的影子被反噬为代价，将自我记忆嵌入石像核心层；在顾衍戒指中隐写反干扰频段编码 |
| **隐藏动机** | "让裂隙不再自动拉人"——把选择权还给人类 |
| **识别特征** | 银哨子（送出者）；五瓣花胸针（她的出口锚点）；刻在哨子上的摩斯电码SOS |

## 顾衍 — 向导 / 情报员 / 未说出口的爱

| 维度 | 内容 |
|------|------|
| **年龄** | 35岁左右 |
| **外表** | 戴眼镜，常年在静默大厅角落看书 |
| **身份** | 裂隙资深聆听者（共鸣六级）→ 姜树的首席向导和情报员 |
| **性格** | 沉默寡言，极度自律，用"一直在看书"来压制自己对血月馆的恐惧和对林小雨的感情 |
| **创伤** | 六年前进入血月馆，看到了自己最怕的东西（被封印在副本里，一旦说出口就会跟着他出来） |
| **隐藏动机** | 对林小雨有超越战友的感情——但选择了克制和成全 |
| **关键时刻** | 读完那本六年没读完的书（最后一页是林小雨的字）；交出戒指中的胶片（林小雨拍的姜树背影）；坦诚自己的感情并"把遗憾重新栓到它该栓的桩上" |
| **特殊物品** | 永远看不完的书（《一切离去之前所说的》）；银色戒指（含林小雨烙痕+反干扰频段编码） |
| **成长终点** | 从"用书逃避恐惧"→"读完最后一页"→共鸣从5级跳到6级 |

## 白鹿 — 医学生 / 追随者

| 维度 | 内容 |
|------|------|
| **年龄** | 24岁左右 |
| **外表** | 白发（后来齐耳），身形偏瘦 |
| **身份** | 医学生 → 裂隙聆听者 → 姜树的绑定追随者 |
| **性格** | 表面脆弱（怕黑、不敢一个人坐电梯），内核坚定——一旦决定就绝不回头 |
| **创伤** | 在现实世界缺乏安全感和被认可——"在沉默教室里，我第一次觉得自己做了点什么" |
| **绑定选择** | 自愿放弃独立成长的无限可能，用共鸣石子将上限锁在姜树身上——"能遇到这种人不跟着他，才是真的会死" |
| **功能角色** | 团队医疗担当；情感温度计（读者通过她的视角感受团队的温暖） |
| **成长终点** | 从"需要被保护"→ 用治愈共鸣替全队扛极限深度测试 → 编写新人急救教材 |
| **识别特征** | 翡翠绿共鸣石子；手腕上的绑带（另一端挂在姜树的银环上，没有压力只是挂着） |

## 九爷 — 静默大厅守门人

| 维度 | 内容 |
|------|------|
| **年龄** | 70多岁 |
| **外表** | 灰色对襟衫，低头写字，语气平淡像在说菜价 |
| **身份** | 静默大厅交易所管理者 |
| **性格** | 看透一切但不说透。提供信息和工具但不替人做选择。嘴上冷硬但行动温柔 |
| **关键台词** | "你想问的不是这个。你想问：她还活着吗？" |
| **功能** | 信息枢纽；安全感的象征（读者知道只要九爷还在柜台，静默大厅就还在） |
| **隐藏设定** | 他的登记簿记录着每一个聆听者的来去；终局在登记簿末页加了一行铜版箴言 |

## 林远 — 现实世界的守门人

| 维度 | 内容 |
|------|------|
| **年龄** | 32岁左右 |
| **身份** | 省厅刑警，姜树最好的朋友 |
| **性格** | 硬汉外表下的极度忠诚。不会说煽情话但会把配枪递给姜树 |
| **关键行为** | 拔出九二式配枪放在桌上给姜树——"拿着。回来还我。"；在子弹上刻姜树的案件代码；挂断电话后把子弹从备用弹匣放回正常弹匣 |
| **功能** | 现实世界的锚点——证明裂隙之外还有人在等姜树回来 |
| **与主线的关系** | 最后接到了九爷的电话——"告诉你们头儿，老姜的案子结了吧" |

## 苏姐 — 沉默的守护者

| 维度 | 内容 |
|------|------|
| **年龄** | 40岁出头 |
| **外表** | 工装外套沾着油渍和血渍但所有破损处被针线整整齐齐缝过；军绿色帆布包底磨出了线 |
| **身份** | 前军人/警察 → 裂隙聆听者 |
| **性格** | 极度自律、沉默、在恐惧中仍试图为他人省下资源 |
| **秘密行为** | 一个人照顾十二个在副本里受伤但没人管的聆听者——帆布包里装的不是衣服，是止血贴、抗生素和注射器 |
| **功能** | 团队的纪律核心和后勤保障；白鹿的实战导师 |
| **识别特征** | 从不离身的帆布包；一把军刀（姐姐留给她的） |

## 老赵 & 阿青 — 后勤组

| 角色 | 特征 | 功能 |
|------|------|------|
