# 第081章 失踪者的名单

四天。沈渡没有去忘名楼。没有去纺织厂。没有回那棵槐树底下。

他在做一件事，一个人。查名单。

他从何也手里拿到了过去半年所有异常体任务发布的记录。他把它们摊在老余面馆最大的那张桌子上，用胶带把打印纸一张一张拼起来，拼成了一张完整的任务地图。然后他拿了一支红色记号笔，在每一项任务旁边标注了四个数据：时间、等级、执行者、结果。

十二个消失的收容员，全部被他在任务地图上圈了出来。红圈像散布在城市各处的血迹。

他蹲在地图前，手里的记号笔没有盖上。他的视线在十二个红圈之间来回扫了大概五分钟，然后他在地上坐了下来。他看出来了。十二个红圈之间的连线，不是随机的，是一个很规则的圆形。圆形覆盖了城北到市中心的大约三分之一城区。圆心，在忘名楼的位置。

"他不仅在收容员身上做实验。他也在测试忘名楼的规则覆盖范围。每一个被改造的异常体被激活的位置，都在那个圆形的边界上。他在测量边界。"

苏遥蹲在他旁边，手里端着一杯水，没有喝。

"测试边界做什么？"

"他要确定忘名楼的规则可以被扩展到什么程度，然后用这些被改造的异常体作为'基站'，把忘名楼的记忆覆盖规则嫁接过去。一个人的记忆覆盖范围是一栋楼。如果你把十二个被改造的异常体作为接收器，覆盖范围可以扩展到整个城区。到时候，这个城市每天会有一半的人忘记一个名字。"

苏遥的手指在杯壁上轻轻敲了两下，她在消化这个信息。

"你能阻止吗？"

"能，如果我能找到他嫁接规则的蓝本。那个规则不是凭空产生的，他是从忘名楼604室的墙上复制下来的。墙上的名字排布方式，本身就是一个规则密码。他只需要复制那个排列模式，把它写进每个被改造的异常体里。"

沈渡站起来，腿蹲麻了，他扶着桌子边缘站了几秒。

"我需要再去一次604，不带裂隙，不带能力，用眼睛看那面墙。用最笨的方法，数。一个名字一个名字地数。"

苏遥看了他一眼。

"我跟你去。"

"你上次去忘名楼之后，有没有记住什么？"沈渡问。

苏遥沉默了一会儿。然后她说："我记住了那老太太的话。她说——'你找的那个人，可能已经不记得自己叫什么了。也可能不记得你了。'那句话，之后我一直忘不掉。"

沈渡看了她一眼。没有接话。他知道她在说什么，她在说她自己。

善终程序在衰退。她的生命锚定在减弱。她不记得自己什么时候开始容易累了，但老余有一次说漏了嘴：她最近吃面总是剩下半碗。

他没有戳破。他把地图从桌面上揭下来，叠好放进文件袋里。

"走。去一趟忘名楼。白天去，白天它的规则没那么活跃。"

孟迟从面馆后厨探出头来，他在帮老余剥蒜。手上全是蒜皮。

"我也去。我不进楼，我在楼下帮你们看着门。有人来了给你们打电话。"

沈渡看了看他。孟迟的眼镜片上粘了一小片蒜皮，他自己没注意到。沈渡没有提醒他。他点了点头。

三个人出门的时候，老余从厨房窗口探出半个身子：

"回来吃晚饭。今天煨了排骨汤。"

白天走进忘名楼和夜晚走进去是两种完全不同的体验。白天它看起来就是一座普通的老居民楼，光线从楼梯间的窗户照进来，灰尘在光柱里漂浮。楼道里甚至有邻居在洗菜的说话声，二楼有人开着门在听收音机，唱的是粤剧。

沈渡上楼的时候在二楼停下脚步。听收音机的是一个六十多岁的男人，坐在一个矮凳子上，面前摆着一张小方桌，桌上一碗白粥、一碟咸菜。他抬头看了一眼沈渡，眼神正常，甚至还点了一下头。

"找谁？"声音有点沙。

"604。"

老人没有追问，低头继续喝粥。但沈渡注意到他捏勺子的手停了一下，很短，不到一秒，然后继续舀粥。他没有回头。

604室的门还是和上次一样，没有锁。沈渡推门进去的时候，房间里的光线比外面暗了一截，窗帘拉着。墙上的名字在昏暗中呈现出一种微微发灰的颜色，像那些字本身就在褪色。

他走到墙面前蹲下。

他从最右下角第一个名字开始，用手指指着，一个一个地读，不出声，用嘴唇默念。

"张建国、李文秀、王桂芳、陈志强、刘小红、赵德柱、吴美兰……"

他数到第五列中间的时候，陆格的名字。他停了一下，记下了它前后的名字，然后继续往上读。第六列、第七列、第八列，墙上一共十四列。每一列大约二十到二十五个名字。总数大约三百个。

三百个被忘名楼"吃掉"的名字。

他的视线扫到第十一列的时候，他看到了一个不应该出现的名字。

"温敏"。

孟迟的母亲，第一个收割者，温伯远的表姐。她的名字被写在第十一列的中段，不是在末端，也不是在最前面。按遗忘顺序，她的位置是在忘名楼的规则里被排在"熟人"之后、"近亲"之前。楼把她归类为一个"被记得但不亲密的人"。

沈渡读完了整面墙。他没有站起来，而是蹲着往后退了半步，他的视线范围扩大到能看到整面墙的布局。

然后他看到了一个新的东西，上次来的时候没有注意到：墙的最上方，靠近天花板的位置，有一行极小的字，不是铅笔写的。是用某种尖的东西刻上去的，可能在刷石灰之前就刻了。石灰盖住了它，但年份久了，石灰层剥落了一小块，露出了下面的刻痕。

他站起来，踮起脚凑近看。

刻痕是三个字组成的符号，不是汉字。一个圆圈，中间三根线，驯兽师的标记。但和陆格、方砚秋用的版本不同：这个版本的圆圈不是封闭的，左下角有一个很小的缺口。

他盯着那个缺口看了几秒。然后他掏出手机，拍了一张照片，发给了何也。

何也的回复在几分钟后到达：你发给我的那个符号，我见过。在那个消失的收容员的手机里。老宋的手机，我们查过照片，他拍过一张窗外夜景。夜景里有一栋楼的楼顶，你放大看。和这个符号一样。

沈渡放大了何也发来的图片。老宋的手机照片，夜景里有一栋楼的楼顶，楼顶上有一个巨大的装置。不是广告牌，是一个钢结构支架，形状和他刚在天花板上看到的符号一模一样。

他盯着照片看了更久，发现那个支架所在的位置，正是温伯远住的楼层。他住在忘名楼附近的一栋楼的顶层。那个支架，不在任何建筑图纸上，是私自搭建的。

温伯远在忘名楼的天花板上留下了他的标记。而他自己的住处，能直接看到这个标记。他每天拉开窗帘，第一眼看到的就是这个符号。

"他在这个符号下面写规则。"沈渡的声音在空房间里显得很轻，"他在符号的'指引'下复制墙上的名字排列模式。每天看一遍，确保自己不会记错。"

沈渡把视线从天花板上收回来，落在墙根处，一个极小的铅笔印记，不是刻的，是铅笔画的，画了一只闭合的眼睛的形状。

和他左手腕内侧的烫伤疤，形状完全一样。

沈眠来过。她也看到了这个符号。而且她在符号下面留下了一个回答，一只闭着的眼睛。

他蹲下来，伸出手指，轻轻碰了一下那只铅笔画的闭眼。铅笔痕迹很淡，像画的人手上没什么力气了。但线条很坚定，没有犹豫。她画完就走了。

他甚至能想象她蹲在这面墙前面的样子。穿什么颜色的衣服，头发是扎着还是披着，他想象不出来。他已经三年没见过她了。

他站起来，拍了拍膝盖上的灰。走出了604室。

在门口等他的苏遥看他出来，说了一句："你的脸色比进去的时候差。"

"正常。"

"你还是不打算睡？"

"睡。回去就睡。"

苏遥看了他三秒。然后她伸出手，不是搭他的手腕，是把他连帽衫帽子上的一个线头摘掉了。动作很随意。然后她转身走在前面。

沈渡跟在她身后下楼。二楼那个听收音机的老人已经不在了，收音机关了，碗收了，矮凳子翻过来扣在墙边。沈渡注意到凳子的背面，用记号笔写着一串数字。

不是电话号码。是一组坐标。

他掏出手机查了一下这组坐标，地图上显示的是一个他从来没去过的地方。不在任何街道上。在城北郊区一个没有名字的废弃厂房附近的空地上。

他弯腰把凳子放好。然后下楼，走到门口，阳光照在脸上，刺得他眯了一下眼。

他回头看了一眼忘名楼的外墙。淡黄色瓷砖在午后的阳光下反射着黯淡的光，像一面很久没有被人认真看过的脸。

他记住了那个坐标。他打算今晚就去。
