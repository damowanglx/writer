# 第246章 孟迟的第一次招新

新规则覆盖的第十三天。大学校园，食堂门口，孟迟摆了一张折叠桌，铺了一块从宿舍床上扯下来的格子床单当桌布，上面放着一沓传单，他自己印的，在宿舍楼下打印店，黑白双面，一页五毛。

传单抬头：

"都市传说研究社，招新"

下面用小字写着——
"你听过凌晨四点的图书馆，书架之间有人翻书的声音吗？你注意过食堂的微波炉，设定的时间总是比别人拧的多三秒吗？你知道学校后门那条路，路灯编号从A-01到A-23，但A-13从来没有人见过吗？"

最后一行——"如果你注意过这些，欢迎加入。我们一起，听一下。"

孟迟坐在折叠桌后面，戴着他那副总是要滑下来的黑框眼镜，面前放着一包开了封的旺仔小馒头，他紧张的时候会吃，今天已经吃了半包了。

路过的学生大多数看了一眼传单，然后笑了一下，走过去了。有人停下来，拿起传单看了看，然后放回去。

"这社团是干嘛的？灵异研究？"

"不是灵异，是，研究那些，大家都注意到了，但没有人认真当回事的现象。"

"那不还是灵异吗？"

"灵异觉得它是'超自然'的，我觉得它是'还没有被归类'的。"

问话的学生，一个剃寸头的男生，看了孟迟一眼，然后从桌上拿了一张传单，折好放进口袋——"行，我回去看看。"

他没有留姓名，也没有扫二维码，但孟迟看到了他把传单放进口袋的动作，不是随手一塞，是折好，放进了内袋，和手机放在一起。

到了中午，他印的一百张传单，发出去了二十一张。二十一个人拿了传单，没有一个人现场扫码加入。但孟迟觉得，够了，因为第一天的目标，不是招到多少人，是让这张传单，出现在一些人的口袋里。

他开始收拾桌子，准备把剩下的传单带回宿舍，这时候，一个女生在他桌前停了下来。

她穿着一件旧棉服，袖子有点长，手里捧着一杯热奶茶。她没有拿传单，她看着孟迟，问了一句：

"你说的'听一下'，是听什么？"

孟迟停下收桌子的动作，抬头看着她。

"听，那些本来就有，但被我们忽略的声音。"

女生看了他几秒，然后把奶茶放在桌上，从书包里掏出一支笔，在桌上那张没有被拿走的传单背面，写了一个号码，放在桌上。

"如果你真的能听到什么，打这个电话告诉我。"

她拿起奶茶，走了。

孟迟低头看着传单背面的号码，数字写的很工整，没有署名，只有十一位数。

他把那张传单，没有折，小心地放进书包的夹层里，和母亲的照片放在一起，不是因为那个号码，而是因为，她问的方式，和所有人都不一样，她问的是"听什么"——不是"什么是都市传说"。

他收好桌子，把折叠桌扛在肩上，往宿舍走。阳光照在校园的主干道上，两排法国梧桐的叶子已经落光了，光秃秃的枝干在灰蓝色的天空上投下细密的线条。他走过一棵梧桐树下的时候，听到头顶的枝干之间，有风吹过的声音，像极远处的海浪，在城市的内陆，在校园的路边，在冬天的树枝间，他以前从来没有注意过这个声音。

他停下来，听了一会儿。

然后他继续往前走，风吹着他的头发，书包的拉链头在背上轻轻晃动着，发出细微的叮当声。

回到宿舍，他把那张写了号码的传单从书包里拿出来，放在桌面上，用一本书压住，不是怕被风吹走，是放在一个他一抬眼就能看到的位置。

他坐在桌前，打开笔记本，在当天记录的最后一行，写道：

"招新第一天。实发22张。收回1个电话。收获：'听什么'，这个问题比所有传单上的文字都重要。"

他合上笔记本，把那包吃了一半的旺仔小馒头封好，放进抽屉。

窗外，食堂的灯光在傍晚的光线中亮起来，学生们三三两两地走向食堂，孟迟坐在桌前，透过窗户看到那些走动的身影，他们中的某一个人，口袋里，也许放着一张折好的传单，还没有扔。

他关上台灯，站起来，拉上外套拉链，走向食堂。

在经过食堂门口的垃圾桶时，他低头看了一眼，桶里没有他印的传单。
