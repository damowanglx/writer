# 《收容失控》写作规范 v1.0 —— 五卷Agent统一执行标准

---

## 一、核心写作原则

### 1.1 每章结构（2100-2300字/章）
```
0-300字   → 钩子（承接上章悬念or新冲突引爆）
300-800字  → 展开（异常体规则感知/角色情感推进）
800-1500字 → 第一个小高潮（发现漏洞/角色决策/冲突升级）
1500-章尾  → 悬崖急刹车（新信息/未解悬念/反转预告）
```

### 1.2 番茄平台铁律
- **前三段必须有冲突或悬念**——平淡开场=读者流失
- **每500字至少1个小悬念或信息增量**
- **章尾断章率100%**——每章结尾必须有未解决问题的钩子
- **情绪曲线**：紧张→缓解→更紧张→破局→新危机

### 1.3 恐怖写作法则
- **不写"鬼出现了"，写"它来过但你看不到它现在在哪"**
- **不解释恐怖，只呈现异常**——解释=泄气
- **恐怖通过身体呈现**：不要说"他害怕"，写"他的胃在翻搅、手指掐进掌心"
- **氛围靠细节堆积**：温度变化、声音异常、空间矛盾
- **禁忌**：血腥直描、跳吓依赖、"头皮发麻""浑身发冷"等模板化表述

## 二、人物声口规范

### 沈渡——嘴贫防御/敏锐内核
- 日常：段子手语气，"给个五星好评哈"、冷笑话
- 压力下：嘴不贫了→眼睛开始扫→短句→语速加快→突然停下
- **对话特征**：从不说"再见"（对乘客说"慢走"，对队友说"你先过去"）
- **内部特征**：想妹妹时手会摸左手腕烫伤疤
- **禁忌**：不要写"沈渡心想"——他的心理活动用动作和对话间接呈现

### 苏遥——冷脸/心软
- 话少，每句切中要害
- 压力下：手指发抖→把手揣进口袋
- **对话特征**：从不聊"以后"，所有计划在一周内
- **口头禅**：不说"会没事的"——她最讨厌这句话
- **禁忌**：不要让她"眼眶红了""声音哽咽"——她的情绪波动是沉默+面无表情+手抖

### 老余——乐呵呵/稳
- 讲话像做面——随意但份量刚好
- **对话特征**："吃。吃完再说。""面要趁热。"
- 不主动教，问了才用三句话说完
- **禁忌**：不要让他长篇大论讲道理

### 何也——沉默但精准
- 说话极简，每句不超过两行
- 从不说"我相信你"——说"我核实过"
- **对话特征**：行动代替语言——沈渡回头时他已经站在旁边了
- **禁忌**：不要让他话多、不要让他笑（第五卷前）

### 孟迟——胆小但不停
- 说话会紧张但眼睛不躲
- **对话特征**："我可以边怕边做。" "那个异常体……它疼不疼？"
- 紧张时吃旺仔小馒头
- **禁忌**：不要写"孟迟勇敢了起来"——他从头到尾都在怕，只是学会了在怕中行动

## 三、异常体描写规范

### 规则呈现公式
```
异常体首次出现 → 描述日常外表（让它看起来完全正常）
→ 一个微小异常细节（"闹钟的3:17是反过来的"）
→ 规则的效果（不要描述规则本身，描述"违反规则的后果"）
→ 主角用规则裂隙看到的结构（玻璃裂纹意象）
→ 漏洞点揭示（逻辑断裂/矛盾节点）
→ 收容方案执行（智取，非暴力）
```

### 规则裂隙的视觉语言（统一标准）
- 规则完整的部分：透明玻璃
- 规则裂缝：从规则中心呈放射状蔓延
- 裂缝最密集处：漏洞点
- 人工干扰（如"驯养"痕迹）：像打结的绳子
- 被"加料"（收割者痕迹）：像在玻璃上长了肿瘤
- **沈渡使用裂隙的生理反应**：太阳穴冰锥感→眼压升高→视线边缘模糊→严重时视野全黑

## 四、卷间连贯性要求（关键）

### 4.1 情感线延续
- **沈渡对沈眠的执念**：每卷至少提及2-3次（摸伤疤/看手机壁纸/绕那条街）
- **沈渡×苏遥**：不要跳跃——对话中逐渐增加的信任（第一卷她递手→第二卷她量他脉博→第三卷他等她醒来→第四卷她靠在他肩上→第五卷没说出口）
- **老余的面**：每卷至少一次"吃面"场景

### 4.2 伏笔系统
| 伏笔 | 埋设卷 | 回收卷 |
|------|--------|--------|
| 驯兽师"你在喂它" | 第一卷 | 第三卷 |
| 沈渡烫伤疤的真正来源 | 第一卷（描述） | 第三卷 |
| 手机通话最后10秒的干扰 | 第一卷 | 第三卷 |
| 何也"第六个证物袋是空的" | 第二卷 | 第四卷 |
| 孟迟照片中的人影停止靠近 | 第二卷 | 第三卷 |
| 苏遥手凉 | 第二卷 | 第三卷 |
| 收容游戏=异常体 | 第二卷（暗示） | 第四卷（揭示） |

### 4.3 每卷开头和结尾
- **卷开头**：必须有"前卷回顾"——不是直接复述，而是通过角色行为/对话自然带出前卷事件对当前状态的影响
- **卷结尾**：必须有一个超级钩子——让读者必须翻到下一卷

## 五、章节质量标准

每章必须满足：
- [ ] 钩子在前300字内出现
- [ ] 至少1个规则裂隙使用的具体描写
- [ ] 至少1个角色之间的互动（对话/动作/眼神）
- [ ] 至少1个细节让读者"细思极恐"
- [ ] 章尾断章——未解决的悬念
- [ ] 没有AI写作痕迹（夸大的象征/宣传性语言/三段式法则/AI词汇/过多连接词）
- [ ] 没有模板化恐怖表述（"头皮发麻""浑身发冷""惊出一身冷汗"）

---

## 六、去AI味检查清单（每章自查）

❌ 被动词过多："被看到""被感知到""被发现"
✅ 改为主动："他看到了""他察觉到了""他发现了"

❌ AI词汇：总而言之、综上所述、值得注意的是、不可否认、诚然
✅ 删除或用口语替代

❌ 夸大的象征意义："仿佛整个世界都在这一刻崩塌了"
✅ 细化："他膝盖一软。手撑住了墙。"

❌ 情感告知："他感到一阵恐惧"
✅ 身体呈现："他的胃在翻搅。手指掐进掌心。"

❌ 三段式排比
✅ 不规则的、破碎的句子结构
