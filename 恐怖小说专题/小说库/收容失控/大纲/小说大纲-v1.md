## 三、世界观设定

### 3.1 核心概念：「异常体」与「收容游戏」

在日常现实的裂缝中，存在着一种被称为「异常体」的存在。

异常体不是鬼，不是怪物，而是**现实规则出了bug的产物**。它们可能是一盏会在特定时间照出不存在的人影的路灯，可能是一部能拨通已死之人的公共电话，可能是一个永远只倒出七分满的自动咖啡机。它们看起来和普通物品没有区别，但遵循着扭曲的、自成体系的规则。

当异常体的规则影响到足够多的人，就会触发「收容游戏」。

「收容游戏」是一个覆盖全球的隐秘系统。被选中的人成为「收容员」——获得识别异常体的能力，同时被赋予收容（或消灭）异常体的任务。完成任务获得积分，失败则被异常体"同化"。

没人知道收容游戏是谁创造的。它是保护人类的屏障，还是更大阴谋的伪装——这是贯穿五卷的核心谜题。

### 3.2 核心机制

| 要素 | 说明 |
|------|------|
| **异常体** | 现实规则的扭曲产物。外表为日常物品/建筑/人，内部有自成体系的规则。分为F级（轻微异常）到S级（城市级威胁）。 |
| **异常规则** | 每个异常体都有自己的一套行为规则。规则一定有逻辑漏洞——找到漏洞是收容的关键。 |
| **收容员** | 被游戏选中的玩家。获得「感知」异常体的能力，完成收容任务获取积分。等级从E到S。 |
| **收容任务** | 游戏发布的指令：定位异常体→理解其规则→找到漏洞→执行收容。失败次数过多会被降级——降到底级意味着被"抹除"。 |
| **规则裂隙** | 主角的金手指。能看到异常体规则的**逻辑漏洞**——不是看穿全部规则，而是感知到规则中的矛盾点和弱点。这一能力在收容员中极其罕见。 |
| **收容积分** | 完成任务的奖励。可在「交易所」兑换情报、装备、临时能力强化。积分不可交易，不可赠予。 |
| **安全屋** | 收容员在城市中的补给点。外表为普通商铺/仓库，入口只有收容员能看到。提供基本补给和情报交流。 |
| **交易所** | 更高层级的安全屋。积分兑换中心。由被称为「掌柜」的神秘人物运营——每个城市只有一家。 |

### 3.3 异常体等级体系

| 等级 | 威胁范围 | 示例 | 规则复杂程度 |
|------|----------|------|-------------|
| **F级** | 个人级，不致命 | 一个永远显示3:17的闹钟 | 1-2条规则 |
| **E级** | 个人级，可能致伤 | 一部拨打亡者号码的电话 | 2-3条规则 |
| **D级** | 场所级，致命风险 | 医院停尸间凌晨会多一具尸体 | 3-5条规则 |
| **C级** | 场所级，高致命 | 整层办公楼的镜子在午夜显示不同内容 | 5-8条规则 |
| **B级** | 区域级，可连锁感染 | 一栋居民楼的居民开始忘记自己是谁 | 8-12条规则 |
| **A级** | 城区级，灾难性 | 全城交通信号灯开始指引通往不存在的地方 | 12+条规则，规则可能嵌套 |
| **S级** | 城市级，存在性威胁 | 规则复杂到足以改写现实本身 | 规则数量未知，可能无上限 |

### 3.4 异常体的运行逻辑

每个异常体都是一套**扭曲的因果链**：

```
触发条件 → 异常现象 → 规则约束 → 漏洞点
```

示例：某便利店的自动门——  
- **触发条件**：午夜0:00-3:00间进出  
- **异常现象**：进入后走出来的人会丢失一段记忆（进门到出门之间的）  
- **规则约束**：如果在进门时说一句"我回来了"，记忆就不会丢失  
- **漏洞点**：门上方监控摄像头在异常时段拍到的是前一天同一时段的画面——说明异常体在时间上存在"盲区"

## 六、反派 / 对抗力量

### 「驯兽师」（The Tamer）—— 第一卷暗线反派

- 一个利用异常体牟利的收容员。他不收容异常体，而是"驯养"——找到它们的规则漏洞后反向控制，用异常体做自己的打手。
- 他在第一卷的最后被沈渡击败，但他遗留的一个"被驯养"的异常体，成为第二卷的核心事件触发器。

### 「收割者」（The Reaper）—— 第二卷 / 第三卷反派

- B级收容员，一个相信"异常体是进化的下一阶段"的极端主义者。他主动帮助异常体升级——用收容员的生命作为养分。
- 他的行为导致多个D/C级异常体连锁升阶，最终引发一个B级异常体的失控——第三卷的核心危机。

### 「管理局」（The Bureau）—— 第三卷 / 第四卷对抗力量

- 收容游戏内部的一个秘密组织。他们发现了游戏的真相的一部分——游戏本身就是一个S级异常体。他们不试图收容它，而是试图控制它。
- 管理局相信：如果人类能控制这个最大的异常体，就能控制所有异常体，甚至控制现实本身。
- 他们的方式：通过操控收容任务发布系统，定向培养收容员，最终夺取游戏的"核心权限"。
- 何也曾是管理局的一员——他离开的原因在第四卷揭示。

### 「初始异常」（Prime Anomaly）—— 第五卷终极存在

- 收容游戏本身。第一个异常体。所有其他异常体的"源头"。
- 不是传统意义上的"反派"。它没有恶意——它只是在运作。就像万有引力不会"恶意"让你摔倒。
- 它的存在让现实产生了裂缝——每一道裂缝就是一个异常体。收容游戏是它创造出来的"免疫系统"——用人类来收容裂缝。但这套系统正在失效。
- 核心谜题：如果游戏是S级异常体，它是被谁"收容"的？或者说——谁在收容员的身份背后，等待着游戏的最终结局？

### 第一卷：《夜车》（约10万字 / 45-50章）

**卷名含义**：深夜的网约车，是一切的开端。也是沈渡找到妹妹的唯一线索。

|--------|------|-----------|
| 第1-8章 | 开局事件：深夜接单，乘客消失。后座留着一部还在通话的手机，电话那头是沈渡自己的声音在说"别停车"。首次触发规则裂隙——看到那部手机表面浮现出的规则裂痕。 | E级异常体：**「同声电话」**——拨打接听者自己的过去。规则：通话持续7分钟，每过1分钟，通话内容向"未来"偏移1天。如果你在第7分钟还握着它，你会听到7天后的自己在说什么。 |

| 第31-40章 | 「驯兽师」初现。沈渡发现有人在暗中干扰收容任务——利用异常体攻击其他收容员。 | 遭遇被"驯养"的D级异常体——一只会在特定时刻咬断影子的"狗"。 |
| 第41-48章 | 第一卷高潮：沈渡第一次直面「驯兽师」。在被对方设下的陷阱中，他利用规则裂隙看穿了"被驯养"的异常体之间共用的脆弱规则——连锁反噬。击败驯兽师。但驯兽师临死前说了一句话："你以为你在收容它们？你只是在喂它。" | 多异常体连锁收容。 |
| 第49-50章 | 卷尾：沈渡在整理驯兽师的遗物时发现一张照片——照片上有三个女孩，其中之一是他的妹妹沈眠。照片背面写着一个地址。而那个地址，他认识：就是他每晚开网约车一定会绕过去的那条街。 | 本卷悬念钩子。 |

**本卷爆点分布**：至少8个高能爆点，均匀分布于关键章节。每5-6章至少一次情节重大推进。

**本卷情感线**：沈渡对妹妹的执念建立。与苏遥的初识互不信任到初步合作。老余的一碗面——"吃面的时候不用想规则"。

### 第二卷：《暗流》（约10万字 / 45-50章）

**卷名含义**：异常体在进化。收容游戏表面之下，暗流涌动。

#### 核心副本与收容事件

| 章节段 | 事件 | 异常体类型 |

**本卷爆点分布**：至少7个高能爆点。每卷核心异常体的"人类面"揭示是最大爆点——忘名楼里最后一位老人还记得妻子的名字。

### 第三卷：《溯源》（约10万字 / 45-50章）

**卷名含义**：追查游戏的本源。异常体的起源、收容游戏的真相，以及沈眠的去向。

### 第四卷：《崩塌》（约10万字 / 45-50章）

**卷名含义**：收容游戏大规模失效。异常体涌入现实。所有人在崩塌中做出选择。

| 章节段 | 事件 | 异常体类型 |
|--------|------|-----------|
| 第1-12章 | 游戏系统开始出现"错误"——收容任务发布乱序、积分系统波动、安全屋的位置开始移动。沈渡意识到管理局的干预正在加速游戏的不稳定。与此同时，城市中异常体出现频率暴增——从一周一次到一天数次。 | 多个C/B级异常体同时出现。城市进入"异常体潮"。 |
| 第13-22章 | 老余的面馆被卷入一个失控的B级异常体。沈渡赶到时面馆已经被规则扭曲——店内的时间流速是店外的12倍。老余在店里待了三天（外面只有六小时），用自己的热量感知稳住了异常体的扩散。但他太老了。沈渡破开规则救出老余时，老余说："没事。面给你留着。加热一下就行。"他的体温降到危险水平。 | B级异常体：**「回忆灶台」**——能让人尝到"已经不存在的人做的饭"。但每一次品尝都会让那个人在你的记忆中淡去一分。老余用规则漏洞反制——他不停地想起战友们。越想起，味道越浓。他撑到了沈渡来。 |
| 第23-34章 | 管理局开始"大收割"——利用异常体潮的混乱，大规模收编或消灭不愿合作的收容员。何也反出管理局，带出一份关键情报：管理局已经从游戏系统中锁定了"核心"的位置——在城市地下，一段废弃的防空洞网络中。而核心的结构——是沈眠三年前进去后用自己"锁住"的。她在用自己的存在延缓游戏的全面崩溃。 | 全城级危机。沈渡的阵营开始集结：苏遥（生命在衰退）、何也（被管理局追杀）、孟迟（能力被管理局盯上）、老余（重伤恢复中）。 |

| 第45-50章 | 卷尾：沈渡在核心中找到沈眠——她不是实体，是"信息态"。她把自己的存在编写进了异常体的核心规则中。她能"活着"，是因为游戏认为"如果删除这个信息，规则将不完整"。沈渡和沈眠只有一次对话的机会。沈眠告诉他："哥，我走不了了。但你可以做一件事——让游戏不再需要我。三种选择：① 摧毁核心——所有异常体消失，但所有被异常体'维持'过的人（包括苏遥）也会消失。② 让管理局接手——人类控制异常体，代价是用不断的人命去维持。③ 成为新的核心——用规则裂隙重写整个系统，但你不再是你。'哥哥'这个词不会再有任何意义。你选哪一个？" | 第五卷终极选择的前置。 |

### 第五卷：《重写》（约10万字 / 45-50章）

**卷名含义**：沈渡的选择将重写一切规则。

#### 核心副本与收容事件

| 章节段 | 事件 | 收容/对抗内容 |
|--------|------|-------------|
| 第1-8章 | 沈渡没有立刻做出选择。他带着沈眠留下的信息回到地面。三种选择的后果开始在他身边显现：苏遥的身体在加速崩溃；何也被管理局追杀到绝境；孟迟发现自己能"听到"核心异常体的频率——那是沈眠的，而那个频率正在减弱。老余的面馆重新开张。他说："不管怎么选，先吃碗面。" | 选择前的缓冲。角色们面对终局的态度。 |
| 第9-18章 | 管理局发动总攻——夺取地下核心的最后尝试。沈渡的阵营+残余的自由收容员+被老余召集来的"老人脉"联合作战。苏遥用自己的最后力量维持战线伤员的生命。何也正面迎接管理局的核心战力。孟迟用频率感知干扰管理局的通讯。 | 正面对抗管理局。每个角色都在用自己最强的能力。 |
| 第19-28章 | 沈渡重新进入核心。这次他不是一个人——孟迟跟着他。"你别一个人选。你选错了，我会骂你一辈子。"在核心深处，孟迟听到了所有人都没听到的东西——初始异常体不是一个意识，是无数被遗忘的人在现实规则中留下的"伤痕"的集合体。每一个异常体，都是某个被现实碾碎的人最后的存在痕迹。收容不是在消除威胁——是在倾听一个无声的"曾经存在过"。 | 孟迟的能力揭示最终意义。他的"频率感知"是唯一能"理解"而非"对抗"异常体的能力。 |

| 第39-46章 | 第五卷高潮：执行"共存规则"的覆盖。沈渡进入核心深处——他必须"写"下新的核心规则。代价是规则裂隙将永远关闭——他再也看不到任何异常体的漏洞。但他会把'看到'的方法写进新规则里——让每一个收容员都能学会用"理解"而不是"对抗"来面对异常体。苏遥在生命最后一刻选择被"新规则"覆盖——她的生命锚定能力被写入了核心，成为新游戏的底层机制：在收容失败时自动触发一次"生命锚定"，给收容员最后一次机会。她的最后一句台词是："沈渡，你欠我一条命。替我看看规则重写以后的世界。" | 全员代价。苏遥牺牲。老余在面馆里感觉到温度变化——新规则上线了，面馆里突然暖和了一度。他知道沈渡成功了。他把煮好的面又加了一勺酱油——"这小子回来肯定饿。" |
| 第47-50章 | 终章：三个月后。城市恢复了平静——但平静不同于以前。异常体仍然存在，但它们不再是无解的恐怖。每个异常体旁边都贴着"收容指南"：不是消灭方法，是理解方法。老余的面馆成了最热闹的安全屋——收容员们出任务前都来吃一碗面。何也成了新游戏的规则仲裁者——用他的轨迹回溯重现异常体的"故事"，帮助新人理解。孟迟在大学里开了一个社团——表面是"都市传说研究社"，实际是收容员预备训练营。沈渡不再开网约车。他在老余的面馆旁边开了一间小店——没有招牌。店里只有一面墙，墙上贴满了照片。每一张照片对应一个被"理解"过的异常体——和曾经被它们困住的人。照片里，沈眠站在纺织厂的楼梯上，穿着白T恤和牛仔裤。她笑了一下。照片不会动。但每天早上，沈渡来开门的时候，照片里的位置会变——有时候在楼梯拐角，有时候在窗前。有时候她旁边的空位上，多了一杯还冒着热气的豆浆。 | 开放式温情结局。留下余味——异常体还在，规则还在，但"理解"取代了"恐惧"。沈眠以异常体的方式"存在"——不是被困住，是被看见。 |

## 九、情感线规划

### 主线：沈渡 × 沈眠（兄妹情）
- 全书的驱动核心。不是爱情，是血缘。
- 三年前他没能保护她，三年后她用自己保护了他。
- 最终他们不能"在一起"——但他们以各自的方式"存在"于新世界中。

### 辅线：沈渡 × 苏遥（爱情）
- 不是甜宠。是两个在恐惧中互相托付的成年人。
- 从"不信任"→"搭档"→"可以把后背给对方"→"没有说出口但都知道"
- 她牺牲了。他没有哭。他只是每天在照片墙上多看苏遥那张空椅子一眼。

### 师徒线：沈渡 × 老余
- 一碗面的传承。老余教会沈渡的不是技巧——是"别被规则变成异常体"。
- 老余没有死。他的面馆还在开。新规则上线那天，他多加了酱油。

### 镜像线：沈渡 × 何也
- 两个因为失去而进入游戏的人。一个选择不再信任，一个选择赌在别人身上。
- 从对手到战友——但他们从来没有正经说过一句"我们是朋友"。

### 良心线：沈渡 × 孟迟
- 孟迟是沈渡在成为"工具"之前最后的镜子。
- "那个异常体疼不疼"——这句话改变了最终的结局。

## 十一、差异化亮点（vs 市场同类作品）

1. **收容物不是鬼怪，是"规则的bug"**：避免了传统灵异恐怖的同质化，恐怖来自逻辑被颠覆而非视觉冲击。
2. **网约车司机视角**：城市各个角落、夜晚的各色乘客——天然的"撞异常体"载体，接地气。
3. **规则漏洞感知而非看穿全部**：金手指有限制、有代价、有升级空间，避免主角过强导致爽感疲劳。
4. **终极反转：游戏=最大异常体**：不在最后一卷才揭示，而是从第二卷开始埋线、第三卷揭示、第四五卷展开——让读者参与"解谜"而不是被动接受。
5. **情感底色：不是拯救世界，是理解被遗忘者**：每个异常体背后是一个无声的"曾经存在过"。这与《凶宅测量员》"叫出每一个名字"共享精神内核，但路径完全不同。
