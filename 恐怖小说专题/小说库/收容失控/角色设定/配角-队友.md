# 配角/队友人设卡：《收容失控：我能看穿一切规则》

---

## 配角一：苏遥 —— 女主 / 搭档 / 治愈系

### 基本信息

| 属性 | 详情 |
|------|------|
| **姓名** | 苏遥（Su Yao） |
| **年龄** | 25 岁 |
| **职业** | 前市中心医院急诊科护士，现C级收容员 |
| **外表** | 身高165cm。纤瘦但站姿很稳——急诊科站出来的。黑发永远扎成低马尾，碎发用最便宜的发卡别住。眼型偏长，不笑的时候显得冷淡，笑了像换了一个人。左手食指有一道旧刀疤——拿手术刀练习时割的。 |
| **一句话描述** | 她见过的最恐怖的画面不是异常体——是病人从她手里一点一点变透明。她没能拉住。从此以后，她拼命拉住每一个快被规则吞噬的人。包括她自己。 |

### 功能定位

| 维度 | 详情 |
|------|------|
| **在故事中的功能角色** | 医疗/后勤：用生命锚定稳住队友生理状态，争取收容时间。情绪支柱：沈渡信任重建的第一块砖。 |
| **在团队中的位置** | 沈渡的搭档+第二指挥。在沈渡过于激进时是刹车——"我不管你看到什么裂缝，你的心跳已经135了。" |
| **不可替代性** | 没有她的生命锚定，团队在第三卷面对的高强度异常体潮中早就团灭。她的医疗背景也让她成为唯一能判断"这个人还能不能救"的人。 |

### 动机体系

**表面动机**：
"被卷进来了，能怎么办？活下去吧。"

**隐藏动机**：
她知道自己已经死了。善终程序维持着她，但那是一个倒计时——程序终有一天会判定她"该走了"，然后关闭。她现在做的每一件事都是在倒计时里做的。所以她不敢停。她也不敢让任何人靠太近——因为"不能让人在乎一个快死的人"。

**与主角的情感连接**：
苏遥和沈渡的情感不是一见钟情——是两个习惯了一个人扛的人在慢慢察觉到"对方也在扛"的过程。

第一个连接点：第一卷医院停尸间事件中，苏遥第一次看到沈渡认真起来。他收起所有嘴贫，盯着停尸间的门，说了一句："你先别进去。让我看看规则。"不是因为她弱——因为他知道她刚才多拼。他在用他的方式保护她。

第二个连接点：第三卷苏遥告诉他真相——"我一年前就死了。"她以为他会松手。大多数人都松手了——"我救不了你"。沈渡沉默了很久，然后说："那你欠我一年。得还。"他用嘴贫替她挡了一下——让她不必在那个瞬间独自承受。

第三个连接点：第五卷她选择牺牲。不是因为爱他——是因为她终于不怕被人在乎了。三年来第一次。

### 性格速写

| 维度 | 详情 |
|------|------|
| **最突出性格特征** | 表面冷脸实际心软。话少但每句都切中要害。",
| **压力下的反应** | 面无表情但手指开始发抖——她无法控制手，所以她会在压力上来时把手揣进白大褂口袋里（一个在急诊科养成的习惯）。",
| **不可触碰的底线** | 不要说"会没事的"——三年急诊科让她知道这句话是最无用的安慰。",
| **最让读者喜欢的细节** | 她会在队友睡着时量他们的心率——手指搭在手腕上，默数三十秒。不是职业病，是想确认他们还活着。 |

### 能力：生命锚定

**描述**：通过触摸感知并稳定一个人的生命体征。在异常体规则导致的身体异变中——比如尸僵蔓延、体温骤降、意识模糊——她能暂时"锚定"住人的正常生理状态。不是治愈，是维持。像在洪水中拉住一只快被冲走的手。

**局限**：
- 只能维持，不能修复——伤了就是伤了，她只是拖延时间
- 每次锚定都在消耗她自己的"善终程序"的余额——她每救一个人，自己就离死亡近一步
- 锚定上限：最多同时维持三个人，超过就会因自己的生理状态崩溃而失效

### 成长弧光

| 阶段 | 状态 | 关键事件 |
|------|------|----------|
| **第一卷** | 封闭：不信任任何人，靠"我需要活下去"支撑。对沈渡第一印象很差。 | 停尸间事件中沈渡用规则裂隙找到漏洞——她第一次看到有人和她一样拼命。 |
| **第二卷** | 松动：开始承认"这个团队不是临时组队"。主动帮老余量血压——被老余笑着推开了。 | 忘名楼事件中，她锚定住一个老人，老人的最后一句是"谢谢"。她当晚没说话。 |
| **第三卷** | 撕裂：告知沈渡真相——自己已死。最大的恐惧不是死——是告诉别人"我快死了"之后，别人眼里的怜悯。 | 善终程序开始失效。沈渡没有松手。她的第一反应不是感动——是生气。"你别这样——你越这样，我越不敢走。" |
| **第四卷** | 燃烧：她知道时间不多了，用最后的余额疯狂救人。老余的面馆事件中她连续锚定五人——事后昏了六个小时。 | 在地下核心，她第一次对沈渡说"我欠你的一年可能还不完了。" |
| **第五卷** | 牺牲：主动走入新规则核心——让生命锚定能力被写入新游戏的底层机制。她的最后一句台词："沈渡，你欠我一条命。替我看看规则重写以后的世界。" | 她的牺牲不是被动的——是选择的。她终于在倒计时结束前，用自己想要的方式结束了它。 |

### 退场设计

| 维度 | 详情 |
|------|------|
| **退场时机** | 第五卷第39-46章（新规则写入阶段） |
| **退场方式** | 不是被杀死——是"选择融入新规则"。新规则需要锚定机制作为底层保险（收容失败时自动触发一次生命锚定，给收容员最后一次机会）。苏遥主动把自己作为"模板"写入规则。 |
| **对主角的影响** | 沈渡失去"信任重建的第一块砖"——但他已经不需要那块砖了。她已经教会了他。 |
| **对读者的冲击** | 不是突然死亡——是缓慢的、有预兆的、她自己选择的。读者从第三卷开始就知道她会死。但真正发生的时候，心碎感不会减少——因为她最后说"替我看看"，不是"不要忘了我"。 |

### 识别特征

| 类型 | 具体特征 |
|------|----------|
| **身体标记** | 左手食指的旧刀疤。生命锚定发动时手指会微微发光——不是特效，是温度。触碰点升温约1度。 |
| **行为习惯** | 搭人手腕量脉博。对所有人——队友、路人、甚至被抓的反派。不是关心，是职业病。 |
| **口头禅** | 从不聊"以后"。她的所有计划都在一周之内。沈渡说"下个月"时她会沉默。 |
| **特殊物品** | 急诊科离职时带走的一个听诊器——不是她的，是一个老医生退休时留给她的。那个老医生是善终程序延长的第一个病人。 |

## 配角二：老余 —— 引路人 / 师父 / 人间温度

### 基本信息

| 属性 | 详情 |
|------|------|
| **姓名** | 余兴旺（没人叫全名——都叫老余） |
| **年龄** | 58 岁 |
| **职业** | 前消防员，现"余记面馆"老板，B级收容员 |
| **外表** | 矮壮身材，常年穿着白色厨师围裙，围裙口袋里有三样东西：温度计、便签纸、一小瓶辣椒油。光头，头顶有道疤——火场留下的，不是异常体。眼神温和但偶尔会突然警觉——拿温度计的手比任何仪器都准。 |
| **一句话描述** | 他煮了一辈子面。一碗热面，让人记住自己还是人。 |

### 功能定位

| 维度 | 详情 |
|------|------|
| **在故事中的功能角色** | 引路人（教导沈渡收容员生存守则）+ 安全屋提供者 + 情绪支柱（父亲替代） |
| **在团队中的位置** | 后勤中枢。面馆是安全屋+情报站+心理辅导室。老余不下副本——但他做的面是下副本前最后一顿和回来后第一顿。 |
| **不可替代性** | 他能用温度变化定位异常体——0.3度的温差在城市噪音中几乎不可能被捕捉，但他可以。因为他做了十五年消防员——他对"热"的理解不是数据，是本能。 |

### 动机体系

**表面动机**：
"面馆开着，人来人往。不差多一双筷子。"

**隐藏动机**：
十五年前火场里，他眼看着三个战友因为异常体规则逐渐忘记彼此——然后互相攻击。他无能为力。他开面馆，是因为一碗热面是最简单、最确定的东西——"让人记住自己还是人"不是比喻。温度是人最本能的记忆。热的东西在手里，你就知道自己还活着。

**与主角的情感连接**：
沈渡第一次进面馆是在第一卷——刚从第一个异常体事件逃出来，左眼视力还没完全恢复，头痛得像有人拿冰锥钻他太阳穴。

老余什么都没问。他端了一碗面过来，说："吃。吃完再说。"

沈渡吃了一口——是西红柿鸡蛋面。沈眠最爱吃的。

他问老余为什么做这个。老余说："冰箱里有啥做啥。你小子运气好——今天刚好有西红柿。"

其实冰箱里永远有西红柿和鸡蛋。老余的冰箱和沈渡来不来没关系——但沈渡不知道。

后来发现沈渡每次执行完任务，老余都会"刚好"做西红柿鸡蛋面。不是刻意的——他也不知道为什么。可能是因为沈渡第一口吃的是这个。可能是因为他看到了沈渡手机壁纸上的沈眠。可能是因为他做了十五年消防员——知道幸存者需要什么。

### 性格速写

| 维度 | 详情 |
|------|------|
| **最突出性格特征** | 乐呵呵的。但笑不代表放松——他是那种在火场里还能给你讲笑话的人。不是轻松，是稳定军心。 |
| **压力下的反应** | 摸围裙口袋——找温度计。温度计在手，他就稳住了。 |
| **不可触碰的底线** | 不要在面馆里说"死了就死了吧"——任何人。他会放下勺子，看着你，不说一句话，直到你改口。 |
| **最让读者喜欢的细节** | 他的面不放味精——放温度。面汤的温度永远正好——夏天比体温低五度，冬天比体温高八度。孟迟问过他怎么量的。他说"手。做了这么多年面，手比温度计准。" |

### 能力：热量感知

**描述**：能通过温度变化感知异常体的位置和状态。异常体扭曲规则时会在局部产生微弱的温度异常——通常下降0.3-0.5度。老余能在一定范围内（约半径50米）精准定位这种温差。

**局限**：
- 只能感知，不能攻击或防御
- 城市环境噪音（空调外机、汽车尾气、人群）会干扰感知
- 年龄增长后感知范围缓慢缩小——现在比十年前缩了约20%
- 他在面馆里的感知最强——因为面馆是他的"领域"，他对每一块砖的温度都烂熟于心

### 成长弧光

| 阶段 | 状态 | 关键事件 |
|------|------|----------|
| **第一卷** | 观察者：不主动教，只在沈渡来吃面时顺带说两句。"规则不是用来记的——是用来琢磨的。" | 沈渡第一次来面馆。西红柿鸡蛋面。他说"吃。吃完再说。" |
| **第二卷** | 指引者：开始主动指点——因为沈渡不是来躲的，是来问的。"你知道一个消防员最重要的经验是什么？别逞英雄。活着回来的人才有资格总结经验。" | 沈渡问他十五年前火场的事。他只说了一句："那天很冷。" |
| **第三卷** | 后盾：沈渡得知真相后精神接近崩溃。老余在他坐了一小时后收了坨掉的面，下了碗新的，多加了蛋。"吃点东西再走。" | 没有大道理。只有一碗面。 |
| **第四卷** | 燃烧：面馆被失控异常体卷入。他在店里用自己的热量感知稳住了异常体的扩散——一个人在三天（面馆内部时间流速）里守住了防线。被救出来时体温降到危险水平。躺着说的第一句话："面给你留着。加热一下就行。" | 沈渡破门而入时，面馆里的温度是34度——比正常低了3度。老余用身体当了热源。 |
| **第五卷** | 守望：伤好后回到面馆。新规则上线那天，他感觉到面馆里的温度突然暖了一度——他知道沈渡成功了。他往煮好的面里多加了一勺酱油，"这小子回来肯定饿。" | 结局：面馆成了最热闹的安全屋。收容员出任务前都来吃一碗面——面汤的温度永远是正好。 |

### 退场/结局

老余是唯一一个**没有死的导师角色**。

原因：老余的功能不是"牺牲以刺激主角"——是"证明人是可以被温度拉回来的"。他的存活本身就是第五卷"共存规则"的核心论据：人类不需要消灭和牺牲，人类需要温度和面。

### 识别特征

| 类型 | 具体特征 |
|------|----------|
| **身体标记** | 光头和头上的火场旧疤。做面的时候疤会微微出汗。 |
| **行为习惯** | 每次沈渡来，都会"刚好"有西红柿和鸡蛋。苏遥来的时候有凉面（"夏天吃凉的，你身体本来就不好"）。何也来的时候是最辣的（"你脸太冷了，辣一辣"）。孟迟来的时候是甜的（"小孩子吃甜的"）。他从来不会端错。 |
| **口头禅** | "吃。吃完再说。" / "面要趁热。" |
| **特殊物品** | 温度计——不是医用的，是老的煤油温度计。用了二十年。核心里有气泡——但他知道气泡在哪，他会扣掉。 |

## 配角三：何也 —— 对手 / 灰色角色 / 终极战友

### 基本信息

| 属性 | 详情 |
|------|------|
| **姓名** | 何也（He Ye） |
| **年龄** | 29 岁 |
| **职业** | 前市局刑侦警察，现B级收容员 |
| **外表** | 身高182cm，身板笔直——警察站姿没丢。常年黑色外套，口袋永远鼓鼓囊囊——不是武器，是证物袋。他从警队带走了一批空的证物袋。脸上表情很少，不是冷——是累了。 |
| **一句话描述** | 他最后一个案子：一家五口在一间反锁的房间里消失。饭还在桌上冒着热气。他看到了他们"走"进了一扇不存在的门——从此以后，他在每一扇门后面找答案。 |

### 功能定位

| 维度 | 详情 |
|------|------|
| **在故事中的功能角色** | 灰色角色（亦敌亦友）→ 终极战友 + 新规则仲裁者 |
| **在团队中的位置** | 追踪/侦察。轨迹回溯能力让他成为最好的追踪者——异常体逃不掉的，何也能追。 |
| **不可替代性** | 团队需要有一个"不信任任何人"的人——不是因为冷漠，是因为他见过的背叛够多。他是团队的免疫系统。 |

### 动机体系

**表面动机**：
"收容异常体。拿到积分。继续下一个。"

**隐藏动机**：
他想要一个解释。不是"异常体从哪来的"——是"为什么那一家五口"。他处理过几十个命案现场——每一个都有逻辑。情杀、财杀、意外——背后都有"为什么"。但那一家五口没有——他们只是打开了一扇不存在的门，走了进去，再也没回来。

没有为什么。

这个"没有为什么"比任何犯罪现场都让他恐惧。他在收容游戏里待了两年——不是为了收容异常体，是为了找到那扇门。他想知道那扇门通向哪。和有没有办法把他们带回来。

**与主角的情感连接**：
何也第一次看到沈渡的时候，看到的是一个"还在找"的人。大多数人被拉进收容游戏后就放弃了现实世界的牵挂——因为回不去了。沈渡没有放弃。他每天晚上还在那条街上绕。

何也没有说，但他羡慕这种"还在找"。因为他自己已经不太会"找"了——他只会"追"。追踪是往前的。找是往后的。他往不了后。

第三卷揭示：何也的搭档——那个叫林柏的警察——在何也被停职后独自调查那扇门。他找到了。他也进去了。何也至今不知道林柏是死是活。这成了他的第二个"没有为什么"。

沈渡在第四卷地下核心门前说了一句："你那个搭档——他可能还在里面。"何也沉默了很久。然后说："那就开门。"

### 性格速写

| 维度 | 详情 |
|------|------|
| **最突出性格特征** | 沉默到近乎社交障碍。不信任任何人——尤其信任那些"看起来可信"的人。 |
| **压力下的反应** | 不说话。完全不说话。但他的手指会动——在空气中划轨迹。他在回溯。不是用能力，是本能。 |
| **不可触碰的底线** | 不要碰他的证物袋。那些是空的——但他装满了。每一个袋子里装着一个他没法"结案"的名字。 |
| **最让读者喜欢的细节** | 他从来不说"我们是朋友"。但沈渡每一次独自冲进危险的地方时，回头看——何也已经站在他旁边了。不是跟上来的，是早就站在那的——他比沈渡还先到。 |

### 能力：轨迹回溯

**描述**：通过触摸物体，能看到该物体过去24小时内被移动的"轨迹残影"。轨迹以半透明的"残影线"形式出现在他视野中——不同时间的残影有不同颜色（近→鲜艳，远→暗淡）。对追踪异常体极为有效——异常体的"规则扭曲"会在轨迹中产生明显的断裂点。

**局限**：
- 只能回溯24小时
- 只能看到被移动的轨迹——不能看到物体"经历过什么"
- 异常体的规则扭曲会干扰轨迹信号——接近S级时完全看不清
- 对没有物理痕迹的事物（纯信息态异常体）无能为力

### 成长弧光

| 阶段 | 状态 | 关键事件 |
|------|------|----------|
| **第一卷** | 未登场 | |
| **第二卷** | 对手：在一个商场异常体事件中与沈渡争夺同一个收容目标。何也先到——但沈渡在最后关头用规则裂隙找到他忽略的漏洞。两人各凭本事，最终被迫合作。 | 镜面写字楼事件后，何也站在楼外看了十分钟。"你刚才看到的漏洞——是规则本身的？"沈渡点头。何也懂了——有些东西自己看不到。 |
| **第三卷** | 揭露：告诉沈渡自己的过去——包括管理局。带沈渡去见管理局的联络人。然后说："我只是带你过来。剩下的你自己判断。"没有要求信任——但给了信息。 | 沈渡问："你为什么离开管理局？"何也："他们让我在一队人和一个异常体之间选。他们选了异常体。我选了人。" |

### 退场/结局

何也没有死。他的存活是"不信任的人学会信任"的证明。他在结局中成为新游戏系统的规则仲裁者——用他追踪的能力，帮人"看见"异常体的过去。他终于从一个"追的人"变成了一个"往回看的人"。

## 配角四：孟迟 —— 新人 / 良心 / 关键钥匙

### 基本信息

| 属性 | 详情 |
|------|------|
| **姓名** | 孟迟（Meng Chi） |
| **年龄** | 19 岁 |
| **职业** | 大一学生（计算机系），E级收容员 |
| **外表** | 身高175cm。瘦，戴一副永远要滑下来的黑框眼镜。T恤上偶尔有动漫印花。书包里除了课本还有一包旺仔小馒头——"紧张的时候吃甜的"。看谁都是先紧张地笑一下。但笑完了眼睛不会躲。 |
| **一句话描述** | 他问的第一个关于异常体的问题是"它疼不疼"——在所有人在算积分和生存率的游戏里，他是最后一个问这种问题的人。而这个问题，在终局里改变了所有人的结局。 |

### 功能定位

| 维度 | 详情 |
|------|------|
| **在故事中的功能角色** | 新人/弱者（需要被保护）→ 关键钥匙（他的共情能力在终局破局）+ 良心（沈渡在成为"工具"之前最后的镜子） |
| **在团队中的位置** | 最弱战力但最重要感知。他的频率感知在早期被团队视为"最没用"——直到第三卷开始揭示异常体的规则源头都是情感创伤。 |
| **不可替代性** | 他是唯一能"听到"异常体情绪的人。其他所有人的能力都是"对抗"——他是唯一的"理解"。这个看似无用的能力，在第五卷成为破局的关键——因为初始异常需要的不是被消灭，是被理解。 |

### 动机体系

**表面动机**：
"我妈妈的照片在变……我想知道那是什么。我觉得那不是坏事。我想保护它。"

**隐藏动机**：
孟迟的母亲三年前死于一场他没有赶上的告别。他母亲在ICU的最后一天，他在学校里考试——没有人告诉他情况恶化了。他知道这不是任何人的错，但他一直在想：如果那天我在，妈妈走的时候会不会少疼一点。

所以他听到异常体的情绪频率后，第一个反应不是恐惧——是熟悉。那个频率听起来像妈妈在他小时候发烧时把手放在他额头上的温度——不是热的，是想让他安心。

他不想消灭异常体。他想知道它们为什么疼。

**与主角的情感连接**：
孟迟是沈渡最初不想管的"闲事"——一个大学生拿着一张家里的老照片跑来安全屋说照片里的人影越来越近了。沈渡说"我没时间"。但孟迟没有走。他坐在面馆外面的台阶上等。等了三个小时。沈渡出门时看到他还在——手里拿着一包吃完了的旺仔小馒头。

"你回去吧。"
"照片里的人影——是妈妈。我不想让她一个人。"

沈渡看着他。看了很久。然后说："你进来。面要坨了。"

不是因为心软。是因为他在孟迟眼睛里看到了自己——对着沈眠的照片坐了三个小时的自己。

### 性格速写

| 维度 | 详情 |
|------|------|
| **最突出性格特征** | 胆小但不退缩。怕很多东西——怕黑、怕高、怕突然响起的电话——但怕的时候他还在往前走。"我可以边怕边做。" |
| **压力下的反应** | 会吃旺仔小馒头。频率感知在压力下会自动增强——异常体的情绪频率像噪音一样涌入。但他学会了在噪音里分辨出"哪一个在叫我"。 |
| **不可触碰的底线** | 不要说"异常体就是该死"——任何人对他说这句话，他的反应不是争论，是看着你。用那种"你只是没有听过它们的声音"的眼神。不咄咄逼人。但让你说不出第二遍。 |
| **最让读者喜欢的细节** | 他给每一个他感知过的异常体起了小名——不是编号，是名字。"那个在地下停车场哭的——叫小雨。因为它听起来像在下雨。"苏遥听到这个之后沉默了很久。然后说"以后别说了"——不是批评。是怕自己哭。 |

### 能力：频率感知

**描述**：能"听到"异常体发出的情绪频率——恐惧、愤怒、悲伤、孤独、甚至渴望。不同情绪有不同的"音色"：恐惧像金属刮擦，悲伤像低音提琴的延长音，孤独像风吹过空瓶子的声音。越接近异常体的"规则核心"，频率越清晰。

**局限**：
- 完全不能用来攻击或防御
- 频率过载：身边异常体超过三个时会产生严重眩晕和恶心
- 不能区分"真实的情绪"和"异常体想要传达的情绪"——异常体可能会用虚假频率误导他
- 在嘈杂的城市电磁环境中感知距离大幅缩短

### 为什么他是关键钥匙

前三卷：孟迟是"团队里最弱的那个"——被保护、被小看、被同情。

第三卷结尾反转：当所有高阶收容员都在试图用武力/规则/智力"对抗"异常体时，孟迟意识到——异常体不需要被对抗。它们需要被**听到**。

每一个异常体都是一个"被现实碾碎的人最后的存在痕迹"。它们的规则不是攻击机制——是一声无声的"我曾经存在过"。孟迟是唯一一个从一开始就"听到"了这些无声之音的人。

第四卷：在沈渡+何也+孟迟三种能力的叠加中，孟迟的频率感知补上了"规则裂隙"和"轨迹回溯"都看不到的维度——情感。初始异常的规则完整面貌中，有沈渡看到的"结构"（裂隙），有何也看到的"历史"（轨迹），有孟迟听到的"情绪"（频率）——恐惧、孤独、和不理解。不是恶意的。是一个从来没有被理解过的东西。

第五卷：沈渡创造"共存规则"的第四选项时，直接使用了孟迟的频率感知模型——"理解"才是终极的收容方式。不是因为善良，是因为有效。

### 成长弧光

| 阶段 | 状态 | 关键事件 |
|------|------|----------|
| **第二卷** | 新人：带着照片来到安全屋，被所有人当成"又一个被吓坏的新人"。但他在沈渡被忘名楼的规则干扰时，用频率感知找到了规则的情绪源——忘名楼里那个还记得妻子名字的老人。他听到的是"不悲伤的孤独"。很奇怪——那个老人不悲伤。因为他还记得。 |
| **第三卷** | 觉醒：他开始理解自己的能力不是"最没用"——是"最不同的"。所有能力都是对抗，他的是倾听。但这个理解让他更怕——因为听懂的越多，想消灭的人就越少。在一个所有人都在"收容"的体系里——收容的前提是"威胁"。如果他不认为它们是威胁——他还有资格留在这个游戏里吗？ |
| **第四卷** | 关键：三种能力的叠加——第一次听到S级异常体的"完整声音"。他用频率感知听到的初始异常：不是恶魔的声音，不是一个怪兽在咆哮。是风声——像一个人对着空房间说话，没有人回答。他说："它不是在攻击我们。它是在问——'有人吗'。" |
| **第五卷** | 传灯者：三个月后，他回到大学。没有离开收容游戏——但选了另一条路。他在大学里开了一个社团——表面是"都市传说研究社"，实际是收容员预备训练营。第一堂课他讲的是："首先，异常体不是恐怖片的怪物。它们是——你忘了关的一扇窗。有人在窗那边站了很久。你只需要走过去，把窗打开。不是关上。打开。" |

### 退场/结局

孟迟是唯一一个**从开头到结尾都没有受过致命伤**的角色。不是因为有保护——是因为他从来不正面作战。他在第五卷成为了新收容员培养体系的核心——教新人"先听，再看，最后才判断"。

他的结局是整本书的注脚：恐惧的回应不一定是战斗——也可以是蹲下来，轻声问："你怎么了。"
