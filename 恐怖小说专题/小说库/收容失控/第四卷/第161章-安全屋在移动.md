# 第161章 安全屋在移动

通知是交易所发的。

「安全屋坐标正在重新分配。当前安全屋将在24小时后失效。请所有收容员注意接收新坐标。」

沈渡吃完面就开车去了城南的交易所。

交易所的伪装是一家二十四小时便利店。货架上摆着过期的面包和落灰的饮料，冰柜的灯管坏了一根，嗡嗡响。收银台后面坐着一个戴鸭舌帽的中年人——没人知道他叫什么，老余叫他“掌柜”。

沈渡推开玻璃门，门上的铃铛响了一声。

掌柜抬头看了他一眼。然后低头继续看手机。

“交易所还在。”沈渡走到收银台前。

“还在。”掌柜没抬头，“但再过二十三小时就不在了。”

“你在收拾东西？”

掌柜抬起手，指了指便利店里的货架。“你看看这些东西——哪个是真的？”

沈渡扫了一圈。货架上的商品标签都是正常的——价格、生产日期、条形码。但他用规则裂隙看了一眼，那些条形码在自动变化。每个条形码下面的数字每隔三秒跳动一次，不是随机的——是按某种序列。

“这些货架——不是货架。”沈渡说。

“是坐标。”掌柜终于放下手机。“每一个商品对应一个安全屋的新位置。条形码的跳动序列就是安全屋的重排顺序。”

沈渡盯着货架看了几秒。日光灯闪了一下。

“多少个安全屋在重排？”

“全市六十二个安全屋，全部在重新分配。”掌柜说。他从柜台下面拿出一个塑料袋，里面装着几瓶水。“连交易所都要搬——我在这里坐了七年，今天第一次接到‘搬家通知’。”

“谁发的？”

掌柜没回答。他把塑料袋放在柜台上，看着沈渡。

“你们前几天的积分——是不是也不对？”

沈渡点头。掌柜从口袋里掏出一张纸——是手写的。上面密密麻麻记着数字。

“这是过去一周，我这边十二个固定客户的积分反馈。”他说，把纸转过来给沈渡看。“积分波动率从正常的正负百分之二——涨到了正负百分之四十。有的人多了，有的人少了。没人知道为什么。”

“系统在失控。”沈渡说。

“游戏在失控。”掌柜纠正他。“积分、任务推送、安全屋坐标——这三个是游戏最基础的子系统。如果这三个同时出问题——”

“不是在出问题。是已经出了。”沈渡打断他。

他拿起塑料袋，看了一眼里面的水——瓶装水，标签正常。水在瓶子里没有晃动。即使他把瓶子倾斜过来，水面也没有跟着倾斜。

“这是什么？”

“新安全屋的钥匙。”掌柜说。“找到对应的商品——开锁的水面会告诉你新坐标。你要是找不到——”

“找不到呢？”

掌柜没说话。他把鸭舌帽往下压了压。“找不到你就没有安全屋了。”

沈渡把瓶子放回塑料袋里，拉上拉链，拎起来。

“还有二十三小时？”

“二十二小时四十分钟。”

沈渡转身往门口走。走到一半他停下来。

“掌柜，你见过这种情况吗？”

“见过一次。”掌柜的声音从背后传来。“七年前。那次全城安全屋重排，是因为有一个A级异常体在扩散——它模仿安全屋的规则，让收容员走进错误的安全屋，然后吃掉他们。”

“那次怎么解决的？”

“没解决。”掌柜说。“那次死了十九个收容员。老余认识其中三个。”

沈渡握住门把手。铃铛又响了一声。

“这次不一样。”他说。

“怎么不一样？”

“上次是异常体在模仿安全屋。这次——是安全屋自己在跑。”

他推门出去了。

沈渡回到车上，把塑料袋放在副驾驶座上。他没有发动引擎——他坐在驾驶座上，看着挡风玻璃外面的街道。

街道是正常的。人、车、红绿灯，一切在运转。但在这条街的地下——在一段废弃的防空洞里，沈眠用自己的存在锁住了收容游戏的核心。

而地面上的系统，正在崩。

手机震了一下。不是任务推送——是孟迟。

「哥，你在哪？」

沈渡打字：「交易所。怎么了？」

孟迟回得很快：「我的安全屋不见了。」

「什么意思？」

「我本来在南山路的安全屋——就是那个三号安全屋，伪装成饮品店的。我刚才去买水，发现那个店拆了。门口的招牌都没有了。我站在那看了十分钟——没有人注意到那里曾经是一家店。」

沈渡盯着屏幕看了几秒。

「你站那别动。我过来接你。」

他发动了车。引擎的声音在早晨的街道上听起来正常，但沈渡注意到了——转速表比平时低了两百转。不是因为发动机冷——是因为引擎在“省力”。像有什么东西在替他分担一部分动力。

他没有深想。他挂了档，车子驶出停车位。

后视镜里，便利店的灯管灭了。

沈渡再看的时候，它又亮了。
