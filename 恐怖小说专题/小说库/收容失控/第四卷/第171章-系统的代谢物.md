# 第171章 系统的代谢物

掌柜把便利店的门锁了。挂上了“暂停营业”的牌子——开店七年第一次。

他坐在收银台后面，面前的桌上摆着一排钥匙、三本账本、和一个黑色的移动硬盘。鸭舌帽摘了放在一边——沈渡第一次看到他摘帽子。掌柜的脑袋是光的，头皮上有一道从头顶到耳后的长疤。

“交易所的核心数据在这里面。”掌柜拍了拍那个移动硬盘。“六十二个安全屋的新坐标、近三年的收容记录、异常体的频率统计、收容员的存活状态。你说要打一个电话的时候——我就知道该打包了。”

沈渡站在收银台前面。苏遥倚在门边，看着外面的街。

“你们在回忆灶台里发现了什么？”掌柜问。

“异常体的最后一个声音。”沈渡说。“交易所在收集它们。”

掌柜没有否认。“你认为交易所在做什么？”

“我以为交易所在帮助收容员。现在我不知道了。”

掌柜沉默了一会儿。他伸手拿过一个账本，翻到中间某一页，转过来给沈渡看。

那一页上写的不是数字——是一条规则描述。手写的，字迹非常小，排列得像代码：

「收容游戏的底层机制（部分推测）：

1. 异常体 = 现实裂缝 = 游戏的‘代谢产物’
2. 收容员 = 清理代谢产物的‘免疫细胞’
3. 收容 = 清除 → 裂缝闭合 → 游戏获得能量
4. 收容员在收容中消耗的生命力 = 游戏的‘能量收益’

结论：游戏不需要收容员活着。它只需要收容员在闭上裂缝的过程中‘燃烧’。燃烧完了，换下一个。”

沈渡看着那页纸看了很久。

“交易所的人——知道这个吗？”

“知道。”掌柜说。“我们从七年前就知道了。我是第一批发现这个机制的收容员之一。发现者是老张——我搭档。”

“老张——面馆的原房东？”

“对。他发现了真相之后——选了和老余一样的路。开了一家店。不问前程。”掌柜把账本合上。“我选了另一条路——留在交易所里。能帮一个收容员多活一天，就是一天。”

他站起来，把移动硬盘放进一个帆布袋里，拉上拉链。“但你妹妹做的事情——改变了这个底层机制。她进入核心之后，游戏的‘代谢’速度被压慢了。异常体数量在三年内趋于稳定——收容员的死亡率下降了百分之四十。”

“但她正在消失。”苏遥说。

“对。她消失得越多——游戏的代谢速度就越快。”掌柜说。“你们现在看到的异常体潮——是三年积压的代谢需求集中释放。”

沈渡的手按在收银台上。他的指节泛白。

“何也在地下三层——找到了什么？”

“他找到了‘录音室’。”掌柜说。“里面存着七年来每一个被收容异常体的‘最后一次频率’。包括你妹妹进入核心前——她留下的那一条。”

沈渡抬头看着掌柜。

“她的频率——是什么？”

掌柜没有立刻回答。他拉开收银台下面的抽屉，翻了翻，拿出一张叠好的纸。

纸上是一条声波图。波形简单——只有两个重复的波形段。像两个音节。

“我听不懂。但何也走之前说——这是频率版本的‘我没事’。”

沈渡接过那张纸。他看着声波图上那两个重复的波形。

我没事。

三年前她最后给他发的那条消息——也是三个字。

“何也去了录音室之后——还出来了没？”

掌柜沉默了一会儿。“他从录音室出来了。但他出来之后——去了另一个地方。那个地方我没有坐标，没有记录。他走之前给我打了一个电话。他说了八个字：‘如果沈渡来了，把这个给他。’”

掌柜从口袋里掏出一个东西——一个小密封袋。袋子里装着的是一枚银戒指。

沈渡认得这枚戒指。

“多余一具”——那个停尸间异常体的戒指。它不应该在这里——应该已经被永久收容在交易所的数据库里了。

“何也把它从灾备库调出来了。”掌柜说。“他说你会需要它。”

沈渡接过密封袋。戒指内侧刻着一个日期——银戒指的银色在水珠里发暗。

他突然明白了。

“戒指是密钥。何也留下的钥匙打开的是物理门——戒指打开的是规则门。”

掌柜没有说话。

沈渡把戒指套在自己的小指上——有点紧，但能戴上。

戒指接触皮肤的一瞬间——正常。没有异样。但沈渡知道——它正在“记录”他。它在等。

“还差什么？”

“还差一个声音。”掌柜说。“打开第三层核心的规则门——需要三样东西：何也的物理钥匙、这枚戒指、和老余的温度计。三样东西同时在场——规则门会认出‘这是被三个最高权限的收容员认证的进入请求’。”

沈渡摸了摸口袋。温度计还在。

“温度计——不是用来破解回忆灶台的？”

“也是。但它的主要功能——是老余留给你的遗产。”掌柜看着沈渡。“他在把一辈子的权限，一个一个往你手里放。”

沈渡没有回答。他把密封袋放回口袋，把戒指戴稳。

“掌柜，交易所搬了之后——你还能联系多少人？”

“六十二个安全屋的掌柜——我能联系到五十八个。”

“帮我发一个通知。”沈渡说。“不是以我的名字发。以何也的名义发。”

“内容？”

沈渡想了想。“‘第六个证物袋已打开。钥匙已取得。三天后——核心。想来的，自己来城北体育场停车场。入口在C区消防通道。’”

苏遥在门口回过头。“你用何也的名字——他们会来吗？”

“何也不用名字被信任。他的人被信任。”

掌柜看了沈渡一会儿。他拿起笔，在一张空白便签上写了几行字。然后把便签折好，递给沈渡。

“这是交易所的新坐标。只告诉你一个人。如果三天后你还活着——来找我。”

沈渡接过便签，没有打开，直接放进口袋。

他转身走向门口。苏遥推开了门——外面的天快黑了。

沈渡站在便利店门口，看着街道上逐渐亮起的路灯。

他的手机震了一下。不是来电——是交易所的系统通知。

「安全屋坐标重分配已完成。请所有收容员自行查询新坐标。」

沈渡没有看。

他又摸了一下小指上那枚银戒指。冰凉的金属。

“十二天了。”他说。“从积分异常到找到第三层入口——十二天。”

“你怕来不及？”苏遥问。

“我不怕来不及。”沈渡说。“我怕来得及的时候——我已经不是那个需要找到她的人了。”

他走向自己的车。引擎发动的声音在黄昏的街道上传出去很远。

后视镜里，便利店的灯熄灭了。

沈渡没有回头。

他挂了档，车子融入了晚高峰的车流。
