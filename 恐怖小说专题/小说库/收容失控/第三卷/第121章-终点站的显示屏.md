# 第121章 终点站的显示屏

电子显示屏上的绿色字体没有闪烁——它在呼吸。

沈渡盯着那行字看了大概十秒，才意识到"呼吸"这个感觉从哪来的。不是屏幕本身的亮度变化。是他的眼睛出了问题——或者说是站台本身出了问题。绿色字体在缓慢地、几乎不可察觉地一明一暗，频率和他的心跳不一样。比他慢。

慢多少，他说不上来。但他在心里计数的时候，感觉自己的心跳在努力去够那个节奏。

2023年6月17日。

他往前走了几步，鞋底在站台地面上发出的声响被周围的空间吸收了——不是回音消失了，是声音被什么东西吞掉了。他踩下去的时候，脚步声的前半截还在，后半截被截断了。

沈渡停下脚步，低头看了看自己的鞋。

地面是干净的瓷砖。白色的，标准的站台用砖。他蹲下来用手指摸了一下瓷砖表面——不是完全光滑的。表面有极细的纹理，网格状的，像是为了防滑刻上去的。

他用指甲在那纹理上刮了一下。

什么也没有。正常的瓷砖。

但他的手指能感觉到一种微妙的震动——不是来自地面，是来自站台深处。像是在地下的更深处，有什么机器在运转，频率太低了，耳朵听不到，但骨头能感觉到。

他站起来，沿着站台走了一圈。

站台大约七八十米长，两端的隧道口都被黑暗堵死了。没有列车。没有轨道灯。但轨道是完好无损的——金属轨面在手机灯光下泛着暗灰色的光泽，没有锈迹，没有灰尘，像是半小时前刚有人打磨过。他蹲下来摸了一下轨道侧面，指尖触感冰凉，光滑，没有任何风化的粗糙感。

一条随时可以通车的轨道。

沈渡站在站台边缘，看着那条轨道延伸到隧道里消失的方向。他突然想到一个问题：如果这是一条还在运营的轨道——那它连接的是哪一站？他又看了一遍轨道两侧的绝缘支架。型号是他在网上查过的地铁标准件，但支架上没有任何生产批号或者日期标识——只有光滑的灰色塑料表面，像刚从模具里取出来的。

尚安路。不在任何地图上的站。

那他面前这条轨道——通向哪里？

他从口袋里摸出那张收割者笔记的复印件，再次看了看沈眠的字。她的字迹在他手电筒的光线下显得更深了——因为纸背面的凸痕被光线从侧面照亮，那些笔画的压力变成了阴影。

2023年6月17日。

如果沈眠在失踪后的第三天来过这里——是她自己来的，还是被带到这里来的？

他翻开手机里沈眠的聊天记录。最后一条消息停在那晚：2023年6月13日，23:47。

"哥，明天我自己打车。你别接我。可能会有点乱。"

沈渡的手指在屏幕上停了片刻。他没有划走那条消息——他在看"乱"那个字。沈眠用"乱"而不是"危险"或者"麻烦"。"乱"是什么意思？

他退出聊天界面，打开相册，翻到沈眠的照片。一张她站在大学门口的侧面照——她手里拿着一杯奶茶，低头在看手机，头发别到耳后，露出半边脸。他记得这张照片是他偷拍的，那天她说奶茶店的麦芽糖波波好喝，他跟她说不健康的。她翻了个白眼，然后笑了。

照片拍摄日期：2023年6月10日。

失踪前三天。

他突然放大照片的边缘。

沈眠的左手里除了奶茶，还攥着什么东西——一个被她的手指遮住大半的纸片。只能看到一角。蓝色底，白色边框。

地铁票。

沈渡盯着那个模糊的角落。蓝色票面——是这个城市的地铁票。一种普通的、用了几十年的单程票。票面中间嵌着磁条，边缘有一个小缺口——是他帮她剪的，她说票角扎手。他记得。

她手里攥着一张地铁票。三天后她失踪了。

他抬起头，看着眼前的站台。

电子显示屏幕上的日期还在。绿色字体的呼吸频率似乎变快了——或者说，他的心跳变慢了。两种节奏在互相靠近。他能听到自己的心跳声，在安静的站台上像一个节拍器。咚、咚、咚。但在某个瞬间——咚和咚之间的间隔变长了。像是他的心脏也犹豫了一下。

沈渡把手机翻过来，打开相机，拍了一张电子屏的照片。

照片里什么也没有。

他低头看着手机屏幕——取景框里能看到电子屏和上面的日期。但按下快门后，照片里只有一张空白的电子屏，绿色的像素全消失了。像是站台把光收了回去。

他又拍了一张。

一样。

他把手机举到眼前，对准屏幕，直接拍——取景框里的字清清楚楚。拍下来的照片里只剩一块黑色的屏幕，像一面擦了粉的黑板。

沈渡慢慢放下手机。他盯着那面电子屏，三秒后绿色字体又恢复了。

它不想被记录下来。

他的太阳穴开始跳了——不是刺痛，是那种熟悉的、暴风雨来临前的闷胀感。规则裂隙要开了。但他没有去关闭它。他放任那股刺痛在颅骨内部扩散，让视线边缘开始发毛——那些透明的、像是玻璃内部生长的裂纹开始在周遭的空气里浮现。

他看向站台。

裂隙视觉中，他看到的不是裂纹。

站台的一切都是完整的。完整的、光滑的、完美的。

没有裂隙。

白瓷砖的连接处没有规则的裂缝，盲道的凸起没有规则的断层，电子显示屏的外壳没有规则的热胀冷缩缝隙——一切都在它该在的位置上，以最标准、最完美的方式存在着。

太完美了。

完美到没有一个漏洞。

沈渡的规则裂隙能看到一切规则中的瑕疵——此刻他什么都看不到。不是因为他看不见。是因为这个站台本身就是一个完整的规则。它不是在执行规则——它就是规则本身。

一个没有漏洞的规则。

这才是最大的不正常。

他能感觉到冷汗从后背慢慢渗出来的过程——不是突然冒出来的，是每一滴都清清楚楚地在他的皮肤上形成了、汇聚了、往下淌了。

尚安路站。

不存在的站。完美的站。沈眠来过的地方。

他站在站台中间，头顶的日光灯明亮而稳定。

但问题是——

他刚刚注意到一件事。

这个站台没有影子。
