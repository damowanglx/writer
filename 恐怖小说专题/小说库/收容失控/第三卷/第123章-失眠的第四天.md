# 第123章 失眠的第四天

沈渡连着失眠了四天。

第四天晚上，他坐在驾驶座上，车窗全开着，外面的风把遮阳板夹层里那张地铁票吹得微微抖动。副驾驶座上那团人形阴影在第三天凌晨消失了——不是慢慢淡掉的，是他在服务区买水回来就没了。但他不确定那团阴影是真的存在过，还是他缺觉缺出来的幻觉。

他已经四十八小时没有闭上过眼。

每次闭眼，他就能看到那个站台。白色瓷砖，绿色显示屏，没有影子的地面。和那个坐在副驾驶座上、没有形状的透明轮廓。

他掏出手机看了一眼时间。23:41。

他需要坐一次地铁。

这个念头不知从哪来的，但它已经在他脑子里驻扎了三天。从尚安路回来的那个晚上起，他每次经过地铁站入口，都能感觉到地下有什么东西在叫他。不是声音。是一种气压的变化——每次他走近地铁站，鼓膜就会微微内陷，像飞机降落时的感觉。

他没法不去。

沈渡把车停在最近的地铁站停车场——三号线。他下车的动作没有犹豫。锁车，刷卡进站，下扶梯。

最后一班地铁的末班车是23:50。他还有九分钟。

站台上稀稀落落站着几个人。一个背着电脑包的上班族，一个戴着耳机的女生，一个靠柱子打瞌睡的中年男人。灯光是那种标准的白色日光灯，地面有水渍，空气里有一股隧道风带来的铁锈味。

普通的深夜站台。

沈渡站在站台边缘，看着隧道深处。轨道上方的电缆在微微晃动。列车还没来。

他把手伸进口袋，摸到那张地铁票的边缘。他没有把票拿出来——只是用指腹压着票的边角，感受那张薄薄的塑料片在他的口袋里被体温焐热。

23:48。

列车进站了。风从隧道里涌出来，吹动站台地面上的一片废纸。纸片翻了几个跟头，停在他脚边。他低头看了一眼——是一张超市小票，打印的字已经模糊了，只有日期还看得清。2023年6月。

他没有捡。

列车门打开。车厢里有零星几个乘客，坐得很分散，各自占据一个座椅。灯光和站台一样，惨白的日光灯管，照亮了车厢里所有人脸上的疲惫。

沈渡上了车。他没有坐下——选了车门旁边靠墙的位置站着。左手握着扶手杆，右手依然插在口袋里，攥着那张票。

门关了。

列车启动。隧道里的灯一盏一盏往后掠，速度感不强，像在一条无限延伸的管子里滑行。

第一站。正常。有人下车，没人上车。

第二站。正常。一个穿校服的学生下了车。

第三站。那个打瞌睡的中年男人下了。

第四站。车厢里剩下三个人。沈渡，一个戴耳机听歌的女生，和一个坐在车厢另一端的老太太。他在后视镜一般的车窗玻璃上看到了他们的倒影——他自己的脸被车厢里的灯光照得发白，黑眼圈像是用炭笔画上去的。

第五站。站名他没听清——广播声音太小，混在列车行驶的噪音里变成了模糊的气音。但他看到站台了。普通的站台。没有人上车。

第六站。耳机女生下车了。

第七站。老太太也下车了。她下车前看了沈渡一眼——那一眼太平静了，平静到他还没反应过来，车门就关了。

车厢空了。

沈渡站在空荡荡的车厢里，列车继续向前。窗外的隧道灯开始变得稀疏——间隔比刚才长了。原本每隔四五米就有一盏的隧道照明灯，现在间隔扩大到了十几米，两盏灯之间有明显的黑暗区间。车轮碾压轨道接缝的声音也变了节奏，从规律的"咔嗒咔嗒"变成了不规律的、间隔越来越长的敲击，像是列车在进入一段结构不同的轨道。

沈渡的左手腕内侧开始发烫。

沈渡的左手腕内侧开始发烫。

他没有坐下。他靠着车门旁边的立柱，盯着窗外的黑暗。列车在加速——他能感觉到。不是速度表上数字的那种加速，是他脚底的震动频率在变。轮轨撞击的声音越来越密，像雨点打在铁皮上，从疏变密。

他数了数经过的灯之间的间隔。

从第三盏到第四盏，隔了大概七秒。

以地铁的速度，七秒的黑暗区间——这段隧道比正常的要长。

或者比正常的要宽。

他突然意识到一个问题。从他上车到现在，他没有听到过广播报站。从一个站到另一个站之间，列车广播应该会提醒下一站。但他没听到。一次也没有。

他抬头看车厢上方的电子线路图。

那幅图上的灯——在闪烁。

不是所有灯都在闪。只有一盏。在一条没有被标注的线路上，一个他不认识的站名在慢速闪烁。那站名的字不是任何一种他见过的地铁站字体——笔画太细了，间距也不对，像是有人在系统里手动打上去的。

那个站名的字体慢慢稳定下来。

尚安路。

闪烁。停止。再闪。

列车开始减速。车窗外的隧道灯完全消失了——取而代之的是一种均匀的、浅色的光，从隧道前方漫射进来。像是隧道出口就在前方。

列车滑入站台。

车门打开了。

沈渡站在没有人的车厢里，看着门外的站台。白色的瓷砖。日光灯。干净的空气。和那个电子显示屏。

绿色的日期。不是2023年6月17日了。

2023年6月14日。

他的脚自己动了。不是大脑在指挥——是他的身体先于他的意识做出了反应。他的一条腿迈出了一步，踩在车厢地板和站台之间的缝隙上。那一瞬间，他的瞳孔骤然缩小。

尚安路站的站台没有阴影。

但这个站台有。

日光灯下方，每个人的影子都在地面上，清晰的、深色的、正常的。但他回头看了一眼车厢——车厢里没有影子。灯在头上，但地板是均匀的白色，没有任何物体在它上面留下投影。

包括他自己。

他的影子被站台这边的光截住了——从半条腿的位置开始，下半截拖在站台地面上，上半截在车厢里消失了。像一个人被从中间切成了两段。一段属于这个站台，一段属于他来时的世界。

沈渡站在车厢和站台的边界上。一半身子在外面，一半在车厢里。他的影子在站台这边，在车厢那边消失不见。他的右脚在站台上，左脚在车厢里。

站台比他刚才站的位置冷了三度。

他感觉到了。
