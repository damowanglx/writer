# 配角团队设定卡：《祠堂》

> 《祠堂》的配角系统围绕"守"这个核心动词展开——每个人用不同的方式在守。

---

## 一、沈父（第五代守祠人）——沉默的传授者

| 属性 | 内容 |
|------|------|
| **姓名** | 沈父（全名未在文中出现——"守祠人不需要名字"） |
| **年龄** | 约55岁（沈渡32岁时） |
| **身份** | 第五代守祠人 |
| **性格** | 沉默寡言、行动代替语言、把最重要的话留在最后 |
| **核心行为** | 走进祠堂门后，以自身为代价封堵菌毯，通过石碑共振向儿子传讯 |
| **标志性物品** | 上海手表（停于凌晨4:03——沈渡出生时刻）、樟木盒（传给沈渡的铜钥匙+铜牌） |
| **标志性动作** | 把烟雾吐进门缝——用烟和门后的爷爷说话 |
| **与沈渡的关系** | 十四年前问沈渡"你闻到什么"——这个问题沈渡用了十四年才答上来 |
| **功能定位** | 引路人+代价承担者——让沈渡看到"守祠人"的真正含义 |

### 关键台词
- "你闻到什么？"（十六岁时问沈渡）
- "记住这个味道。以后变浓了——就回来。"
- "不要用火。它不怕光。它等你叫它名字。"
- "以后不再说话了——越说话它越能捕捉声带的震动频率。它在学——它在练。"

## 二、陆知遥（母亲）——远方的守望者

| 属性 | 内容 |
|------|------|
| **姓名** | 陆知遥 |
| **年龄** | 约50岁（沈渡32岁时） |
| **身份** | 镜面都市第47,383号居民，湖心亭守护者 |
| **性格** | 安静的、用行动表达的——"全村最安静的沈家媳妇" |
| **核心行为** | 每天泡两杯茶——一杯自己喝，一杯给死去的丈夫。倒生柏叶片把茶的温度变成光封回路里的一道弧线，传到祠堂门上 |
| **标志性物品** | 红毛线（未织完的毛衣）、竹针（沈渡小时候磨牙咬过）、倒生柏叶片 |
| **标志性动作** | 坐在藤椅上喝茶，对着窗外山的方向——"不是看——是听。你爸在祠堂里。" |
| **与沈渡的关系** | 缺席的母亲——不是去世，是"出差"了二十多年。在裂隙中以另一种方式守护 |
| **功能定位** | 情感锚点+光学理论基础——光封方案的理论原型来自她对倒生柏的观察 |

### 关键细节
- 用茶水测震——茶温过喉的震动频率=门缝红泥冒泡的频率，"茶比人的感官更早收到那个频率"
- 照片背面字迹："给第六代守祠人织毛衣。这团线没用完——留给以后需要红线的守祠人。"
- 沈渡带去单丛茶，倒生柏蜡膜对高香型茶反应更强——那天光弧更亮更持久了半分钟。"妈妈在给爸爸送茶。"

## 三、二叔——人间的锚

| 属性 | 内容 |
|------|------|
| **姓名** | 二叔（全名未出现） |
| **年龄** | 约60岁 |
| **身份** | 沈渡父亲的弟弟，退伍修路工，面馆老板 |
| **性格** | 粗糙、忠厚、不识字但有自己的一套逻辑——"两只手都得出力" |
| **核心行为** | 替沈渡攥铜牌、每天送粥（葱花三粒）、把烟搁石鼓上给门后的哥哥、把光封说明折成巴掌大塞围裙口袋 |
| **标志性物品** | 围裙（胸口口袋放烟+打火机+光封说明）、烟（不是瘾——是节奏） |
| **标志性动作** | 蹲在祠堂门口石鼓旁边点烟、端碗送粥、攥铜牌攥到手心出汗 |
| **与沈渡的关系** | 替代父亲——沈渡小时候罚站，二叔蹲在旁边陪抽烟。一根烟七分钟，"行了想通了没。想通了就进屋吃饭。" |
| **功能定位** | 现实锚点+人间温度——"守祠人他弟弟能做的事：松树、面馆、每天早上门口石桌上那杯粥" |

### 关键细节
- "粥给你放在门口石鼓上。葱花三粒。没放多。"
- 修铁路修出的烟瘾——"隧道里唯一能让人扛住黑暗的是烟头的那个红点"
- 不识字但会把"封"字写半页——"封只要一个横折钩再点一笔竖。锁的笔画太多"
- "如果以后有一个戴眼镜拿全站仪的女孩子来找你问祠堂——你请他们喝碗粥，葱花三粒。然后带他们上山看门。"

## 四、二婶——日历里的守护

| 属性 | 内容 |
|------|------|
| **姓名** | 二婶 |
| **年龄** | 约55岁 |
| **身份** | 二叔的妻子，面馆老板娘 |
| **性格** | 安静、细致、用行动在最底层撑起一切——"习惯是二婶对这个世界最完整的态度" |
| **核心行为** | 三十一年手写日历，每初一十五盖章六瓣花（对应守祠人给铜锁上油的日子），默默蒸馒头放在厨房桌上——"不通过手递手，通过错开步" |
| **标志性物品** | 手写日历（三十一年，三十一朵六瓣花）、腊月最后一页留白——"白的是给光的" |
| **标志性动作** | 写日历、蒸馒头→换肉包（看到沈渡父亲膝盖受伤）、留白——"不写字。白的是给光的。" |
| **与沈渡的关系** | 无声的后勤——沈渡小时候吃她的馒头长大，长大了才知道馒头怎么来的 |
| **功能定位** | 沉默的补给线——证明守祠人的坚持不是一个人的事，是一个系统 |

### 关键细节
- 光封取代铜锁后，日历倒过来写："给何也留间屋（要换新被），给松树下放一把折叠椅，祠堂门口多放一个搪瓷杯"
- "日历上以后没有'上山送面'这个日期了。她把这个格子留白——不写字。白的是给光的。"

## 五、周婆——最古老的知情者

| 属性 | 内容 |
|------|------|
| **姓名** | 周婆 |
| **年龄** | 80岁以上 |
| **身份** | 隔壁村老人，每两天走四里山路上山烧香 |
| **性格** | 锋利、不废话——眼睛很亮，"像一只在黑暗里待了太久的猫突然被光照到" |
| **核心行为** | 烧香"喂地老爷"——不是供奉，是"喂"。转交爷爷的铜铃给沈渡 |
| **标志性物品** | 拐杖、铜铃（爷爷沈观海的，由她丈夫保管→传给她→传沈渡） |
| **标志性动作** | 把三炷香整把插进香炉，"不是烧给你的——是喂" |
| **与沈渡的关系** | 信息传递者+最后的警告——"地老爷不是吃供品的——它吃声音" |
| **功能定位** | 口述历史载体——六十年一轮回的活见证 |

### 关键台词
- "不是烧给你的。是喂。地老爷不能挨饿。挨饿了它翻身。"
- "你爸说：不要用火。地老爷不是吃供品的——它吃声音。"
- "对着铃喊名字不要太大声，它会听到。但如果它跟着过来的话——跑。"

## 六、爷爷沈观海（第一代守祠人）——种树的人

| 属性 | 内容 |
|------|------|
| **姓名** | 沈观海 |
| **年龄** | 已故（嘉靖三十八年建祠堂） |
| **身份** | 第一代守祠人、祠堂建造者、铜锁设计者 |
| **性格** | 远见——种一棵松树等四百年后的人用树根探测菌毯 |
| **核心行为** | 建祠堂、造六瓣铜锁、种松树（"观海植松一株于祠前"）、半口气压在碑底 |
| **标志性物品** | 铜铃（后来传给沈渡）、怀表（停于同一时刻4:03，壳内羊皮纸标注"睡眠层"） |
| **与沈渡的关系** | 隔着五代人的对话——沈渡用光封替他完成了"锁不再需要铜"的遗愿 |
| **功能定位** | 起点——整个故事因为他当年建了这座祠堂 |

## 七、何也（第七代守祠人）——光的孩子

| 属性 | 内容 |
|------|------|
| **姓名** | 何也 |
| **年龄** | 约25岁 |
| **身份** | 研究生（测绘/光学方向）→ 第一个非沈姓第七代守祠人 |
| **性格** | 理性、精确——用全站仪测绘光封数据；但也会在阳台装私人校准镜"用偏角差调校" |
| **核心行为** | 用全站仪证明一度角差是400年前预留；继承铜铃；论文答辩核心发现是"光封不需要血脉" |
| **标志性物品** | 全站仪、卷尺（沈渡传的旧尺）、铜铃（沈渡传的） |
| **标志性动作** | 测数据、写论文、用精确到毫厘的方式做最浪漫的事（证明光封稳定） |
| **与沈渡的关系** | 传承者——"如果以后偶然遇到一扇不需要开也能封住的门……替我量一下门宽，多半是1.2米" |
| **功能定位** | 未来——证明沈渡做到了"第七代不限姓沈" |

### 关键细节
- 论文答辩被问光封原理，她答："妈妈在给爸爸送茶。光只是替她跑了一趟。"
- "光封稳定的条件是：任何两个人能同时在一面镜子里看到对方，也看到自己。"
- 在阳台装私人校准镜——"也在用偏角差调校。不是不放心——是习惯。习惯是最完整的担当。"
