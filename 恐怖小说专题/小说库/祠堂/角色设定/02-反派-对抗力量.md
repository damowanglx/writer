# 反派/对抗力量设定卡：《祠堂》

> 《祠堂》没有传统意义上的"反派"。对抗力量是分层的——从可触摸的菌毯到不可名状的存在。

---

## 一、对抗力量分层

### 第一层：菌毯/红泥（物理层）

| 属性 | 内容 |
|------|------|
| **本质** | 地老爷的表皮分泌物——地下巨大生物体的"皮肤"和"消化系统" |
| **形态** | 暗红色泥浆状，有弹性（踩上去像"绷紧的皮鼓"），含多糖类物质（"甜的"） |
| **行为** | 子时渗出、天亮停止；有方向性（向山下蔓延）；有触须状边缘（"像菌丝在培养基表面扩张"）；能吞噬水泥（"泥把水泥吃了"） |
| **弱点** | 石灰脱水（石灰吸水→泥干→结壳→地下气流减弱）；切断热泉供能→菌毯钙化变白 |
| **恐怖机制** | 环境恐怖——不是攻击你，是在你脚下缓慢蔓延。你踩在上面，它"接纳你的重量" |

### 第二层：地老爷（本体层）

| 属性 | 内容 |
|------|------|
| **本质** | 比神更古老的不可名状存在——在地下睡了六百年，每六十年翻一次身 |
| **形态** | 不可见。通过菌毯、甜味、红泥、石碑、地震间接感知。本体在热泉层以下——"睡眠层" |
| **行为** | 吃声音（不吃供品，不靠香火）；学人说话（捕捉声带震动频率）；翻身=地震=村里死人 |
| **弱点** | 可以被命名（"沈渡"二字刻入石碑→锁住）；失去热泉供能→菌毯钙化→"名进了锁。锁在门后面。" |
| **恐怖机制** | 认知恐怖——它不是恶的，它只是在"存在"。它的存在本身就压迫一切。地老爷不是神——是比神更古老的东西。 |

### 第三层：时间/熵（哲学层）

| 属性 | 内容 |
|------|------|
| **本质** | 六代人的消磨——铜锁在第六代用尽，不是地老爷变强了，是人撑不住了 |
| **形态** | 铜牌三道裂纹（六代扭力）；松树枯死（菌毯堵死根系）；"锁尽名竭则物出" |
| **恐怖机制** | 传承恐怖——不是地老爷要出来，是锁撑不到第七代了 |

## 二、对抗力量的行为逻辑

地老爷不是传统反派——它没有任何"恶意"。它的行为逻辑是：

```
本能（非恶意）         反应（非攻击）           后果（对人是灾难）
─────────────────────────────────────────────────
菌毯蔓延（生长）    →  泥土变红、植物枯死    →  土地不能耕种
吃声音（感知方式）  →  捕捉声带震动频率      →  学人说话（恐怖谷）
翻身（生理节律）    →  地震                    →  村里死人
被命名（认知收束）  →  石碑锁住部分意识      →  暂时抑制
热泉被切断          →  菌毯钙化、本体休眠    →  不再蔓延
```

**核心设定**：地老爷"不怕光——怕火"（或火会触发不可控反应）。这条规则从未被打破，也从未被解释——是全书最重要的留白。

## 三、六代人的对抗史

| 代 | 人物 | 对抗方式 | 代价 |
|----|------|----------|------|
| 第一代 | 沈观海 | 建祠堂、造铜锁、种松树 | 半口气压在碑底 |
| 第二代-第四代 | 沈家先人 | 维护铜锁、守门 | 铜牌裂纹两道（四代扭力） |
| 第五代 | 沈父 | 走进门后封菌毯、传讯儿子 | 三年困门后，不能说话（越说菌毯越学） |
| 第六代 | 沈渡 | 命名地老爷、光封取代铜锁 | 十四年回避+双手建光封 |
| 第七代 | 何也 | 非沈姓掌铃人，光封维护 | 不再需要血脉 |

## 四、恐怖美学定位

| 维度 | 定位 |
|------|------|
| **恐怖类型** | 中式民俗克苏鲁——客家祠堂 × 不可名状 |
| **恐怖节奏** | 慢恐怖——不是跳吓，是"踩在菌毯上的每一步都被它接纳" |
| **恐惧来源** | 存在本身 —— 地下的东西不是要出来，是它一直在那里 |
| **核心意象** | 祠堂门缝往外渗红泥、甜的——六百年来一直在渗 |
| **最恐怖的一幕** | 沈渡发现菌毯在学他爸说话——"它在学——它在练" |
