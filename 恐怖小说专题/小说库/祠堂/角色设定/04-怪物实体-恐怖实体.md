# 怪物/恐怖实体设定卡：《祠堂》

---

## 一、实体总览

《祠堂》的恐怖实体是一个**分级系统**，从可触摸到不可名状：

```
地表层：菌毯/红泥（可触摸）
   ↓
岩壳层：菌毯根网（半可感知）
   ↓
热泉层：地老爷本体（不可见但可定位）
   ↓
睡眠层：地老爷意识（不可名状）
```

---

## 二、第一级：菌毯/红泥（表皮系统）

### 基本信息
| 属性 | 内容 |
|------|------|
| **名称** | 菌毯（村里人叫"红泥"） |
| **本质** | 地老爷的表皮+消化系统——地下生物体的"皮肤"和"口器" |
| **首次出现** | 第001章——祠堂门槛下往外渗红泥 |
| **规模** | 渗透方圆几公里土壤层，最深可达30米以下 |

### 物理特性
| 属性 | 描述 |
|------|------|
| **颜色** | 暗红色——"像猪血放了太久快干之前的那种红" |
| **触感** | 有弹性——"像踩在很厚很韧的果冻上"；有张力——"像踩在一面被绷紧了的巨大皮鼓上" |
| **温度** | 温的——体温，不是冷的（吸收地热） |
| **味道** | 甜的——多糖类分泌物，"像泥土自己在泌糖" |
| **活性** | 渗出的泥有方向性（向山下蔓延），触须状边缘（"像菌丝在培养基表面扩张"），能拉出有弹性的黏丝 |
| **腐蚀性** | 能吞噬水泥——"泥把水泥吃了" |

### 行为模式
| 行为 | 描述 |
|------|------|
| **渗出节奏** | 子时开始渗出，天亮停止——日周期 |
| **方向性** | 向山下蔓延——"每片泥印边缘都有触须状边缘朝下山方向延伸" |
| **吸附性** | 接触皮肤时会缠绕手指——"很细的比头发丝还细但有韧性"（被动附着，非主动攻击） |
| **植物影响** | 通过根系往上泵送色素——"所有从这片土里长出来的新生命都在被地下那股红泥渗染变成了它的颜色" |

### 弱点
| 弱点 | 原理 | 效果 |
|------|------|------|
| **石灰** | 石灰遇水反应吸热，脱掉红泥水分 | 泥干结壳→酸性气体被抑制 |
| **切断热泉** | 菌毯主干根依赖热泉供能 | 菌毯脱水钙化→变白色磷酸钙粉末 |
| **铜** | 铜的共振频率干扰菌毯感知 | 铜钥匙在附近时地面不会拱 |

## 三、第二级：菌毯根网（传导系统）

### 基本信息
| 属性 | 内容 |
|------|------|
| **本质** | 菌毯的地下根系网络——地老爷的"神经系统" |
| **分布** | 从热泉层向上辐射，主干根插在热泉上泵送热水 |
| **与铜铃的关系** | 铜铃骨舌可感知根网方向——铃舌偏角=菌毯根网在罗盘上的方位角 |

### 物理特性
- 从砖窑挖穿的地下岩壳向外延伸到整个沈家村地下
- 根的延伸方向具有导航性——每个节点都是一条从热泉向外辐射的路径
- 触丝断裂后能在皮肤表面保持短暂活性（"被动附着，像静电吸附灰尘"）

## 四、第三级：地老爷本体（生命系统）

### 基本信息
| 属性 | 内容 |
|------|------|
| **名称** | 地老爷（村里人的称呼，不是真名——真名不可知） |
| **本质** | 比神更古老的不可名状存在，在地下睡了六百年，每六十年翻一次身 |
| **位置** | 地下热泉层——本体压在热泉上面，"被压在第三层的一口地底热泉上面" |
| **规模** | 未知——"不止是菌毯——菌毯只是它的表皮" |

### 生理特性
| 属性 | 描述 |
|------|------|
| **能源** | 热泉——"六百年一直在缓慢向下蚕食岩层，想把热泉整个包进去" |
| **代谢产物** | 糖分（多糖类分泌物——"甜的"源自此）；红泥（菌毯排泄/分泌物） |
| **感知方式** | 吃声音——不是吃供品，是捕捉声带震动频率 |
| **学习能力** | 能学人说话——"在学——在练"；能用被命名的声波传回对话 |
| **周期** | 每六十年翻一次身——翻身=地震=村里死人 |

### 意识特征
| 属性 | 描述 |
|------|------|
| **意识状态** | 半梦半醒——"六百年只是它打的一个盹" |
| **对人类的认知** | 非恶意——它的行为是本能而非攻击。菌毯"接纳你的重量"，石碑"问你的名字" |
| **被命名后的反应** | 石碑收束——"六百年来第一次有生物用这个名字回应了石碑上的提问" |
| **终极恐惧** | 如果它从热泉里完全浮出来——"它就不再需要菌毯了。它会直接从热泉里浮出来。" |

## 六、恐怖机制分解

### 6.1 "甜味"的恐怖设计

甜不是恐怖的常规联想——这正是它的恐怖来源：

```
甜（预期 = 安全/食物/愉悦）
  ↓
甜从门缝里出来（位置异常）
  ↓
甜来自地下（来源异常）
  ↓
浓度高到刺激吞咽反射（生理失控）
  ↓
甜的 = 活的生物的分泌物（本体暗示）
  ↓
"像泥土自己在泌糖"（不可名状）
```

### 6.2 "它不是恶的"的恐怖

地老爷没有任何恶意。它的行为全部是本能：
- 菌毯蔓延 = 生长
- 吃声音 = 感知方式
- 翻身 = 生理节律
- 学人说话 = 认知能力

**最恐怖的地方**：你无法跟它谈判。它不恨你——它甚至不知道你在。

### 6.3 "命名"的双刃剑

```
命名 = 给地老爷一个名字（"沈渡"）
  ↓ 正面效果
锁住它——"名进了锁。锁在门后面。"
  ↓ 负面效果
它听到了名字——它知道有人在。从此它会等人。
```

### 6.4 "脚步声反方向"的恐怖

门槛红泥上的小脚印——五趾，小孩大小，"脚尖是反过来对着门的"。不是人。是从门里走出来的。它在你来之前——已经出去了。

## 七、恐怖美学对比

| 维度 | 《祠堂》地老爷 | 传统克苏鲁 | 中式恐怖 |
|------|---------------|------------|----------|
| **命名** | 地老爷（乡土称呼，降格为"可以对话的东西"） | 不可名状（不可命名） | 民间信仰中"不可直呼其名" |
| **位置** | 祠堂下面——家族祭祀空间正下方 | 深海/外太空（遥远） | 家/宗族下面（亲近→颠覆） |
| **对抗方式** | 命名+铜锁+光封（技术方案） | 逃避/疯狂（不可对抗） | 镇压/封印（可暂时控制） |
| **恐怖来源** | 存在本身就是威胁 | 不可理解 | 打破日常秩序的异常 |
| **结局** | 光封——不是消灭，是持续封印 | 不可战胜 | 付出代价的暂时平息 |
