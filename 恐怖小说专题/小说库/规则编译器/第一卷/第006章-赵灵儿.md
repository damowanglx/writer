# 第一卷：《编译者》

## 第006章 赵灵儿

时代广场负一层。地下停车场。

下午三点。停车场的灯管坏了一半，剩下的在头顶发出嗡嗡的电流声，照得地面一块亮一块暗——每隔三四米就有一段被阴影吞没。空气里混着汽车尾气和潮湿水泥的味道。

陈渡把电动车停在楼梯口，按着林昭发的定位往里走。

C区。最里面的角落。靠近消防通道，旁边堆着几摞废弃的轮胎。那台机器就贴在墙上——灰色的金属外壳，屏幕只有巴掌大，按键是一块覆膜的薄膜键盘。不像他在那栋不存在的写字楼里见到的ATM式终端机——这台更像是打卡机，或者老旧的电力计费终端。

普通人从这里走过去，会以为它是消防栓或者废弃的电表箱。

但陈渡看一眼就知道。那种灰白色的荧光。屏幕边缘微微发热的气息。那股淡淡的金属烧灼味——和第一台一模一样。

他走近的时候，看到一个人。

一个女孩蹲在终端机前面。扎着马尾辫，穿着浅蓝色卫衣，帽沿有一圈被洗得发白的磨损痕迹。她没注意到他。她正盯着屏幕——很小的一块绿色字符屏——手指放在键盘上，悬在半空，犹豫。

他停在她身后大约三米远的地方。

她没回头。但她知道他来了——她的肩膀动了一下，像被突然从身后叫了一声但不想表现出被吓到。

"你也是？"

她没回头，问出这句话。

"什么？"

"看得见这台机器。"

陈渡没回答。她能这么问——说明她知道自己看见的东西不是每个人都能看见的。她已经测试过了。

她转过头来。

二十一岁左右的脸。不是精致那种——圆脸，鼻梁不高，眼睛很大。不是漂亮——是亮。眼睛里有一种他没预料到的东西：好奇。不是恐惧，不是警惕——就是好奇。像一个在野外发现新物种的人。

"你手上有纹路？"她问。

他下意识把右手往口袋里缩了一下。但没来得及。她站起来，走近他，仰头看了看他的右手食指。

然后笑了。

不是紧张的笑。不是礼貌的笑。是真的觉得有趣——嘴角往上弯了一下，眼睛也跟着弯了一点。

"你的也是？我前天发现的。我以为只有我一个人。"

"你是——前天写的？"

"对。我在这边打工——帮人看车，地下二层到负一层都归我管。那天休息的时候走到这个角落，看见屏幕亮着。我以为是什么设备故障，按了一下。"

"然后？"

"然后它让我写一句话。我写了。按了回车。手指上就多了一行——"她伸出左手。食指上有一根极细的黑线，位置和陈渡的一模一样。

"你写了什么？"

她转向终端机，手指在键盘上敲了几下。屏幕显示了她之前输入的内容：

```
此层停车场——不得抢劫。
```

范围：地下B1层全层。内容：禁止抢劫行为。类型：行为规则。

很直接。没有修饰，没有转弯——就是一条最简单的禁令。

陈渡看了看她的脸。她不紧张。没有他那种"我到底卷进了什么东西"的表情。她写规则的时候手没抖。

"你不害怕？"

"怕什么？"

"这条规则——它会在你身上留下痕迹。每多一条——"

"我知道。"她打断他。"我今天来查了一下系统说明书。虽然语言很奇怪——但大概意思是，每一条规则都会消耗你的身体容量，用完了就结束了。对吧？"

她知道。她知道代价。但她还是写了。

"那你为什么还写？"

赵灵儿想了一下，把卫衣袖子拉下来盖住手腕。

"这个停车场上周出了两起抢劫案。有一个阿姨下夜班回来，在地下二层被抢了包。她摔了一跤，腿断了。她女儿是我同事——每天哭着来上班的那种。所以我写了。"

她说得理所当然。不是豪言壮语，不是自我感动——就像说"地上有水所以拖一下"一样自然。

陈渡看着她。不是自大。不是愚蠢。她单纯觉得：我有这个能力，我看到了问题，所以我就用。直线思维。没有中间环节。

"你叫什么？"

"赵灵儿。你呢？"

"陈渡。"

她把头发拨到耳后。"你也是新来的？"

"嗯。"

"那我们一起。"

她说"一起"的时候不是客套。是她在这句话之前是一个人在面对一台发光的机器，不知道这意味着什么——这句话之后就不是了。

她的手机响了。她低头看了一眼——屏幕亮光照在她脸上，上面是一条语音消息的提示。

"我得去上班了——晚上还要帮人看车。加个微信？"

他们加了微信。她的头像是一只胖橘猫蹲在纸箱里。

她跑走了。浅蓝色卫衣在停车场昏暗的灯光里晃了几下，消失在消防通道的拐角。

陈渡站在原地。

他转身看那台终端机。灰白色的屏幕亮着，上面显示着刚才的操作记录：

```
规则已接受。
编译者#48。
当前纹路：1。
此层停车场——不得抢劫。
```

#48。

不是#47。

不是他以为的"唯一新编号"。

还有其他人。不是只有他一个。这个城市里——或者说，这个系统里——还有更多新人在觉醒。他不知道这意味着什么。但赵灵儿出现之前，他以为自己是唯一的。现在不是了。

他站在终端机前，伸手摸了一下屏幕边缘的温度。温的。和她刚写的规则一样新。

他能感觉到——这座城市的规则网络正在扩张。不是终端机在扩散——是有人或者有某种力量，在引导新的编译者进入系统。赵灵儿是#48，他是#47。两个月前这台终端机才出现。两个月——出现了两个新人。

赵灵儿消失后的通道里，灯光闪了两下。

他的手机震了。

一条规则通知——不是他写的。不是赵灵儿写的。第三条。在同一栋建筑里。距离不远——可能就在二楼，或者三楼。

有人也在这栋楼里，刚刚写了一条规则。

陈渡抬起头，扫了一圈停车场的角落。

灰白色的灯光在头顶嗡嗡响。消防通道的指示灯泛着绿光。通风管道里有风的声音。

灯光又闪了一下。

有人在看他。

（第006章 完）
