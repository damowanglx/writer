# 第038章 防空洞第一层

铁栅栏门在他们身后自动关上了。

没有声音。没有锁扣的咔嗒声——就像门自己知道他们进去了，不需要锁。

通道向下延伸。大约三米宽，两侧是粗糙的混凝土墙壁，每隔五米有一盏应急灯，发出昏黄的、仿佛随时会熄灭的光。地面是水泥的，表面有细密的裂纹，裂纹里积着陈年的灰尘。

陈渡和赵灵儿并排走。

脚步声在通道里产生了回音，每一步都像在告诉别人他们在哪。

走了大约三十米后，通道向左拐了一个弯，然后突然开阔起来——一个大约五十平方米的地下空间。

防空洞第一层。

地面是压实的泥土。墙壁上还保留着七十年代的标语——"深挖洞 广积粮"——红油漆写的，大部分已经剥落，只剩下几个字的残影。空间中央摆着一张木桌——不是古代那种——是一张普通的、学校教室里用的那种木桌，桌面上刻着几个名字缩写，还有一些歪歪扭扭的几何图形。

木桌上放着一台终端机。

不是ATM外壳。不是打卡机。不是金属壳。

是木头的。

一台外壳是实木的终端机——深棕色，表面有漆面剥落的痕迹，露出下面深色的木纹。屏幕是嵌入在木头里的——一块大约iPad大小的液晶屏，黑底绿字。

赵灵儿走到它前面。她没有碰——只是低头看屏幕。

屏幕亮着——不是待机画面——是一张图。

一张系统拓扑图。

上面画着五个节点——时代广场、春柳路、居民楼设备层、旧法院、电视塔——以及它们之间的连接线。连接线有粗有细——粗的是数据流量大的通道——细的是几乎不使用的废弃通道。

拓扑图的中心——不是电视塔——是防空洞。

也就是他们现在站的位置。

"这台终端知道我们来了。"赵灵儿说。

屏幕上的文字变了：

```
欢迎，编译者#47。
欢迎，编译者#48。
系统检测到双人同步。
当前同步率：74%。
已满足第一层通行条件。
```

屏幕下方——木桌的侧面——无声地弹出一个抽屉。抽屉里放着两样东西：一把钥匙——铜制的，很旧，表面有一层绿色的氧化层。一张纸条——叠得很整齐。

陈渡拿起钥匙。铜的，大约三指长，齿痕不对称——不是普通的门锁钥匙，是一种特制的锁。

然后他打开纸条。

纸条上的字是手写的——不是系统打的——是人写的——墨水是蓝黑色的，钢笔字：

"第三层的门需要两把钥匙。一把在这里。另一把——在#12口袋里。"

陈渡看着那行字。

在苏墨口袋里。

苏墨已经有一把钥匙了。他早就来过第一层——但他没有拿走第一层的钥匙。他留在这里——等陈渡来拿。

这从一开始就是一条双人路径。一个人拿不到完整的钥匙。两个人——一个拿第一把——另一个口袋里有第二把——才能在第三层汇合。

苏墨在等他在第三层碰面。

赵灵儿也看到了那条消息。

她沉默了几秒。然后她开口了——语气里没有恐惧，没有愤怒——是一种接近分析的冷静：

"他给你的第六条规则留言——说'你犯了一个错误'——不是真的在批评你。是在确认你写规则的速度。他在测量你的发育进度。就像老周说的——你在被他培养。"

陈渡把钥匙握在手心里。铜很凉——比这个空间的温度还要凉——像刚从冰箱里拿出来的。

"他现在在哪？"

赵灵儿感知了一下——她的五条规则在狂写期，感知范围扩大到了平时的两倍。大约用了五秒——她睁开眼睛：

"在入口外面。和老周在一起。"

陈渡拿着钥匙的手紧了一下。

苏墨在入口外面。和老周在一起。他在等他们拿到第一把钥匙。

如果他现在下来——他和赵灵儿被困在防空洞里——前后夹击。

但如果他不下来——他们拿到了钥匙也出不去。

"他在等我们拿到钥匙——然后下来找我们。"陈渡说,"他知道第一层钥匙是诱饵。但他需要你拿到它——因为你拿着钥匙进第三层——他才能在你进去的时候从你手里接过去。"

"或者——"赵灵儿说,"他在等我们两个同时站在第三层门口——然后他出现，用他的钥匙配合我的钥匙——打开门——自己进去。"

陈渡把钥匙放进口袋。

"不管哪种可能——我们都需要先看到第三层的门。"

他走向第一层深处的第二扇门。

那扇门和进来的铁栅栏门不同——是实木的。深棕色，和木桌同一种木材。门上有两个钥匙孔——左右各一个。需要两把不同的钥匙同时转动才能打开。

他站在门前，手握着口袋里那把铜钥匙。

另一把——在苏墨口袋里。

"如果我现在用这一把——"

"会卡住。一把钥匙打不开。需要两把同时转。"

陈渡把手从口袋里拿出来。

"那我们等他来。"

赵灵儿没有说话。

她站在他旁边，面对着那扇实木门。头顶的应急灯发出持续的、低沉的嗡嗡声，和防空洞深处传来的某些频率混合在一起——像整座城市在通过这个通道呼吸。

大约过了不到两分钟。

通道入口的方向——第一层的外侧铁门——传来一声轻微的金属碰撞声。

然后是脚步声。

不急。不慢。节奏均匀——每一步之间的间隔完全相等。

苏墨从通道的阴影里走出来。

他站在第一层空间的入口处，手插在口袋里，脸上没有任何表情。

"你们拿到了。"

不是问句。是确认。

他伸手从自己的口袋里掏出一把钥匙——铜制的——和陈渡手里那把一模一样——只是氧化程度稍微浅一些，说明他拿到的时间比陈渡早不了太久。

"我在三个星期前拿到这把钥匙。"苏墨说,"然后我等了三个星期——等另一个人拿到另一把。"

他向前走了几步。在距离陈渡大约四米的地方停下来。

"你们有两个选择。第一——我们现在一起开门。进去。看到里面有什么。然后——我们再决定下一步。"

"第二呢？"

苏墨的手从口袋里拿出来——掌心里握着那把钥匙:

"第二——你们现在跑。我追。追到之后——我自己用两把钥匙开门。"

他说话的语气像在介绍超市里的两种商品。

陈渡没有说话。

他转头看了赵灵儿一眼。

赵灵儿看着苏墨——她的表情不像在看一个猎杀者。像是看一个她研究很久的问题，现在终于到了解答的时候。

"开门。"她说。

苏墨没有意外。他走到门前，站在右侧的钥匙孔前。

陈渡站在左侧。

两把钥匙同时插入。

两把钥匙同时转动——方向相反。

木头门内部发出一声沉闷的机械声——锁芯转动了。

门——没开。

但门上浮现出一行字——不是刻上去的——是从木头纹理内部渗出来的——

白色的光——像液体一样在木纹之间流动——

形成了一行字：

"编译者#47 和 #12 —— 同步率检测中 —— "

进度条出现——从0%开始走——

走到56%——停住了。

门没有开。

文字更新了：

"同步率不足。当前：56%。需要：75%以上。"

陈渡站在门前。

他和苏墨的同步率——只有56%。

他之前和赵灵儿一起站在终端机前——同步率是73%。

他需要的不是苏墨。

他需要的是赵灵儿。

他低头看着自己插入钥匙的手——然后转头看赵灵儿。

她把她的右手——覆盖着五条规则纹路的手——放在了木门的表面。

门上那行字变了：

"检测到编译者#48的规则频率——正在覆盖#12的频率——"

进度条重新开始走——从56%往上爬——57——63——71——

停在了74%。

门——开了。

不是向里开——是向两侧滑动——像一扇隐藏在木纹中的推拉门。

门后是一条更窄的通道——只够一个人通过。

通道尽头——有光。

不是灯的光——是屏幕的光。

青白色的——像旧电脑显示器发出的那种光——在通道深处稳定地亮着。

赵灵儿的手还没有从门上放下来。她的五条规则纹路在接触木门之后——亮了一下——然后暗下去。

"苏墨——你同步率不够。"赵灵儿说,"不是门不认你——是门选了别人。"

苏墨没有回答。他站在门右侧的位置——口袋里的钥匙还在，他的手已经拿了出来。他看着洞开的通道，和通道尽头那青白色的光。

他的表情——第一次——出现了一种陈渡从未见过的东西。

不是恐惧。

是饥饿。

一种长期没有得到满足的、深不见底的、沉默的饥饿。

（第038章 完）
