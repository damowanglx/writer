# 第一卷：《编译者》

## 第005章 痕迹

陈渡回到日常生活的第二天。

早上七点。闹钟响了。他起床洗脸，看了一眼镜子里的右手——那根黑线还在。他不再试着擦掉了。

上午跑了九单。中午回站点充电的时候，姜南递给他一瓶水。

"你这几天脸色不对。是不是你爸那边又有事？"

"没有。"

"屁。你脸都绿了。"

陈渡拧开瓶盖喝了一口。"真没事。"

姜南没追问。他来站点不到半年，和姜南算不上多熟——但送外卖的人之间有一种默契：你不想说，我不问。但你撑不住的时候，我可以顶你班。

换了电池，陈渡继续跑单。

但今天不一样了。

他开始用意识去碰周围的环境。

不是闭上眼睛的那种感知——是睁着眼睛的时候，在正常的视觉之上叠一层额外的频道。像在听一个微弱电台信号。起初全是噪音——三十二条街区的交通声、人声、店铺音乐声——都混在一起。但他慢慢学会了调频。

不找声音。找共振。

规则在这个城市里，像低频电流。不发声，但如果你把意识放得足够轻，能感觉到一种规律的震动。

第一站——春柳路往北三百米的一条老巷子。

这条巷子他跑过三十多次。老居民区的背街，墙上有青苔，地上永远没有垃圾。不是没有垃圾产生——是没有垃圾存在。他之前以为是环卫扫得勤。现在他把感知放出去——墙根处有一层几乎不可感知的"膜"，贴在地面和墙体交界的位置。

他读到了：

```
此巷——垃圾自动转移至市政垃圾站。
范围：整条巷。
类型：物理规则。
```

自动转移。不是清洁工来扫——是垃圾本身会被传走。六年了。这条规则已经稳定运行了六年。六年来每一个在这条巷子里居住的人，扔在路边的垃圾都会在落地后五分钟内消失。没有人觉得奇怪——他们只是觉得这条巷子特别干净。

第二站——解放路拐角那家便利店。

不是林昭去的那家。是另一家。这家店永远有人在排队——不管几点。早上七点有人排队，中午有人排队，凌晨两点还是有人在排队。三五个人站在门口，不多不少，表情平静。他之前以为是生意好。

现在他感觉到了——那家店门口有一道几乎和墙体融为一体的规则。不仔细感知根本发现不了——因为它嵌在"正常"里。

```
此店——顾客自发排队。
不可插队。不可拥挤。
类型：认知规则。
```

认知规则。不是强制行为——是让每一个人"自然而然"地觉得应该排队。写这条规则的人很清楚自己在做什么——用了认知型而不是行为型，让人不会感到被控制，只会觉得"这家店值得等"。

第三站——时代广场对面那栋写字楼的电梯。

他送餐到十五楼。等电梯的时候，门开了，里面站着两个白领。他从十七楼下到一楼——整个过程平稳到不真实。每一层开门，关门，加速，减速——都精确到感觉不到惯性。

他感知了一下电梯井。

有一条细如发丝的规则缠绕在缆绳上，像藤蔓一样贴着钢索表面蔓延。他贴近了读：

```
此电梯——每层停留时间误差不超过±0.01秒。
类型：物理规则。
```

精确到百分之一秒。写这条规则的人要么有强迫症，要么在保护什么东西——一栋电梯不需要这种精度，除非有人在用它做比运输人更精确的事。

他不是在扫描城市。

他是在发现自己的盲区。这座城市——他送了三年外卖、跑了上万条路线、闭着眼睛能画出城区地图的城市——从来不是他以为的那座。规则层一直存在。贴在墙上，嵌在地面里，缠在缆绳上，像微弱的电流在所有缝隙里流动。

三十二年来，几十个编译者在这座城市里写下了上百条规则。普通人感知不到，也永远不会知道。但他们一直在被规则影响——只是他们以为"这条巷子本来就这么干净"。

下午跑完最后一单，他停在路边，打开手机备忘录。

他看着自己的配送路线——三年积累的轨迹。如果把每条路线都画出来，它们就是一张覆盖全城的网。一个交叉的、重叠的、几乎覆盖所有街区的网格。

他意识到一件事。

送外卖是这个城市最完美的探测行动。没人会注意一个骑手在每栋楼之间穿梭——外卖骑手本来就哪里都去。他每天跑的路，就是他测绘的规则地图。

他在备忘录里画了一个粗略的城市分区图。标注了四个主节点——旧法院、电视塔、城南钢厂、城西防空洞。然后用自己常跑的路线把四个点连起来。

对角线交叉的地方——时代广场。

不是任何一个主节点。是中心点。

他盯着那个点看了很久。

时代广场。他每天至少经过两次的地方。地下一层是停车场，再下一层是地铁站，地面是购物中心和写字楼群。如果他爸的事故不是意外——如果某条规则覆盖了他爸的死亡——

那那条规则在哪里？

谁来写的？

为什么写着那条规则的人——要覆盖一个工地上的意外？

他的手机震了。林昭的短信。

他低头看：

"明天下午三点。时代广场负一层。停车场西南角。有一个终端机。去看看。"

他盯着屏幕。时代广场。他刚画完的中心点。

他打字回复：

"为什么是那里？"

过了大概两分钟。回复：

"因为那是最新加入网络的终端机。两个月前才出现。你去看看——它上面写着什么。"

他锁了屏幕。把手机插回支架。

时代广场负一层。终端机。

他拧了油门。

（第005章 完）
