# 第044章 苏墨的刀

他们走出电视塔维修通道的时候——苏墨在出口外面等着。

他站在阳光里，靠着路灯杆，手里拿着一杯便利店的咖啡。纸杯壁上凝结着水珠。他喝了一口，放下杯子，看着赵灵儿从通道口走出来。

他的目光在她的右臂上停了一下。

“你写了。”

“写了。”

苏墨没有说恭喜。没有说遗憾。他喝了一口咖啡，然后看着陈渡。

“你现在感知一下她的规则链。”

陈渡不解。但他按照苏墨说的——闭眼感知赵灵儿的六条规则。

在他的感知中——赵灵儿的六条规则原本是六根独立的频率线——各自运转——互不干扰。但现在——在她写完第六条规则之后——六条规则像被一根看不见的线串在了一起。它们不再独立运转——它们互相缠绕——形成了一个闭环。

六条规则——闭合了。

像一个被拧紧的螺母。

她的规则链——不再是六条单独的规则——是一个整体。一个规则系统。

这就是编译前兆。

她的规则不再是她“写的”东西——它们变成了她本身。

“你知道这意味着什么？”苏墨问。

陈渡没有说话。

“意味着——如果有人攻击她的任意一条规则——等于攻击她全部六条。攻击会沿着规则链传导——六条规则同时承受冲击。防御力不是叠加——是平分。她全链中最弱的一条——就是她整体的防御值。”

苏墨说这些话的时候——语气像在教课。

“我教你一个规则基础课没讲的知识点——当一个狂写期的编译者规则闭合之后——她的规则链会自动寻找下一个要‘吸收’的规则来源。通常是——离她最近的编译者。”

苏墨的目光落在陈渡身上。

“她现在离你最近。”

赵灵儿的手臂——六条规则——在苏墨说这句话的时候——突然亮了一下。

不是她在催动。

是自动反应。

她的规则链——在寻找他的频率。

陈渡后退了半步。

“控制一下。”他对赵灵儿说。

“我在控制。”赵灵儿的声音——第一次——带上了一种陌生的东西。她握着自己的右手臂——手背上的纹路在自发地一亮一暗——像呼吸。

“她在控制。”苏墨重复了一遍这句话，“但她的规则链——已经不完全是她在管了。你站得离她越近——你的规则频率就越会被她的闭合链吸引。系统会以为——你是她规则链的一部分。”

他喝了一口咖啡。

“你现在有两个选择。第一——离开她——至少五十米。这样她的规则链感知不到你的频率。她独自承受编译前兆——十七天后——编译完成。”

“第二呢？”

“第二——你靠近她——写你的第七条规则。你用新规则的频率覆盖她的闭合链——打破她的闭环。但代价——”

他把咖啡杯放在路灯杆的底座上。

“——你的纹路会推到上臂。”

陈渡站在正午的阳光下。温度大约三十度——但他感觉不到热。

赵灵儿站在他旁边——她的右臂纹路在持续发着微弱的灰白色光。她能听到——他也能听到——六条规则在她体内运转的声音——像六条河流在地下交汇的低沉轰鸣。

他还没做出选择。

苏墨替他选了。

苏墨没有动——但他的十七条规则——在没有任何前兆的情况下——同时激活了。

不是攻击陈渡。

是攻击赵灵儿。

一道规则——陈渡来不及看清是什么——像一根无形的针——刺入了赵灵儿的规则链。

她闷哼了一声——膝盖弯了一下——用手撑住了路灯杆。

不是物理伤害。

是规则层面的撕裂。

苏墨写了新规则——不是写在终端机上——是直接用他的纹路输出——一条针对赵灵儿规则链的“干扰规则”。规则内容他看不到——但效果他能看到。

赵灵儿的六条规则——在抵御那条干扰规则的时候——产生了共振。六条规则同时在对抗——但每条对抗的力度不同。三条强的在抵抗——两条中的在颤抖——第六条——刚写的那条——在承受全部冲击。

苏墨没有杀她。

他在攻击她规则链里最脆弱的那一环——她刚写的那条“开放权限”规则。

如果那条规则断裂——她不仅会失去对管理员的开放权限——她的整个规则链会因为第六条的塌陷而崩解。六条规则全部崩塌——她会瞬间进入完全编译——不是十七天——是十七秒。

陈渡冲上前——挡在赵灵儿前面——但他知道这没有用。规则攻击不是物理攻击——挡不住。

他必须写规则。

在他的纹路输出范围之内——在没有任何终端机的情况下——用他的身体直接写。

他低头看自己的右臂。

六条规则的纹路在皮下发热。

第七条——正在成型。

不是他决定要写——是他的身体在应激反应下——自动开始了写入。

他的右手不自觉地握紧。规则文本——不是从键盘上打出来的——是在他脑子里——被身体直接编译成了纹路——出现在他的皮肤表面——从肘部上方开始——

他开口了。声音沙哑——不像自己的：

```
编译者#47的第七条规则——
在编译者#48处于规则链震荡状态时——
编译者#47自动启动频率覆盖——
以其规则链的全部频率——
覆盖编译者#48的规则链震荡区域。
```

他说完最后一个字的瞬间——他的右臂肘部以上的皮肤——裂开了一道黑色的纹路。

不是线——是裂口。

像皮肤被什么东西从内部划开了——黑色的纹路从裂缝里渗出来——沿着上臂外侧向上蔓延——停在了肩关节下方大约三厘米的位置。

他的第七条规则——在没有终端机的情况下——用身体直接写入了。

系统的确认消息——不是出现在屏幕上——是直接出现在他的感知中：

```
编译者#47。
第七条规则——写入完成。
规则类型：紧急频率覆盖型。
纹路进度：指尖至上臂。
警告：该规则为应激写入——规则文本未经系统完整性校验。
可能包含未检测到的副作用。
```

陈渡的右臂——从肘部到上臂——新纹路的区域在持续发热。皮肤表面有一种撕裂后的刺痛感——像被刀割过的伤口在愈合。

但规则生效了。

赵灵儿的规则链共振——在他的频率覆盖下——稳定了下来。

她松开路灯杆，站直了。

“好了。”她说。声音有点喘——但稳定下来了。

苏墨看着她——又看了看陈渡。

“你写第七条了。”

陈渡低头看自己的手臂——第七条规则的黑色纹路——从肘部到上臂中段——比其他六条更粗——颜色更深。像一条河的主干道。

“你满意了？”陈渡问。

苏墨没有回答。他弯腰拿起路灯杆底座上的咖啡杯——已经空了——他把杯子扔进旁边的垃圾桶里。

然后他转过来，看着陈渡——

眼神里——第一次——出现了一种陈渡看不懂的东西。

不是饥饿。

不像在防空洞第二层门口的那种饥饿。

是接近——

接近满足。

“我一直在等你写第七条规则。”

他说完这句话——转身走了。

陈渡站在原地看着苏墨的背影消失在街角。

他低头看自己的手臂——七条规则从指尖一路延伸到上臂——在皮肤表面构成了一张不对称的、黑色的地图。

然后赵灵儿说了一句话——声音很低——像在自言自语：

“陈渡——你的第七条规则——不稳定。”

“什么？”

“你的频率覆盖——覆盖了我的震荡。但覆盖的同时——你的规则链也和我产生了连接。我现在能感知到你的七条规则——比我自己写的六条还要清晰。”

她抬起头看着他——她的眼睛里——灰色的线在持续蔓延——已经快要覆盖到整个眼白了。

“你的第七条规则——有一个漏洞。不是系统没检测到——”

她停了停。

“——是系统故意没告诉你。”

陈渡的手握紧了车把。

“什么漏洞？”

赵灵儿没有回答。

她低头看自己的手臂——她的六条规则——在陈渡的第七条规则覆盖她之后——纹路的颜色发生了变化。

从灰色——变成了一种和陈渡第七条规则一样的黑色。

她的规则链——正在被他的频率改写。
