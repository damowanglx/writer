# 第一卷：《编译者》

## 第001章 不存在的写字楼

雨是在晚上七点开始下的。

陈渡把电动车停在便利店门口,扯了扯雨衣帽子。手机支架上弹出新单——配送距离四点二公里,从春柳路到时代广场。备注栏只写了两个字:加急。

他扫了一眼路线。正常走大路要十五分钟。穿过那片老写字楼区能省五分钟——只不过路不好走,全是单行道和死胡同。他把手机往支架上卡好,拧了油门。

雨越下越大。雨刷刮不干净,红色尾灯在水雾里糊成一片。拐进写字楼区的时候路面已经积了水,电动车轮碾过去溅起的水花打在裤腿上,冰凉的。

他骑了大概三分钟才注意到不对劲。

这条路他跑过至少两百次。春柳路往西,经过一个废弃的保安岗亭,左拐进两栋写字楼之间的窄巷,再右拐上大路。但他拐进巷子的时候——巷子不在原来的位置。

不对。不是巷子移了。是多了一栋楼。

他捏了刹车。电动车在雨里停下来,前轮刚好碾过一滩积水。他抬头。

巷子左边的位置——原本是一块围起来的空地,铁皮围挡上写着"施工中"——现在立着一栋写字楼。不是新楼。灰色的外墙上爬满了黑色霉斑,楼顶的广告牌锈得只剩骨架。门口没有保安。整栋楼一盏灯都没亮。但一楼是亮的——不是灯光。是某种灰白色的荧光,从一楼的玻璃门后面渗出来,在雨雾里看着像电视机没信号的雪花屏。

他低头看手机。导航上他的定位光标停在一片空白区域。没有路名。没有建筑标注。连卫星图层都是模糊的——像被人从地图上抠掉了。

他把手机翻过来。信号满格。但地图上确实没有这栋楼。

雨砸在头盔上噼里啪啦。他应该继续开的。这单快超时了,超时扣钱。但他看了一眼那栋楼——玻璃门后面,荧光还在闪。频率很规律。每三秒亮一次,每次亮大约半秒。不是日光灯。是某种他没见过的东西。

他把电动车停到对面楼檐下,锁了车,脱下雨衣。手机拿在手里,手电筒打开。玻璃门没有锁。他推了一下,门往里开了——不是往外。是往里。门后面不是大堂。

是电梯。

一部老式电梯。不锈钢门,门缝里渗出灰白色的光。电梯按钮只有一个——不是数字,是一个符号。他盯着看了大概三十秒才意识到那是什么:一个回车键。

他不应该进去。

他自己也不知道为什么按了那个按钮。可能是跑单跑太久了——太累了。可能是他爸的事还没结果——心里一直堵着。可能是那栋楼不该存在——而不存在的东西,应该有人去看看。

电梯门打开了。

里面不是电梯井。是一间大约二十平米的机房。灰色的墙壁,灰色的地板,灰白色的荧光从天花板均匀地打下来。房间正中央立着一台机器——看起来像九十年代的银行ATM,外壳是浅灰色的塑料,屏幕是暗绿色的CRT显示器。

屏幕亮着。上面只有一行字。白色字符,黑色背景:

```
第47条规则已失效。请补充。
```

光标在闪。

陈渡站在机器面前。雨水从他的衣领往下滴,在灰色地板上积了一小滩。他的手机显示距离超时还有六分钟。机房里的空气很干——不是空调的干,是旧电脑机房里那种电子元件发热的干。他闻到一股淡淡的金属味。像电路板被烧过之后的味道。

光标还在闪。

他把手指伸到屏幕前。不是触摸屏。下方有一个机械键盘——灰白色的键帽,回车键是深绿色的。键盘很干净。没有灰尘。像有人在用。

他应该走的。他真的应该走。

但他想到一件事。今天下午在劳动局门口等了三个小时,前台说包工头的公司注销了——找不到了。他在雨里站了十分钟,然后在手机上接了一单又一单,靠跑单把脑子放空。他不相信自己能改变什么事。世界有一套规则——我爸死了,我讨不到赔偿,我今天超时要扣钱——这些规则他改不了。

但面前这台机器,在问他。

他把手指放在键盘上。犹豫了大概十秒。然后打了:

```
此路口禁止鸣笛。
```

他按了回车。

屏幕闪了一下。字符消失了。然后一行新字浮上来:

```
规则已接受。
编译者#47。
当前纹路:1。
```

他低头看自己的右手。食指指尖——握车把磨出茧的那个指头——上面多了一道极细的黑线。很细。像一根头发压在皮肤下面。他用拇指搓了一下。不疼。但刮不掉。

手机震了一下。超时提醒。

他转身跑出机房。电梯门关上的瞬间,他从门缝里回头看了一眼——那台机器的屏幕上,又多了一行字。不是他写的。是系统自己打上去的:

```
欢迎回来。
```

电梯下降。门再次打开的时候他站在写字楼外面。雨还在下。电动车停在原处。他跨上车,拧了油门。脑子一片空白——不是吓的。是他看见的那行"欢迎回来"——语气不是新的。是旧的。像这台机器一直在等他。

他骑到春柳路和解放路的交叉口——他今天早上在劳动局门口就是在等这条路的路口。平时这里从早到晚全是喇叭声,堵车堵得人想骂娘。但现在——静的。

不是没有车。车都在。红灯变绿灯的时候,最前面的出租车司机犹豫了一下,手放在喇叭按钮上——然后放下来了。后面的公交车也没按。再后面的私家车也没按。所有的车安安静静地通过了路口。不是自觉。是到了路口——喇叭不响了。每一个司机都下意识地放弃了按喇叭——然后又下意识地觉得好像没什么不对。

陈渡把电动车停在路边。手在雨衣口袋里攥着手机。手机屏幕湿了,指纹解不了锁。他输密码——手在抖。不是冷。是他刚才写的那行字——那句"此路口禁止鸣笛"——生效了。不是人的意识里生效。是物理上生效了。到路口的车,喇叭不响。

他深吸一口气。打开手机备忘录。翻到最上面那行——他今天早上在劳动局门口打的。那时候只是在想:要是我能写一条规则就好了。要是有个系统能让我改这个世界就好了。

他没想过这个系统真的存在。

他把手机收起来。拧了油门。脑子里的导航还在——但地图变了。他现在不光知道哪条路快,哪条路堵。他需要知道另一件事:

这个城市里还有多少台这样的机器？

雨停了。后视镜里,那栋不存在的写字楼正在被新起的水雾慢慢遮住。他最后看了一眼——一楼玻璃门后面的灰白色荧光灭了。整栋楼又回到了黑暗里。

像从来没亮过一样。

（第001章 完）
