# 第015章 共振

三天后。暴雨后的第一个晴天。

赵灵儿坐在陈渡电动车后座上，一只手抓着他的腰，另一只手举着手机看地图。地图上被她用红色标注了十几个点——是她靠感知找到的可能有终端机的位置。她在时代广场停车场值夜班的时候，用感知扫了整栋楼，发现停车场西南角的终端机会向其他方向发出微弱的频率。

"你确定是终端机的信号？"陈渡拧了油门，车子从巷子里窜出去。

"我跟你描述一下：像一只蝙蝠在墙里叫。每隔两秒一次——不是心跳，是某种脉冲。从电梯井的方向传出来的。"赵灵儿的声音从背后传来，很兴奋，"我蹲在消防通道里数了四百次。频率没变过。不是市电的干扰——市电是五十赫兹，这个是零点五赫兹。"

零点五赫兹。两秒一次。

陈渡之前感知规则的时候，感受到的频率都是连续的——像电流在整面墙上流动。但赵灵儿描述的是脉冲。有节奏的、规律的单次信号。那不是一条正在生效的规则在运行——而是一台终端机在"发送"什么。

他们停在一栋旧居民楼前面。六层，外墙贴着白色瓷砖，一楼是五金店和裁缝铺，二楼以上是住户。铁门上装着老式对讲机，按键面板已经发黄，有几个键帽掉了，露出下面的金属触点。

"就是这里。"赵灵儿跳下车，指着居民楼入口,"感知最清晰的位置——单元门正中央。终端机不在地面层——在地下。"

陈渡蹲下来，手掌贴着单元门前的混凝土地面。感知放出去——穿透水泥层，穿过管道层，触到地下空间的边界。在地下大约四米的位置——一个信号在规律地跳动。不是规则的频率，是终端机本身的脉冲。

他睁开眼睛。

"下面有空间？"

"化粪池上面有个设备层。"赵灵儿说,"我查过这栋楼的建筑图纸——地上六层，地下一层是储物间，再下面是设备层。但图纸上没有标注通往设备层的入口。老楼的建造者留了一堵假墙。"

她推开单元门。楼道很窄，墙上贴满了疏通下水道的小广告，空气中有一股潮湿的、混着洗衣粉和油烟的味道。赵灵儿走到楼梯拐角处，在一面看起来和周围墙体完全一样的墙前面停下来。

墙壁用水泥抹平了，刷着和周围一样的白色涂料。但赵灵儿用指关节敲了两下——声音不一样。实心墙应该是沉闷的"咚"声——但这面墙的回音很空，像敲在一块薄铁皮上。

陈渡摸了摸墙面。灰泥下面摸到一条垂直的缝隙——极细，肉眼几乎看不见，但指尖能感觉到。是一扇伪装成墙壁的门。

"你什么时候发现的？"

"昨天凌晨两点。睡不着，就骑车出来扫街。"赵灵儿咧嘴笑了一下,"我做事比较直接。"

她用力推了一下墙的右侧边缘——墙壁无声地向内打开了。不是普通的铰链门，是滑轨式的。门后面是一条向下的楼梯，窄得只够一个人侧身通过。楼梯没有灯，黑暗像固体一样填满了整个空间。

陈渡打开手机手电筒。光束切进黑暗里，照出楼梯的尽头——大约八米下方，有一扇灰色的铁门。

他们一前一后走下去。赵灵儿在后面，步伐很轻，没有犹豫。陈渡在前面，每走一步都先用脚试探台阶是否结实。楼梯的台阶是水泥的，表面有细密的裂纹，裂纹里积着黑色的污垢。空气越来越闷，温度明显比地面层低了三四度。

铁门没有锁。陈渡推了一下，门开了。

里面是一个大约十平方米的房间。天花板很低，陈渡一米七八的身高伸一只手就能碰到顶。地面是粗糙的水泥，墙角有积水干掉后留下的白色水渍。房间正中央——放着一台终端机。

但这台和之前见过的都不一样。

不是打卡机那种灰白色塑料外壳——这台终端机的外壳是金属的。深灰色，表面有一层细密的氧化纹路，像暴露在海水里很长时间的船用钢材。屏幕也不是CRT——是一块大约A4纸大小的液晶屏，黑底绿字。屏幕边缘镶着一圈极细的金属边框，边框上没有螺丝孔，没有任何接缝——像一整块金属被掏空后嵌入了屏幕。

最奇怪的——这台终端机连着线。

一根小指粗的黑色线缆从终端机底部延伸出来，穿过地板上的孔洞，通向地下更深处。不是电源线。陈渡顺着线缆的方向感知——它向下延伸了至少十米，然后分叉，像树根一样向四个方向散开。

"它在联网。"赵灵儿蹲在线缆出口的位置,"其他终端机是独立的——每一台都是单独运行的。但这台——它在跟什么东西通信。"

陈渡伸手碰了一下屏幕。

屏幕亮了。

不是待机画面——已经在运行中了。屏幕上显示的不是"第XX条规则已失效"的标准提示，而是一个列表：

```
当前网络节点共振状态：
时代广场 | 信号强度：76% | 延迟：0.3s
春柳路   | 信号强度：42% | 延迟：1.1s
本站     | 信号强度：100%| 延迟：0.0s
旧法院   | 信号强度：--%  | 延迟：——(节点休眠)
```

赵灵儿凑过来，呼吸在陈渡耳边变得急促："这是——节点拓扑图。"

陈渡盯着屏幕。时代广场那台终端机——赵灵儿写第一条规则的地方——信号强度76%。春柳路那台——他第一条规则诞生的地方——信号强度42%。旧法院那台最大的终端机——信号休眠。

这台终端机能"看到"其他终端机的状态。

他把手指放到屏幕右侧的一个选项上。选项写着："执行共振探测。"他点了一下。

屏幕闪了。

然后整个房间的空气震动了一下——不是声音，是频率。低频的、几乎不在听觉范围内的震动，从终端机的机箱深处发出，透过地面传到他们的脚底，再沿着墙体向上蔓延。

陈渡手腕上的纹路同时亮了起来——四条规则的纹路在皮肤下发出微弱的灰白色光。他转头看赵灵儿——她的手腕也在发光。两条规则的纹路，和她自己的频率——在共振。

"陈渡——"赵灵儿的声音变了,"你看屏幕。"

屏幕上的列表变了。在"本站"那一行的下面，新出现了一行：

```
检测到编译者双人同步。
编译者#47 —— 纹路进度：4条。
编译者#48 —— 纹路进度：2条。
同步率：73%。
建议操作：执行双人规则写入。
```

陈渡的呼吸停了一瞬。

双人规则写入。这个词他从来没听林昭提过。终端机能检测到他们两个同时站在这里——而且检测到了同步率。73%。不是100%，但足够高到触发了某个隐藏功能。

屏幕下方弹出一个输入框：

```
请输入双人规则。
编译者#47 和 #48 将共同承担代价。
规则强度：约1.7倍。
```

"双人规则。"赵灵儿念出来,"我们可以写一条——两个人一起写的规则。威力更大——代价也——"

"分担。"陈渡接上她的话,"代价不是翻倍——是两个人平摊。"

墙面上的灰尘在终端机的低频震动中簌簌落下，在手机电筒的光束里像一场微型的雪。地下的空气带着混凝土粉末的味道，混着终端机散发出来的那种熟悉的金属烧灼味。

赵灵儿的手指已经放在了键盘上。她抬头看了陈渡一眼——不是询问，是在等他同意。

陈渡的手悬在键盘上方。

双人规则。他不确定这意味着什么。林昭没教过——方诚也没提过。这是系统里更深一层的功能。如果他和赵灵儿一起写一条规则，他们两个的命运就更紧密地绑在了一起。一个人的规则被覆盖——两个人的代价同时触发。一个人的纹路前进——两个人的生存时间同时缩短。

而她完全没有犹豫。

"你的手在抖。"赵灵儿说。

"我知道。"

"你每次犹豫的时候——都在想什么？"

陈渡没有回答。他看着屏幕上那行字——"编译者#47 和 #48 将共同承担代价。"他想起了林昭手腕上那道疤，想起了苏墨坐在便利店椅子上的姿态，想起了方诚说的"替身"。他不知道自己能不能相信任何人。但他知道——如果他永远不迈出这一步，他永远只能一个人面对苏墨。

他深吸了一口气。

"写什么？"

赵灵儿几乎没有思考——手指已经在键盘上敲出了一行字：

```
此楼地下空间——禁止被任何规则锁定。
```

范围：仅仅这一栋楼的地下设备层。内容：不能被任何规则"锁定"——不能被覆盖、不能被追踪、不能被标记为任何规则的作用目标。像一个规则层面的安全屋。

陈渡看着她打完这行字。精准。经济。没有多余的字符。她天生就知道怎么把一条规则写到最干净。

他把自己的手指放到键盘上，和她并排。两人的手指接触到同一个键盘的瞬间——终端机的屏幕闪了一下，键盘上的字母键全部变成了灰白色，只有回车键亮着——深绿色。

他按了回车。

屏幕上的字符没有像普通终端那样直接显示"规则已接受"。它显示了一个进度条。进度条缓慢地从0%走到100%，用了大约五秒。在走到100%的那一刻——

陈渡的右手腕内侧，纹路之下，传来一阵温热的刺痛。不是之前写规则时那种"刻入"的感觉——更像是一根针从皮肤内部向外刺了一下。他低头看——第四条规则留下的纹路旁边，多了一行极细的银色线。不是黑色。是银色。

银色的纹路。

赵灵儿同时撸起袖子——她的左手腕上，同样的银色纹路出现在第二条规则下方。两条黑色规则文字，一条银色规则文字。

终端机上跳出最终提示：

```
双人规则已接受。
编译者#47，#48。
共享规则位：1。
当前同步率：73%。
建议：同步率高于70%时，双人规则有效。低于50%时自动失效。
```

陈渡盯着手腕上那条银色的线。它在皮肤下微微发光，然后慢慢暗下去，变成了和黑色纹路相近的灰色。但底色不一样——凑近了看，还是能看出它泛着极淡的银色光泽。

"我们现在——"赵灵儿看着自己的手腕,"连在一起了。"

她抬起头。脸上不是恐惧——是一种接近于新奇的表情。像发现了一条从未有人走过的路。

"双人规则。终端机能识别两个人同时写规则。"陈渡说,"林昭不知道这个——"

"或者她知道——但她没说。"赵灵儿接上。

他们沉默了几秒。

头顶的设备层通风管里传来一阵风声，像这座城市在通过管道呼吸。铁门外面的楼梯间里，声控灯突然亮了一下，然后又灭了。

陈渡转头看了一眼铁门的方向。

有人在外面。

他感知放出去——频率很模糊，像隔着一层水在看东西。不是编译者的频率——但也不完全是普通人。那个频率被什么东西挡着，像经过了一层过滤。

铁门外的脚步声很轻——是刻意放轻的。一个人。没有上楼梯，没有下楼梯，就站在楼梯拐角处。在听。

陈渡把手电筒关了。

他和赵灵儿在黑暗中并排站着。终端机的屏幕也自动熄灭了。整个地下空间陷入了完全的、固体一样的黑暗。只有通风管里的风声，和他们自己的呼吸声。

铁门外的脚步声向西移动——沿着设备层走廊，逐渐远去。过了大概二十秒，单元门关闭的声音从上方传来。

陈渡重新打开手电筒。赵灵儿的表情在光线恢复的那一刻平静而专注——她在感知。

"走了。"她说。

"你没感知到频率？"

"感知到了——但不是编译者。"赵灵儿皱起眉头,"频率很怪——像一个人身上缠着好几层规则，每一层都不属于他自己。像被别人的规则包裹着走路的。"

陈渡没有接话。

他把手机收起来。手电筒关了。地下空间重新陷入黑暗。

但他记住了那个频率——如果下次再遇到，他能认出来。

（第015章 完）
