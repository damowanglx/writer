# 第036章 第六条规则

他们没直接去防空洞。

电动车骑到半路的时候，陈渡让赵灵儿停了车。他们停在老城区一条窄巷里——左边是一家打烊的理发店，右边是一家关门的水果摊。路灯的光只照亮了巷口的一半，剩下的部分浸在黑暗里。

"怎么了？"赵灵儿问。

"我需要写一条规则——在去防空洞之前。"

赵灵儿没有问为什么。她感知了一下他身上的频率——他的五条规则在体内运行，第五条规则——遮蔽规则——在持续运转。但遮蔽规则下面，他的原生规则频率正在出现一种她没见过的波动。

"你的纹路——在准备进入新规则。"

"不是准备——是在推我写。"

陈渡把右手袖子撸起来。路灯的光照在他的手臂上——五条规则从指尖延伸到小臂中段。但在第五条规则的末端——新纹路的起点——皮肤表面有一层极淡的灰色光晕。不是规则的运行状态——是规则在体内"堆积"的迹象。

他的身体在积蓄规则能量。如果他不主动写第六条规则——那些能量会在体内自行释放，导致不可控的规则输出。老周在公交总站没说的后半句话——他现在亲身体会到了。

"你知道写什么吗？"

"知道。"

陈渡环顾了一下四周。这条巷子周围没有终端机——但他不需要。赵灵儿带来的那台便携终端——金属外壳，从电视塔下面取的——就放在电动车脚踏板上。

他蹲下来，把终端机放在地上，开机。

屏幕亮了。输入框弹出。

他把手指放在键盘上。

但他没有立刻打字。他低着头看着键盘，沉默了几秒。

赵灵儿站在他旁边，没有催。

然后他打了：

```
编译者#47的所有规则——在其死亡或失去意识后——自动转移至编译者#48名下。
```

赵灵儿看到这行字的时候——她的呼吸停了一下。

"你在做什么？"

"写第六条规则。"

"把规则全部转移给我——那你怎么办？你死了之后——你的规则不会落到苏墨手里——但全部会压到我身上。"

"你不会死在我前面。"陈渡说,"你在你的第五条规则里写了'死后锁定'。我刚才想到——如果你的规则全部锁定了——那我的规则转移给你——你锁定的规则和我转移给你的规则——会相互覆盖。在系统层面产生一个谁都拿不到的空白区域。"

赵灵儿盯着屏幕上的文字。

她明白了。

陈渡写的不是"遗产"——他写的是"障碍"。他把自己死后可能被苏墨继承的所有规则——全部转移到赵灵儿的锁定规则体系里。两条规则在系统层面相互冲突——产生一个规则空隙。苏墨无法继承他的规则——因为规则全部被赵灵儿的锁定规则"卡"住了。

这是他在读#18的笔记本之后想到的。

用两条不可撤销的规则——制造一个系统无法解决的矛盾。

然后他在第六条规则下面打了一行补充：

```
转移规则生效条件——编译者#48的第五条规则处于已锁定状态时——本规则自动激活。
```

他按了回车。

屏幕的亮度闪了一下——不是普通的接受提示——是系统在更高级别上确认这条规则的逻辑。

然后屏幕显示：

```
规则已接受。
编译者#47。
当前纹路：6条。
纹路进度：指尖至前臂末端——接近肘部。
规则类型：转移型（条件触发）。
特别提示：该规则触发的目标对象——编译者#48——当前纹路状态为「狂写期」。
在狂写期内的编译者——继承规则的速度和效率将提升300%。
```

三百。

陈渡看着那个数字。

赵灵儿在狂写期继承他的规则——速度会提升三倍。六条规则会在极短的时间内全部压进她的纹路系统。他不会让那个触发条件成立——但如果它成立了——赵灵儿会在几秒内承受六条规则的同时加载。

他低头看自己的手臂——第六条规则的纹路正在成型。

位置——第五条规则的上方，接近肘部的位置。

他的纹路——从指尖出发——现在已经覆盖了整个前臂。

肘部在等他了。

他的手机震了一下。

新消息——苏墨。

只有一行字:

"你写第六条规则的速度——比我想象中快。但你犯了一个错误。"

陈渡盯着屏幕。

一条新消息接着出现:

"你把所有规则都转移给了#48——但你没给自己留后路。如果#48先死了——你的规则会直接判给系统。系统继承——比苏墨继承更糟。"

陈渡握着手机的手指收紧了。

他没有考虑过这个可能。

如果赵灵儿先死了——他的所有规则直接归系统所有——管理员终端就会拥有他全部六条规则的控制权。

他写了防苏墨的规则——但没写防系统的。

他放下手机。

赵灵儿站在他旁边——也看到了那些消息。

"他不会让我死的。"她说,"我活着——他才有机会在我狂写期结束之前收割我。我死了——他什么都没了。"

陈渡把手机收起来。

他站起来,把终端机放在电动车的脚踏板上。

巷子里很安静。远处的街道上有汽车驶过的声音,有几声狗叫,有一个醉汉在唱歌。

他抬头看了一眼夜空——看不到星星。城市的灯光照亮了低空的云层,云在缓慢移动,在黑暗中显出轮廓。

"走吧。"

他跨上电动车。

赵灵儿拧了油门。

车子向前冲去——方向——

城西防空洞。

（第036章 完）
