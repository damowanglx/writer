# 第033章 第五条规则

去劳务公司之前，赵灵儿说她要先回一趟时代广场。

"我的第四条规则写完到现在——系统日志应该更新了。我想看看规则链的记录内容。"

陈渡没有阻止她。他骑车载她到时代广场，把电动车停在路边的自行车停放区。两人坐电梯下到负一层。

停车场和昨天一样。车不多，灯光昏暗，空气里有水泥和汽车尾气的味道。

但陈渡一踏进停车场就感觉到了不对。

赵灵儿的第四条规则——那个和她第三条规则自动链接的规则链——正在以超出正常范围的强度运行。不是规则本身出问题了——是系统在强化它。

赵灵儿走到终端机前。屏幕亮着——显示的不是待机画面，是一行实时更新的数据：

```
规则链#48-A 当前运行状态：
- 链接规则数：2
- 覆盖面积：时代广场负一层（已饱和）
- 数据同步频率：实时
- 当前已记录的编译者信号数：7
```

七个编译者。

时代广场负一层的透明化规则——在这段时间内已经记录到了七个编译者的信号。不一定是七个人来过——可能是同一个人多次经过被重复记录。

赵灵儿把屏幕上的数据拍了下来。

然后她说了一句让陈渡后背发凉的话：

"我准备在这里写第五条规则。"

"现在？"

"现在。"

"你第四条规则才写了不到二十四小时。"

"我知道。但系统在等我。"

陈渡站在她身后，感知她身上的频率。她的四条规则在体内运转——第四条是最活跃的，链接到系统日志之后，她的纹路像通了电一样在持续发热。他能感觉到她的第五条规则正在纹路里"成型"——不是她已经写好了——是她的身体已经在自动准备写入。

老周说的"狂写期"——

不是她会进入。

她已经进入了。

"赵灵儿——你听我说。你现在不能写第五条。你的纹路还没稳定—"

"如果我停下来——我的纹路会自己写。"

陈渡的声音卡在喉咙里。

"什么？"

赵灵儿把终端机的屏幕转向他。屏幕下方有一行极小的字——不是系统提示——是她自己写的备忘录：

"第4条规则写入后11小时——右臂纹路在小臂处出现自主频率。未接触终端机的情况下，纹路在尝试自行生成规则文本。"

"它自己在写？"

"对。我睡觉的时候——纹路在自动激活。我没有碰终端机——但我的手臂在夜间亮了三次。每次持续大约两分钟。"

她的声音没有恐惧。甚至没有紧张。

"所以我必须现在写——在我的纹路自己写出一条不受我控制的规则之前。"

陈渡的手按在终端机的边缘。金属表面的温度和平时不同——比正常温度高一些。赵灵儿的频率太高了——高到终端机都在发热。

"写什么？"

赵灵儿看着他，眼睛里有光——不是泪光，是纹路在眼眶周围轻微发亮导致的生物荧光。不是人类眼睛该有的光。是规则能量从体内渗透到了眼球周围的组织。

"写一条我自己的'保管'规则。覆盖我的前四条规则——让它们不能在我死之后被任何人继承。"

陈渡愣住了。

她不是在写攻击规则。不是在写防御规则。她是在写遗嘱。

她在保护自己死后——她的规则不会落入苏墨手中。

"你知道你在说什么？"

"知道。"

"你的前四条规则——覆盖之后——就变成不可继承的。如果苏墨杀了你——他拿不到你的任何一条规则。"

"对。"

"那你拿什么保护你自己？"

"第五条规则不是保护我的——是保护我死了之后的事。"

赵灵儿把手放到键盘上。她的手指——陈渡注意到——在微微发抖。不是因为恐惧。是因为她的纹路在写规则之前就已经开始释放能量了。她的身体在告诉她自己——这是她能自主写最后一条规则的机会。

狂写期。

她还有最后一次自主选择写什么规则的能力。

之后——就是纹路自己写。系统接管。

赵灵儿打了：

```
编译者#48的所有规则——在编译者死亡或失去意识时，自动锁定。
锁定状态：不可被任何编译者读取、覆盖、继承或转移。
锁定持续时间：永久。
```

她按了回车。

屏幕闪了一下——比平时更久。三秒。四秒。五秒。

然后屏幕显示：

```
规则已接受。
编译者#48。
当前纹路：5条。
纹路进度：指尖至前臂中段偏上。
规则类型：元规则前置型（不可逆）。
特别提示：该规则写入后——编译者#48的所有现有及未来规则，将在触发条件下自动锁定。

警告：该规则不可删除，不可覆盖，不可撤销。
```

赵灵儿读完了最后一行字。

"不可撤销。"

她把手从键盘上放下来。

然后她低头看自己的手臂——第五条规则的纹路在皮肤下成型。

不是黑色。

不是银色。

是一种介于两者之间的颜色——像灰色，但表面浮着一层极淡的银色光泽。

她看了很久。

然后她拉下袖子，转身。

"走吧——去劳务公司。见于莉。"

她先走出去。

陈渡站在原地。他看着终端机上的"不可撤销"四个字——然后在心里补上了老周没说完的那句话：

"如果你进入了狂写期——你就已经不在'人'的范畴了。你是一个正在编译自己终端的宿主。"

（第033章 完）
