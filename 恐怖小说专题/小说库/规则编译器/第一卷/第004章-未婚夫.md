# 第一卷：《编译者》

## 第004章 未婚夫

时代广场旁边有一条老巷子。巷子尽头是一栋旧居民楼，一楼夹在五金店和废品站之间，开着一家茶馆——白天不开门，晚上才亮灯。

陈渡到的时候林昭已经在了。不是昨天便利店门口那种随意的坐姿——她坐在靠里的卡座，面前放着一壶铁观音，两个杯子都倒好了。铁观音的香气在空调冷气里凝成一层薄薄的雾。

他坐下。她没说话。他把茶杯拿起来喝了一口。烫的。放了回去。

"苏墨。三十五岁。前算法工程师。"

她开头了。

"你昨天晚上感知过你那条规则吗？"

陈渡愣了一下。感知——这个词她昨天就在用。

"闭上眼睛。"

他闭上了。

"找到你写的那条规则。不是路口那条。你身上那一条。"

他找了大概二十秒才感觉到。胸口偏右的位置——一个微弱的震动。像心跳，但频率不对，节奏不对。他在意识里把它拉近了一些。那行字浮现出来：

```
此路口禁止鸣笛。
编译者#47。
状态：已覆盖。
```

覆盖。不是删除。系统还认这是他写的，但已经不在生效状态了。像程序里被注释掉的代码——还在那里，但什么都不干。

"所有规则在系统里都有记录。包括失效的。包括被覆盖的。包括——作者已死的。"

她说完这句话，把左手袖子撸上去。

手腕内侧。一道宽约一厘米的旧疤。不是刀割的——是某种东西从内部烧穿的痕迹。疤痕边缘不整齐，像皮肤被高温气体从里面顶开过。颜色比周围的皮肤浅一个色号，在茶馆昏黄的灯光下，像一块被漂白过的印记。

"三年前。苏墨写的规则。他杀了我未婚夫。"

她说得很平静。像在陈述一个已经结案的案子。

陈渡的视线从疤痕移到她脸上。她在看自己的手腕，不是看他。

"他也是编译者。比我早一年。他教我的第一条规则——和你一样：在终端机上写，按回车，看它生效。我们那时候不知道系统是用来干什么的——以为发现了什么秘密武器，能随便改世界。我写的第一条规则是'此路段雨天减速'——幼稚的东西。他笑了我三天。"

她喝了一口茶。

"苏墨那时候还不叫苏墨。他叫苏逢春。#17。猎杀开始后他改了名字。具体什么时候——没人记得。只知道有一天，之前认识的'苏逢春'不再回消息了。#17这个编号开始出现在其他人的规则覆盖记录里。覆盖就是猎杀——他覆盖你的规则，等于在你的规则空间里开了个洞。他再用他的规则填上。你的规则被覆盖三次——你的身体会被系统标记为'规则空洞'——纹路停止生长，人也停止意识。"

"他为什么这么做？"

"因为每猎杀一个编译者，被杀者的规则记录不会消失——会保留在系统里，变成系统数据。苏墨在收集规则。他不是因为恨才猎人——他相信自己在优化系统，收集最完整的规则组合。不够好的编译者，他提前结束。够好的——他等到他们写够了再结束。"

陈渡的手指停在茶杯边缘。茶已经不烫了，但他没端起来。

"他现在多少条？"

"二十三。纹路到肩膀。他三种类型都写——物理的，行为的，认知的。他的规则覆盖范围横跨大半个城区。三年来他没有停止过。"

"他为什么没杀你？"

林昭放下茶杯。发出一声很轻的磕碰声。

"因为我还有用。系统每过一段时间就会出现新编译者。总得有人教他们基本规则，告诉他们什么能写什么不能写。他杀教规则的人——不划算。"

"谁在安排这些？"

"没有人在安排。系统自己选人——终端机在哪里出现，新人就在哪里觉醒。苏墨在等。等这些新人长成他能收割的样子。等他们的规则量值得他出手。"

她抹平袖子，遮住疤痕。

"你说他——他相信自己在做正确的事？"

"对。他不嗜血。他真诚地相信自己在优化一个不够完美的系统。他跟我面对面说过：'这个城市三十二年来积累了上百条规则，大部分是自私的，短视的，互相矛盾的。如果规则系统不够干净——城市会先坏掉。我在清理。'"

她看着陈渡的眼睛。

"他用'清理'这个词的时候——他是认真的。"

茶馆的空调开得很低。陈渡的手臂上起了一层细密的鸡皮疙瘩。不是温度的原因。

一个有信仰的杀人者。他做的每一件事都有逻辑支撑——而且他自己不认为那是问题。这才是最恐怖的部分。

"所以你教新人规则——他就让你活着？"

"我不确定他一定让我活着。我只是——"她停顿了一下，"在做我该做的事。有人知道了系统的存在却不知道怎么用——他们会死。我的未婚夫就是这么死的。他写了一条认知规则——以为只是改了一个人的记忆——结果导致了连锁裂隙。苏墨来'修复'的时候，顺便结束了。"

"你恨他吗？"

林昭没有马上回答。她端起茶杯，喝了最后一口。把杯子放回桌上，杯底磕在木桌面上，发出一声闷响。

"恨。但不是他让我恨的——是我自己。我恨他，我就还会写规则。我写规则，就在他的计划里。"

她站起来。

"你已经感知过自己的规则了。下一步——去感知这座城市。"

她往外走。走到门口的时候停下来，没有回头。

"你爸的工地事故——你确定那不是一场普通的意外？你确定——没有规则在覆盖它？"

门关上了。

陈渡坐在卡座里。茶已经彻底凉了。空调还在吹。他低头看自己的右手——那条黑线。

他从来没有想过这个角度。他爸的死——手续齐全的死亡证明，注销的公司，找不到的包工头——一切都很正常。正常到像是被安排好的。

如果规则可以覆盖现实——那有没有一种规则，覆盖了某个人的死？

他掏出手机。打开备忘录。在"爸的事——查"下面加了一行：

```
找那条规则。
```
