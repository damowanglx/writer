# 第027章 第四条规则

下午两点五十分。时代广场负一层。

陈渡到的时候，赵灵儿已经站在终端机前面了。她今天穿了一件黑色短袖——露出整条右臂。三条纹路从手腕延伸到小臂中段，黑色和银色交错。她没戴手表，没戴任何遮挡的东西。

她在展示她的纹路。

"你确定要在苏墨知道的时间和地点写？"陈渡站在她身后两米的位置。

"他在城西那条日志规则还在运行。"赵灵儿没有回头，"他随时都能感知到我们在做什么。时间地点对他没区别。对我来说——有区别。"

"什么区别？"

"我要让他在感知里看到——我在写。"她转过头来看着陈渡，"我要让他知道他锁定的目标在往前走。不是躲着写——是在他看得到的地方写。"

陈渡没有回答。他感知了一下停车场内部的规则层。赵灵儿的三条规则还在运行——第一条"禁止抢劫"，第二条双人规则，第三条透明化规则。三层覆盖在这个空间里交织，密度不算高，但边界清晰。

停车场的车不多。下午时段，商场客流少，地下一层只有零星几辆车停着。远处的电梯间传来电梯到站的提示音——叮的一声——然后安静了。

赵灵儿把手放在终端机的键盘上。

屏幕亮了。规则输入框弹出。

她的手指悬在键盘上方，停顿了大约三秒。不是犹豫——是在组织语言。

然后她打了：

```
本停车场——任何编译者在此区域内持有的规则数量，每增加一条，必须在十秒内公开其规则类型。
```

陈渡看完这行字，后背的汗毛竖了起来。

这不是一条独立的规则——它是她的第三条规则的补充。第三条规则要求编译者在停车场写规则时必须披露自己的规则数量。第四条规则把要求升级了——规则数量变化的瞬间，必须公开类型。

两条规则组合在一起——在这个停车场，任何编译者的规则变动都会暴露。

透明化的程度翻倍了。

赵灵儿按了回车。

屏幕闪了一下。但这一次——屏幕没有立刻显示"规则已接受"。屏幕变暗了，然后变亮，然后是另一行提示：

```
规则已接受。
编译者#48。
当前纹路：4条。
规则类型：认知型（信息强制公开）。
效果范围：与第三条规则叠加——覆盖时代广场负一层全层。
特别提示：该规则将自动与第三条规则形成「规则链」。
规则链效果——当两层透明化规则同时生效时，范围内编译者的规则数据将自动同步至系统日志。
```

规则链。

陈渡盯着屏幕上的那行字。这不是他之前见过的任何机制——系统明确表示这条规则会和第三条规则"形成链"。两条规则组合在一起的时候，效果不是叠加——是质变。

从"强制披露规则数量"变成"自动同步规则数据至系统日志"。

赵灵儿也看到了那行字。她没有立刻拉下袖子——她低头看着自己的手臂。第四条规则的纹路正在皮肤下成型——位置在第三条规则上方两厘米处。黑色，但边缘带着银色的细线——双人规则的残留。

她的纹路——从手腕出发，第四条规则的末端已经到达了小臂中段。

林昭说过——纹路过肘部之前，人的思维方式会开始变化。

赵灵儿的第四条规则——已经写到了小臂中段。

"你看到规则链了。"陈渡说。

"看到了。"

"你知道这意味着什么——你的第三条规则和第四条规则组合之后，所有在这个停车场写规则的编译者，他们的规则数据会自动同步给系统。"

"我知道。"

"那你——"

"我不是为了抓苏墨。"赵灵儿打断了他，声音很平静，"我是为了让系统知道我在这里。我在写。我在改变规则。如果系统在记录所有编译者的行为——那我要我的行为被记录得最清楚。"

她转过身来面对着他。停车场的灯光照在她脸上——她的表情不是狂热，不是恐惧。是平静。一种接近清醒的平静。

"你不怕被系统盯上？"

"系统已经盯上我们了。"赵灵儿说，"春柳路路口那个冲突标记——不是只标记了你的规则。它标记了所有参与竞争期的人。包括#12——苏墨。我是通过双人规则的关联感知到的——你被标记的时候，我的银色纹路也跟着亮了一下。我们在系统里已经被关联记录了。那我写规则——还有什么好怕的？"

陈渡站在她面前，伸出手，但没有碰到她。

他停住了。

赵灵儿低头看着他的手，然后抬起头看着他："你怕什么？"

"我怕你写到停不下来。"

"我已经停不下来了。"她的声音很轻，没有悲伤——只是陈述事实，"从我在这个停车场写第一条件规则那天开始——我就停不下来了。"

她拉下袖子，遮住纹路。动作很自然——像一个习惯了自己有纹路的人。不再觉得新奇，不再需要展示。

"走吧。这里没什么好看的了。"

她转身往停车场出口走。步伐不快不慢，运动鞋踩在水泥地面上的声音在空旷的空间里回响。

陈渡站在原地，看着她的背影消失在楼梯拐角。

然后他感知到一件事。

在她的第四条规则写入之后——停车场西侧那台打卡机样式的终端机——它的屏幕亮了。

不是待机画面。

是一行字。

他走过去看。

终端机的屏幕上显示着：

```
系统日志已更新。
编译者#48——规则链已建立。
系统记录开始。
```

然后下面还有一行小字：

```
同类规则链检测：编译者#12——已建立规则链数量：3条。
```

苏墨有三条规则链。

而赵灵儿——刚建立起第一条。

她写第四条规则的时间——是苏墨写第一条规则链速度的大约十分之一。

他拿出手机。翻到通讯录里一个没有备注的号码——老周上次发消息用的那个号码。

他打字："你是老周？"

等了大约三十秒。回复来了：

"是。你终于找我了。"

（第027章 完）
