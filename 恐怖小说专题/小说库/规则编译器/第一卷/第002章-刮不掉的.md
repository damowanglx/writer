# 第一卷：《编译者》

## 第002章 刮不掉的

陈渡回到出租屋的时候已经凌晨一点。

雨衣挂在门把手上，水顺着下摆滴到地板上，积了一小滩。他没开灯，直接走进卫生间，拧开水龙头，把右手食指伸到水流下面。

冷水冲了大概两分钟。他用拇指搓——那根极细的黑线还在。嵌在皮肤下面，像纹身，但比纹身浅——像有人用极细的针在表皮底层画了一道。

他找肥皂。洗手液。搓了三十秒。黑线还在。

他翻柜子找到一瓶医用酒精，倒在纸巾上用力擦指尖。皮肤擦红了，火辣辣的疼。黑线没变淡——反而在酒精浸润后更清晰了。像墨水遇水显色。

他去厨房拿漂白水。倒了一瓶盖，把指尖泡进去。

刺痛从指尖窜到手腕。他咬着牙泡了十秒。

拿出来冲洗干净。黑线还在。

他靠在灶台边上喘气。整个出租屋只有冰箱压缩机的声音。右手食指指尖有股漂白水的呛味，皮肤被酒精和漂白水轮番烧过之后，皱起来一小片。

他终于抬头看了镜子。

镜子里那张二十四岁的脸被卫生间惨白的灯照着——眼圈发黑，嘴唇干裂，头发被雨淋成一绺一绺贴在额头上。但他没看自己。他在看指尖。那行符号——在镜子里被翻转了——他看得懂。不是中文，不是英文。是一种他从未学过但能直接理解的文字。像阅读底层代码。

他读出来了：

```
此路口禁止鸣笛。
```

他写的。在指尖上。不是伤口，不是疤痕——是代码刻在皮肤上了。

他关了灯。卫生间陷入完全的黑暗。他没动。站在黑暗里，闻着自己指尖上漂白水的味道。那行字是他写的——但刻在他身上之后，它就跟他分开了。它是一条独立的规则。在城市某个路口运行着。而他只是一个载体。

他在黑暗里站了大约两分钟。然后走出去，在床边坐下。雨衣还挂在门把手上滴水——滴答，滴答。节奏稳定。他低头看手机。打开地图，找到春柳路路口。然后点了附近的实时监控——有的红绿灯路口有城市监控画面。他点开一个。画面加载了几秒。凌晨一点的车流稀疏，但他看到那些车到了路口——真的没有按喇叭。不是少——是零。一秒钟的喇叭声都没有。

他锁了屏幕。把手放到膝盖上。在黑暗里坐着。

手机屏幕亮了。

一条短信。陌生号码。

没有称呼，没有问候，只有一行字：

```
第四条规则已被覆盖。编译者#12。你的规则被改了。
```

他盯着屏幕看了很久。凌晨一点二十三分。

第四条规则。也就是说#12至少写了三条规则——可能更多。至于"被覆盖"——他花了几秒才明白这个词的意思。他写的那条规则，被别人改了。不是修正，是覆盖——在同一个路口，在同一个规则空间里，叠了一条新的上去。

他穿上雨衣，下楼。电动车还在老地方。雨已经小了，变成雾一样的水汽，在路灯下飘。

骑到春柳路的时候他先听到的——是喇叭。

凌晨一点半，这个路口不该有太多车。但仅有的几辆车都在按喇叭。出租车在按。私家车在按。深夜送货车也在按。不是正常交通里偶尔一声——是失控的、集体的、停不下来的鸣笛。一辆出租车在红灯变绿之后停在原地，司机双手按在喇叭上，像忘了怎么松手。

他把电动车停在昨天站过的位置。

路口的氛围变了。他能感觉到。不是视觉上的变化——是某种更底层的改变。空气里有一种细微的震动，像低频噪音，从地面升上来，从墙体里渗出来。

他闭上眼睛。

然后他读到了：

```
此路口——鸣笛者将听到自己的鸣笛声被无限循环。
```

不是禁止。是惩罚。

范围没变——还是这个路口。但内容从"不能做"变成了"你做，然后承受后果"。而且后果不是物理的——是感知层面的。你在自己的脑子里永远循环那一声喇叭。永远。精确到每一条神经。

这是升级。更精准，更残忍。

覆盖不是删除。是在原规则上叠了一层更狠的。

他站在路边，手插在雨衣口袋里。凌晨的路口，被改写的规则正在运行——每一声喇叭都在制造一个被困在自己鸣笛声里的司机。他们自己不知道。他们只是觉得刚才那声喇叭响得特别刺耳，然后不耐烦地又按了一下。又一下。

楼上某扇窗户亮了一下又灭了——有人被吵醒了。但醒的人不知道自己为什么醒。

陈渡抬头扫了一圈街对面的楼。

四楼。一扇窗户。窗帘动了一下。

他的手机震了。同一个号码。一行新消息：

```
第一条规则写得粗糙。但框架可以。——#12
```

他骑上电动车往回走。不是回家——是在街上绕圈。送了三年的外卖，他对这座城市熟悉到能闭着眼睛走完一条街——但在这一条短信之后，这座城市不是他以为的那座了。

他的纹路，他的规则，他以为是他和那台终端机之间的秘密交易——现在第三个人看见了，评论了，改写了。

这个城市里还有其他编译者。

而且他们比他强。至少#12比他强。

他的右手握车把的时候，食指上的那根黑线轻轻硌了一下握把的橡胶。不是疼。是提醒。那行字——此路口禁止鸣笛——现在变成了别人手里的一层基底，在那上面盖了更重的东西。

他不知道自己是在跟什么打交道。

但他知道一件事：#12的第四条规则已经生效了。#12这个人，至少写了四条规则。每条规则在身体上留下一行纹路。

四行。

而他才一行。

他回到出租屋楼下，锁了车，站在楼道里掏出手机。那个陌生号码没有归属地显示。他没有拨回去。他把手机收起来，上楼，打开门，脱雨衣的时候看到镜子里自己的脸。

他低头看右手食指。那行符号还在。

这次他没试着擦掉。

（第002章 完）
