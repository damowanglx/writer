# 第024章 第三条规则

下午两点四十五分。时代广场负一层。

停车场里和平时一样——车不多，灯光昏暗，空气里混着水泥和汽车尾气的味道。消防通道指示灯在墙角泛着绿色的光。远处偶尔传来车辆驶过减速带时减震弹簧的闷响。

赵灵儿已经在了。她蹲在那台打卡机样式的终端机前面，卫衣帽子翻起来盖住头，手指放在键盘上——但没有按。她在等三点。

陈渡从楼梯口走过来。

他没有走到她旁边——停在距离大约五米的位置，靠在墙柱上。他知道今天不是他来陪她——是她来做她要做的事。他在旁边看着就好。

赵灵儿没有回头。她感知到他的信号了——遮蔽规则虽然模糊了她的信号，但共享规则的存在在两个方向都保持了一条微弱的通道。

她把手放到键盘上。

但她的手指没有立刻开始打字——她看着屏幕，停顿了大约十秒。

不是犹豫。

是在思考怎么表达。

陈渡从未见过她写规则之前停顿这么长时间。她写第一条规则的时候，直接打了回车。第二条——在斑马线前——用了大约三十秒感知现场，然后直接写。这一次——她在想措辞。

她打了几个字，删掉。又打，又删。

第四遍的时候——她停下来。把手指从键盘上移开。转过身来看着他。

"我在想——写一条规则，让这台终端机不接受苏墨的输入。"

陈渡没有说话。他等她自己把话说完。

"但我想过了——那是防御。不是进攻。防御性规则——林昭说太多防御规则会让人缩在原地。我不想缩。所以我换了一个方向。"

她转回去。重新把手放在键盘上。

这一次她没有停顿。

她打了：

```
此层停车场——任何编译者在此范围内写规则时，必须同时披露自己当前持有规则的数量。
```

陈渡看到这行字的时候，他先是一愣——然后他明白了。

不是保护她自己。不是攻击苏墨。

是一条透明化规则。在这个停车场写规则的任何编译者——包括苏墨——必须公开自己的规则数量。你不能隐藏你的实力。你不能伪装成比你弱的样子。

赵灵儿按了回车。

终端机的屏幕亮了一下——比平时更亮的白色光，像在确认这条规则的优先级。

屏幕显示：

```
规则已接受。
编译者#48。
当前纹路：3条。
规则类型：认知型（强制透明）。
效果范围：时代广场负一层全层。
特别提示：该规则将影响所有在本区域执行规则操作的编译者——包括原作者。
```

包括原作者。

赵灵儿低头看自己的手臂。

她把袖子撸上去——手腕到小臂之间，两条黑色纹路下方，第三条规则正在成型。黑色的线条从皮肤内部浮现，但在线条中间——混杂着极细的银色丝线。

银色的丝线和黑色纹路混在一起，像两种颜色的墨水在同一支笔里混合着写出来的字。

"银色的——"

"双人规则的残留。"赵灵儿说,"我们写那条共享规则的时候——我的纹路里混进了你的频率。黑色的是我的。银色的是你的一部分。"

陈渡蹲下来。

她的第三条规则——和他有关的证据。不是绑定——是他们在同一台终端机上同时按回车时，两个人的规则能量在终端机内部短暂混合，各自带走了一部分对方的规则频率。

"你的纹路——以后每一条都会有银色。"

"嗯。"赵灵儿把袖子放下来。"所以你看——我现在写什么规则，你都能感知到。你写什么，我也能感知到一点。不是坏事。"

陈渡没有说话。他在感知周围。

停车场的频率变了。

不是很大的变化——是细微的调整。苏墨的规则信号——如果他在附近——会被这条"透明化规则"强制披露。他必须公开他的规则数量。不能隐藏。

但如果苏墨不在这个区域的覆盖范围内——这条规则影响不了他。

他感知了整个负一层。没有苏墨的频率。

赵灵儿站起来，拍了一下裤子膝盖上的灰。

"他不在。"

"你怎么知道？"

"因为如果他在——刚才我写规则的时候，他一定会趁我纹路暴露的那一刻做什么。他没出现——说明他不在这个覆盖区内。"

陈渡看着终端机的屏幕。苏墨凌晨两点发的定位——时代广场负一层。但他不在这里。他不是来狙击赵灵儿的——他是来告诉他们：他知道他们下一步要在哪里做什么。

他在展示他的情报覆盖范围。

"他凌晨发定位的时候——可能就不在这里。"陈渡说,"他是提前发的。像是——通知。告诉我们在什么地方做什么事会被他看到。"

赵灵儿没有接话。她把终端机屏幕上的操作记录拍了下来，然后把手机收进口袋。

停车场西侧——就是赵灵儿第一次发现终端机的那个方向——传来脚步声。不急，节奏均匀。一个人的脚步在空旷的地下空间里显得格外清晰。

脚步声在距离他们大约十来米的位置停住了。

不是苏墨。

是一个女人。

大约三十出头，短头发，穿着一件灰色防晒外套。陈渡在孟轲书店见过她——徐盈。#33。五条规则。她站在两根柱子之间的阴影里，手插在口袋里。

"苏墨刚才在城西。"徐盈说,"他在#31的原节点附近。我正在看那个位置——他应该感知到你们在这里写规则了——但他没过来。"

"你怎么知道他在城西？"

"因为我的规则覆盖区覆盖了城西一部分。刚才有一条新规则在那里写入——规则密度高到把我的感知都弹开了。"

徐盈停顿了一下:"他也在写新规则。"

赵灵儿的手指从键盘上移开了。停车场里安静了几秒。

"他在城西——同一个时间段——也在写规则。"赵灵儿看着陈渡,"他在对标我们的节奏。"

陈渡想起苏墨的纸条:"你们的规则结构已记录。"

他说的不只是赵灵儿。

他说的是"你们"。

双人规则把两个编译者的规则频率锁在一起。苏墨读到了赵灵儿的规则结构——也读到了他的。两个结构在系统里相互关联。赵灵儿写规则的时候——他的数据也会暴露一部分。

陈渡的遮蔽规则只遮蔽了他的信号——但没有遮蔽双人规则的关联通道。

赵灵儿写第三条规则——她的纹路暴露了——通过双人规则的关联——陈渡的规则数据也被部分暴露了。

"他今天早上就知道了你要写规则。"陈渡说,"他凌晨发的定位——不是威胁——是确认。他在确认你会不会在他说的地方写，会不会在他预测的时间内写。"

赵灵儿看着自己的右手腕。三条纹路静驻在那里，但银色线的边缘正在微微发亮：

"他在用我——当测量工具。"

不是"样本"。是"测量工具"。苏墨不只是在收集规则类型——他在测试新编译者的行为模式。他发一个定位，看她会不会来。他在她写了规则之后感知规则的变化——像在实验中读取数据。

她在她的节点写规则——苏墨在城西同步写规则——然后观察两条规则之间有没有共振，有没有冲突，有没有关联。他在测量她和陈渡之间的双人规则连接强度。

陈渡握紧了自己的右手腕。

遮蔽规则还在运行——但他的规则数据已经被读到了。遮蔽规则挡不住共享规则的暴露通道。

他们被测量了一次。

而他不知道测量结果是什么。

（第024章 完）
