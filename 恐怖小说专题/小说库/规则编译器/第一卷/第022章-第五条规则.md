# 第022章 第五条规则

深夜。城西那栋旧居民楼的设备层。

陈渡一个人来的。没有告诉赵灵儿，没有告诉林昭——只给孟轲发了一条消息："我要用一下你店里那本老周的手记。"

孟轲没回。但他经过书店门口的时候，那本书被放在门口的信箱上面，用一根黑色皮筋捆着。

陈渡在地下设备层的铁门前停下来，解下皮筋，翻了翻老周的记录。找到他需要的那一页——关于"规则信号遮蔽"的笔记。

老周写得很克制，没有多余的解释，像工程师写操作手册：

"规则信号遮蔽原理：所有写入的规则都会向外辐射特定频率。频率包含编译者的纹路特征。特征无法消除——但可以用另一种频率覆盖它。像把收音机调到同一个频率的白噪音频道。信号还在——但接收端无法从噪音中提取有效信息。"

"方法：写一条常驻规则——不作用于现实——只作用于你的规则信号本身。范围越小越好。效果越专一越好。不要试图覆盖整个城市的信号——只覆盖你自己。"

"警告——信号遮蔽规则会因为持续运行而加速纹路向心脏前进。使用前确认你已经准备好承担这个代价。"

陈渡把书放在地上。用手机手电筒照着那几行字，又读了一遍。

"加速纹路前进。"

他抬起右手。在手机电筒的光下，四条黑色纹路从指尖延伸到小臂内侧。第四条规则的末端——覆盖#31规则时留下——在小臂内侧的起点处停住了。

第五条规则——如果再往前写——会正式进入小臂。

林昭说过，纹路过了肘部，"你就停不下来了"——纹路会改变你的思维方式。但林昭说的不是小臂。小臂还安全。小臂到肘部之间还有一段距离。

但陈渡不确定自己的纹路会在哪里开始"改变思维方式"。

他用手机的光照了一下自己的右上臂——没有纹路。手腕以下全部被覆盖了。但从这里到肘部——大约还有十五条规则的距离。他现在有四条。

第十五条——纹路到肘部。然后开始变化。

他在心里给自己画了一条线。

他先碰了一下那台金属外壳的终端机。屏幕亮了——显示的是那天的节点状态列表：

```
时代广场 | 信号强度：71% | 延迟：0.4s
春柳路   | 信号强度：38% | 延迟：1.3s
本站     | 信号强度：100%| 延迟：0.0s
旧法院   | 信号强度：--%  | 延迟：——(节点离线)
```

旧法院——从"休眠"变成了"离线"。有人在那之后碰过旧法院的终端机。

他把手指收回来。

终端机上有一种频率——不是规则的频率——是他手腕上纹路的频率。终端机在自动记录他的数据。苏墨说过他能读到规则结构——如果终端机能记录，那他可能确实有办法读取。

那么——老周说的遮蔽规则——是有效的。

他用指尖触碰终端机的输入界面。

屏幕上跳出规则输入框。

陈渡的手指悬在键盘上方。

他在想一件事。老周说"范围越小越好"。但他的目的是遮蔽他和赵灵儿的双人规则信号——不是只遮蔽他自己。遮蔽两个人的信号——需要比单人的范围大。范围大了，代价会不会翻倍？

他的纹路在手腕上亮了一下——不是主动亮的，是在他思考要不要写规则的时候，身体自动进入了"预热"状态。

他深吸一口气。然后把手指放在键盘上。

打字。

```
编译者#47和#48的双人规则信号——不可被任何编译者读取。
范围：二人所在节点的规则覆盖区域。
```

不可以被任何"编译者"读取——不是不可被"系统"读取。的区别是很微妙的。系统可以知道他们在哪——但编译者不能。

苏墨是编译者。他不是系统。

这意味着苏墨读不到他们的信号——但系统知道他们在哪。

陈渡按了回车。

屏幕闪了一下。

然后——什么都没发生。

不是"规则已接受"。不是"规则已生效"。屏幕闪了一下之后恢复了之前的节点状态列表——像是拒绝了他的输入。

他皱起眉，重新用手指触碰屏幕。

屏幕下方弹出一个提示框：

```
规则格式正确。
当前纹路容量不足。
建议：先增加规则位再试。
```

纹路容量不足。

他的身体——当前四条规则——已经满了。不能承载第五条规则。

但林昭有十七条。苏墨有二十三条。他们怎么做到的？

他想到了方诚说的话——"所有权的转移"。规则在位，但可以转移到别人名下，腾出位置。

他没有规则可以转移。他的规则是自己的。

但他有一条被覆盖的规则。

"此路口禁止鸣笛"——他的第一条规则。被苏墨覆盖了。在系统里，这条规则的状态是"已覆盖"——但依然是他在持有。占着一个规则位。

被覆盖的规则——占用规则位吗？

他走出设备层，在楼梯间拨了林昭的电话。响了五声。接了。

"你在哪？"

"城西。设备层。"

"——你是不是又在终端机前面？"

"问你一件事——被覆盖的规则算不算占用纹路容量？"

电话那头沉默了几秒。只有电流声。

"算。"

"那被删除的规则呢？"

"……你知道了？"

"知道什么？"

林昭的声音变得很低:"在系统里——覆盖不算释放容量。删除才算。你把一条规则从系统里真正移除——那条件规则的'占用位'才会空出来。但删除规则——必须和原规则同级别或更高级别。你做不到。整个城里只有一个人能做到。"

"苏墨。"

"对。他能删除规则——所以他能一直写新规则。他猎杀——不光是收集——也是清空自己的容量。"

陈渡靠在墙边。地下设备层的灯管在他头顶发出极其微弱的嗡嗡声，频率和终端机的脉冲几乎一致。

"如果我的第一条规则被覆盖了——但覆盖我的那个人，他的规则也是占用容量的？"

"对。覆盖者和被覆盖者的规则都是'在位'的。"

"那如果我把那条覆盖的规则——反过来覆盖呢？"

林昭没有立刻回答。电话里传来打火机的声音——她在抽烟。

"理论上可以。但覆盖苏墨的规则——需要你的规则强度比他高。你的规则比他新，比他弱。你做不到。"

"那如果我不覆盖他——我覆盖原来的规则呢？"

"什么意思？"

"我的第一条规则被他覆盖了——但原规则还在系统里。如果我在原规则上嵌套一条新规则——绕过他的覆盖层——让系统认为我的原规则被我自己升级了——"

林昭打断了他:"你在说的不是规则编写——是在找系统的漏洞。"

"有漏洞吗？"

"……我不知道。但你如果成功了——你会知道——因为你的纹路会同时反映出两条规则的状态变化。"

陈渡挂了电话。

他站到终端机前，重新打开输入界面。

他没用老周说的遮蔽规则——暂时放弃了。他在编辑框里输入了另一行文字：

```
编译者#47的第一条规则——其覆盖状态被重新评估。覆盖者#12的覆盖层级——降为次要级。原规则恢复主要级。
```

不是覆盖。不是删除——是一行"重新评估"的指令。不是在规则层面打架——是在系统层面申请仲裁。

他按了回车。

这一次——屏幕没有拒绝。

进度条走了三秒。

然后屏幕显示：

```
规则覆盖层级已调整。
编译者#47第一条规则——状态从「已覆盖」变更为「竞争期」。
编译者#12的覆盖层级——调整为次要级。
竞争期将持续：72小时。
在此期间，两条规则的强度将实时对比。强度更高者胜出。
```

竞争期。

七天。两条规则同时生效。谁的强度高——谁赢。

陈渡低头看自己的手腕。

原来的四条纹路还在。但在第一条规则的旧位置上——那根最开始出现的、被覆盖后变成灰色的纹路——重新亮了起来。不是黑色。是一种介于灰色和黑色之间的颜色。浅墨色。

纹路容量增加了。从四条变成了五条。

他重新在输入框中输入遮蔽规则——这次只改了范围：

```
规则不可被任何编译者读取。
范围：编译者#47当前所在位置半径三米内。
```

很小的范围。几乎只覆盖他一个人。

他按了回车。

屏幕这一次完整地显示了接受提示：

```
规则已接受。
编译者#47。
当前纹路：5条。
纹路进度：指尖至小臂中段。
新增效果：规则信号遮蔽。
距离肘部剩余距离：约14cm。
```

他看到了最后那一行。

"距离肘部剩余距离：约14cm。"

系统在追踪他的纹路进度——精确到厘米。他之前不知道系统会这样报告。这意味着系统在记录每一毫米的移动。

他把袖子放下来。遮蔽规则已经生效了——他能感觉到自己的规则信号被一层白噪音一样的东西包住了。不是消失——是被模糊了。

他的手机震了一下。

赵灵儿: "你的信号——我刚才读不到了。你在哪？"

他没有回复。

他走出设备层。爬上楼梯。推开单元门。凌晨三点的街道上空无一人。路灯照在潮湿的地面上，反射出一片模糊的光。

他抬头看了一眼夜空——没有星星。城市的灯光把夜空染成一种浑浊的橙红色。

十四厘米。从指尖到心脏的距离。他现在走了大约五分之一。

剩下的路——他会一步一步走完。

（第022章 完）
