# 第025章 信号

从时代广场出来已经是傍晚。

陈渡没有直接回书店。他骑着电动车在城里绕了一大圈——从时代广场到春柳路，从春柳路到城西他父亲的工地，从工地到劳务公司。王德发的楼还在。规则被覆盖之后的这段时间里，劳务公司楼上的电视机修好了，灯照常亮着。

他停在马路对面看了一会儿。二楼的窗户开着——有人在里面走动。王德发还在。但#31——于莉——她不在了。她的保护规则被他覆盖之后，她退出了这个链条。

他拧了油门继续走。

骑到春柳路路口的时候——他第一次写规则的地方——他捏了刹车。

路口和以前不一样了。

不。不是路口变了——是他感知到了不同的东西。

"此路口禁止鸣笛"——他写的那条原规则——重新出现在感知中。不是覆盖状态——是"竞争期"状态。系统在执行72小时的竞争。他的原规则和#12的覆盖规则同时生效。喇叭声在路口很少——但不是全部消失——偶尔有短促的按喇叭声，然后马上停止——像司机自己也觉得不该按。

两条规则在同时作用。一条禁止，一条惩罚。两个编译者的规则在一个路口拉锯。

他站在路边感知了大约五分钟。

然后他从口袋里摸出手机——翻到苏墨的号码。上次的短信还在。

他打了一行字:"你凌晨发的定位——是想测双人规则的数据？"

发送。

等了一分钟。没有回复。他把手机放回口袋。

手机在口袋里震了一下——不是电话，是消息提示音。他拿出来。

苏墨:"不是想测——是已经测完了。分享一个小知识:双人规则会让两个人的纹路在系统里被标注为'同组'——但双人规则双方写的新规则，会产生交叉共振。共振强度超出我的预期。"

下面附了一张图——不是照片，是某种类似频谱图的截图。横轴是频率，纵轴是强度。两个波峰在图上接近的位置几乎重叠，形成了一个更高的峰。

陈渡盯着那张图。

"交叉共振"——赵灵儿写规则的时候，他的纹路也跟着亮了一下。不是误差——是双人规则的自然联动。系统把他们的规则频率合并计算了。

苏墨的新消息又来了:"你现在感知不到我在哪里。但你每次和#48同步写规则的时候——我都能感知到。不是因为你弱——是因为你们的信号太亮。双人规则像一对绑在一起的手电筒。黑暗中，一个人打手电筒不容易被发现。两个人——亮一倍。"

陈渡打字:"你到底想要什么？"

发送。这次等待的时间更长。

然后苏墨回了:"我想要你走到能和我对话的位置。不是现在的你——你现在的规则太浅。太基础。但你有潜力。我在等。"

"等我写够你收割的量。"

"不。等你写够——你就不会说'收割'这个词了。你会理解我在做什么。"

陈渡盯着这句话看了很久。路口红灯变了绿灯，后面的车按了一声喇叭——很短促，然后停了——竞争期的规则生效了。

他收起手机，拧了油门。没有回消息。

他骑回孟轲书店的时候天已经全黑了。书店的灯亮着——暖黄色的光从窗户透出来，照在门口的槐树根上。赵灵儿不在——她回出租屋了。林昭在内间沙发上睡着了——手臂上的规则擦伤还在恢复，但在她自己的规则作用下，灰色的光晕已经褪了一半。

孟轲在收银台后面看书。看到他进来，合上书:

"城西那边——苏墨写的新规则，我在感知中读到了。"

"什么内容？"

"不是攻击性的——是一条记录规则。内容是'此区域所有规则写入行为——自动记录并归档至系统日志。'"

陈渡站在收银台前面。书店门口的风铃在晚风中发出几声低沉的铜响。他听懂了那条规则的意思。

苏墨在城西写的不是攻击规则——是一条日志规则。把城西区域所有编译者的写规则行为自动记录到系统日志。那样的话——他不用自己在现场守着——系统会替他记录。

他抬起头看着孟轲:"他在构建基础设施。不是猎杀——是监控。他在城西建了一个监控节点。"

"嗯。"

"那他之前猎杀贺冬——也是在测试监控节点是否有效。"

"嗯。"

陈渡把手撑在收银台边缘，手指在木质台面上轻轻敲了两下:"他在把整座城市改造成——他能实时监控所有编译者的环境。"

孟轲没有说话。他把书翻开到折角的那一页，沉默了一会儿，然后说:"上次你说——苏墨在构建一个操作系统。我现在觉得你说对了。"

陈渡垂下眼睛。店内间传来林昭平稳的呼吸声。

"你饿不饿？"孟轲合上书站起来,"隔壁还有一家面馆开着。我请客。"

他们坐在面馆门口的塑料桌旁。两碗牛肉面，加蛋。热气在路灯下升腾，辣油的香味随风飘散。

孟轲用筷子挑开面条上的葱花:"老周那本书——你看了多少？"

"前三分之一左右。"

"那你还没看到关键部分。"

"关键在哪？"

"老周的记录里有一个模式——所有被猎杀的编译者，在死之前的一到两周，都会有一次'规则狂写'——突然写很多规则，比平时快，比平时多。像是在被某种东西推着走。他说那是——'编译前兆'。"

孟轲夹起一块牛肉，在汤里涮了一下，然后放进嘴里，嚼完，咽下去。

"老周自己在记录里写——他已经停写规则三年了。但他能感觉到，如果有一天他开始写了——大概也会进入那种状态。所以他停着。"

陈渡的手停在筷子上面。

"你觉得我在接近那种状态？"

"我不是在说你。"孟轲说,"我是在说#48。"

陈渡的筷子落在碗沿上，发出一声清脆的碰撞。

"她的纹路比你快。三条规则——一周多。而且她不犹豫。犹豫是人体内的刹车——她没装那个零件。如果她继续以这个速度写规则——她会在你之前到达那个'狂写期'。"

陈渡的手从筷子上垂下来。碗里的面还冒着热气，但他已经不想吃了。

"你为什么不直接跟她说？"

"我说了。今天下午——她来书店放东西的时候。她说——'我知道。但停不下来。'"

面馆的电视上放着地方新闻——明天城西部分区域要停电检修。字幕在屏幕底部滚动，没有人注意到这条新闻下面的另一行字——只有陈渡看到了。

电视塔的灯光会在明晚进行年度检修。届时电视信号可能中断。

年度检修。但电视塔不是供电视信号的——它有一个地下楼层，藏着四台终端机。

他站起来。

"面我吃完了——谢谢老板。"

陈渡骑上电动车，在夜色中穿过老城区的街道。路灯一盏一盏从头顶掠过，把他的影子在地面上拉长、缩短、再拉长。

他没有回书店。没有回出租屋。

他骑向城西的方向——不是劳务公司——是城西的边界线。苏墨写日志规则的位置。他想亲眼看看那条规则是怎么运行的。

骑了大约二十分钟，他在一条老路的路口停下来。这里是城西老工业区的边缘。周围几乎没有路灯，只有远处工厂废弃的烟囱在夜空里露出一个黑色的剪影。

他闭上眼睛。

感知。

苏墨的日志规则像一层半透明的薄膜——覆盖了大约两平方公里的范围。薄膜的质地很薄，但密度很高——不是普通规则那种"阻挡"的结构——是"记录"结构。

他伸手感知了一下。信息流入薄膜——进入系统。没有返回。单向的。

一条单向监控规则。只进不出。

他站了几分钟。夜风吹过废弃工厂的铁丝网，发出细碎的声响。

然后他的手机震了。

新消息——苏墨。

他低头看。显示屏的蓝光照亮了他的脸。

"第二条消息。"

"我看到你在成长。继续。"

和你写第四条规则的时候我说的不一样——第四条规则我评价你'有勇气'。现在我看到你不只有勇气——你有结构思维。你在理解系统怎么工作。这比会写规则更重要。"

"城西的日志规则——你感知到了。对。我在建基础设施。你觉得我在监控别人——其实我在等你来碰它。你碰了。我知道你在哪。"

"不要停。"

陈渡看完消息。

他没有回复。

他把手机收起来，骑上电动车，调头往回骑。晚风把他的外套吹得鼓起来，在身后像一面晃动的旗。

他骑了大约五分钟，手机又震了一下。

他以为是苏墨的第三条消息。

但不是。

是赵灵儿发的一段语音。

他点开。听。

背景音里有风声和她走路时鞋子踩在地面上的声音。她的声音很轻，带着一种接近清醒的平静:

"我刚在出租屋里想——他一直在说'继续'。他不是在鼓励你。他是在推你写规则——推到你的纹路足够接近心脏。然后在你最膨胀的那一刻——收走。"

语音播放完了。

陈渡把车停在路边。发动机在脚下微微震动。

他看着前方的路灯。灯罩里有几只飞虫在绕着光源打转。

然后他又看了一遍苏墨最新的那条消息——

"我在等你来碰它。"

不是"如果你来碰"。

是"等你来碰"。

他预测到了他会来。

那座城市的规则网络，织得比他想象中更深。它并不均匀，也不公平。但它有自己的逻辑——通过一条又一条规则的嵌套、覆盖、记录、归档，系统正在形成一种接近于"意识"的东西。

不是人的意识。

是规则本身的意识。

他拧了油门。电动车向前冲去，进入路灯照亮的区域，然后穿过去，进入下一段黑暗。

他知道自己在走向什么。

他也在等，等自己能走到足够近的时候。

（第025章 完）
