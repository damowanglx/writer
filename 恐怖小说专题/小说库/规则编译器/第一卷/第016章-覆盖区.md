# 第016章 覆盖区

第二天清晨。陈渡在出租屋里被手机的震动吵醒。赵灵儿发来一条消息：

"我又去了那栋楼。终端机上的信号列表更新了——春柳路那台的信号强度从42%降到了39%。旧法院从休眠变成了'离线'。有人在我们之后碰过旧法院的终端机。"

陈渡坐起来。旧法院那台最大的终端机——管理员终端——从"休眠"变成了"离线"。休眠是待机的意思。离线的意思是被人操作过之后进入了某种锁定状态。

他洗漱用了不到三分钟，然后骑车出门。没有早餐。路过春柳路便利店的时候买了瓶水，三块钱，拧开喝了一口，在脑子里把昨晚的事过了一遍。

他们发现了一台能联网的终端机。他们写了一条件双人规则。有人在暗中监视他们——一个身上裹着多层规则的人。现在旧法院的管理员终端状态变了。

三件事之间有没有关联？

他到赵灵儿说的地点时，看见她蹲在那栋旧居民楼门口的台阶上，手里捧着一杯豆浆，膝盖上摊着一个打开的本子。本子上画着一张手绘的地图——城区的街道、建筑、节点位置都用不同颜色的笔画了出来。

"你还画画？"

"只有地图。"赵灵儿把本子转过来给他看，"我昨天晚上睡不着，把感知到的所有信号位置都标出来了。"

地图上有十三个点。赵灵儿用红笔圈了四个最大的一圈，旁边写着"时代广场""春柳路""我们这栋""旧法院"。另外九个点她用蓝色铅笔画了虚线圆圈——那些是她隐约感知到但不完全确定的信号位置。

"九个不确定的信号源。分散在城西和城北——正好是林昭说的钢厂和防空洞的方向。"

陈渡蹲下来，仔细看那张地图。时代的信号点分布有明显的规律——它们不是随机散落的。四个信号点——时代广场、春柳路、旧法院和他们刚发现的这栋楼——形成了一个近似四边形的形状。另外九个点在这个四边形之外，有的远在城郊。

"你发现没有——"赵灵儿用圆珠笔尖点着四边形中心的位置,"这里。"

圆心位置。

标注着两个字——"电视塔"。

城市广播电视台的发射塔，坐落在城东老城区。高约一百八十米，是老城区最高的建筑。陈渡每天都经过那附近——塔下是一个小公园，白天有老人下棋，晚上有人跳广场舞。

"电视塔。"陈渡说，"如果它是一个节点——"

"那它就是五个节点组成的中心点。"赵灵儿接上,"时代广场、春柳路、我们这栋、旧法院——四个点围成一个四边形。电视塔在四边形正中央。如果这是地图上的节点分布规律——那电视塔下面也应该有一台终端机。"

"我们去看。"

电视塔所在的小公园早上人不多。晨练的老人们已经把场地占了——太极、广场舞、遛鸟。公园不大，地面铺着红色地砖，每隔五米种着一棵法国梧桐。

陈渡和赵灵儿绕着塔基走了一圈。

塔基是混凝土浇筑的，大约八米见方，表面刷着灰色涂料。入口处有一扇铁门，上着锁。锁是新的——不锈钢挂锁，在周围所有设施都生锈的环境中显得格外突兀。

"锁是新的。"赵灵儿蹲下摸了一下锁的表面，"但是——没有钥匙孔。"

陈渡凑近看。确实没有钥匙孔。锁的正面是一块光滑的不锈钢面板，背面——什么都没有。像一块锁形的铁疙瘩，挂在门扣上。不是锁——是一种装饰。假装这里被锁着，实际上任何人都可以拧开它。

陈渡伸手握住锁。轻轻一拧——锁开了。内部没有机械结构，是一个磁吸的卡扣。

铁门后面是一条向下的楼梯。和前一栋楼那条楼梯几乎一样——窄、暗、水泥台阶。只是更长。

他们走到最下层的时候，看到的不是房间。

是一个通道。

通道大约二十米长，两米宽，两侧墙壁是粗糙的混凝土，每隔三米装着一盏应急灯，发出昏黄的、电压不稳的光。通道尽头——不是终端机。

是四台终端机。

四台间隔两米并排放置。每台的型号都不一样——一台是老式ATM那种灰白色外壳，一台是打卡机样式，一台是金属壳的，和昨晚见到的那台同款。第四台——陈渡从未见过——它的外壳是透明的。像玻璃，但比玻璃厚，表面的手感像树脂。透过外壳能看到内部的电路板——金属线路像血管一样分布，没有芯片，没有处理器——线路直接通向屏幕。

四台终端机的屏幕全部亮着。不是在待机——是在同时运行。四块屏幕上的文字不一样，但滚动的速度完全同步。像四声部合唱。

陈渡的纹路在进入通道的那一刻就开始发热。不是单条规则在响应——是所有四条规则同时在共振。

赵灵儿的第三条规则——银色的那条——也在发光。

"四台——"赵灵儿的声音在这条通道里带了回音,"同时在一个位置。这地方不是节点——这是——"

"交换站。"陈渡说。

他走到透明外壳那台终端机前面。屏幕上的文字和另外三台不一样——它显示的是一行他从未见过的字符。

```
欢迎，编译者#47。
系统检测到你的纹路已进入第二阶段。
建议：开始学习元规则编写。
```

第二阶段。元规则。

他伸手碰了一下屏幕。屏幕上的文字消失了，换成一行新的：

```
元规则定义：修改其他规则的规则。
前置条件：纹路已达4条，同步率73%，双人规则位1。
尚未解锁条件：——需要与至少3名不同编号的编译者完成规则交互。
```

三台不同编号的编译者。他交互过谁？苏墨覆盖过他的规则——那算交互。他和赵灵儿写过双人规则——算交互。还有谁？林昭？林昭没有直接和他的规则交互过，她只是引导。

还差一个。

赵灵儿也在看自己的屏幕——透明终端机在她靠近时自动切换到了她的界面：

```
欢迎，编译者#48。
系统检测到你的纹路已进入第二阶段。
建议：开始学习元规则编写。
前置条件：纹路已达3条或以上（当前：3条）——已满足。
尚未解锁条件：——需要与至少3名不同编号的编译者完成规则交互。
```

赵灵儿需要三条纹路——她已经有两条黑色加一条银色，加上今天会写的——但她也需要交互三个不同的编译者。

"电视塔不是终端机的位置——"陈渡说，"它是编译者的升级站。每个进入第二阶段的编译者——都会被引导到这里。"

"被谁引导？"

"系统。"

通道尽头的应急灯闪了一下。四台终端机的屏幕同时切换——不再是个人信息，而是同一行字：

```
警告：检测到区域内规则密度超过安全阈值。
当前覆盖区：春柳路至城西段。
建议编译者暂时远离该区域。
```

覆盖区。

从春柳路——陈渡写第一条规则的地方——到城西——那家劳务公司所在的方向。连接他第一个节点和他最近覆盖#31规则的位置。有人在这条线段上的某个位置——写了一规则，导致规则密度超过了安全阈值。

陈渡站起来。

赵灵儿合上本子。

他们同时想到了同一个人。

苏墨。

在春柳路和城西之间的某个位置，苏墨写了新规则。不是普通的规则——是一条规则密度高到被系统判定为"超过安全阈值"的规则。这意味着那条规则的强度或者覆盖范围，是正常规则的好几倍。

陈渡掏出手机。林昭的号码。响了四声——没人接。

又响了两声——接通了。

"——别来城西。"林昭的声音很急，背景音里有风声，还有什么东西在金属表面爬行的声音，"苏墨在写新规则。不是普通规则——是——"

通话断了。

陈渡看着手机屏幕。信号满格——但通话断了。不是挂断，是被某种力量切断了。

赵灵儿站在他旁边，手里握着那支圆珠笔。笔尖在不锈钢地面上轻轻点着——她在数节拍。零点五赫兹。两秒一次。

"他在用规则干扰通信。"赵灵儿说,"编译者本身不会受规则影响——但电子设备会。手机、对讲机、监控——苏墨写了一条阻断通信的规则。"

陈渡盯着手机屏幕。三秒后，手机自动重启了。开机动画播放到一半——卡住了。屏幕定格在品牌Logo上，不再前进。

赵灵儿的手机也一样。

他们站在原地。四台终端机的屏幕还亮着——它们是唯一还在正常运行的电子设备。

透明外壳的那台终端机屏幕底层，一行小字在闪烁——像在屏幕玻璃内部刻上去的：

```
规则#23已写入。作者：苏墨。
规则内容——检测中——
尚未完全生效。当前进度：67%。
建议：在规则完全生效前离开该区域。
```

第三条规则。苏墨还在写——还没写完。但如果规则完成，整条春柳路到城西的连接线都会被封锁。通信会中断。信息会静止。那些在规则覆盖范围内活动的普通编译者会被切断一切联系——像是被困在一座信息孤岛。

陈渡转身。

"去哪？"赵灵儿问。

"城西。在林昭被关进去之前。"

赵灵儿没有犹豫。她把本子塞进包里，跟着他走过通道，爬上楼梯。铁门外的阳光照在他们脸上的时候，手机同时恢复了正常——重启完成，信号重新出现。

赵灵儿的手机弹出一条微信消息。陌生号码。没有备注。

消息只有一行字：

"孟轲书店在等你。——老周"

（第016章 完）
