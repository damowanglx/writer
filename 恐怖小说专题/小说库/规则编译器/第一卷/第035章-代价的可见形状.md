# 第035章 代价的可见形状

陈渡走出劳务公司的时候，赵灵儿不在电动车上。

他站在楼门口环顾四周——她在马路对面的路灯下面，背对着他，面前站着一个穿灰色外套的人。

老周。

陈渡穿过马路走过去。老周看到他了，但没有打招呼。他在跟赵灵儿说话——声音不大，语气不像上次在公交总站那样从容，带着一种认真的、接近于叮嘱的成分。

"你第四条规则写完之后——你感知过自己的纹路走向吗？"

"感知过。"赵灵儿说,"它在向肘部方向推进——速度比以前快。写第五条规则之后，速度翻了一倍。"

"你知道这意味着什么？"

"知道。我在进入狂写期。"

老周沉默了一下。他从口袋里掏出那包白色包装的烟，抽出一根，点上。吐了一口烟，烟雾在路灯下呈现出一种浅蓝色的形状。

"你第五条写的是保管规则。"

"你感知到了？"

"全城的人都感知到了。"老周说——话里没有责备，只有疲惫,"你的第五条规则——包含了不可逆标签。系统在写入那条规则的时候，整个城区的规则网络都震了一下。不是所有人能感知到——但能感知到的人，都知道有人在写一条不可逆转的规则。"

他吸了一口烟:

"苏墨也感知到了。"

陈渡走到赵灵儿旁边。她把那台便携终端机抱在怀里——屏幕上显示着一张地图。地图上有两个闪烁的点。

"防空洞的位置——和我们现在的位置。我在算时间。"

陈渡站在路灯的光照范围内。他手里还握着那本#18的笔记本。他低头看了看——封面的边缘磨损得很厉害，纸张发黄，带着一种陈年的气味。

老周看了他一眼，目光从他手上的笔记本扫过:

"那本东西——她从#18那里拿到的。"

"于莉给我的。"

"于莉——"老周重复了一下这个名字,停顿了半秒钟,"她现在是全城最危险的编译者。不是因为她强——是因为她的规则被覆盖之后,她在系统里的'权重'降低了。系统会更快地回收她的规则。她现在能活着的每一天——都是在跟时间赛跑。"

陈渡的手握着笔记本的封皮。他想起了于莉眼睛里那根灰色的线。

"那她现在能做什么？"

"什么都做不了。"老周说,"她的规则一旦被覆盖——她的活性就归零了。她不再能写新规则。不再能感知到系统的变化。她变成了——一个普通人——带着四条死规则在身体里。"

烟在夜风中燃尽了一截。老周把烟头弹进路边的排水沟里，火星在黑暗中划出一道短暂的弧线。

"而这不是最糟的。"

"最糟的是什么？"

"最糟的是——她以后会开始忘记自己是个编译者。忘记自己写过规则。忘记自己曾经能改变世界。"老周的声音像一块铁,又冷又硬,"这是规则被覆盖之后的最大代价——不是物理损伤——是记忆的退化。系统在清除所有和'写过规则'有关的记忆。到你死的那天——你会连自己是谁都不记得。"

风吹过长街。路灯把四个人的影子拉得很长,在柏油路面上构成一个不规则的几何形状。

赵灵儿把终端机屏幕收起来:"那我们没有时间了。"

"什么？"陈渡问。

"如果系统在回收被覆盖者的记忆——那于莉的时间窗口在收缩。管理员终端在她完全忘记之前——必须被打开。否则——世界上唯一知道#18的笔记本位置的人——会彻底忘记它。"

她转过来看着陈渡。路灯的光在她的瞳孔里折射出一点微弱的光:

"我们今天晚上就去防空洞。"

"现在？"

"现在。趁林昭还在拦苏墨。趁苏墨还在处理城西规则冲突的后续。趁我们还知道自己是谁。"

老周没有说话。他站在路灯的灯光边缘,一半在光里,一半在阴影里。

他最后说了一句话:

"你去了——你就回不来了。管理员终端一旦被激活——你的纹路就会和系统永久绑定。你不能再过普通人的生活。你会变成系统的一部分。"

赵灵儿没有回答。她跨上电动车,启动了发动机。

陈渡站在路灯下,握着那本笔记本。他口袋里的牛皮纸信封——老周给的关于他父亲事故规则的原始记录——他还没看。

他拿出那个信封,撕开了封口。

路灯下,他抽出那几张纸。

纸上是#18写那条"施工安全豁免"规则时的原始日志。日期、时间、规则文本——全部记录在内。

他翻到第二页。

在日志的底部,有一行备注——不是#18写的,是系统自动添加的：

```
该规则的触发条件——当工地现场管理人员超过三人的情况下,安全豁免自动生效。
```

触发条件。

不是#18主动写了"安全豁免"——是系统设置了触发条件:当工地现场管理人员超过三人的时候——#18的规则自动生效。

他父亲死的那天——工地上有几个管理人员？

三个。

正好三个。

触发的那天——不是有人在规则层面刻意杀他——是系统在#18死后自动执行了一条符合触发条件的规则。他父亲的死——是系统的自动执行。

不是苏墨的猎杀。

不是#18的恶意。

是系统的机械性。

陈渡把纸放回信封。

他抬起头。路灯的光刺得他眯了一下眼睛。

"走。"他说。

他跨上电动车后座。

发动机轰鸣。车子向前冲去,冲进夜色中。

老周站在路灯下,看着他们的尾灯消失在街道尽头。

他掏出手机。发了一条消息:

"他们去了。"

收件人——没有备注。只有一个#号码。

#12。

（第035章 完）
