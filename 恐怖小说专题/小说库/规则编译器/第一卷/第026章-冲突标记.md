# 第026章 冲突标记

凌晨三点二十七分。陈渡被纹路的灼烧感惊醒。

他从出租屋的床上坐起来，右手腕内侧的第五条规则——那条遮蔽规则——正在发热。不是正常运行的发热，是被什么东西触碰之后的应激反应。

他翻身下床。拉开窗帘。

窗外一切正常。路灯亮着，街道空无一人，街对面的便利店亮着白炽灯，收银员在刷手机。

但他的纹路在持续发热。遮蔽规则在告诉他——有人在靠近它的边界。

他穿上外套，拿了钥匙，没有洗漱。出门的时候看了一眼手机——赵灵儿凌晨两点发了一条消息，他没看到：

"我的纹路亮了。银色那条——在闪。"

他没回复。他现在需要先搞清楚自己的纹路为什么在发热。

他骑着电动车沿着老城区走。纹路的温度随着方向变化——往东降低，往西升高。城西方向。他拧了油门。

骑了大约十五分钟，他在春柳路路口停下来。

就是他的第一条规则——那个被苏墨的规则覆盖后进入"竞争期"的路口。七十二小时的竞争期还剩大约四十个小时。

但他一到路口就感觉到了不对。

竞争期的规则状态——变了。

他闭上眼睛感知。路口残留的规则频率本来是一个平稳的拉锯状态——他的原规则和苏墨的覆盖规则在同一空间里以固定的频率交替。但现在——频率变了。有一层新的东西覆盖在路口上方，像一个盖子。

不。不是一个盖子——是一个标记。

他重新睁开眼睛，仔细感知那层新东西的结构。

不是编译者写的规则。

是系统写的。

他伸手触摸路口的信号灯柱。指尖触到金属的瞬间，他的感知被拉进了一条系统信息：

```
冲突标记已激活。
春柳路路口——当前规则密度：2.4倍标准值。
竞争期规则#47 vs #12——冲突强度超出正常范围。
系统裁定：该区域升级为「规则冲突测试区」。
在冲突测试区内——所有规则写入行为将受到额外监控和记录。
```

陈渡把手收回来。

冲突测试区。系统在监视他们的竞争期——因为两条规则的冲突强度太高，系统把这个路口标记成了"测试区"。在这个区域内写规则——所有的行为都会被系统更详细地记录。

这意味着什么？

如果苏墨知道这个区域被标记了——他可以在这里写一条规则，诱导陈渡回应，然后系统会完整地记录陈渡的每一次写入。

这是一个陷阱。

而苏墨——他不可能不知道。系统升级区域状态的时候，他一定也感知到了。

陈渡的手机震了一下。

他低头看——苏墨的消息：

"感觉到了吗？系统在看你。它在你的第一条规则位置上画了一个圈——'此区域值得观察'。你被系统标记了。#47。不是因为我——是因为你的规则写了之后，路口真的安静了。系统注意到了。它在追踪所有能改变现实的编译者。你是最新的一个。"

陈渡没有回复。

但他在路口站了更久。感知全开。

然后他发现了另一件事——在冲突测试区的边缘，有一层极其微弱的频率，不属于系统，不属于他的规则——是苏墨的规则频率。苏墨也在附近感知这个区域。

不是"也在"。

苏墨是第一个发现的。他在凌晨就已经来过了。

陈渡骑上电动车，调头。

骑了两条街之后，他停车，拿出手机，给赵灵儿发了条消息：

"你的银色纹路还亮吗？"

等了一会儿。赵灵儿没有回文字——直接回了一段语音。他点开听。

她的声音很清醒——不是在睡觉被吵醒，是也没睡：

"亮。灭不了。像一根灯丝通了电，关不掉。你的呢？"

"我的发热。第五条规则——遮蔽规则——从凌晨三点开始发热。"

语音发出去之后，赵灵儿秒回了一段文字：

"因为你的遮蔽规则在保护你不被系统标记——但系统已经标记了你的第一条规则。遮蔽规则在试图掩盖一个已经被系统看到的东西。它在过载。"

陈渡盯着那行字。

她在凌晨三点想通了这件事。

他没想通。他在路口站了半天才弄明白——而她在收到他消息的几秒钟之内就分析出了原因。

他打字："你怎么还没睡？"

赵灵儿回了三个字："睡不着。"

然后又是一条："我在想第四条规则要写什么。我想好了。"

"写什么？"

"明天下午三点。老地方。你来就知道。"

陈渡看着屏幕。

她选在时代广场——她的节点。在苏墨知道的地点。在这个系统已经开始标记他们的时候。

她不停。

她不会停。

他收起手机，看了一眼夜空。城市的灯光把天空染成浑浊的橙红色，看不到星星。但春柳路的方向——那个路口——在他感知的深处亮着一个微弱的信号标记。

他的第一条规则在那里。

他的一个部分被锁在那里了。

而苏墨——他知道。他来过。他感知过。他已经开始在他的棋盘上移动了。

下一步——不在春柳路。

在时代广场。

（第026章 完）
