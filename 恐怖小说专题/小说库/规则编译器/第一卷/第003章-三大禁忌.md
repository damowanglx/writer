# 第一卷：《编译者》

## 第003章 三大禁忌

第二天下午。送完午餐高峰的单，陈渡在春柳路便利店门口的台阶上坐着喝水。

手机响了。陌生号码——但不是昨晚那个。

接起来，一个女人的声音：

"陈渡。"

不是问句。她知道他名字。

"你是谁？"

"林昭。你昨晚感知到的规则变动——在我的区域里。你被#12覆盖了。"

她说话很快，但每个字都清楚，像念法律文书。

"你知道#12是谁？"

"苏墨。不要去找他。"

"你怎么知道我——"

"因为你是#47。昨晚全市所有能感知规则的人都在看你的那条规则。你是三十二年来唯一的新编号。"

三十二年。唯一。

"晚上七点。春柳路同一家便利店。来。"她挂了。

晚上七点。他去了。

天还没全黑。便利店门口的荧光灯已经亮了，在黄昏里照出一块惨白的地面。一个女人坐在门口的塑料椅上喝豆浆。二十八岁左右，灰色运动外套，扎着低马尾——不是那种在等的姿势。就是刚好坐在那里，刚好喝完了。

她看了他一眼。"坐。"

他坐到另一张塑料椅上。

林昭把豆浆杯放到一边，伸出右手。从指尖到手腕——黑色的纹路密密麻麻，像电路板的走线。不是一行一行排好的——是交错的，叠加的，有些深有些浅。有的老纹路已经被新纹路压住了，像版本迭代。

"多少条？"

"十七。"

"到什么程度才会——"

"到肘部——三十条左右会开始有生理反应。到肩——四十。到锁骨——四十五。到心脏——"她停顿了一下，"循环结束。"

她说"循环结束"四个字，跟说"豆浆三块五一杯"一个语气。

她站起来走进便利店。陈渡跟进去。

林昭走到收银台前，拿起货架上的笔，在收银台角落的便签纸上写了一行字：

```
此店今日全部免费。
```

她把纸条按在收银台上。

收银员是个四十多岁的中年男人。他低头看了一眼纸条，抬头看了看林昭——然后没有任何质疑地开始操作收银机。他把已经收进去的现金拿出来，退给正在结账的顾客。

"不好意思，今天免费。"

顾客愣了一下，接过钱，转身走了。没有多问。没有困惑。收银员也低头继续整理收银台——好像刚才发生的事完全合理。

林昭拿起纸条，当着店员的面撕了。

店员抬头看了她一眼，然后低头擦柜台。

他们走出便利店。林昭站在路灯下，从兜里掏出一包烟。抽出一根，点上。

"三件事。记住，不要打破。

"第一——不要在主节点范围内攻击。主节点——旧法院、电视塔、城南钢厂、城西防空洞。这四个地方是系统的核心支柱。你在里面写攻击性规则——系统会把你视为威胁。你不是在打规则战，你是在攻击系统本身。"

她吐了一口烟。

"第二——不要在别人的规则里嵌套你的规则。你以为你在叠加效果——实际上你在制造裂隙。两条矛盾的规则在同一区域同时生效——产生规则裂隙。裂隙不可预测：扭曲空间，反转时间，抹除存在。有一个叫方磊的编译者，在那个永远干净的巷子里嵌套了一条'此巷禁止清洁'——整条巷子消失了三天。三天后恢复，巷子中间多了一棵树。树不在原来的位置。那棵树现在还在。你去看。"

烟灰掉进灭烟器。

"第三——不要让纹路过肘。过了肘部——你就停不下来了。纹路的蔓延会改变你的思维方式。你会开始觉得规则比人重要。你每写一条规则——无论什么类型——所有编译者都能感知到。包括你在哪里写的。所以大多数编译者只写防御性规则。守在自己的地盘里，不出去，不挑衅。"

她抽完那根烟，把烟头按进灭烟器的不锈钢面板里。金属面板上已经有一圈一圈的烫痕——她不是第一次在这里抽烟。

"你昨天写的那条——'此路口禁止鸣笛'——粗糙。幼稚。范围太小，类型选错了。但框架是对的。你对规则有直觉。大多数人的第一条规则连生效都不会。"

"那我应该写什么？"

"先别写。你现在只有一条规则——已经被人覆盖了。等于零。太弱了。你需要先学会看——看清这座城市的规则网络是怎么织的。"

她拿出手机，给陈渡发了条消息。他低头看——是一段文字。

"今天晚上，从你自己家开始，向东延伸三公里。每个路口停三十秒。用你的感知去碰——看你能找到几条不在生效的旧规则。明天告诉我。"

她转身走了。没有说再见。

陈渡站在原地，看着灰色运动外套消失在巷子里。他低头看自己的右手——那根黑线稳稳地嵌在食指上。

但他的心态变了。昨天它是个意外。今天它是一张入职通知。

他骑车回出租屋。打开手机备忘录，写下今天得到的信息：

```
编译者#12 = 苏墨。最强。至少四条规则。
林昭：17条。前律师。女性。
三大禁忌：主节点攻击 | 嵌套规则 | 过肘
规则写在身上——纹路。
所有编译者能感知新规则——包括位置。
防御性规则是生存策略。
```

他盯着屏幕。想了想，又加了一行：

```
爸的事——查。
```

然后他站起来。走到门口。骑上电动车。他决定从现在开始执行林昭说的——感知城市。从自己家开始。向东三公里。每个路口停三十秒。

他骑到第一个路口的时候，手机震了。

林昭的短信：

"还有一件事。你昨晚被覆盖——不是偶然。#12在测试你。苏墨测试每个新人。"

他低头看了一秒。把手机插回支架。拧了油门。

（第003章 完）
