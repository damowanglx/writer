# 第043章 编译前兆

陈渡骑到电视塔附近的时候——他感知不到赵灵儿了。

不是她离开了——是他的感知被切断了。在他和她之间的频率连接——像一根被剪刀剪断的线——突然断裂了。他不是失去了方向——他是完全感知不到她的存在了。

她在写规则。

她的六条规则闭合的那一刻——狂写期的频率达到了某个阈值——切断了她和外界的所有感知连接。

他在电视塔底下的广场上停下来。

苏墨不在。

电视塔底层的入口——那扇钢化玻璃门——关着。但透过玻璃，他能看到一楼大厅里的终端机屏幕在闪烁——青白色的光——和他第一次见到终端机时一模一样。

他推门。锁着的。

他退后一步，看了看手机。没有信号。

然后他听到了——不是声音——是另一种感知。

从地下传来的——低频——像一台巨型发电机的运转声——从电视塔的地基深处传上来。

赵灵儿在地下。

电视塔下面的第二层——和防空洞一样的结构——有第二个终端机。

他绕着电视塔走了一圈——在东侧看到了一个维修通道入口。门没锁。

他推开门，走下去。

通道很窄，只够一个人通过。应急灯每隔三米一盏——和防空洞不同——这些灯是亮的——白色的LED光——冷——像手术室里的灯。

通道向下延伸了大约二十米。

然后空间突然开阔——

一个大约三十平方米的地下室。墙壁是混凝土的，表面涂了防潮漆，地面铺着灰色的防滑瓷砖。房间中央放着一台终端机——不是ATM外壳——是一台放在铁架子上的老式CRT显示器，键盘是黑色的机械键盘，和九十年代的网吧键盘一样。

赵灵儿站在终端机前面。

她的右手放在键盘上——左手握着自己的右前臂——像在压制什么。

屏幕上的文字——

```
编译者#48。
当前纹路：5条。
第六规则写入确认——Y/N？
```

光标在Y后面闪烁。

她没有按Y。她的手悬停在键盘上方——手指在微微发抖——不是恐惧的发抖——是她的身体在抵抗系统强制写入的指令。

“赵灵儿。”

她转过头。她的眼睛——眼白的边缘——灰色的线已经蔓延到眼角了。比于莉的更密——也更亮。像细密的蛛网从瞳孔向外扩散。

“我没有按确认。”她说。声音正常——但音调比平时高了一点——像在努力维持平静。

“那就别按。”

“系统在我脑子里——已经在编译了。我的手不按——系统会用我的纹路直接写。我按住自己的手臂——是在阻止我的纹路自己打字。”

陈渡走到她旁边。

屏幕上——在Y/N的选项下面——有一行不断闪烁的警告文字：

```
警告：未完成的强制写入将在17分42秒后自动执行。
```

十七分钟。

“如果你写了——会怎么样？”

赵灵儿没有回答。她低头看着自己的右臂——五条规则的纹路在皮肤表面发着微弱的灰色荧光。第五条规则的末端——那根银灰色的线——正在自动向肘部延伸。

每秒大约一毫米。

不是她写的。

是系统在推她的纹路前进。

“老周说的‘编译前兆’——不是进了防空洞才有。是在我写第六条规则的时候——就已经开始了。”赵灵儿说。

她的声音第一次出现了一丝裂纹：

“我现在能感觉到系统的想法了。”

“什么想法？”

她转过头看着陈渡。她的眼睛——灰色的线在眼白中像活的藤蔓一样缓慢延伸——但她的眼神还是她的。那种不属于恐惧的、接近分析的眼神。

“它在高兴。”

陈渡的呼吸停了一下。

“什么？”

“系统。在高兴。因为我要写第六条规则了——我的规则链闭合率要到百分之百了。系统在等这一刻——等了很久。”

她把手从键盘上放下来——不是放——是松开。她的右前臂上有五条纹路同时发着光——第五条纹路的银灰色光已经覆盖到了肘关节。

“如果我不写——系统在十七分钟后自动执行。如果我写——我会进入编译前兆阶段——大约有十七天。”

她看着陈渡。

“或者——你用管理员的身份——覆盖我。”

“我没有接受管理员。”

“你接受不接受——系统已经认了。你在防空洞第二层碰了那个石头终端机——权限已经激活了。只是你没有按确认。你现在——是准管理员。你的规则——能覆盖我的。”

陈渡的手悬在键盘上方。

他能做到。

他是#47。系统在防空洞第二层认了他。他的规则——如果他现在写一条覆盖赵灵儿全部规则的指令——系统会执行。

覆盖之后——赵灵儿的规则不会消失——但会变成“由#47管理”的状态。她不再需要自己承受规则运行的压力。

代价是——她会变成和于莉一样的人。

记忆退化。规则能力归零。变成一个带着死规则的普通人。

“覆盖我之后——你就看不到第三层门口的密码了。”赵灵儿说，“#18的笔记本上说——密码需要两个活着的编译者同时输入。我是你唯一的人选。你覆盖了我——你就永远进不了第三层大门。”

“我不需要第三层。”

“你需要。”赵灵儿说，“因为管理员不是在防空洞第二层接任的——是在第三层。你没有踏过第三层的地面——你的权限就只是一半。你只能看到规则——不能修改根规则。”

她的右手——悬在键盘上方——手背上的纹路在自发地向前延伸了一小段。

“我写完第六条——然后你去防空洞第三层。我在外面等你。你的二十四小时——和我剩下的十七天——够你接任管理员了。”

她的手指碰到了键盘的按键——不是她主动碰的——是纹路的延伸把她的手指推了过去。

“现在——我要写了。”

她没有按Y。

但系统的自动执行——在十七分钟前——已经开始了。她的手被纹路牵引着——以不属于她的力量——按下了回车键。

屏幕闪了一下。

然后显示：

```
第六条规则写入中——
编译者#48。
规则类型：规则链闭合型。
规则内容：编译者#48的所有规则——在编译者进入编译前兆阶段后——自动向管理员#47开放全部权限。
开放范围：全部规则内容、全部规则链、全部感知频率。
```

陈渡看着屏幕上的文字。

第六条规则不是攻击性的。不是防御性的。是一个“钥匙”。

她在把自己的全部——规则、感知、频率——全部开放给他。

如果他接任管理员——他就能通过她的规则链——看到整个系统。

代价是——她会在开放过程中——被编译。

屏幕上的确认提示消失。取而代之的是：

```
规则写入完成。
编译者#48。
当前纹路：6条。
纹路进度：指尖至肘部。
规则链闭合率：100%。

系统提示：编译者#48已进入「编译前兆」阶段。
预计编译完成时间：17天。

管理员#47——该编译者的规则权限已全部向您开放。
请在此窗口期内完成管理员接任流程。
```

赵灵儿把手从键盘上放下来。

她的右臂——六条规则的纹路从指尖延伸到肘部——在新的规则写入之后——纹路的颜色变了。从灰色变成了一种接近白色的、半透明的颜色。不是黑色——不是银色——是像玻璃一样的透明线条。

她低头看了自己的手臂。

“好看吗？”

陈渡没有说话。

赵灵儿把袖子拉下来。她转身看着陈渡——她的眼睛里的灰线又蔓延了一步——和刚才相比，她眼眶周围的灰色纹路已经从眼角延伸到了太阳穴。

她看起来不像一个快要被编译的人。

她看起来像一个考试结束了的考生——放松——甚至有一点轻微的疲倦。

“走吧。你想看的——在防空洞里等你。”

她先走出去。

陈渡站在地下室里面，看着终端机的屏幕。屏幕上最后一行字——在自动滚动消失之前——还有一句他没看到的系统日志：

```
管理员#47的未选择状态——已被注册为「主动推迟」。
该状态自当前时刻起——剩余有效期：19小时32分钟。
```

他追上去。

赵灵儿已经走到了通道的出口。正午的阳光从维修通道的门缝里照进来——照在她身上——在她的轮廓边缘镀了一层过曝的白光。

他没有看清她的表情。

只看到了她的轮廓。

和落在身后地面上的、六条纹路在阳光下投射出的——
——异常清晰的影子。
