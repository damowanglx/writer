# 第041章 倒计时

晨光从防空洞入口照进来的时候，陈渡终于感觉到冷了。

他在门口站了多久——他不知道。老周的水泥管空了，苏墨不在树下了。赵灵儿坐在入口旁边的台阶上，抱着那台便携终端机，屏幕的光映在她脸上。

她说了一句：“你站了四十七分钟。”

陈渡没有回答。他把手从口袋里掏出来，看了一眼自己的手臂——六条规则的纹路在晨光下清晰可见。从指间到肘部。第七条的起点就在肘关节上方，像一根等待落笔的线。

他拉下袖子。

“我回去睡一觉。”他说。

“你要一个人待着？”

“对。”

赵灵儿没有拦他。她站起来，把终端机夹在腋下。晨风吹起她额前的头发——她看起来不像一夜没睡的人。她像是被什么能量托住了——那层不是精神，是她体内的五条规则在加速运转。

“二十四小时。”她说，“你决定了再告诉我。”

她转身走了。不是朝城里的方向——是朝另一个方向。陈渡看着她的背影在晨光中变小——她的脚步不快，但每一步都很稳。

他没有问她要去哪。

他骑上电动车，往出租屋的方向开。

---

出租屋在老城区一栋八十年代的居民楼里。四楼。没有电梯。楼道里的声控灯坏了一半，白天也暗着。他走楼梯的时候放轻了脚步——隔壁的老人听力不好，但狗能听到。

他开了门。

房间大约十五平米。一张床，一张桌子，一把椅子。墙角放着一个塑料衣柜，柜门半开着，露出叠得不整齐的衣服。桌上放着一台旧笔记本电脑，键盘缝里夹着外卖订单的油墨印。

他关上门，没开灯。窗帘拉着的，房间是昏暗的灰色调。

他坐在床上。

没有看手机。没有看时间。没有动。

他只是坐在那里。

六条规则在他体内运转——它们不再是“外来”的东西了。它们像身体的一部分，像呼吸和心跳，不需要意识去维持。第五条遮蔽规则在持续运行，第六条的转移规则——沉默地等待着触发条件。

他父亲的事故日志——那三页纸——从他口袋里露出一角。

他没有拿出来看。

他已经记住了。触发条件：工地现场管理人员超过三人。他父亲死的那天，正好三个人。不是谋杀。不是苏墨的猎杀。是系统的机械性——一条沉睡的规则在#18死后自动激活，符合条件，执行。

他父亲的死——是一行代码。

不是人的恶。

是系统的“正常运行”。

他的手机在口袋里震了一下。

他拿出来。

苏墨：

“你还有二十二小时十七分钟。”

陈渡盯着屏幕。没有回复，把手机放回口袋。

过了一会儿——大约五分钟——手机又震了。

苏墨：

“不想知道她在哪吗？”

陈渡的手按在手机边缘，没有解锁。

又过了几秒——第三条消息来了。不是文字——是一张照片。

照片拍的是赵灵儿。

她坐在一个公交站台的长椅上。阳光从顶棚的缝隙照下来，她的右臂袖子卷起来了——五条规则的纹路在小臂表面发着光。不是她在催动它们——是它们自己在发光。

她的表情平静。嘴角甚至有一点弧度——不是笑，是一种接近专注的松弛。

她看起来不像在被纹路吞噬。

她看起来像在享受。

陈渡把照片放大——看她的纹路。第五条规则的末端——灰色的银线——正在向前推进。不是向手腕方向——是向肘部。纹路在移动。

她在不接触终端机的情况下——纹路在自行延伸。

陈渡把手机锁屏。

他站起来，走到窗边，拉开窗帘的一角。街道上的人和平时一样——早饭摊、遛狗的人、送小孩上学的家长。他们不知道这座城市有一套看不见的规则系统在运转。不知道有一个二十四岁的骑手正在决定要不要接管它。

如果他不接管——系统等下一个周期。不知道要等多少年。

如果他接管——他就不再是一个24岁的外卖骑手了。

他会变成管理员。

一个活在系统里的人。

一个不能再犹豫的人。

他的手机在口袋里连续震了好几下——不是消息提示音——是电话。他拿出手机——来电显示是一个他没存过的号码。他接了。

“喂。”

对面沉默了三秒。然后一个声音——他听出来了——是于莉。

“你拿到#18的笔记本了。”

不是问句。

“拿到了。”

“那你应该知道管理员密码需要两个人。”

“我知道。”

于莉在电话那头喘了一口气。声音里有一种陈渡以前没听到过的东西——不是疲惫，是一种接近终点的平静。

“我快忘了。”

陈渡握着手机的手指紧了一下。

“什么？”

“我跟你说的——规则被覆盖之后，记忆会退化。”于莉的声音很稳，像在陈述一件她已经接受的事实，“我今天早上醒来——不记得#18的名字了。我记得有这个人。记得他给了我笔记本。但我叫不出他的名字了。”

陈渡没有说话。

“陈渡——你还剩二十一小时。但我的时间可能更短。你决定之前——先来见我一面。”

她把电话挂了。

---

陈渡在房间里又站了一会儿。

然后他锁上手机，抓起外套，推门出去。

电动车停在楼下。他跨上去，拧了油门——方向不是劳务公司——是时代广场。

他需要先找到赵灵儿。

电动车在早晨的城市里穿行。阳光照在湿漉漉的柏油路面上，反光刺眼。他穿过三个路口——在第四个路口的红灯前停下——然后他感觉到了。

不是看到——是感觉到。

赵灵儿的位置。

他在狂写期内的频率连接——她体内的五条规则像五根琴弦——他在几个街区外就能感知到她在哪里。

她在时代广场负一层。

她在终端机前面。

他拧了油门——闯了红灯——电动车在一声刺耳的刹车声中穿过路口。后面有司机按喇叭，他没有回头。

他骑到时代广场，把车扔在路边，跑下台阶。

负一层的通道灯亮着——和昨天一样昏暗潮湿。

但终端机前没有人。

赵灵儿不在。

他站在空荡荡的停车场中央，环顾四周。空气里有水泥和汽车尾气的味道。通风管道发出低沉的嗡嗡声。终端机的屏幕亮着——不是待机画面——是输入状态。

输入框里有一行没打完的字：

```
编译者#48的第六
```

字没打完。

光标停在“六”字的后面。

赵灵儿写了一半——被中断了。

陈渡低头看屏幕底部的系统提示——一行小字：

```
规则写入中断——编译者#48感知到以下警告：
当前规则链闭合率：91%。
第六条规则写入后——闭合率将达到100%。
警告：第六规则写入将触发「编译前兆」阶段。
```

屏幕下方的出口方向——传来脚步声。

轻。慢。均匀。

陈渡没有回头。

脚步声在距离他大约五米的位置停下来。

苏墨的声音——

“她在公交站台。我把她劝走了。”

陈渡转过身。

苏墨站在停车场出口的光照边界——一半在阴影里，一半在光里。他的外套换了一件——深灰色的夹克，看起来不贵，但整洁。不像一个猎杀者。像一个在周末早晨出来散步的普通人。

“你对她做了什么？”

“什么都没做。”苏墨说，“我只是告诉她——她写完第六条的那一刻，会触发编译前兆。她问我——”

他停了停。

“——‘编译前兆之后，我还有多少时间？’”

陈渡的手握紧了。

“你怎么回答的？”

“我说——平均十七天。最长的一个——沈拓——撑了九个月。”

苏墨的声音没有起伏，像在报一组实验数据。

“她写了多少？”

“她没写完。”苏墨说，“她问我——‘如果我写完第六条，陈渡能看到什么？’”

陈渡的呼吸停了一下。

苏墨看着他——那张没表情的脸上第一次出现了一种接近“好奇”的东西。

“我说——‘他会看到管理员候选人和一个即将被编译的人之间，那条线的终点。’”

苏墨从口袋里拿出一张折叠的纸条，扔在终端机的台面上。纸条落地的声音像一片树叶。

“这是她让我给你的。”

苏墨说完，转身走进阴影里。脚步声——又轻又慢又均匀——沿着通道向上，消失在地面方向的光晕中。

陈渡站在空荡荡的停车场里。

终端机的光标还在闪。

他低头看那张纸条——展开。

赵灵儿的字——和她在终端机上打字时的风格一样——干净、整齐、没有多余的笔划：

“我去写第六条了。我知道你不想让我写。但我写完——你才能看到管理员终端第三条规则的密码。那是#18笔记本里说的——一个人的密码进不去第三层。我们两个——你接管之后——我给你开门。”
