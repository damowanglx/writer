# 第021章 逼迫

老周那本书的最后几页是空白的。

陈渡把书翻到最后一页有字的地方——不是规则记录，是一封信。写给"下一个看到这本书的人"。

他坐在书店角落的旧沙发上，台灯的光只照亮了书页的一部分。其他部分在阴影里。

"如果你能看到这本书——说明你已经在孟轲书店里坐下来了。那你至少活过了一周。大多数新人在第一周就死了——被覆盖的，被编译的，被自己写规则写死的。你活过了——所以你有资格知道下面这些。"

"系统不是最近才出现的。它一直在。三十二年前第一台终端机出现在城西防空洞——1994年11月7日。我那天刚好在附近。不是因为系统选了我——是我选了它。"

"系统不会主动选择任何人。它只是在那里。能看到它的人——某些大脑结构异常的人——会自然被它吸引。像某些昆虫被特定的光吸引。"

"第一批有十二个人。现在活着的不超过三个。"

陈渡翻到下一页。一行字被横线划掉了——划得用力到纸面被笔尖戳破了。划掉之后，下面重新写了一行，字迹比前面潦草：

"四十七个编译者——有一半死于相互猎杀。三分之一死于代价。剩下的——被编译了。"

"如果你在考虑要不要继续写规则——答案是不要。停在这里。你的纹路越多，你在系统里的权重越大。权重越大——你被管理员注意到的概率越大。"

"管理员是谁？"

"我不知道。但我知道它存在。"

"它在看。"

台灯的光跳了一下——不是电压不稳，是一只在灯管附近飞的小蛾子碰到了灯泡。陈渡把书合上。

他现在对这本书的内容有了一个基本的了解。但他需要更多。

他站起来的时候，看见赵灵儿从书店内间走出来。她换了一件衣服——灰色的短袖，右手臂上的纹路边缘在袖口下露出一点点。她手里拿着一张纸条，面色有点不对。

"怎么了？"

赵灵儿把纸条递给他。

纸条上的字是用打印机打的——不是手写。没有抬头，没有署名，只有三行字：

```
#48的规则结构已记录。
下一条规则——不要在孟轲书店写。
在你们自己的节点写。
```

苏墨。

他知道他们在书店里。知道赵灵儿在这里。

纸条用什么方式送进来的？书店里八条防御规则——外部规则无法穿透，电子设备无法被追踪。但纸条在赵灵儿口袋里出现了——她换衣服的时候发现的。

苏墨不光能感知到他们在哪里——他能在孟轲书店的防御层内部留下东西。

"早上换衣服的时候——不在口袋里。"赵灵儿说,"我出门前刚穿的这件——口袋是空的。到了这里之后我没脱过外套——但这张纸条就在口袋里。"

"他在你身上留的。"陈渡说,"在你来书店的路上——你经过某个人多的路段——他可能碰到了你。"

"我没感觉到任何人碰我。"

"那就不是物理接触——是规则接触。"

陈渡把纸条翻过来。背面有一行极小的字——不是打印的，是手写的。笔迹和之前纸杯底部那行字一样：

```
双人规则——共享代价也共享位置。
你以为你们在分担重量。
实际上你们在暴露两个人的坐标。
```

陈渡的手指收紧了一下。

共享位置。双人规则——他和赵灵儿共享了一条规则。这意味着系统在记录他们的时候，会把他们的位置数据关联在一起。苏墨不是追踪到了赵灵儿——是追踪到了"双人规则信号"，然后同时定位了他们两个人的位置。

"他把我们也当成样本了。"赵灵儿的声音没有颤,"他说'你们的规则结构已记录'——不只是我。是我们两个。双人规则——他以前没见过。"

书店的温度没变。但陈渡的后背有一层薄汗。

"他让你在自己的节点写——为什么？"陈渡问。

"不是在帮我——是在帮我写规则之后——他能通过节点的信号更清晰地读到规则结构。"赵灵儿说,"在书店里写——八条防御规则会干扰信号。他读到的不完整。他在等我在一个干净的地方写——他好完整地读取。"

"那你别写。"

赵灵儿看了他一会儿。

然后她笑了——和第一次在停车场见面时一样的笑。不是紧张。不是自嘲。是真的觉得有什么地方有趣。

"他会一直等。如果我一直不写——他会来找我。到时候不是在节点写的问题——是在他面前写的问题。"

"那就躲。"

"躲多久？你躲过吗？"

陈渡没有回答。

她说对了。他没有躲过。他一直在往前写规则——虽然犹豫，虽然害怕——但他一直在写。

因为他停不下来。他父亲的死还没有真正的答案。包工头的和解协议还在法律文书里。苏墨知道那条规则在哪里——但没告诉他。如果他不继续走——他永远找不到那条覆盖了他父亲死亡的规则。

"你先回书店里面。"陈渡说。

"你呢？"

"我想点事。"

赵灵儿看了他一眼，没追问，转身走了。脚步声在书店的木地板上走了几步，然后被书架之间的狭道吞没了。

陈渡走进卫生间。拧开水龙头。冷水冲在他的手指上——他低头看着水流冲过手腕上的纹路。四黑色，一银色。

第五条规则——他需要写。不是为了苏墨。不是为了解锁什么能力。

是因为他需要一条能保护赵灵儿的规则。

不是因为双人规则让他们绑在了一起。是因为她不该被苏墨当做一个样品来记录、追踪、等待收割。

他关掉水龙头。用袖子擦了手上的水。

走出卫生间的时候，他的手机震了一下。

林昭的消息："来内间。有东西给你看。"

他走进去。林昭坐在沙发上，膝盖上放着一本打开的笔记本——不是老周那本，是她自己的。手指压着翻开的页面上的一张图。

"这是我在城西的规则感知记录——从苏墨开始猎杀算起。"

陈渡低头看。页面上是一张时间线图，标注着每次猎杀的时间和对应的规则类型。苏墨的猎杀——从三年前开始——被林昭按时间顺序排列。

"三次猎杀之间平均间隔四个月。但——"

林昭的手指点在最近的三个点上。时间间隔急剧缩短。

"上个月他猎杀了两个。这周他猎杀了贺冬的一部分——但不是全部。他在加速。"

"因为他找到了想要的东西？"

"或者——他在赶时间。"林昭看着他,"系统里有某种东西在变化。我说不上来——但我能感知到——规则网络的整体频率在上升。像机器在升温。"

她翻到笔记本的下一页。

上面画着一幅图——四个主节点和一个中心点的位置关系。和赵灵儿画的那张几乎一模一样。

"你知道为什么是五个节点吗？"

"四个角加一个中心——电视塔。"

林昭摇头:"不。电视塔是我们后来发现的。系统的原始设置是——四个主节点。城中旧法院——立法。城东电视塔——发布。城南钢厂——执行。城西防空洞——存储。"

"第五个呢？"

"第五个不是节点——是管理员终端。"

"在哪里？"

林昭的手指落在图纸的中心位置——但不是电视塔。

"城西防空洞。最深处。"

她的手指在图纸上画了一条线——从旧法院穿过电视塔，经过城南钢厂，最终指向城西防空洞。

"管理员终端不在任何新位置。它在第四个主节点的最深处。"

陈渡看着那张图。四个节点的交叉线全都汇聚到防空洞。电视塔不是中心节点——它是一个入口。一个引导编译者走向真正终端的指引标志。

"你去过那里？"

"去过一次。两年前——在苏墨帮我之前。"林昭的声音很轻,"我只到了第二层。第三层的门是关着的——上面有一行字。"

"什么字？"

"管理员密码需要在两个活着的编译者之间同时输入。"

两个活着的编译者。

双人输入。

他和赵灵儿已经演示过双人同步了——在发现共振的那台终端机上。那不是巧合。系统在他们身上看到了一种特定的组合方式。这种组合不是偶然发生的——是系统在引导他们走向那个输入口。

赵灵儿在某个时候会站在他旁边。两个人同时输入密码。打开管理员终端的门。

"他知道。"陈渡说,"苏墨知道密码需要双人输入。所以他需要你和我——"

"都活着。"林昭说,"到我写够了规则。到你掌握了足够的信息。到我们两个刚好达到系统的最低门槛。然后在那个节点上——他收割我们两个。继承我们所有的规则——包括双人规则。"

卫生间的滴水声在安静的房间里格外清晰。一滴。又一滴。

三秒间隔。

像一台计时器。

（第021章 完）
