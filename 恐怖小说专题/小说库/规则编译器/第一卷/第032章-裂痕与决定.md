# 第032章 裂痕与决定

他们没有直接去防空洞。

赵灵儿骑着陈渡的电动车，陈渡坐在后座——她的驾驶风格比陈渡野得多，在巷子里穿梭的时候几乎没捏过刹车。她骑到了孟轲书店后面一条窄巷里，熄火，跳下车。

"你从哪弄的终端机？"陈渡问。

"电视塔下面的第四台。"赵灵儿把那台金属外壳的终端机放在地上，蹲下来打开屏幕,"我回去了一趟。在苏墨布置城西规则的时候——我去取的。"

"你疯了？"

"没疯。我算了时间——苏墨在城西同时写三条规则的时候，他的注意力覆盖了整个城西区域，但他不可能同时感知电视塔。终端机的位置在电视塔下面四层——他的感知密度不够穿透那个深度的混凝土。"

她打开屏幕。屏幕上显示的不是规则输入界面——是一张地图。

赵灵儿用手在屏幕上划了一下——地图放大了。

"电视塔的终端机——我发现了它的一个功能。它不只能写规则——它能读取系统日志。我今天凌晨写的第四条规则触发规则链之后——系统日志里多了一条记录。"

她把屏幕转向陈渡。

屏幕上显示着一行字：

```
管理员终端访问日志——上次访问时间：2026-06-27，00:47:33
访问者：编译者#12。
访问结果：被拒。——纹路编码不匹配。
```

苏墨在昨晚凌晨——去过防空洞的管理员终端。他被拒绝了。

"他被拒绝了——但他还是去了。"赵灵儿说,"他明知道进不去——他还要去试。为什么？"

"因为他在等——等我够资格。"

"或者——他需要确认管理员终端还在运作。他怕系统在他准备好之前就关闭了。"

陈渡蹲下来，看着屏幕上的日志。管理员终端——林昭去过一次，只到了第二层。苏墨也去过——被拒绝了。现在轮到他了。

但他没有管理员密码。林昭说管理员密码需要在两个活着的编译者之间同时输入。

他和赵灵儿就是那两个人。

他抬头看着赵灵儿:"你知道如果去防空洞——可能会发生什么。"

"知道。"

"苏墨可能就是在等我们两个同时出现在那个门口。"

"知道。"

"那你为什么还要去？"

赵灵儿看着他，表情平静得像在说一件她已经想了一整夜的事:

"因为如果他不让我们进去——那防空洞里就没有我们需要的东西。如果系统真的需要我们——它会让我们进去。"

她停顿了一下。

"我想知道——系统需要我做什么。"

陈渡没有回答。他站起来，在巷子里走了几步。巷子很窄，两侧的墙壁上爬满了青苔，老旧的水管沿着墙面排列。空气中有一股潮湿的、混着油烟的气味。

他的手机响了——林昭。

他接起来。

林昭的声音很疲惫，但不急:"我拦了他两分钟。不是一分钟。我现在在书店。孟轲在帮我稳定纹路。"

"你的纹路——"

"到肘部了。"

陈渡握着手机的手紧了一下。

"你现在什么感觉？"

"……还能思考。但——有些事情在变。"林昭的声音里有一种他在电话里从未听过的陌生感——不是恐惧，是困惑,"我在跟你说话的时候——脑子里有一部分在同时计算规则结构。不是我在算——是我的纹路在算。像另一颗大脑长在我的手臂里，在我没有同意的情况下开始工作。"

"你还能控制吗？"

"暂时可以。但——"她停了一下,"我不确定明天还能不能。"

电话那头传来孟轲的声音——在说什么，听不清。然后是林昭的声音重新出现:

"别来书店。别来找我。去做你应该做的事。"

"我应该做什么？"

"你知道的。"

她挂了。

陈渡把手机放回口袋。巷子里安静了几秒。赵灵儿站在他旁边，手里拿着那台终端机。屏幕的光映在她的脸上，照亮了她眼睛里的某种无法言说的平静。

"林昭的纹路过肘了。"陈渡说。

赵灵儿没有惊讶。她低头看着自己的手臂——第四条规则纹路在小臂中段:

"我也会。如果你继续写——你也会。"

"那你怕吗？"

赵灵儿想了一下。然后她说:

"怕的不是过肘——怕的是过了肘之后,我还能不能认得自己。"

她把终端机抱在怀里。

"走吧。防空洞的位置——你知道在哪。"

陈渡没有立刻回答。他站在巷子里，手里握着老周给他的牛皮纸信封——那几张关于他父亲事故的原始记录。他一直没看。

他低头看着信封。

然后他把信封塞进口袋。

"先去一个地方。"

"哪？"

"劳务公司。我要见于莉。"

（第032章 完）
