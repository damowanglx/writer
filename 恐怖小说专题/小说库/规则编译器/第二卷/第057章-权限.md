## 第057章 权限

十分钟。

陈渡只知道苏墨在来的路上——不知道具体多久。但他用了十倍加速之后，他感知中的时间被压缩了。每一秒都像被拉长了两倍。

不是因为速度——是因为感知精度。

十倍规则执行速度让所有规则信号都变得更清晰。像一台收音机从模糊调到了清晰频道——他能听到以前听不到的杂音。

但这个区域的规则在打架。

他提交的“不受追踪”覆盖规则——#31还没有完成拒绝操作。系统一直在等。十倍加速下，等待时间从几十秒缩短到了几秒。但如果#31在这几秒内完成了拒绝——

终端机亮了一下。

陈渡屏住呼吸。屏幕显示的不是拒绝——是#31发过来的数字：

“31”。

什么意思？

手机震了一下。#31发了一条消息：

“你的编号是47。我的编号是31。差16号。你猜为什么我比你早注册？”

陈渡没有回。

“因为系统是按注册顺序编号的。我是第31个发现终端机的人。你是第47个。但我比你多写了17年规则。你知道17年里我学到了什么吗？”

陈渡看着屏幕。等着下文。

“覆盖的核心不是范围——是深度。”

他的手机屏幕突然闪烁了一下。

他写的那条“此区域所有信息传递——不受追踪”规则——状态从“等待确认”变成了“已被覆盖”。

不是拒绝。

是覆盖。

#31没有拒绝——他直接写了一层新规则压在陈渡的规则上面。像叠罗汉。他拒绝的是“接受陈渡的覆盖”，不是拒绝系统——他用行动覆盖了陈渡的规则。

陈渡盯着屏幕。

“此区域所有信息传递——不受追踪——已被覆盖。”

新规则的内容是什么？

他闭眼感知。

空气里，#31的新规则像一层更厚的油膜浮在表面。他读取它的文本——

“此区域所有信息传递——受本节点最高优先级规则约束。”

不是针对性的。是通用规则。

#31没有写一条“恢复数据采集”的规则——那太弱了，陈渡还能再覆盖回去。他写了一条更抽象的规则：#31自己的规则在#31的节点上拥有最高优先级。

这是一条元规则。

陈渡站在黑暗里。他感觉到了——他的“不受追踪”规则还在，但被压在了下面。像两块磁铁同极相斥——他的规则在向上顶，但#31的规则更厚、更重、更老。

这就是十七年的差距。

不是谁更聪明——是#31写的每一条规则都在为下一条规则铺路。他布局了十七年。每一条规则都是他棋盘上的一颗子。

陈渡只有五条。

他的前四条规则分散在不同的节点、不同的方向、不同的逻辑层面。没有互相关联。没有形成体系。老周说得对——他是在应激，不是在创造。

但他有一个#31没有的东西。

管理员权限。

他不知道自己到底有什么权限。林昭说过，老周暗示过——但没有人给他一份明确的“管理员功能列表”。他只知道自己是在雨中的写字楼里、在灰色区域意外“继承”了那个身份。

如果他有什么权限是#31没有的——那会是什么？

他打开手机APP。不是写规则的界面——是设置界面。

他以前从来没有认真看过APP的设置。他打开它，快速扫描菜单。

基本功能：写规则、查看规则、感知增强。

隐藏功能——

陈渡的手指停住了。

屏幕最下方有一行极小的字，小到几乎看不见：

“系统诊断模式——仅管理员可用。”

他点了下去。

屏幕闪了一下。整个界面变了——不再是简洁的APP风格，变成了一行行滚动的代码。

终端模拟器。

他不懂代码。但上面的信息是中文的——不是编程语言，是系统在实时输出它内部的运行状态。

他看到了这个区域的规则堆栈：

```
[节点00231] 规则堆栈（按优先级降序排列）：
  1. 规则#0056 "本节点最高优先级规则约束"  — 作者：#31 (覆盖中)
  2. 规则#0047 "不受追踪"  — 作者：#47 (等待生效)
  3. 规则#0031 "数据采集"  — 作者：#31 (运行中)
  4. 规则#0022 "终端机禁令"  — 作者：#31 (运行中)
  5. 规则#0019 "加速惩罚"  — 作者：#31 (运行中)
```

他看懂了。

规则堆栈是按优先级从上到下排列的。#31的元规则在最顶层，排第一。他自己的规则排第二——被压着，但在等待生效。#31的三条老规则在下面继续运行。

如果他能在堆栈顶层插入一条自己的规则——从上面往下压——就能一口气覆盖#31的所有规则。

但怎么插进去？

他盯着那行代码看了很久。

管理员权限到底能干什么？他试着在终端模拟器里输入了一个命令：

`/rules insert`

屏幕回应：

```
权限验证通过。请输入规则文本。
```

陈渡的手抖了一下。

通过了。

#31的权限只是“节点所有者”——他的权限是管理员。在系统层面，节点所有者和管理员不是同一个层级。节点所有者可以管理自己的节点。管理员可以管理所有节点。

他站在黑暗的路灯下，右臂的纹路已经加速到肘上五厘米。右前臂的一半被黑色覆盖了。但在终端机屏幕的冷光里，他输入了一行规则：

“此区域规则优先级判定标准——以编译者系统登记时间为准。”

他点了一下“插入”。

屏幕上显示：

```
规则正在插入目标节点[00231]堆栈顶层...
检测到权限冲突：节点所有者(#31)未授权此操作。
正在调用管理员权限覆盖...
权限覆盖成功。
规则已插入堆栈顶层。
```

成功了。

陈渡甚至没反应过来——屏幕上已经显示“权限覆盖成功”。

他往后退了一步。低头看自己的右臂。纹路不动了。不是加速——是不动了。

因为他插在堆栈顶层的那条规则改变了优先级判定标准——不是按谁的规则更抽象，是按谁在系统里登记的时间更早。

而他——#47——比#31早了大约三十年。

那是管理员终端登记的时间。不是他注册的时间。管理员终端存在的年份比所有编译者都早。

整栋分拣站的灯亮了。不是恢复供电——是终端机的屏幕全部亮了。

#31的所有规则——三条主规则、一条元规则——全部被压在了他的规则下面。

陈渡的头突然一阵眩晕。

不是累——是信息。

#31的规则数据开始涌入他的感知——十七年的规则记录，像一本被压缩到极致的书，在他的大脑里快速翻页。不是每条规则的完整文本——是概要。像读目录。

他能感觉到#31的愤怒。规则之间是通感的——覆盖者的情绪会通过规则传递给被覆盖者。他感觉到了#31的震惊和恐惧。

十七年的编译者，被一个四个月的新人用管理员权限压制。

陈渡按着太阳穴。信息量太大了。他靠在路灯杆上，大口呼吸。

手机震了一下。

#31的消息只有一句话：

“你不是普通编译者。”

陈渡没有回复。

他知道这意味着什么。从今天开始——所有能感知到规则信号的编译者都会知道：有一个管理员在活动。

包括苏墨。

他又站了起来。分拣站的灯光照在他身上。右臂的纹路停在了肘上五厘米——不算太多，但这只是开始。他知道覆盖#31的规则不是没有代价的。只是代价还没来。

他低头看手机。

终端模拟器还在运行。堆栈顶部是他刚插入的规则。

但他不知道的是——在系统日志里，在陈渡看不到的地方，有一行记录：

```
管理员#47 — 首次使用覆盖权限。
权限等级：完全。
目标节点：00231.
结果：成功。
耗时：3.7秒。
```

这条日志被系统自动上传到了所有主终端机的同步日志里。

包括城中心灰色区域。包括城北的某个地下室。

包括苏墨正在看的那台终端机。

（第057章 完）
