# 配角团队设定卡：《规则编译器》

---

## 一、林昭 —— 引路人/女主

| 属性 | 内容 |
|------|------|
| **年龄** | 28岁 |
| **身份** | 前律师 → 全职编译者 |
| **纹路** | 17条,到肘部 |
| **性格** | 冷、精准、说话从不废话。但她的"冷"不是无情——她曾经因感情用事害死过队友(未婚夫) |
| **核心行为** | 教陈渡规则基础。开始是利用(新手可替她试探规则),中间是愧疚(陈渡替她扛了一条),最后是——她说不清楚 |
| **标志性物品** | 左手腕一道旧疤——苏墨写的规则留下的。她留着不修复,提醒自己 |
| **与陈渡关系** | 引路人→战友→说不清楚的羁绊。她在他身上看到了自己三年前的样子——区别是陈渡还没有失去任何人 |

### 关键台词
- "写第一条规则的时候,你有没有想过——这个系统,是谁在管？"
- "我教你规则不是为了让你替我挡。是想让你活下去。"
- (第五卷,看着陈渡从管理员终端走下来)"你还有纹路到心脏的最后一厘米。那一厘米——别给我。"

## 二、赵灵儿 —— 人性锚点/代价承担者

| 属性 | 内容 |
|------|------|
| **年龄** | 21岁 |
| **身份** | 大学生,同批编译者 |
| **纹路** | 从1条到5条(进度比陈渡快) |
| **性格** | 乐观、纯粹、不害怕代价——"如果我能用规则帮一百个人,变成代码也值" |
| **核心行为** | 用规则做善事——帮老人过马路、阻止家暴。她的纯粹是陈渡的人性锚点。但也因此写规则太快 |
| **命运** | Vol4末尾替陈渡挡编译——"你还有半本书没写完呢。我写得少——就五条。不亏。" Vol5在系统内部为陈渡留后门,以规则形态继续存在 |

### 功能定位
- 代价展示——她越是不怕,读者越怕
- 情感重击——第四卷末尾是全书情感最高点
- 系统内的希望——她不是"死了",是换了一种方式存在

## 三、老周 —— 知识宝库/代价活证据

| 属性 | 内容 |
|------|------|
| **年龄** | 45岁 |
| **身份** | 第一代编译者(1994年第一批) |
| **纹路** | 22条,到锁骨。已停止写新规则三年 |
| **性格** | 沉默、观察、说话只说七分——"你知道的越少,越安全" |
| **核心行为** | 观察系统三十二年。知道所有编译者的历史。在陈渡出现后开始透露真相——"系统在等你" |
| **标志性物品** | 一本手写的规则日志——记录了三十年来所有他观察到的规则变更。最后几页是空白——留给陈渡的 |

### 关键台词
- "系统不是在选最强的。是在选——最不愿意写规则的人。"
- "你问为什么是你——因为你每次写规则之前都犹豫。前46个——没有一个犹豫的。"

## 四、姜南 —— 普通人视角

| 属性 | 内容 |
|------|------|
| **年龄** | 26岁 |
| **身份** | 外卖站点同事 |
| **纹路** | 无(普通人) |
| **性格** | 话多、热心、看起来大大咧咧但心细 |
| **核心行为** | 替陈渡顶班(不知道他在干什么,只知道他"有事")。最后发现真相时的反应是全书情感重击 |
| **功能** | 读者代入视角——普通人面对"规则世界"的反应。提醒陈渡(和读者):这世界还有正常人 |

### 关键台词(第五卷发现真相时)
"你三个月前那单超时的——我帮你赔了。你那天是不是在跟那个姓苏的打？"

## 五、孟轲 —— 安全区守护者

| 属性 | 内容 |
|------|------|
| **年龄** | 33岁 |
| **身份** | 中立编译者,旧书店主 |
| **纹路** | 8条,全写在书店里 |
| **性格** | 平和、不爱出门、相信"有的书不该被打开,有的规则不该被写" |
| **核心行为** | 书店是编译者公认的安全区——"此店禁止争斗"是第一条写在书店里的规则。不参与战争,但提供场地 |
| **标志性物品** | 收银台抽屉里有一本空白的规则记录本——每个来书店的编译者可以自愿留下一条规则 |
