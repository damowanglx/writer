# 第175章 归零

第三天。

陈渡在凌晨四点半醒来——不是自然醒，是系统警报。

他的视野里弹出一行红色的文字:

```
系统状态更新 | 管理员 #02
格式倒计时: 0天 0小时 0分钟

格式化程序状态: 已暂停（管理员干预）
格式化预期时间: 已过

系统询问:
格式化已被取消。请提交替代方案。
回复期限: 48小时。

逾期未回复——系统将自动执行默认操作。
默认操作: 重新激活格式化程序。
```

陈渡坐在床边，看着那一行字，很久没有动。

不是没有预料到——但他以为系统会给他更多时间。四十八小时。他需要在四十八小时内给系统一个“替代方案”——一个不靠格式化维持系统运行的方法。他不确定自己有没有方案。他甚至不确定自己理解了系统的“方案”是什么意思——是一段代码？是一套规则？还是一个方向？

他站起来。赵灵儿还在睡觉——侧躺着，呼吸均匀。她的右臂露在被子外面——纹路已经越过了肩膀，正朝锁骨的方向蔓延。在月光下，那些纹路不像墨水——像血管。蓝色的，微微发光。他在床边站了一会儿，没有叫醒她。

他穿好衣服，轻轻带上门。

凌晨四点半的街道很空。路灯还亮着，但天边已经开始泛白。他骑上电动车，没有目的地。骑了大约十分钟，他在路边的早点摊停下来。老板娘正在蒸包子，蒸汽在路灯下像一团白色的雾。

“两个包子。一杯豆浆。”

“这么早？”老板娘随口问。

“睡不着。”

老板娘看了他一眼——不是看他的脸，是看他的右手。他下意识地把手缩了一下，但已经晚了。她从蒸笼里夹了两个包子，用纸袋装好，递给他。

“年轻人，别熬太狠。”

陈渡接过包子。他没有解释——解释不清的。他付了钱，站在路边，把包子吃完了。豆浆是甜的，热的。很简单。但这是他在倒计时归零之后吃的第一顿饭。他在路边站了几分钟。然后他的手机响了——不是系统通知，是孟轲。

“你知道了吗？”

“知道了。”

“四十八小时。你有什么想法？”

陈渡沉默了一会儿。“我在想——如果我不提交替代方案——格式化重新激活——会怎么样？”

电话那头安静了几秒。然后孟轲的声音传来，不是惊讶，是确认:“你在考虑格式化？”

“我在考虑‘如果不提交方案’意味着什么。”

孟轲沉默了一会儿。“这意味着——你选择了让系统自己决定。”

“系统会重新激活格式化。”

“然后呢？”

“然后——所有人都会被执行格式化。”

“那些选相信你的人呢？许乐？那个主动编译的老编译者？他们的选择就白费了。”

陈渡没有说话。他站在早点摊旁边，看着街道上开始多起来的车辆。早高峰快到了。这座城市又要开始新的一天了。他在想——如果他提不出方案，这座城市底下那个系统会在四十八小时后重新启动格式化。然后一切都会消失。

“你手里有一百多条规则。”孟轲说。“一百零七条冲突。系统不稳定。你不提方案——系统会重新格式化。你提方案——系统可能会接受，也可能会拒绝。但你至少试了。”

“如果我的方案不够好呢？”

“那你就和沈拓一样。”孟轲说。“尽了全力。但不是所有的努力都会成功。”

陈渡挂掉电话。天已经亮了。他把空豆浆杯扔进垃圾桶，跨上电动车。他没有去防空洞。他先回了家。

赵灵儿还在睡觉。他站在卧室门口，看了一会儿她的睡脸。纹路从她的手臂蔓延出来，在晨光中泛着微弱的蓝光。他在她额头上很快地碰了一下，然后转身出门。

他骑到防空洞的时候，太阳已经完全升起来了。废弃农贸市场的地面上有一层薄薄的露水，踩上去脚感是湿的。他走进防空洞。管理员终端机的指示灯亮着。他在它面前坐下，把双手放在键盘上。

系统弹出了提示:

```
管理员 #02
请提交系统稳定化方案。

你的方案将成为系统新的运行基线。
所有现有规则将根据此方案重新评估。
```

他的手指悬在键盘上方。

他想到赵灵儿。想到许乐。想到那个主动编译的老编译者的哥哥。想到沈拓。想到沈铮。想到苏墨——他的二十三条规则，他收缩的网络，他在电视塔上说的话。

他想到沈拓信里的那句话:“系统不是工具。它是活的。”

他没有写技术方案。没有写规则补丁。没有写任何关于冲突处理、稳定性算法或权限分配的东西。

他写了八个字:

“让规则服务于人。”

然后他按下了回车键。

系统没有立刻回应。屏幕闪烁了一下。然后出现了一行字:

```
方案已接收。正在评估。

评估维度: 可行性 | 一致性 | 可持续性 | 系统稳定性影响

评估中……
```

光标在闪烁。

陈渡坐在防空洞里，面前是管理员终端机。

他不知道系统会怎么评估他的八个字。

但他知道——这是他作为管理员写下的，最重要的八个字。

（第175章 完）
