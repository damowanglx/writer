# 第151章 广播

陈渡从防空洞出来的时候，天快亮了。

他靠在洞口外墙上，低头看自己的右手。从指尖到肘部，黑色的纹路像血管一样趴在皮肤下面。九条规则——九道纹路。不算多，不算少。但每一道都在提醒他:离心脏还有多远。

他回头看防空洞入口。最深处那台管理员终端，屏幕还亮着，光标在"请输入管理员密码"后面一闪一闪。他已经在那台机器前坐了七个小时，试了所有能想到的组合——沈拓的生日、沈拓的名字、系统的编号。都不对。

他拧开一瓶水喝了一口。水是温的。夏天清晨的风吹过来,带着柏油路面和早点摊的气味。城市快醒了,但他在的城市和别人的不是同一个。

手机震了一下。不是单子——他已经关掉了接单软件。是一条系统通知,来自终端机:

「所有编译者:请于今日内确认你的规则状态。」

他盯着这行字看了三秒。什么东西不太对。

这则通知的语气不对。不是编者的声音。过去的通知都有作者——是某个编译者写的规则被触发了,或者系统在回应一个操作。但这一条——像系统自己在说话。

他正要关掉手机,屏幕突然闪了一下。

新的消息覆盖了上一条。不是在他手机里——是在他的视野里,叠加在现实世界上。就像视网膜上突然多了一层显示:

```
—————————————————————————————
系统全局广播 | 优先级:最高
—————————————————————————————

所有编译者:

格式化倒计时已启动。

剩余时间:29天23小时59分钟。

在此期间,任何编译者可选择主动编译 —— 你的全部规则将被保留在系统的新架构中。

被动编译者的规则将被删除。

此消息由系统自动生成。不来源于任何编译者。
—————————————————————————————
```

陈渡的手停住了。

不是震惊——是那种确认了某件事之后的平静。倒计时。沈拓的日志最后一条写的就是这个。"不要接任。格式化的倒计时已经开始了。"他以为倒计时是一年、两年,或者至少几个月。他没想过是三十天。

他把手机锁屏又打开。那行字还在视野右上角,像一块永远消不掉的弹窗。

他骑上电动车,没有目的地。只是骑。

街道的样子没变:环卫工在扫落叶,早餐摊的老板娘在蒸包子,公交站台有人在等车。普通人不会收到那条消息,他们看不到视野里那行倒计时。对这个世界来说,今天和昨天一样。

但他能感觉到——不是看到,是感觉到——整个城市的规则网络在震动。像一面大鼓被敲了一下,余波还在扩散。每一个编译者都收到了同样的消息。他能感觉到他们的反应:有人恐慌,有人在终端机前跑来跑去,有人……

有人在编译。

不是主动编译——是被动的。有一个人,在收到消息的瞬间,纹路突然加速蔓延。那个人应该写了很多规则,纹路本来就离心脏很近。消息的冲击让他——或者她——失去了控制,最后一条规则自动写入,纹路跨过了最后一毫米。

陈渡捏了刹车。

他感觉到了那个人的纹路走到终点。不是杀人的感觉——是被编译者的意识融入网络的时候,整个系统会发出一声极其微弱的"嗡",像一块石头扔进水里。然后那个人就变成系统的一部分了。不到三十秒,他感知到那个编译者的规则出现在全局网络中——几行代码,功能明确,很稳定。但人没了。

第一个。消息发出后不到五分钟,第一个。

他的手机响了。孟轲。

"你来书店。"孟轲的声音很稳,但比平时快一点。"所有人都来了。"

"苏墨呢？"

沉默。

"他不在。"孟轲说。"我联系不上他。他去了电视塔。"

陈渡挂掉电话,拧了油门。去书店的路上他经过两个节点——一个在商场负一楼,一个在地铁站的通道里。两台终端机前都有人。一个年轻女人站在机前,手放在键盘上,但没有打字。她只是看着屏幕上的倒计时。路过时他看清了她的脸——哭着,但没有声音,眼泪滴在键盘上。

他没停。

到书店门口的时候,他数了一下:大约十五个编译者挤在孟轲这间不到三十平的旧书店里。有人站在书架中间,有人坐在收银台旁边的地上,有人靠着门框抽烟——书店里不让抽烟,但孟轲没阻止。窗户关着,烟雾出不去,灰色的烟在日光灯下晃。

他看见林昭。她站在最里面的柜台旁边,面前放着一杯没碰过的水。她看到他的第一眼什么也没说,只是轻轻摇了一下头。不是打招呼——是警告。

然后她朝他走过来,低声说了一句:

"刚才有人被编译了。两分钟前。感知到了吗？"

"感知到了。"

"那是老李。城东那个,写了十五年规则,纹路到脖子。消息来的那一刻他正在写一条新规则——来不及停手。"

陈渡没说话。他不认识老李。但他知道那种感觉——规则自动写下去,收不住,眼睁睁看着纹路往前走。

林昭看了一眼身后的人群。"现在的问题是——不是要不要阻止格式化。"

"问题是什么？"

"问题是有多少人会主动帮系统执行格式化。"

陈渡顺着她的目光看过去。人群中,有几个人没有在恐慌。他们在低声交谈。表情不是害怕——是在思考。

他在想:消息发出才十五分钟。十五分钟就能让人开始考虑"主动编译"。

三十天后,还会有多少人站在他这边？

他的手机屏幕亮了一下。视野右上角的倒计时跳动了一格:

28天23小时58分钟。

已经过了一分钟。

（第151章 完）
