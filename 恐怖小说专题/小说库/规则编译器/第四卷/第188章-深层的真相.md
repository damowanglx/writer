# 第188章 深层的真相

当天晚上——陈渡一个人又去了一趟防空洞。

不是系统叫他去——是他感知到了管理员终端在召唤他。不是声音——是一种低频的共振——像一台没有被完全激活的机器——在等待最后的启动信号。

他到了之后发现——管理员终端的屏幕亮着。不是之前那种待机状态的亮——是完全亮。屏幕上没有光标——只有一行字:

```
管理员 #02 —— 完整权限验证已完成。

系统核心层——正在解锁。
请等待。
```

他等了大约两分钟。屏幕闪烁了一下——出现了新的界面。不是之前那个简单的命令行——是一个完整的控制面板。左侧是系统状态——右侧是操作日志——中间是一行大字:

```
格式化程序状态: 深层休眠中(管理员方案生效中)

注意:
格式化程序未被删除——仅被延迟。
管理员方案每90天评估一次——若评估不通过——格式化自动恢复。

永久取消格式化——需要管理员执行以下操作:
1. 确认格式化取消
2. 提供替代指令
3. 替代指令消耗一名编译者的全部规则——该编译者将被编译

替代指令消耗的是编译者的规则——不是寿命。
编译者的规则全部移除后——编译者本身不会死亡。
但——目前没有任何编译者在规则全部移除后存活超过24小时。

警告: 此操作不可逆。
```

陈渡看着屏幕上的文字。

格式化程序没有被删除——只是被延迟了。永久取消——需要消耗一名编译者。

他想到的第一个人是苏墨。苏墨有最多的规则——二十三——他的规则消耗掉之后——系统会获得巨大的稳定性能量。苏墨自己说的——"我的每一条规则都是在打扫垃圾"——如果他的规则被消耗了——等于系统一次性清除了二十三堆垃圾。

但苏墨会死。

没有人在规则全部移除后存活超过二十四小时。

他想到的第二个人是自己。他是管理员。他的规则虽然不多——但他写的每一条都是元规则——影响范围大。如果他消耗掉自己的规则——管理员权限怎么办？

他想到了第三个人——没有人名——是一种想法:不能有任何编译者被消耗。

但系统说——永久取消格式化需要替代指令。替代指令需要消耗一名编译者的全部规则。

没有免费的选择。

他关掉了终端机的屏幕。防空洞里陷入了黑暗。

他坐在黑暗中，坐了很久。

他知道——他必须做出选择。不是现在——但他的方案只是暂缓。九十天后——如果冲突数没有降下来——格式化恢复。如果冲突数降下来了——系统接受方案——但格式化程序还在深层休眠——没有被删除——永远是一颗定时炸弹。

永久取消——需要一个人消失。

他站起来，走出防空洞。凌晨的街道很安静。他在路边站了一会儿，抬头看天。城市的天空看不到星星——只有几盏远处大楼的航空障碍灯在一闪一闪。

他的手机震了一下——赵灵儿。

"你不在家。"

"在防空洞。"

"发现了什么？"

陈渡看着那行字。他犹豫了一下——然后打字:

"永久取消格式化——需要消耗一名编译者的全部规则。"

赵灵儿的回复很快:

"多久？"

"二十四小时——存活时间。"

她没有再回复。

陈渡锁了屏幕。他骑上车，往家的方向去。他骑得很慢。

他在想——他不可能让赵灵儿做这件事。他不可能让苏墨做这件事。他不可能让任何编译者做这件事。

但如果他自己来做——他是管理员——他的规则被消耗之后——系统可能会重置。

他到家的时候——客厅的灯亮着。赵灵儿坐在沙发上——她没有睡。她面前放着那杯已经凉透的水。

她看到他进来，说了一句话——不是问问题——是陈述:

"你不会让任何人做这件事。"

陈渡没有说话。

"你打算自己来。"

陈渡仍然没有说话。

赵灵儿站起来。她走到他面前——很近。

"你想好了——怎么跟我说？"

陈渡看着她。

他没有想好。

他从来都没有想好——怎么跟她告别。

（第188章 完）
