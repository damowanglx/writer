# 第154章 密码线索

老周住在城北一个老小区的六楼,没有电梯。陈渡爬上去的时候,在楼梯间里遇到一个拎着菜篮子的老太太,看了他一眼——可能是被他的表情吓到了。

他敲门。老周开门的速度很快,像是已经站在门后面等了很久。

屋子里很暗。窗帘拉着——不是大白天拉窗帘的那种暗,是住了三十年以上的人家那种习惯了半明半暗的暗。客厅里全是纸箱子、笔记本、发黄的打印纸。墙上贴满了标注过的城市地图,有些是老的地图,有些是手绘的。

老周走到客厅中央,从一摞笔记本底下抽出一本比别的都旧的——封皮是棕色的硬纸板,边角磨白了,拿在手里像拿着一块砖头。

"第一批编译者名单。"他说,把本子放在桌上翻开。

陈渡低头看。一共五个人。钢笔写的,蓝色墨水,字迹工整但有些地方墨水分叉了,说明写的时候笔尖压得很用力。

```
#01 - 周？(图书馆管理员)
#02 - 刘？(已编译 · 1997)
#03 - 方？(已编译 · 2001)
#04 - 赵？(规则冲突死亡 · 1999)
#05 - 孙？(已编译 · 2005)
```

名字都是代号。没有全名。不是刻意隐瞒——是三十多年过去,老周自己也记不清了。

陈渡指着第一行。"#01。"

老周点头。

"他是谁？"

"第一代的第一个。1992年——终端机出现之前两年——他就在图书馆的系统里发现了某种'规则模式'。不是他写的规则——是他观察到的东西。他管它叫'书的排列对读者的影响'。后来他发现,那种影响是可以被写入和修改的。就像——他在终端机出现之前,就已经在无意识地编译了。"

"终端机出现之后呢？"

"他是第一个用的人。也是第一个意识到代价的人。"老周翻开本子后面几页,指着一段用铅笔写的备注——字迹比前面急,有些涂改。"第二天他发现指尖多了一道线。洗不掉。不用医院看也知道——那就是代价。"

陈渡看着那几行字。1994年的标注。距今三十二年。

"后来呢？"

"后来他教出了沈拓。再后来五个人里,四个没了。一个在自己想办法活下来。"

"#01还活着。"

"对。"

"他在哪里？"

老周看了一眼地图。不是看具体位置——是看地图上某个方向。城西。那片区域陈渡跑单很少去——太偏了,老小区多,路窄,没有集中的外卖市场。

"城西老图书馆。"老周说。"你应该路过过。灰色五层楼,门口种了两棵梧桐。正门上方有一块看不清楚的牌子,写着'城西区图书馆'。从外面看像个废弃的建筑——因为也确实废弃得差不多了。但二楼靠东的窗户,晚上会亮一盏灯。他住在图书馆里。"

陈渡回忆了一下。他确实路过过。那次是因为导航导错了路,拐进一条窄巷,一抬头看到一栋被藤蔓包了大半的建筑。他当时以为是废弃的办公楼,没多停留。

"他是#1。"陈渡说。"那台终端机呢？"

"在最深处的书库后面。但——"老周顿了顿,"不是你以为的那种终端机。"

"什么意思？"

"你去看了就知道了。但我得提醒你一件事。"

老周看着陈渡的眼睛。

"他等了三年了。从系统选你那天起,他就知道你会来。但他不是你要找的答案——他是你要找的问题的一部分。密码不在他手里——密码的线索在他手里。"

"密码的线索？"

"沈拓把密码拆成了两半。一半的线索在#01那里。另一半——在苏墨手里。"

陈渡愣住了。

"苏墨一直都知道密码的事？"

"他不知道他知道。"老周说。"他以为自己只是收藏了#23的规则记录——但那里面,有沈拓留下的一半线索。苏墨到现在都没意识到他手里有什么。"

陈渡靠在椅背上。椅子发出吱的一声。

倒计时:25天6小时。

"你把所有事情都串起来了,但每次到最关键的地方就停——这就是你的方式？"陈渡的声音没有怨气,是陈述事实。

老周没有否认。他低下头,翻了两页旧本子,然后合上。

"因为有些事——你知道得越晚,你活得越久。"他说。"但我告诉你一件我已经确认的事:苏墨不是你的对手。"

"他不是？"

"他是你的镜子。你对规则犹豫,他不犹豫。你写规则是为了解决问题,他写规则是为了让系统更完美。你们做同一件事,从不同的方向走——越走越远。但到了终点,你们会站在同一个地方。"

陈渡没有说话。

"去找#01。他等你等得够久了。"

陈渡站起来,走到门口,停了一下。

"老周。"

"嗯。"

"你刚才说苏墨手里有密码的一半线索。如果我找到他——"

"他不会给你的。"

"为什么？"

"因为他的那半密码——不是文字。是一个选择。他不会替你做。你要自己走到那个选择面前——然后决定要不要走那条路。"

陈渡打开门。楼梯间的光涌进来,把地板上那些纸箱和旧地图的影子拉得很长。

他下楼的时候脚步声很重。楼道里回荡着那种老房子特有的、混凝土与铁栏杆共同发出的共鸣。

电动车停在楼下。他跨上去,拧了油门,往城西的方向骑。

骑了大约十分钟,他突然放慢速度。

他看到了那条巷子。三个月前,暴雨夜,他因为导航导错路拐进这里,抬头看见一栋灰色的建筑。当时他只觉得那楼诡异——不像普通的老楼,有一种"不应该存在"的气息。但他在赶单,没多想就骑走了。

现在他停在那条巷子口。

巷子尽头,灰色的建筑立在那里。梧桐树的枝条从围墙里伸出来,遮住了半边路面。午后的阳光穿过树叶的缝隙,在灰色的墙面上落下晃动的碎影。

招牌还在——"城西区图书馆"。字迹已经淡得几乎看不清楚了。

他推着电动车走进去。越接近,越觉得这栋建筑不是普通的图书馆。

它在呼吸。

不是比喻。他走进大门的那一瞬间,感觉到整个建筑像一台巨大的机器——缓慢运转、安静、有规律。他不确定这是不是错觉。但他记得老周说的那句话:

"不是你以为的那种终端机。"

他推开第一道门。灰尘和旧纸的气味涌出来。

（第154章 完）
