# 第168章 管理员的话

陈渡在防空洞的管理员终端前坐了四个小时。

他没有写规则。没有看冲突数。没有回应任何系统通知。他只是在看——管理员终端的三十二层日志目录，他一层层读完。

第三层的日志记录的是编译者的纹路迁移数据——每条规则被写入后，系统的内存占用变化。第五层是格式化的底层代码——三十二年前谁写的第一行。第七层是已经被编译者的意识残片——像深夜电台的微弱信号，在系统深处循环播放。

他本来在找稳定系统的方案。但他看到了第十二层的一行备注:

```
>> 此层下有管理员留言。仅管理员可见。
```

留言。

陈渡点开了第十二层的隐藏目录。里面只有一个文件——没有标题，没有时间戳，只有一段文本。

他读完了全部内容。然后他靠在椅背上，很久没有动。

屏幕上的文字没有消失，光标在最后一行字符下面安静地闪烁着。

他站起来，从口袋里掏出手机，拍了一张屏幕照片。然后他把手机放回去，又坐下了。防空洞里很安静，只有终端机的散热风扇在发出细碎的声响。通风管道里的低频嗡鸣比刚来时减弱了一些——不是系统变好了，是系统在适应新的状态。

他重新看了一遍屏幕上的文字。

不是沈拓写的。

是沈拓的前一任。

---

办公室的门锁坏了十一年了，一直没人修。

我也不想修。因为如果门能锁上，我就有理由把自己关在里面不出去。但门锁坏了——外面的人可以进来，我也就不好意思一直把自己关着了。

这是我在管理员日志里写的最后一篇。不是工作报告——是一封信。

写给以后会读到这段话的人。写给接替我的人。写给可能走和我同一条路的人。

我当了七年管理员。

这七年里，我学会了三件事——

第一件:系统不是工具。它是活的。

我用了差不多三年才确认这件事。不是比喻——它是真的活的。它有偏好。它喜欢某些规则超过另一些。它会在你夜里写规则的时候，帮你补完一个你忘记了后半句的条件从句。它会等你人类写完了，在执行之前帮你检查一遍——像助教帮你批改作业。它希望你写得好。

我花了很多年才敢把它称为"活的"。因为我怕说出来别人觉得我疯了。但你知道吗？如果一件事走起来像鸭子、叫起来像鸭子——它可能真的是一只鸭子。

系统的"活"不是人类的"活"。它没有自我意识，至少不是我们理解的那种自我意识。但它在生长。它的偏好来自它学到的模式——三十二年来，所有编译者写的规则训练了它。它学会了哪些模式是"好的"（可执行、低冲突、高效率），哪些是"坏的"（模糊、自相矛盾、不可维护）。

它不是神。它是一只被三十二年的数据训练出来的——狗。它按你教它的方式去理解世界。

第二件:格式化的真正目的不是维持系统秩序。

格式化是系统的免疫反应。

它不清理规则——它清理"教坏"系统的人。

当系统的"偏好"开始偏离可维护的轨道——当编译者写的规则越来越差、越来越自私、越来越短视——系统会判定:这一批人类训练者不合格。全部清除。换下一批。

格式化不是为了重启系统——是为了重置训练数据。

第三件:管理员不是操控系统的人。

管理员是系统用来判断"要不要格式化"的基准。

你的每一次操作——写的每一条规则、做的每一个决定、每一次倾向性的选择——系统都在学。你在教它什么叫"好"，什么叫"坏"。你写了一条保护弱者的规则——系统学会了"保护"是好的。你猎杀了另一个编译者——系统学会了清除是许可的操作。你袖手旁观——系统学会了不作为也是选项。

管理员不是驾驶员。管理员是样本。

我在意识到这件事的时候，我已经当了五年管理员了。我的纹路到了脖子。

我这辈子写过很多规则。有些让我后悔，有些让我骄傲。但我知道——无论我写的规则是好的还是坏的——系统都记住了。我会成为它衡量后来管理员的标准。

如果我是一个好管理员——它会倾向于"不需要格式化"。

如果我是一个坏管理员——它会得出结论:人类管理员不可靠，格式化是必要方案。

我写了这封信。

不是因为我想告诉你"小心系统"。

是因为我想告诉你——你已经改变它了。

从我之后到你上任之间的这二十几年——系统一直在用我留下的"样本"做基线判断。你接任的那一天——系统会重置基线。从现在开始——你是新的样本。

格式化会不会发生——取决于你教会它什么。

——沈拓

写于1994年7月。第三年管理员任期的最后一周。

灯光:桌上那盏台灯。灯泡在闪但没灭。

---

陈渡把屏幕上的文字读了三遍。

沈拓的留言不是技术文档——是一份遗书。他写这封信的时候，已经知道自己在教系统什么。他已经预见到了系统会变成什么样。他留下密码，留下后门程序，留下"不要接任"——不是因为他想阻止后来的人接任。因为他希望接任的那个人，能比他做得更好。

管理员不是驾驶员。

管理员是样本。

赵灵儿今晚刚写完第七条规则。她的纹路已经到肘上三厘米——接近透明。她不是在写规则——她是在教系统。

陈渡把双手放在键盘上。屏幕的光照着他的脸。防空洞里只有他和终端机。

系统在等他写一条规则。

不是关于交通的。不是关于冲突的。不是关于任何编译者个人的需求——是关于他自己。

他在键盘上敲了第一个字。

（第168章 完）
