# 第156章 守门人

陈渡从负一层上来的时候,天已经全黑了。

图书馆的走廊里没有灯,但他不需要——他的眼睛已经适应了黑暗。走廊尽头,四面目录柜还在运行,绿光从木纹缝隙里渗出,像四堵发光的墙。

沈铮坐在目录检索室正中央的一把木椅上。

椅子的位置变了。陈渡下去前这把椅子是靠墙放的,现在放在房间正中央,正对着四面墙的交点。沈铮坐在那里,像坐在一个十字路口。

"你找到了吗？"沈铮问。

陈渡把那张纸条放在桌上。沈铮看了一眼,没碰。

"密码线索的起点的起点。"陈渡说。"负一层的铁门后面,只有一堵墙。墙上有字——一段话。沈拓写的。最后一句是:'不要接任。'"

沈铮的脸上没有表情。但他的手动了一下——不是抽搐,是那种被某句话击中后、身体比表情更先反应的动作。

"他写的是对的。"沈铮说。

"你知道他写了什么？"

"我不知道具体内容。"沈铮的声音不重,但每个字都很稳。"但他被编译之前,我去看过他。不是去告别——是去阻止他。我没能进去。他把自己关在地下研究站里,整整三天。出来的时候——他已经不是他了。他变成了系统的一部分。但他的意识还在——那三天里,他把自己看到了什么、做了什么,全部写进了系统底层。不是规则——是记录。你们看到的那些日志,就是他离开之前最后的痕迹。"

陈渡沉默了几秒。

"你是他弟弟。"

"亲弟弟。"沈铮说。"同一个父亲。他大我十一岁。1994年,他带我找到了第一台终端机。"

"你也是第一批？"

"第一批的第五个人。"沈铮伸出右手。从指尖到手腕到手肘——填满了纹路。密密麻麻,层层叠叠,像旧纸上的复写。"我写了比我哥更多的规则。但我在编译的前一天——停了下来。"

"怎么停的？"

"我选择了不写。"

陈渡皱眉。"不写？"

"不是停止。"沈铮说。"是选择。"他站起来,走到最近的一面墙前,用手指敲了敲木质目录柜的一格。"你看这些。33万册书的目录。每一册书——从建馆到现在——全在这里。但书本身在五楼书库里。你想知道为什么要保留纸质的书,而不是全部数字化？"

陈渡没有回答。

"因为规则写在纸上看不到的地方。"沈铮说。"你翻开一本书,页边距里——密密麻麻的字符。不是印刷的。是系统写入的。每一条被写过的规则,都出现在某一本书的页边距里。不是全部的书——是特定的书。系统选择书籍作为缓存的载体。像把数据写在最不可能被注意的位置。"

沈铮从口袋里掏出一串钥匙——旧的,生锈的,每一把都磨得发亮。他走向房间最深处的门。五楼书库。

陈渡跟上去。楼梯是木质的,踩上去吱呀作响。墙上的标语还是八十年代那种配色——蓝底白字。"书籍是人类进步的阶梯。"

五楼书库很暗。日光灯只亮了一半。高大的书架从地板延伸到天花板,中间的过道窄得只能一个人通过。空气里是旧纸和灰尘的气味——不是陈腐的,是那种被封存了很久的、干燥的、没有被打扰过的气味。

沈铮走到第二排书架,从中间抽出一本书。

封面是深蓝色的,书名已经褪色了。陈渡接过来。那是一本1987年版的《计算机程序设计艺术》——第三卷。书脊已经开裂,纸张发黄,边角卷曲。

"翻开第671页。"

陈渡照做了。

第671页的正文是排序算法。但在页边距里——从顶部到底部,密密麻麻写满了字符。不是中文,不是英文——是一种陈渡从未见过的符号。像数学符号和某种古老文字的结合体。字符是手写的,墨水是黑的,笔迹非常稳定——没有颤抖,没有涂改。

"这是沈拓的字。"

陈渡抬头看沈铮。不是问句。

"对。"

"这些符号写的是什么？"

"我不知道。"沈铮说。"没有人知道。这是沈拓被编译前留在系统里的最后一段意识痕迹。他写的不是规则——是代码。一种用符号表达的底层代码。三十二年来,我试过所有方法去翻译它——找语言学家、找密码学家、找程序员。没有人能解读。直到我意识到——这段代码不是给人类看的。"

陈渡看着页边距里的字符。一行一行,排列整齐。每行的末尾有一个相同的符号,像一个句号,但多了一横。

"是给谁看的？"

"给管理员终端看的。"沈铮说。"这是一段后门程序。他在被编译前的最后一刻写的——他知道自己会变,所以提前在系统里留下了一个只有后人能找到的漏洞。"

陈渡的手指停在页边距上。他能感觉到什么——不是触觉,是一种非常微弱的"共鸣"。他的纹路在指尖跳动了一下。那些字符,在接触他皮肤的瞬间,像活了一样——极轻微地变动了排列。

他猛地抽回手。

字符又恢复了原来的位置。

"它们认识你的纹路。"沈铮说。陈渡能看到他的表情——不是震惊,是确认。"你写了几条规则？"

"九条。"

"你的纹路来源于你自己写的规则。每条规则有一个基因——你写规则的方式、你选择的措辞、你的逻辑结构——构成了你的编译者签名。#47。沈拓留下的这段代码,检测的不只是管理员权限——它还在等一个特定的编译者编号。"

"我。"

"你。"

陈渡看着页边距里的字符。它们安静地排列在那里,像睡着了一样。

"我刚才碰它的时候——它的排列变了。"

"对。因为你的纹路匹配了它的第一道锁。"沈铮说。"还有两道锁。"

"什么锁？"

"第一:管理员权限。你有。第二:#47的纹路。你是。第三:格式化倒计时的第28天。"

陈渡在心里算了一下。今天是第3天。还有25天。

"为什么是第28天？"

"不知道。"沈铮说。"但沈拓把时间定在那里——说明那一天会发生某件事。或者——说明他从一开始就知道,格式化不会是突然发生的。倒计时本身就是一个过程。每一步都有人在行动。你要在所有人之前,在第28天到达那道锁。"

陈渡合上书,夹在腋下。

"我能带走吗？"

"你最好记住它。"沈铮说。"但你不必现在就带走。等你拿到密码——你会用到这段代码。"

陈渡看了沈铮一眼。

"你刚才说'等你拿到密码'。你知道怎么拿？"

沈铮没有直接回答。他走到书库深处,从另一排书架上抽出一本更旧的书——封面是黑色的,没有书名。他的动作不紧不慢,但每一步都有一种"早就准备好了"的笃定。

他翻开最后一页,从夹层里抽出一张纸。纸很薄,透明到几乎看不清——像硫酸纸。上面用钢笔写着一串符号:

```
∫·∂·∑·∏·→·√·∞
```

七个符号。不是字母,不是数字。是数学符号。

"密码的第一半。"沈铮说。

陈渡接过硫酸纸。七个符号,排成一行。他看不懂。

"这不是给你看的——是给管理员终端看的。你只需要把这串符号输入进去。"

陈渡盯着那些符号看了很久。突然,他想到一个问题。

"你刚才说密码有两半。这一半你守了三十二年——另一半呢？"

沈铮没有立刻回答。他走到窗前,看着外面黑透的街道。梧桐树的影子在路灯下晃动。

"另一半在苏墨手里。他猎杀#23的时候,从规则记录里拿到了。但苏墨不知道那是什么。他以为那是#23的一条私藏规则。"

陈渡听到自己的心跳声在耳边变得清晰。

"如果苏墨知道了真相——"

"他不会给你。"沈铮打断他。"他会自己用。因为他相信只有他能正确地管理这个系统。"

"你见过苏墨。"

"两年前。他来过这里。问了我同样的问题——怎么活。"

"你告诉他了？"

"告诉他了。他没听最后一句。"沈铮的声音里有一种说不清的东西——不是遗憾,不是后悔。"他听完说:'你选择了保存自己。我选择使用自己。'他走的时候,带了图书馆里的一本书——《系统论导论》。封面是灰色的。书脊编号004。他以为他在学系统——其实他在学的是一面镜子。他在那本书里看到的,全是他自己的影子。"

陈渡攥着手里的硫酸纸。

"如果苏墨拿到了完整密码——他会做什么？"

沈铮转过身。在月光和路灯的混合光线下,他的脸看起来比实际年龄更老。纹路从领口延伸到下巴——灰黑色的线条,像血管,但不是血管。

"他不会取消格式化。"沈铮说。"他会按下确认。然后——成为新的管理员。"

"然后呢？"

"然后他会写第一条新规则。在所有人都没有反应过来之前——他已经把系统锁死了。新的系统。新的规则。新的编译者。只有他一个人是管理员。"

陈渡的手机震了一下。倒计时跳了一格。

24天22小时。

他抬头看沈铮。这个老人站在书库深处,四面全是书。33万册。每一册的页边距里都有一条规则。这是三十二年来的全部记录——编译者的生,编译者的死,以及在系统里留下的一切。

"你为什么帮我？"

沈铮沉默了很久。

"因为我哥让我不要写。他说:'不要写最后一条规则。'我没听。我写了——然后我在最后一刻停了下来。"他低头看自己的手。"我活下来了。但三十二年来,我每天都能听到系统的声音。不是命令——是低语。它在说:'你本该被编译的。你逃了。但你会回来的。'"

他抬起头。

"你不一样。你不是想逃——你是想找到另一个可能。"

"你怎么知道？"

"如果你只是想活——你不会来找我。你会像苏墨一样,找一台终端机,写满规则的纹路,让自己强到谁也碰不了你。但你没那么做。"

沈铮看着他。

"你来找密码。不是因为你想活——是因为你想让别人也活。"

陈渡没说话。

但沈铮说对了。
