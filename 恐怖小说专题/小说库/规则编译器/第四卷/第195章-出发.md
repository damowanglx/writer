# 第195章 出发

晚上十点。陈渡站起来。

赵灵儿坐在沙发上——她没有站起来。她的手指在膝盖上交握着——关节发白——但她的表情是平静的。

"要走了？"

"嗯。"

陈渡走到门口,换好鞋。他弯腰系鞋带的时候——动作很慢——不是犹豫——是在记住这个动作——记住在门口弯腰系鞋带的时候——客厅的灯照在地板上的角度——记住她坐在沙发上的姿势——记住这一刻所有细节。

他站直了。回头看了一眼——赵灵儿还坐在沙发上。

"你——"

"我不送你到门口。"她说。"送到门口——我就跟你一起走了。"

陈渡站在门口。门把手在他手里——金属是凉的。

"赵灵儿。"

"嗯。"

"如果我——"

"别说了。"她打断他。"你快走。趁我还没改主意。"

陈渡握着门把手——没有拧开。

他站在门口——站了很久。客厅的灯从她身后照过来——在地板上拉出一道暖黄色的光。她的轮廓——在光里——像一张他看过无数次的照片——但他知道以后再也看不到了。

赵灵儿的声音从客厅传来——很轻——但很稳:

"你写给我的那条保护规则——第七条——还在运行。你走了之后——它也会运行。只要系统还在——那条规则就在。所以——你没走远。"

陈渡拧开了门。

"我走了。"

"嗯。"

他走出去。门在他身后轻轻关上了。锁舌卡进门框的声音——咔哒一声——在楼道里很清晰。

他在楼道里站了两秒——声控灯亮了——黄色的光。他走下楼梯——一层一层。脚步声在楼道里回荡——一楼的感应灯坏了——他在黑暗中走了几步——摸到了单元门的把手。

外面——夏天的夜风——带着柏油路面的余温和草丛里的虫鸣。

他骑着电动车——凌晨的街道很空。路灯一盏一盏往后退。风是凉的——吹在脸上——让他清醒。

他骑过那些他送过外卖的街道——白天挤满车的路——现在空荡荡的。他经过那个他常停车的路口——经过那家赵灵儿说好吃的饺子馆——卷帘门拉着——门口的招牌灯还亮着。

他在等红灯的时候——低头看了一眼自己的手。手指上——纹路还在——从指尖到锁骨——在路灯的光线下——像安静的黑色河流。

他想到赵灵儿说的——"你写给我的第七条规则——还在运行。"

是的。还在运行。

她也会一直运行——如果她走了——她的规则会保留在系统里。

但规则不是人。

规则不能回答他。

规则不能坐在他对面吃饺子。

红灯变绿。他拧了油门。

他骑到废弃农贸市场的时候——铁门上的链条锁和他离开时一样——挂着——没有拧紧。他解开——推门进去。

市场里面——空空荡荡。月光从破损的天窗照进来——在地面上投下白色的光斑。他穿过大厅——推开那扇通向地下的铁门——铰链发出的声音——和第一次来时一样——刺耳——在深夜特别响。

台阶向下延伸。寒意从脚下升起来——越往下越凉。到了一定的深度——地面的热风完全消失了——只剩下地下那种稳定的、不流动的冷。空气里有一股混凝土和铁锈混合的气味。

管理员终端机的屏幕——亮着——绿色光标在黑暗中一明一灭。

他坐下来。屏幕的光照亮了他的脸。

屏幕上——文字还在——和他离开时一样:

```
永久取消格式化 —— 剩余时间: 44小时。

确认取消？ [Y/N]
```

他输入了Y。

屏幕:

```
确认取消格式化。

请提供替代指令。
替代指令将消耗一名编译者的全部规则。

请在下方输入编译者编号。
```

他的手悬在键盘上空。手指没有发抖——很稳。

他输入了#47——他自己的编号。

屏幕:

```
编译者 #47 | 陈渡
规则数量: 10条(含1条元规则)
预计转化能量: 足够。

确认消耗编译者 #47 的全部规则？
此操作不可逆。编译者将在规则全部移除后的24小时内意识消散。

确认？ [Y/N]
```

他的手指放在Y键上——指尖贴着键帽——微凉——塑料的触感。

防空洞里很安静。通风管道里的嗡鸣——像系统的呼吸——低沉——均匀。

他没有按下去。

因为他听到了一声很轻的——脚步声。

在台阶上。

不是幻觉——是真实的——有人从台阶上走下来——步子不快——但很稳——没有犹豫。

他转过头。

防爆门的缝隙里——一个人影站在外面——背着光——看不清脸——但他知道是谁。

赵灵儿。

（第195章 完）
