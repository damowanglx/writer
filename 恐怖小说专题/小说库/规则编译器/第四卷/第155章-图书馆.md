# 第155章 图书馆

门里面不是大厅。是一条走廊。灰色水磨石地面,泛黄的墙皮,日光灯隔三米一盏——有些亮着,有些灭了,有些在闪。

陈渡走了大约三十步才意识到一个问题:这栋楼从外面看没有这么大。他跑单三年,进过一千栋楼,脑子里自动建地图是基本功。但这栋楼的内部结构对不上外部轮廓。

他停下来。

安静。不是一般的安静——是图书馆那种没有声音的安静。但他能感觉到地面在极其轻微地振动。不是地震——是某种大型机械运转的振动。像站在服务器机房里。

走廊尽头是一扇深棕色木门。他推了一下,门无声地开了。

门后面是目录检索室。很大,层高将近四米。四面墙全是木质目录柜——从地板到天花板,几百个小抽屉。

但全是空的。

他随手拉开一个——空的。再拉一个——也是空的。三层、四层——所有的目录抽屉都是空的。他正要关上一个抽屉,手指碰到底部木头的时候,触感不对。

不是木头的触感。是温的。人的体温。

他低头看。抽屉底部的木纹在变化——不是自然纹理,是字。一行行极细的等宽字体从木纹缝隙里浮现。他认出那种字——终端机绿幕上的字体。

他关上抽屉,后退了一步,看向整面墙。

整面墙都在"运行"。

几百块木面板,每一块都在显示文字。不是全部同时亮——是在扫描,像显示屏逐行刷新。信息量太大,眼睛无法一次处理。但他认出其中一行:

```
该读者于2003年借阅《城市地理》。逾期未还。
该读者已无法归还——编译者#19。规则覆盖范围:城中区。
借阅记录已锁定。
```

他转身看另一面墙。同样的。四面墙——四面终端。整栋图书馆不是"安装了终端机"——图书馆本身就是终端机。每一本借阅记录、每一个读者的名字、每一次逾期——全部被系统提取,转化成了规则结构。

"你是#47。"

声音从背后传来。不响,不尖——是很旧的声音。像一个人很久没说话之后第一次开口。

陈渡转身的速度很快——是本能的防御反应。但他看到那个人的时候,本能告诉自己:这个人不是威胁。

目录检索室最深处的阴影里,站着一个老人。

白色短发,梳得整齐。穿着一件旧式的灰色对襟外套——布面扣子,领口洗到发白。他站得很直。右手垂在身侧——从指尖到手腕到手肘到肩膀,一直到领口里面,全是黑色的纹路。不是线条——是满了。整只手都是灰黑色的,像被密集的代码覆盖了一层又一层,深到皮肤失去了原本的颜色。

老人看到了他在看自己的手,没有缩回去。

"你是#47。"

陈渡点头。

"我等你三年了。"

声音里没有情绪。不是激动,不是释然。像在说一个确认了很久的事实。

"你怎么知道是我？"

"你写第一条规则的时候,我感知到了。那个路口——禁止鸣笛。不到五百米处有一条平行规则没被触发——那条规则是我三十二年前写的。两条规则在没有重叠的情况下同时生效——巧合的概率是四十七分之一。当时我就知道,最后一个到了。"

"四十七个编译者。"

"对。"老人说。"三十二年前,第一批五个人。后来四十二个。最后是你。#47——不多不少。"

陈渡环顾着四面正在运行的目录柜。"你一直在这里？"

"我选择不出这栋楼。"老人说。他转身,走向最深处的墙。陈渡跟上去。墙边有一扇小门,推开后是一个极小的房间——五平米,一张木板床,一张书桌,一把椅子。桌上放着一杯冷掉的水,旁边是一本摊开的书。墙上刻满了字——密密麻麻,层层叠叠。

"我把所有规则写在这面墙上。不是建立新规则——是反复修改已经写过的内容。纹路的推进速度会慢到几乎停滞。代价是我不能离开这栋楼——我在图书馆里,规则有效。我走出去,规则断裂,所有修正同时失效——和被编译没有区别。"

陈渡看着墙上那些深浅不一的刻痕。"苏墨来过？"

老人第一次露出不是平静的表情——很细微,但陈渡看到了。"他来过。两年前。问了和你一样的问题——怎么活。"

"你告诉他了？"

"告诉他了。他听完说:'你选择了保存自己。我选择使用自己。'然后他走了。"

老人走到床边,从枕头底下拿出一张对折的旧纸,递给陈渡。

"这是春柳路的档案。1990年的。老城区规划局记录。"

陈渡打开纸条。上面写着一行圆珠笔字:

```
春柳路 —— 因区域规划文件编号94-0001得名。
编号94-0001文件存放处:城西区图书馆 · 负一层 · 旧档案室
```

他抬头。

"沈拓把密码线索的开始,放在了这座城市的第一条规则里。而那一条——是你开始的地方。"

老人看着陈渡的眼睛。

"你下楼。走廊尽头右转,两层台阶,铁门。钥匙在上面插着。"

陈渡把纸条折好放进口袋。走到门口,回头。

"你为什么等了三年才告诉我？"

老人站在那个五平米的房间里,四面墙上全是他的规则。他低头看了看自己的右手——那只被纹路填满的手。

"因为早三年——你不会懂苏墨。现在你懂了。你读完他的演讲了。"

陈渡没说话。

老人说的是对的。三个月前,他会在苏墨的演讲里看到"疯子"。现在他看到了一个逻辑封闭、自洽、没有漏洞的系统。不是苏墨疯了——是苏墨选择了另一条路,把它走完了。

"他走的那条路——你也看到了。"老人说。"但你没跟上去。"

陈渡走出目录检索室。走廊尽头右转,一扇铁门出现在视野里。钥匙插在锁孔里。他拧开锁,手指碰到铁门表面——冰凉的。但凉意传上来的同时,他感觉到:铁门另一边,有某样东西比他更早、更接近这个系统的本质。

他拉开门。

台阶向下延伸,消失在黑暗中。

（第155章 完）
