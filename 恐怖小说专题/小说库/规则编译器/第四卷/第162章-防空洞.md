# 第162章 防空洞

城西废弃的农贸市场——外墙上的白色瓷砖发黄了,铁门上挂着一把生锈的链条锁。看起来锁了很久。但陈渡用手轻轻推了一下——链条锁的另一端,绑在门上的部分已经被人剪断过,重新接上了,但没拧紧。

有人来过。

他拆开链条锁,推门进去。

市场里面是一个空荡荡的大厅。原来的摊位都拆掉了,地上残留着白色的方形印记——每个摊位原来的位置。天花板很高,日光灯灭了大半,剩下的几只发出微弱的嗡嗡声。大厅尽头是一扇铁门——通向地下室,上面挂着一块掉漆的牌子:"防空洞"。

铁门没锁。拉开的时候,铰链发出一声刺耳的金属摩擦声,在地下空间里回荡了很久。

台阶向下延伸。没有灯。陈渡打开手机的手电筒。光圈照亮了脚下的水泥台阶——台阶很宽,大约是标准楼梯的两倍,两边墙面是毛坯的,有些地方渗水,长了青苔。空气的温度在下降——每往下走一层,就凉一点。外面的夏季被隔绝了,取而代之的是地下特有的那种潮湿的、不流动的冷。

他往下走了大约四层楼的高度,台阶终于到头了。

面前是一条走廊。水泥拱顶,两边是房间——防空洞的标准结构。有的房间门开着,里面是空的;有的门锁着,透过门缝看不到任何东西。走廊尽头是一扇钢制防爆门——厚重的圆形门,像潜艇上的那种。门上有一个转轮把手,表面布满了灰尘,但转轮把手的某一段——被擦亮了。

有人来过这里。最近。

他握住转轮,用力转动。很沉,但能转动。三圈之后,锁芯弹开,发出一声沉闷的金属撞击声。他推开门。

门后是一个巨大的圆形空间——大约一个篮球场那么大。天花板是拱形的,混凝土裸露,没有装饰。地面的中央——有一台终端机。

不是他见过的任何一台终端机。

它太大了。机器的底座占据了大约两平方米的面积,外壳是深灰色的金属,表面有散热格栅——不是装饰,是真的散热用,格栅里能感觉到温热的空气在流动。屏幕是横向的,宽屏,熄着。屏幕下方有一排指示灯,全部暗着。再下面是一排插槽——陈渡数了一下,七个插槽。每个插槽的形状和大小都不同。

他在终端机前站了大约十秒。

然后他的目光落在屏幕旁边的一块铭牌上。金属铭牌,螺钉固定。上面的文字:

```
管理员终端 #0
系统核心节点
输入密码以激活。
```

他伸出手,手指在铭牌上划过。金属表面很凉。三十二年了。这台机器三十二年来没有被激活过。沈拓是第一个——也是最后一个使用它的人。

他把手指收回来,深吸了一口气。然后他按下终端机屏幕下方唯一的一个按键——电源键。

屏幕亮了。

不是慢慢亮,是一瞬间。冷白色的背光,黑色的背景,绿色的光标在屏幕左上角闪烁。然后出现了一行字——不是中文,不是英文——是他不认识的字体。但系统自动识别了他的管理员权限,文字切换成了中文:

```
管理员终端 #0 | 系统核心节点
状态: 休眠中 (已休眠 32 年 7 个月 23 天)

请输入管理员密码:
```

光标在"请输入管理员密码:"后面安静地闪烁。

陈渡从口袋里掏出硫酸纸,展开,放在终端机旁边的金属台面上。然后他在记忆中定位了那七个从苏墨规则网络中提取的符号,把它们写在硫酸纸上的七个符号旁边。

左边:∫·∂·∑·∏·→·√·∞。右边:∇·Φ·Ψ·Ω·≈·Δ·Λ。

十四个符号,两组,每一组七个。它们应该怎么拼？是先输入左边再输入右边——还是反过来？是交替排列——还是每组作为一个独立片段输入？

他不知道。

但他知道——他的管理员权限可以让他看到终端机的输入格式。

他把手放在键盘上。键盘不是普通的键盘——没有字母,没有数字。每个键帽上刻的是一组符号。他在那些键中找到了和硫酸纸上匹配的七个——∫、∂、∑、∏、→、√、∞。然后找到了另一组的七个——∇、Φ、Ψ、Ω、≈、Δ、Λ。

他决定:按照被发现的时间顺序。先沈拓的——再苏墨的。

左手按住了∫。右手按住了∇。然后他依次输入:

∫·∂·∑·∏·→·√·∞·∇·Φ·Ψ·Ω·≈·Δ·Λ

最后一个符号输入完成。终端机的屏幕闪烁了一下——不是出错——是确认。

屏幕上的文字消失了。

取而代之的是一行新字:

```
密码验证通过。

系统正在启动...

启动完成。

欢迎。

管理员 #02。
```

陈渡的手指还停在键盘上。

屏幕上继续滚动:

```
系统状态: 不稳定。
格式化倒计时: 21天14小时。

建议操作: 取消格式化。
确认取消格式化？ [Y/N]
```

光标停在"Y"后面,一闪一闪。他只需要按一下回车——或者输入Y——格式化就会取消。一切回到原来的轨道。编译者继续存在。规则继续运行。城市继续不知道发生了什么。

他的手悬在回车键上方。

然后他想起沈拓的最后一句话——"不要接任。"

他停住了。

不是因为害怕。是因为——他接任管理员之后,会发生什么？沈拓没告诉他。沈铮没告诉他。笔记本里只写了"不要接任"——但没有说为什么。

他低头看着自己的纹路——从指尖到手腕到手肘到肩膀到锁骨,越过锁骨,在胸口上方停住了。第十条规则写完了。代价是三分之一的胸口到心脏的距离。他还有多少时间？三十天？或者更短——如果格式化取消之后,系统需要他继续保持管理员身份？

但赵灵儿还在外面等他。

她说"你回来的时候叫我"。

他输入了Y。

屏幕没有反应。光标消失了一秒。然后出现了新的文字:

```
格式化确认取消。

当前状态: 格式化程序已暂停。
系统管理员: #02 (陈渡)。

警告: 系统稳定性仍在下降。建议在7天内执行稳定化操作。
```

然后屏幕暗了一秒——重新亮起。新的文字:

```
陈渡。

我知道你会看到这行字。
或者——这不是巧合。你在读这段话之前,你已经输入了密码,取消了格式化。

你做对了。

但也做错了。

取消格式化是对的——如果你还想让编译者活着。
成为管理员是错的——因为系统的管理权不应该属于任何一个人。

我做了错的那个选择。
我以为我可以控制它。
但它控制了我。

如果你读到了这句话——别走我的路。

——沈拓
```

陈渡盯着屏幕上的文字。那个在负一层墙上的"不要接任"——不是结束,是开始。沈拓留了一段更长的话在管理员的欢迎页面上——在他被编译之前,他就写好了。他知道会有人接替他的位置。他知道那个人会看到这段话。

他站在防空洞深处,圆形穹顶下,通风管道里传来极低沉的嗡鸣声。系统在运行。三十二年来第一次,管理员终端在运行。格式化的倒计时——停止在21天14小时。

他不是来终止系统的。

他是来替代沈拓的。

陈渡慢慢地把键盘推回原位。

这时他听到了声音——不是从终端机里发出的,是从防空洞入口方向传来的。脚步声。很多人。

他转身看向防爆门的方向。

门没关。

门外站着苏墨。他身后——走廊里,至少还有十几个人。陈渡认得其中几张脸——书店里见过的编译者。#31。灰色衬衫女人。还有几个他不认识的。

苏墨没有走进来。他站在防爆门外面,站在走廊里,表情平静得像一潭不流动的水。

"你选了你自己的路。"苏墨说。

陈渡没有说话。

苏墨低头看了一眼——陈渡输入密码之后,系统暂时关闭了所有编译者的"规则实时推送"功能。所有编译者暂时成了"盲人"。

但苏墨不在乎。

"你还记得我说过的吗？"苏墨说。"格式化不是终点——是起点。"

陈渡站在原地。管理员终端的屏幕在他身后亮着——绿光打在他后背上。

"你没有取消格式化。"陈渡说。

"我取消的是你的延迟。"

苏墨的表情没有变。他往前走了一步,但没跨进防爆门。

"我在电视塔上看规则网络的第28天会变成什么——你知道吗？沈拓的后门程序会在第28天启动。系统会暂停格式化二十四小时。之后——格式化继续。"

陈渡的心沉了一下。

"然后——如果你取消了格式化——你不会退出管理员身份。你会被系统绑定。像沈拓一样。"

"那如果我取消了格式化——你做什么？"

苏墨没有回答。他往后退了一步。他身后的编译者们——他们没有动,但陈渡感觉到——他们在跟着苏墨。

"你接任了管理员。"苏墨说。"你现在是系统的一部分了。"

他转身。走廊里的脚步声渐渐远去,最后消失在台阶上。

陈渡独自站在防空洞深处。

管理员终端机的屏幕上,光标闪动着。倒计时停止在21天14小时。

但他知道——他不是倒计时的那个人。

他是倒计时本身。

（第162章 完）
