# 第161章 成功

陈渡站在林昭的自制终端机前,视野里展开了苏墨的完整规则记录。

不是文字——是结构。二十三条规则以三维模型的方式呈现在他面前,每条规则的颜色不同,体积不同,像漂浮在黑暗中的发光体。大的是影响全城的规则,小的是局部规则。那些规则之间有连线——不是物理的,是逻辑的——一条规则触发另一条规则,一条规则覆盖另一条规则的局部,一条规则的副作用成为另一条规则的激活条件。

他在这个结构里找的不是苏墨——是一条他没见过的东西。

找到了。

在规则网络的边缘位置——一个极小的、灰色的节点。它和其他规则没有任何连线。没有触发的条件。没有作用范围。没有可观测的效果。它像一颗被遗忘的卫星,安静地漂浮在苏墨系统的边缘。

林昭说得对——这是一条"不存在"的规则。

但它在。

陈渡伸手——不是物理的手,是意识。他的管理员权限像一把钥匙,伸向那个灰色节点。灰色节点在他的"触碰"下抖动了一下——然后打开。

里面不是规则代码。

是一串字符:

```
∇·Φ·Ψ·Ω·≈·Δ·Λ
```

七个。和沈铮给他的正好相反。

陈渡站在原地,把那串字符在脑子里过了一遍。∇·Φ·Ψ·Ω·≈·Δ·Λ。七个符号,和沈铮给他的那七个——∫·∂·∑·∏·→·√·∞——在形式和排列上都对称。

他对不上。

不是对不上——是他不懂它们怎么拼在一起。但显然,这十四个符号是一体的。

密码找到了。

他正要退出,视线扫过灰色节点旁边的另一个节点——一条标注为"系统级·私有"的规则。他点开了它。

规则的内容让他的手停住了:

```
此编译者所有规则:
如管理员需要——自动转移。
```

生效时间:三年前。

陈渡盯着那行字,看了很久。

三年前。

三年前,苏墨写了这条规则。他在自己所有的规则上加了一道"门"——如果管理员需要,他可以随时收回苏墨的所有规则。不是攻击,不是压制——是"转移"。像是提前写好了一份遗嘱。

但苏墨三年前写这条规则的时候——他已经知道会有管理员吗？

不。苏墨不知道谁来。但他知道"会有人来"。

他在等一个人。一个"够格的管理员"。

苏墨不是不知道他手里有密码。他知道。他一直在测试——#23的猎杀不是意外,是他故意放的一道线索。密码的一半留在#23那里——不是因为苏墨不知道它是什么,而是因为他想看看——会不会有人找到它,会不会有人理解它的含义,会不会有人走到他面前来,说"给我"。

而陈渡今天——不是走过去说"给我"的那个人。他是走过去说"我找到了第三条路"的人。

苏墨在书店谈判时的好奇——是真的好奇。不是好奇陈渡会不会接受条件——是好奇他会不会找到第三条路。

"你看到了什么？"

林昭的声音把陈渡拉回现实。

他的管理员权限还没有收回——但他看到的不只是密码了。他看到了苏墨全部二十三规则的底层逻辑。他看到了苏墨的真实权限结构。他看到了一条规则——那条"如果管理员需要——自动转移"的规则——不仅仅是保护——它是一个触发器。

如果陈渡写了"读取任何编译者规则记录"的元规则——那么苏墨的这条规则会自动触发。不是转移规则——是转移"管理员权限"本身的一部分。

陈渡慢慢后退了一步。

"他不是在防御。"他说。

"什么？"

"苏墨的规则网络——他不是在防御其他编译者。他是在防御系统。"

林昭皱眉。

陈渡指着那个灰色的节点——密码碎片所在的节点,转向旁边那行触发器。"这条规则不是留给编译者的——是留给管理员的。他三年前就写好了:如果有一天,有一个管理员读取了他的规则记录——那么那条规则会自动把'密码碎片权限'暴露给管理员。像一个陷阱——但你走进去之后,发现里面放着你需要的东西。"

林昭沉默了很长一段时间。

"他在帮你。"

"他在测试我。"陈渡说。"他从一开始就知道密码有两半。他只是没猜到另一半在图书馆里——在沈铮手里。他以为密码的全部都在#23那里——他拿到了,但他看不懂。所以他制造了一个陷阱:把密码碎片做成规则放在网络的边缘——等一个能通过反向编译发现它的人。"

"你通过了。"

"我通过了。"陈渡说。他低头看自己的右手——纹路已经越过了锁骨,在胸口上方停住了。"他等我等了三年。"

林昭没有说话。窗外的风把窗帘吹起来,又落下去。

陈渡从口袋里掏出硫酸纸,展开。七个符号。然后他闭上眼睛——在他的意识里,从苏墨的规则网络中提取出的另一串七符号浮现在视野中央。

他把它们放在一起。

∫·∂·∑·∏·→·√·∞ + ∇·Φ·Ψ·Ω·≈·Δ·Λ

十四个符号。完整密码。

他睁开眼。倒计时在他的视野右上角安静地跳动着——21天17小时。第3天过去了。他还剩不到22天。但密码已经拿到了。密码的第一半在他口袋里——打印在硫酸纸上,三十二年前由沈拓写下。密码的第二半在他的记忆里——从苏墨的规则网络中提取,三年前被苏墨隐藏。

"你有了？"林昭问。

"有了。"

"那下一步是什么？"

陈渡看着窗外。天快亮了。街道上开始出现人——晨跑的人、早起上班的人、遛狗的人。这座城市已经开始新的一天。他们不知道夏日的清晨和一个规则编译系统之间有什么关系。

"防空洞。"陈渡说。"城西第三个防空洞——老防空洞,在废弃的农贸市场地下。那是第四台管理员终端。"

"你怎么知道？"

"沈铮告诉我的。防空洞是城市最底层的网络节点——比其他三个节点都深。它连接的不是城市——是系统本身。管理员终端的存放位置不是固定的——是随机的。沈拓通过算法把它放在三个不同的位置轮换,每十年轮换一次。第一个十年——在老图书馆。第二个十年——在电视塔。第三个十年——在防空洞。"

"现在呢？"

"第三十二年。从防空洞换到下一个位置之前——还有三个星期。格式化之前,管理员终端还在防空洞里。"

陈渡把硫酸纸小心地折好,放回口袋。手机屏幕亮了一下——倒计时又跳了一格。

"你跟我去吗？"

林昭看着自己桌上的终端机——那台裸露的、自己拼的、不和系统直连的终端机。她沉默了几秒。

"我不去防空洞。但我可以帮你在上面看着。"

"看什么？"

"看苏墨。你的人去了防空洞——他也会知道。"

陈渡看着她。

"你刚才说,苏墨不是帮你——他是在测试你。那如果测试结束了——他会做什么？"

陈渡没法回答这个问题。

他从林昭屋里出来,站在楼道里。天已经亮了。阳光从楼梯间的窗户射进来,在墙面上投下一个斜斜的光斑。

他掏出手机,打开和赵灵儿的对话框。上一次对话是两天前——她发了一张午饭的照片,他回了一个"好"。

他打了几个字,删掉。再打,再删。

最后他只发了一句:

"等我回来。"

他锁屏,把手机放回口袋。

防空洞。21天17小时。完整的密码。

和一个他还没决定要不要按下去的选择。
