# 第157章 沈拓的遗物

沈铮没有直接回答陈渡最后那个问题。

他只是转身,往书库深处走。灯光越来越暗,书架之间的通道越来越窄。走到最里面,那排书架贴着墙,看起来和其他书架没有区别——但沈铮在书架侧面摸了一下,手指找到了一个极小的凹槽。

他按下去。墙面上无声地滑开了一扇暗门。

门后是一个房间,大约十平米。没有窗户。一盏悬在中间的灯泡亮了——不亮,发黄,但足够让陈渡看清屋里的东西。

一张桌子。一把椅子。一台终端机。

但那台终端机不是陈渡见过的任何一台。它更旧。外壳是灰白色的,不是通常的那种植入式的深灰色。屏幕是凸面的——像九十年代的那种老显示器。键盘是机械的,键帽上的字母已经全部磨掉了,露出下面白色的塑料底。

"这是第二台终端机。"沈铮说。"1996年,沈拓在地下研究站找到的,他搬到这里来。这台终端机没有连接到公共网络——它只连接系统底层。"

陈渡走到桌前。键盘很干净——有人定期擦拭。屏幕上覆盖着一层薄薄的尘,但通电后亮了起来。没有光标闪烁,没有欢迎信息。只有一行字:

```
>等待输入。
```

不是中文。是英文。陈渡不认识那个字体——不是终端机的标准字体,是一种更古老的等宽字体,像从某个更早的系统里移植过来的。

"这台终端机记录了三十二年来所有的管理员操作。"沈铮说。"包括沈拓的——全部。"

他在终端机旁边摸了一下,拿出一样东西。

一个黑色封面的笔记本。很厚。边角磨烂了,封面上的字几乎看不清。陈渡接过来翻了一下——里面不是文字。全是代码。

不是那种规则代码——是更低层的代码。陈渡看不懂全部,但他认出了中间的大段注释——中文注释。沈拓的字。

"这不是规则。"陈渡说。

"不是。"沈铮说。"这是沈拓在被编译前最后三天写的东西。他把自己关在地下研究站里,没有写任何一条规则。他在写这段程序。全部写完之后——他打开了主终端,主动触发了编译程序。"

陈渡翻到笔记本的第一页。沈拓写的第一行注释:

```
如果你读到这段文字——
你已经接任了管理员。
或者你正在走向这条路。
不管哪一个——
你先听我说完。
```

陈渡的手指压在那一行字上。墨水已经渗透了纸背,在下一页留下了模糊的痕迹。

他继续翻。

后面是完整的代码逻辑——用C语言和一种陈渡没见过的汇编混合写成的。注释很密,像沈拓知道读这段代码的人可能不是程序员,所以每一段都做了详细说明。

大约翻了二十页,陈渡看到一段注释被框了起来——不是画框,是用字符拼的框:

```
+----------------------------------------------------+
| 这是你要找的东西。                                   |
| 激活条件:                                            |
|   (1) 管理员权限 — 你已经是                            |
|   (2) #47号编译者的纹路 — 就是你                       |
|   (3) 格式化倒计时的第28天 — 你有25天                  |
|                                                      |
| 三条件全满足后:                                       |
| 在主终端输入以下命令序列 ——                            |
+----------------------------------------------------+
```

下面是一串命令。陈渡看不懂。但他把每个字符都记了下来。

沈铮在他身后开口:"这个东西——我研究了很多年。它不是单纯的漏洞——它是一个替身。"

"替身？"

"如果你在第28天激活这段代码——系统会认为'管理员的操作已经完成'。不是真正的管理员操作——是模拟的。系统会以为格式化已经被取消了,从而暂停格式化程序。但这只是临时效果——真正的取消需要管理员密码。"

陈渡合上笔记本。"那密码必须在那之前拿到。"

"对。你只有最多24小时的时间窗口——从第28天激活到第29天中午。在这24小时内,系统不会执行格式化。你有时间在取消后稳定系统。"

"如果我没在第28天激活呢？"

沈铮看着他。

"格式化正常执行。所有规则、所有编译者的记录、所有被编译者的意识——全部删除。不是暂停——是清理。彻底。"

陈渡的右手不自觉地握紧了笔记本的封皮。纹路从指尖蔓延到手腕,他能感觉到它们的存在——一种微弱的、持续的跳动。

"你为什么告诉我这些？"

"因为我哥让我保留这段程序。"

"他什么时候说的？"

"不是说的。"沈铮说。"他写在第671页的页边距里。在最后一行符号下面——他用中文写了一行字。很小。像备注。"沈铮顿了一下,像在回忆那行字的准确表述。

"'如果我弟来找你——告诉他,我从来不是自愿的。'"

陈渡愣住了。

沈铮没有看他。他低头看着桌上那台老旧的终端机。屏幕的光照在他脸上,把皱纹拉得很深。

"我不明白那句话的意思——直到你把苏墨的演讲带来给我看。我哥被编译之前看到了苏墨。他从一开始就知道——会有像苏墨这样的人出现。不是苏墨本人——是这种人。那种认为'完美系统比人更重要'的人。"

"所以你哥留下的这段程序——"

"不是给管理员用的。"沈铮说。"是给管理员用来对抗系统的。他不是在帮系统完善自己——他是在给下一个管理员留一把刀。"

陈渡站在原地。面前是那台终端机。左手指尖夹着硫酸纸——密码的第一半。右手握着笔记本——沈拓留给他的后门程序。

他低下头。不是看笔记——是看自己的手。纹路从指间延伸出来,在手臂上爬行,到领口之下。每一条都有来处。每一条都对应着他写过的某一条规则。

他马上要写一条新的规则吗？还是他已经不需要再写了？

"我不需要写规则来对抗系统？"

"你需要写一条。"沈铮的声音在黑暗中显得格外低沉。"但不是你现在想的那种。"

"哪一种是我想的？"

"你想写一条'让所有编译者都不被格式化'的规则。"沈铮说。"你做不到。格式化是系统底层程序——任何编译者写的规则都覆盖不了它。你只能通过管理员终端取消它。"

他往前走了一步,站在陈渡面前。

"你需要写的规则很简单:让系统认为你正在完成格式化。"

陈渡皱眉。

"你激活了后门程序之后,系统会进入等待状态。但它不会永远等待——如果长时间没有收到管理员的操作确认,它会自动重启格式化程序。所以你需要写一条规则——一条覆盖时间感知的规则。让系统认为从倒计时第28天到第29天之间——只过了五秒钟。"

陈渡听懂了。

这是一条极端危险的规则。时间感知的覆盖——不是改变客观时间,是改变系统对时间的主观感知。这种元规则级操作需要的权限接近于系统根权限。

"代价？"

"接近你所有的剩余距离。"沈铮说。

陈渡没说话。他知道自己的纹路已经蔓延到哪里——从肩到锁骨的直线距离,大约还剩三指宽。写这条规则——可能只剩一指。或者零。

他想起赵灵儿。她还在等他回去。

"你活不过这个夏天。"沈铮说。不是威胁,不是预言——是陈述事实。"但你能活过这个夏天——如果你只是把密码输入进去,取消格式化。"

"然后呢？"

"然后系统恢复正常。编译者继续存在。新的规则继续产生。新的管理员会接替你。而你——回到自己的生活。"

"那些已经被编译的人呢？"

沈铮没有说话。

"他们还在系统里。苏墨说的有一部分是对的——被编译者的意识融入了系统。他们不是死了——他们变成了系统的一部分。如果格式化了——他们就被删除了。如果取消格式化——他们永远困在系统里,既不是活的也不是死的。"

沈铮看着他。"你同情他们？"

"我不知道。"陈渡说。"但如果我不做点什么——我就和苏墨没有区别。"

他拿起桌上的笔记本,放到自己口袋里。

"密码的第二半在苏墨手里。我需要和他谈一次。"

"他不会给你。"

"我知道。"

"你知道他会提什么条件。"

"我知道。"

陈渡转身往外走。

"陈渡。"沈铮的声音从后面追上来。

陈渡停在门口,没回头。

"你是对的。"沈铮说。"我哥不是自愿被编译的。他以为自己可以操控系统——系统比他快一步。你在走他的路。"

"我该走哪条？"

"我不知道。"沈铮说。"三十二年来,我一直在想——如果1994年那天我哥没带我进那扇门——我会是什么样的人。一个普通的人。一辈子没写过任何一条规则。不知道这个城市底下有八台终端机。不知道页边距里藏着的文字。不知道系统在说话。"

他的手按在旧终端机的键盘上。

"我有时候觉得——那才是最好的结局。"

陈渡走出了门。书库的灯光在他身后一盏一盏暗下来,像有人在他经过之后关掉了开关。等他走到一楼的时候,整栋图书馆只有目录检索室的四面墙还亮着。绿光从木纹缝隙里漏出来,照亮了沈铮独自坐在椅子上的身影。

口袋里,那本笔记本顶着他的肋骨。七个符号在硫酸纸上安静地等待着。
