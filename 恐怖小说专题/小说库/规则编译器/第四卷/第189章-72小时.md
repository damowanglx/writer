# 第189章 72小时

第二天早上——陈渡醒來的时候,赵灵儿已经不在家了。

茶几上留了一张字条。不是纸——是写在便利贴上的,压在遥控器下面。

"我去城东帮一个编译者改规则。中午回来。"

没有多余的话。没有问昨晚的事。

陈渡把便利贴折好,放进口袋。他去厨房倒了一杯水,站在窗前喝。阳光照在地板上——夏天早上的光,又亮又白。

他的手机震了一下。不是赵灵儿——是系统通知。

他打开管理员视野——终端机在凌晨四点钟的时候自动推送了一条消息:

```
管理员 #02:

附加信息:
格式化取消的替代指令——需在管理员完全激活后的72小时内确认。
超时——替代指令选项将关闭——格式化程序恢复默认执行。

倒计时开始。

剩余时间: 67小时。
```

72小时。已经过去了5个小时。

他要么在这67小时内——找到一个人消耗全部规则——永久取消格式化。要么超时——格式化程序恢复默认——他的方案作废。

他放下水杯。没有喝。

他坐在沙发上,双手交握,低着头。他想了很长时间。

第一个方案:消耗自己。他是管理员,规则不算多——九条规则加一条元规则。如果消耗他——系统会失去管理员,但格式化永久取消。系统可能会在失去管理员后自动生成新的管理员——他不确定。

第二个方案:消耗苏墨。苏墨二十三条规则——消耗之后——系统会获得巨大的稳定性能量。但苏墨会死。

第三个方案:消耗一个已经接近被编译的编译者——比如许乐。许乐只剩几天的寿命——如果他主动消耗自己的规则——他可能在规则全部移除后还是活不过二十四小时——但他的死——至少有意义。

陈渡把这个念头赶出了脑子。

他不可能让许乐死。许乐把权限转移给他的时候——他说过:"别让我的规则变成无人认领的孤儿规则。"他答应过。

他在沙发上坐了很久。阳光从地板慢慢移到他脚边。

赵灵儿在中午回来了。

她推门进来的时候,手里拎着两个塑料袋——一个是蔬菜,一个是水果。她把袋子放在厨房台上,然后走到客厅,看到陈渡坐在沙发上——姿势和她走的时候一样。

"你没动过。"

陈渡没有说话。

赵灵儿在他旁边坐下。她也没有说话。两个人并排坐着,看着对面那面白墙。

过了很久——赵灵儿开口了。

"你还有多少时间？"

"67小时。"

她点头。没有问"你打算怎么办"。

她站起来,走到厨房,开始洗菜。水龙头的声音哗啦哗啦的——夏天的中午,阳光很亮,水声很清晰。

陈渡站起来,走到厨房门口。

"赵灵儿。"

她没回头。"嗯。"

"如果我——"

"如果你什么？"

陈渡没有说下去。

赵灵儿把洗好的菜放在案板上,开始切。刀落在砧板上的声音——规律,稳定。

"你昨晚没说完的那句话——'你不会让任何人做这件事'——你想说的是:你想自己来。"

陈渡站在门口,没有否认。

赵灵儿继续切菜。刀切到第二个番茄的时候——她停了一下。

"那如果我告诉你——我昨晚也没睡着。我一直在想——系统说的'消耗一名编译者的全部规则'——不是消耗编译者本人——只是消耗规则。如果我把我的规则全部转移到你身上——消耗的就是你——但规则还在——规则没有消失。"

陈渡愣住了。

"规则转移——不算是消耗——因为规则还在——只是换了一个人持有。系统说的是'消耗一名编译者的全部规则'——它没有说'消耗一名编译者'。"

她转过身。手里拿着刀——但没有在比划。她只是站在那里。

"如果我把它写成一个逻辑漏洞——你的规则和我的规则合并——消耗你——但规则总量不变——系统稳定。"

陈渡看着她。

"代价是什么？"

"代价——你在24小时内的存活概率——低于百分之五十。"

她看着他的眼睛。

"但我活着。"她说。"你也会活着——因为你的规则和我的规则合并后——你就不是"全部规则被消耗"的编译者了——你是一个拥有两个人规则的编译者。"

她转过身,继续切菜。

"这是一个漏洞。不是系统留的——是我找到的。"

陈渡站在厨房门口。阳光照在赵灵儿的背上。她的右臂——十二条纹路——在光线下泛着蓝色的光。

她找到了一个漏洞——一个让她活、让他也活的漏洞。

但代价——是他的存活概率。

"你为什么不早说？"

赵灵儿没有回头。

"因为我昨晚才找到。而且——告诉你之后——你肯定会说'不行'。"

（第189章 完）
