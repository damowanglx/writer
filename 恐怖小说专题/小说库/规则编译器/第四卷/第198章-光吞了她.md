# 第198章 光吞了她

陈渡不知道他在那里站了多久。

可能是几秒——可能是几分钟——可能是更久。

防空洞里什么声音都没有了。终端机的风扇在转——光标在闪——但赵灵儿的声音——没有了。

他慢慢转过身。

赵灵儿站过的地方——空着。

地面上——没有痕迹——没有灰——没有光——什么都没有。连她来时穿的拖鞋的印子——都被光抹掉了。

她站在那里——然后系统确认——然后光吞了她——然后她消失了。

不是死了。

是被编译了。

像沈拓一样——变成了系统的一部分——以规则的形式——永远存在系统底层。

他在最后一刻没有看到那束光——因为他没有回头。但他听到了——他听到了光的声音——不是视觉——是那种极高极细的嗡鸣——像琴弦绷到极限然后松开——然后是安静——完全的安静——连通风管道的嗡鸣都停了一瞬。

像系统在那一刻——也屏住了呼吸。

终端机的屏幕上——显示着确认信息:

```
替代指令已接收。

格式化取消确认。

消耗编译者: #48 | 赵灵儿
规则数量: 12条
转化能量: 充足 —— 系统稳定性提升至82%。

格式化程序状态: 已永久取消。

编译者 #48 状态: 已接受。

规则保留: 12条规则已保存至系统底层。
```

陈渡看着那行字——"规则保留:12条规则已保存至系统底层。"

她的规则还在。

但她不在了。

他慢慢跪下来。

不是跪在终端机前——是跪在她站过的那个位置。膝盖碰到了水泥地面——凉的——硬邦邦的——和这个城市所有水泥地面一样——没什么特别的。

但她刚才站在这里。

他低着头——看着面前的地面——灰色的水泥——有一点灰尘——有一道很浅的划痕——可能是她的拖鞋留下的——也可能是多年前的某个编译者留下的。

他伸出手——手掌按在她站过的那块地面上。

凉的。

什么都没有。

她已经不在这里了。

屏幕上的文字——在他跪下来之后——又出现了一行:

```
管理员 #02:

系统检测到——编译者 #48 的最后一条规则——于消耗指令执行的瞬间——被系统自动注册为永久规则。

规则内容:
"此规则允许任何人提出修改建议。"

此规则——无法被删除。

原因: 规则注册时——编译者 #48 正在被编译——系统将她的最后一条规则识别为"系统的自我声明"——而非普通规则。

永久规则——不受管理员权限影响——不可修改——不可删除——无条件运行。

建议: 接受此规则作为系统底层逻辑的一部分。

——系统
```

陈渡看着那行字。

她的最后一条规则——不是她写进系统的——是系统在她被编译的瞬间——自动注册的。

系统在最后一刻——把她写过的"此规则允许任何人提出修改建议"——识别成了它自己的声音。

一个系统——说——"允许被修改"。

赵灵儿用自己——教会了系统最后一件事:可以被改变。

陈渡跪在那里。他的手按在水泥地面上——手指张开——压着地面——指甲发白。

"系统。"

他开口了。声音在防空洞里回荡——不是很大——但很哑。

"她在哪儿？"

屏幕没有回应。

"你把她——放在哪里了？"

屏幕闪烁了一下——

```
编译者 #48 的规则——已分散集成至系统底层逻辑中。
她不存在于任何单一位置——但她的影响——存在于所有位置。
```

陈渡低下头。

"她——还能说话吗？"

```
编译者 #48 的意识残片——集成于规则结构内。
规则可以执行——但不能对话。

建议: 接受。
```

建议:接受。

系统说的。

不是残忍——是事实。

她已经不在了。留下的——只有十二条规则——和一条永久规则——和一个被改变了底层逻辑的系统。

陈渡跪在那里。终端机的绿光照在他身上。他跪了很久——久到膝盖开始发麻——久到他感觉不到地面的温度。

然后他慢慢站起来。

他的膝盖——有点疼。

他走到终端机前——把屏幕上的信息——读了一遍——又读了一遍。

然后他伸出手——用手指——在屏幕的灰尘上——画了一条线。

不是写字——是画了一条横线。

像在说:我知道了。

像在说:好的。

然后他转身——走出了防空洞。

他走得很慢。每一步都踩实了。台阶一级一级往上——他数着——不是刻意数——是他的脑子自动在数。

二十四级台阶。

他来的时候走了二十四级。

走的时候——还是二十四级。

但走出地面的时候——他觉得——这个世界——已经不一样了。

（第198章 完）
