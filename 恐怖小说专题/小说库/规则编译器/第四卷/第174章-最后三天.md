# 第174章 最后三天

倒计时三天整。

陈渡没有去防空洞。没有去见苏墨。没有去任何终端机前。他在家里坐了整整一下午——客厅的地板上，背靠着沙发，面前放着沈拓的笔记本。

赵灵儿在房间里睡觉。她昨晚写到凌晨四点——第九条规则。纹路已经越过了肩膀。

陈渡没有阻止她。不是因为不想——是因为他发现，每次他想开口说“别写了”，她都会先看他一眼。那个眼神不是请求——是阻止。她知道他要说什么。她不想听。

他翻开沈拓的笔记本。第671页的前面几页——他之前没有细看的内容。沈拓在密码和代码之间，写了一些不是技术内容的东西。像是日记片段。

他翻到一页，上面只有一段话:

```
你问我后不后悔。

不后悔。

但我希望我写第一条规则之前，有人告诉我——规则会改变你，比你会改变规则更快。

我现在看世界的方式和以前不一样了。我看到一扇关着的窗户，我想的是‘打开它需要什么规则’。我看到一个人站在路口，我想的是‘他站的位置是随机的还是有规律的’。

我已经不会像一个正常人一样看世界了。

但我不后悔。

因为这是我选的。
```

陈渡把那段文字读了两遍。然后他合上笔记本，走到阳台。

外面是夏天下午三点钟的太阳。热。明晃晃的。楼下有小孩在骑自行车，铃声叮叮当当地响。一个老人拎着菜篮子慢慢走。世界正常运转。

他的手机震了一下。系统通知——冲突数：一百零七。他从管理员视野里能看到那些冲突的位置——像地图上密密麻麻的红色标记，分布在城市的不同角落。有的在小巷里，有的在主干道上，有的在居民楼里。一百零七处冲突意味着有一百零七对规则在同时运行但互相矛盾。

稳定系统需要的不是暴力删除——是在一百零七处冲突中找到那个“如果不改动任何规则、让它们自动收敛”的平衡点。他不知道这个平衡点存不存在。

他在阳台上站了很久。然后他听到赵灵儿在屋里说话——不是对他说的。是她在睡梦中写规则。

陈渡走回屋里。赵灵儿侧躺着，眉头皱着，嘴唇在动——发出极轻的声音。不是梦话——是规则的文本。她在念。

他坐在床边，没有叫醒她。

她的右臂露在被子外面。纹路从指尖到肩膀——他数了一下，九条规则的纹路。第九条——今天凌晨写的——从肩头往下延伸了几厘米。颜色比之前的几条都深。不是接近透明的蓝——是深蓝。接近黑的深蓝。

她在加速。

不是“越写越多”——是“越写越深”。

陈渡伸手，轻轻碰了一下她的肩膀。赵灵儿没有醒。但她的眉头放松了一点。唇语停止了。

他坐在床边，一动不动。

下午变成傍晚。傍晚变成晚上。他没有开灯。赵灵儿在晚上七点左右醒了。她睁开眼睛，看到陈渡坐在床边，没有惊讶。

“你坐了一下午？”

“嗯。”

赵灵儿坐起来。她低头看了一眼自己的右臂——黑暗中也能看到纹路的轮廓——然后她放下了袖子。

“倒计时还有多久？”

“三天。”

赵灵儿沉默了几秒。然后她说：“我想吃饺子。”

陈渡看着她。

“我现在就想吃。”她说。不是任性——是一种确认自己还是普通人的方式。

陈渡站起来。“我去买。”

他下楼，骑上电动车，去了两条街外那家东北饺子馆。打包了两份韭菜鸡蛋饺子。回来的时候，赵灵儿已经把客厅的灯打开了，碗筷摆好，醋倒在小碟子里。

两个人坐在餐桌前吃饺子。谁都没有说话。

吃到第五个饺子的时候，赵灵儿开口了。

“我有一个问题。”

“你问。”

“如果我写满了——你会怎么办？”

陈渡的筷子停了一下。

“写满”是什么意思——他知道。从指尖到肩膀，从肩膀到锁骨，从锁骨到心脏。纹路走过最后一段距离——人就被编译了。

“你不会写满。”

“你没办法阻止我。”

“我可以。”

“你不能写一条阻止我的规则。”赵灵儿说。“因为你写的每一条规则都在教系统。你写‘阻止赵灵儿写规则’——系统学的不是‘保护赵灵儿’，是‘管理员可以阻止任何人’。”

陈渡没有说话。

赵灵儿夹起一个饺子，蘸了醋，吃掉。嚼完，咽下去。

“你看，你比我清楚。所以你不会写那条规则。”

陈渡放下筷子。

“那你告诉我——我怎么办？”

赵灵儿看着他。餐厅的灯是暖黄色的，照在她脸上，她的表情不是在难过——是在思考。

“你什么都不用做。”她说。“你只需要活着。管理员#02。”

她拿起醋碟，又倒了一点醋。

“我写规则的速度越来越快——不是因为我快了。是因为系统在加速。它在我身上测试一种新的规则写入方式。如果我能在不借助终端机的情况下写规则——其他编译者也可以。系统在开发新功能。我是一台测试机。”

陈渡看着她。

“所以我不是在消耗自己——我是在帮系统找到一种更快的规则写入方式。如果我成功了——以后所有编译者都可以不用终端机写规则。”

“代价呢？”

“代价是我可能撑不到测试完成。”

赵灵儿说这句话的时候，语气没有变化——像在说“这道饺子有点咸”。她低头吃饺子，筷子夹得稳。

陈渡看着她。他知道赵灵儿说的是真的。她不是在牺牲——她是在测试。她是系统选中的测试者。而他是管理员——负责在测试者倒下之后，收集数据。

“你不会倒下的。”陈渡说。

赵灵儿没有回答。她只是把最后一个饺子夹到他碗里。

“吃完早点睡。明天还有事。”

“什么事？”

“系统在等你的答案。”

陈渡看着碗里的饺子。赵灵儿已经站起来，开始收碟子。她的右臂——纹路在灯光下泛着微微的蓝光——在动着。不是她控制的——是纹路自己在动。像一条活着的藤蔓，在她皮肤下伸展。

陈渡低头，把那个饺子吃掉了。

他不知道自己还能和她坐在一起吃几次饺子。

（第174章 完）
