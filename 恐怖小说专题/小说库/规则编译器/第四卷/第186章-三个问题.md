# 第186章 三个问题

方案提交后的第三个小时——系统给陈渡推送了一条新的通知。

不是关于方案执行的——是关于管理员密码的。

```
管理员 #02:

你在第162章时使用的管理员密码——是系统预设的初始密码。
该密码仅用于激活管理员终端。
完整的系统管理权限——需要以下验证。

请输入以下问题的答案:

问题一: 系统为什么存在？
问题二: 规则为什么需要管理员？
问题三: 你为什么要当管理员？

请依次回答。无时间限制。
但连续三次错误——管理员权限将被锁定。
```

陈渡看着这三个问题。

他不是在看问题——他是在看问题背后的意图。这不是系统在测试他的知识——是沈拓留下的。这些问题——是沈拓写的。

因为系统不会问"为什么"——系统只问"怎么"。"为什么"是人类的问题。

他坐在防空洞的管理员终端前。赵灵儿在旁边的地上坐着——她没有跟进来，但她说"我在外面等你"。

他一个人面对屏幕。

第一个问题:系统为什么存在？

他想了很久。不是因为不知道答案——是因为他知道很多个答案。

系统存在——因为沈拓在三十二年前发现了规则网络。
系统存在——因为编译者需要记录规则的地方。
系统存在——因为城市底层有一个由规则构成的操作系统。
系统存在——因为——人需要被管理。

他输入:

"系统存在——不是为管理规则——是为服务人。"

屏幕没有反应。没有说对，没有说错。

然后第二个问题出现了——在他输入第一个问题的答案之后:

"规则为什么需要管理员？"

陈渡看着这个问题。

规则不需要管理员。规则自己运行。就像老周说的——完美的规则引起冲突，不完美的规则留余地——人自己会适应。规则不需要管理员——需要管理员的是系统。因为系统需要一个样本——来判断"要不要格式化"。

他输入:

"规则不需要管理员——系统需要。因为系统需要一个样本——来学习什么是'好'。"

屏幕仍然没有反应。

第三个问题出现了——在他输入第二个问题的答案之后:

"你为什么要当管理员？"

陈渡的手指悬在键盘上空。

他为什么要当管理员？

他没有主动申请过。老周告诉他"系统在等你"的时候——他没说"好"也没说"不要"。他只是在做每一件被推到他面前的事。取消格式化。提交方案。接受管理员权限。

他从来没有——主动说:"我想当管理员。"

他输入:

"我没有选择当管理员——是系统选了我。我接受了——不是因为我想管理——是因为没有人比我更适合。"

他停了一下。

然后他补了一行:

"但我不会一直当。等系统不需要管理员的时候——我会退出。"

他按下了回车键。

屏幕安静了几秒。

然后——

```
答案验证通过。

管理员 #02 身份确认。

权限状态: 受限管理员。

说明:
你选择了'服务'——而非'管理'。
系统已记录此偏好。
你的管理员权限将被限制为:
  - 规则审核: 通过
  - 规则写入: 通过
  - 规则删除: 通过
  - 系统底层修改: 受限
  - 格式化控制: 受限
  - 编译者权限回收: 受限

你验证了密码——但你的选择限制了你。
这不是错误——是你自己选的。

建议: 你可以在任何时候申请完整权限。
申请方式: 在终端机前输入命令: LEVEL-UP。
代价: 纹路至心脏——被编译。

请继续。
```

陈渡看着屏幕。

"受限管理员"。

他输入了三个问题的答案——系统接受了——但因为他的答案倾向于"服务"而非"管理"——系统自动降低了他的权限等级。

不是惩罚。

是他自己选的。

他的答案——系统不是工具——系统是为人服务的——所以他不应该拥有完整的控制权。

他靠在椅背上。

"LEVEL-UP"——只要输入这个命令，他就能获得完整权限。代价是——纹路走到心脏——他被编译。

他不会输入那个命令。

他站起来，走到防空洞门口。赵灵儿坐在台阶上，背对着他。她听到脚步声，没有回头。

"通过了？"

"通过了。但权限是受限的。"

赵灵儿转过头。"受限？"

"因为我回答了'管理不是目的——服务才是'。系统把我的权限降低了。"

赵灵儿看着他。她的表情不是担心——是一种他很少在她脸上见到的表情——柔和。

"你做了对的事。"

"但我权限不够——可能影响方案执行。"

"够的。"赵灵儿站起来。"你需要的权限——审核规则、写入规则、删除规则——都有。你不需要改系统底层——你需要的是让系统自己学会。"

陈渡站在防空洞入口。傍晚的光从外面照进来——地下出口处的光线像一层金色的幕布。

"你知道——如果我输入LEVEL-UP——"

"我知道。"

"——我会被编译。"

"我知道。"

赵灵儿看着他。她的眼睛没有躲闪。

"你不会输入的。因为你不是那种人。"

她转身，往外走。

"走吧。你提交方案的第一天——你还有很多事要做。"

陈渡跟在她后面走出防空洞。

（第186章 完）
