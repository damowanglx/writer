# 第178章 第二条路

赵灵儿写第十条规则的那个晚上，陈渡没有睡着。

他躺在沙发上——她让他睡床，但他睡不着。客厅的窗帘没有完全拉上，路灯的光从缝隙里照进来，在天花板上投下一条细长的亮线。

他盯着那条亮线，在想一件事。

赵灵儿在教系统什么是“人”。但系统学会“人”之后——会不会得出结论：人应该被保护？如果会——那它就不会容忍管理员被编译。它会为管理员开辟第二条路——不是成为系统的一部分，而是被系统“保存”。

但代价是什么？

谁来替代管理员的位置？

他闭上眼睛。客厅很安静。卧室的门虚掩着，他听到赵灵儿在翻身——呼吸平稳，她睡着了。规则写完之后的睡眠很沉，像一台机器在写完之后进入了节能模式。

他慢慢坐起来。

不是因为他想到了什么——是因为他在管理员权限里感知到了一个变化。

系统对“让规则服务于人”的评估——有结果了。

他从茶几上拿起手机。屏幕亮着——不是通知，是一条系统推送，只出现在管理员的视野里：

```
评估结果 
方案: “让规则服务于人”
评估状态: 原则接受

系统已记录此方案为最高优先级原则。
所有现有规则——将在不影响此原则的前提下继续运行。
新规则注册——需通过此原则的一致性校验。

警告:
当前无足够样本定义“服务”与“人”之间的完整关系链。
系统将在运行中持续学习。

学习来源: 管理员操作 | 编译者行为 | 赵灵儿（已标记为主要样本）

其他提示:
系统检测到管理员#02的纹路已覆盖锁骨至胸骨区域。
当纹路到达心脏位置——管理员权限将自动转移。
建议在权限转移前指定继任者。

当前已检测到符合条件的继任候选人: 1人。
```

陈渡的目光停在最后一行。

“当前已检测到符合条件的继任候选人: 1人。”

他的手指按在屏幕上。他没有往下翻，但他知道那个人是谁。

赵灵儿。

系统检测到她是唯一的继任候选人。不是因为她写规则最快——是因为她的纹路颜色已经接近系统底色。系统已经把她识别为自己的延伸。如果陈渡出了什么事——系统会自动把管理员权限交给她。

但代价是——她会被系统彻底吸收。

他锁了屏幕。把手机放回茶几上。他没有告诉赵灵儿这件事。

但他知道——系统不会凭空检测到一个继任候选人。系统一直在观察。一直在记录。一直在评估谁配得上管理员的位置。

它选了赵灵儿。

不是因为她在教系统“人”的定义——是因为她教的那些定义，让系统觉得：她来做管理员，会比陈渡做得更好。

陈渡靠在沙发上。天花板上的光线亮线还在。

他想：如果他猜错了——系统其实没那么聪明——那最好。

如果他猜对了——

他慢慢闭上眼睛。

第二天早上，陈渡醒来的时候，赵灵儿已经做好了早餐——煎蛋、面包、牛奶。简单。标准。

“系统评估通过了。”他在餐桌前坐下的时候说。

“我知道。我感知到了。”

“你还感知到了什么？”

赵灵儿没有立刻回答。她端起牛奶喝了一口，放下。

“感知到了系统在准备管理员权限转移的预案。”

陈渡没有说话。

“预案的触发条件——是你的纹路到达心脏。”

“对。”

“我是候选人。”

陈渡看着她。赵灵儿没有回避他的目光。

“你知道成为管理员意味着什么？”

“知道——意味着我被系统彻底吸收。不是死亡——是成为系统的一部分。像沈拓一样。”

“那你——”

“我不拒绝。”赵灵儿打断他。“不是因为我准备好了——是因为如果我不接，系统会重新格式化的概率很高。”

陈渡沉默了几秒。

“我可以不让纹路走到心脏。”

“你可以。”赵灵儿说。“但你能控制多久？”

他回答不了。

赵灵儿站起来，把盘子收到水槽里。她背对着他洗碗。水龙头的声音在早晨的厨房里很清晰——哗啦哗啦，像在掩盖某种不该被听到的东西。

“我不希望你出事。”她说。“但如果必须有人接替你的位置——我希望是我。”

她把水龙头关上，甩了甩手上的水，转过身。

“因为如果是别人——他不知道怎么保护你。”

陈渡坐在餐桌前。早餐还摆在他面前，但他没有吃。

赵灵儿走过来，把牛奶杯往他面前推了推。

“喝完。凉了。”

陈渡端起杯子，喝了一口。牛奶是温的。

“我不会让你走到那一步。”他说。

赵灵儿没有回答。

她只是微微笑了一下——那种他见过的最浅、最轻、最没有负担的笑。像在说：“我知道你会这么说。”

但她的眼睛里——不是同意。

是她已经做好了准备。

那条他看不到的第二条路——不是他自己走出来的。是她先走上去的。

她是继任候选人。

她是系统的“人”样本。

她是他在这个世界上唯一不想失去的人。

也是唯一一个注定要为这个系统消失的人。

陈渡把牛奶喝完。杯子放回桌上的时候，发出很轻的一声响。赵灵儿已经转过身，在收拾厨房。她的背影很瘦，右臂上的纹路在晨光下泛着微弱的蓝光。十条纹路。从指尖到肩膀。他数过的。

他站起身，走到厨房门口。

“赵灵儿。”

她没回头。“嗯。”

“如果有一天——系统让我选——你死或者系统格式化——”

“你不会有的选的。”赵灵儿说。她仍然没有回头。“因为在你选之前，我已经选完了。”

陈渡站在厨房门口。水龙头又开了。赵灵儿在洗碗。白色的泡沫在她手指间流动。她的手指上也有纹路——从指尖开始。

他知道她已经选完了。

从她写第一条规则的那天起，这个选择就已经做完了。

他只是现在才看懂。

（第178章 完）
