# 第182章 缺陷的证据

老周没有立刻让陈渡走。

他让陈渡坐下，然后从那一摞笔记本里抽出一本——封面上没有年份，只有一行手写的字:"附录:系统崩坏记录"。

"你看完再走。"

陈渡接过来。翻开第一页。

上面是一个表格——时间,触发事件,崩坏程度,恢复方式。

第一条记录:一九九六年三月。

```
触发事件: 沈拓写了一条覆盖全城的"规则优先级判定"规则
崩坏程度: 严重——所有新规则在注册时被自动判定为"低优先级"
恢复方式: 沈拓手动删除了该规则——但纹路代价已支付
```

陈渡看了老周一眼。

"沈拓写过完美规则。"老周说。"那一条他写了三个月。他能预测到每一种规则之间可能的冲突——然后自动排序。理论上——它会让规则网络永远不冲突。但实际运行的时候——它把所有新注册的规则都判定为低优先级——包括消防、医疗、紧急通道的规则。系统停了三天。"

陈渡继续翻。

一九九八年九月。

```
触发事件: 老李写的"街边摊十点收摊"规则与另一条规则冲突
崩坏程度: 中等——两条规则同时运行时造成逻辑死循环
恢复方式: 自然恢复——摊贩自己找到了解决方案
```

自然恢复。没有管理员干预。没有编译者写新规则。系统自己找到了平衡——因为人自己找到了办法。

"你看到没有——人比规则聪明。"老周说。"规则做不到的事——人自己会想办法。"

陈渡继续翻。

二零零五年七月。

```
触发事件: 系统底层数据溢出——因为规则记录太多
崩坏程度: 严重——终端机全部离线
恢复方式: 无——系统在离线状态下自我压缩了数据
```

"那次没有人做任何事。系统自己压缩了旧数据。不是管理员的操作——是系统发现自己快死了——自己写的压缩程序。"

陈渡抬头。"系统自己写了程序？"

"对。它学到的。从三十二年里所有编译者写的规则里——它学会了自我维护。"

陈渡看着那行字。他没有继续翻了。

"所以系统不需要格式化——它本来就会自我维护。"

"会——但不够快。如果规则冲突的速度超过了自我维护的速度——系统还是会崩溃。"

"那怎么才能让维护速度超过冲突速度？"

老周看着他。"减少完美的规则——增加留有余地的规则。"

他拿出另一张纸——一张手画的折线图。

"你看。一九九四到二零零零——规则完美度很高，但冲突率也高。二零零零到二零一零——规则开始变粗糙，但冲突率下降了。二零一零到二零二零——粗糙规则占多数——冲突率最低。二零二零到现在——完美规则又开始多了——冲突率重新上升。"

他把纸推过来。

"你看到趋势了吗？"

陈渡看到了。

"完美规则——冲突率高。不完美规则——冲突率低。"

"对。因为完美规则不允许例外。不允许例外的规则——当现实出现例外的时候——要么规则被破坏——要么人必须写一条新规则来修补。新规则和旧规则不一定兼容——于是产生冲突。"

老周靠在椅背上。

"但不完美规则——自带例外条款。人不需要写新规则——他们已经活在例外里了。"

陈渡看着那张折线图。粗看像一个普通的统计——但细看——他看到了一个系统层面的规律。

完美——导致冲突——导致更多规则——导致更多冲突——导致格式化。

不完美——减少冲突——减少规则——减少管理系统。

"你是怎么知道这些的？"

"我没有写规则——所以我有时间观察。"老周说。"别人在写的时候——我在看。别人在冲突的时候——我在记。别人被编译的时候——我在算。"

他伸手拿起桌上的一杯凉茶，喝了一口。

"三十二年了。我看着一个系统从零到有、从有到满、从满到快崩。我什么也没做——除了记笔记。"

他放下杯子。

"但你现在需要这些笔记。"

陈渡看着桌上那一摞笔记本——三十二年。每一本都是手写的。记录着每一条规则的出现、每一起冲突、每一次编译、每一个死去的编译者的名字。

"如果我把这些——作为我的方案的一部分提交给系统——"

"系统会读。系统能读。"老周说。"但系统不会因为你提交了数据就改变自己——你需要让系统理解——这些数据说明了什么。"

"说明什么？"

老周看着他。

"说明最好的规则——是不需要被执行的规则。"

陈渡愣住了。

"不需要被执行的规则——"

"它的存在——不是为了约束——是为了提供方向。它不是警察——是指南针。"

老周站起来。他把那本附录合上，放回桌子上。

"你一直在想着写一个方案。你女朋友在用自己当教材。苏墨想当建筑师。但你有没有想过——也许最好的方案——不是写出来的——是引导出来的？"

陈渡沉默了很长时间。

"引导——系统？"

"引导系统——让它不把所有规则当指令——而是把一部分规则当建议。"

"系统不会理解'建议'。"

"你女朋友不是在教它吗？"

陈渡没有说话。

窗外的天开始发白。凌晨四点了。倒计时——十九小时。

他站起来，把那本附录夹在腋下。

"我能带走吗？"

"你不能带走。"老周说。"但你走之后——我会把每一页拍下来，发到你的手机上。"

陈渡看着他。

"你不是来借笔记的——你是来确认的。"老周说。"确认你走的方向是对的。"

陈渡没有否认。

老周笑了笑——不是笑给陈渡看的——是笑给自己看的一种很轻的笑。

"方向是对的。但你走的路——比苏墨的路——更难。"

"为什么？"

"因为苏墨只需要写一条规则。你需要——让所有人都不需要写规则。"

（第182章 完）
