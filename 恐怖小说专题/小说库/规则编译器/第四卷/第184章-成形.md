# 第184章 成形

陈渡没有立刻去防空洞。他坐在客厅里，拿了一张纸，一支笔。赵灵儿坐在他对面，没有出声。

他写了三行字:

```
让规则服务于人。
让规则可以被任何人修改。
以上两条——系统自动生成执行细则。
```

他看了一会儿，把第三行划掉了。

"为什么划掉？"赵灵儿问。

"因为'自动生成'——不应该是系统的事。应该是人的事。"

他在第三行的位置重新写了一行:

```
系统应记录所有修改建议——由管理员定期审核。
```

赵灵儿看完。"你还是在做决定的人。"

"对——但我只决定'是否采纳建议'——我不决定'应该有什么规则'。"

"有区别吗？"

"有。我不提出规则——我只筛选规则。提出规则的人——是所有人。"

赵灵儿沉默了一会儿。她端起水杯喝了一口——然后她说:"你知道苏墨不会接受这个方案。"

"我知道。"

"他想要的是从零开始——由他来写第一条规则。你的方案——是从现在开始——让所有人写规则。"

"对。"

"他接受不了。"

陈渡看着他写的三行字。"他不是必须接受——他是必须看到——这个方案比他的好。"

"你怎么证明？"

陈渡没有回答。

他还不知道怎么证明。他的方案——苏墨的方案——系统会怎么评估？系统会倾向于完美——因为系统的底层逻辑就是追求稳定。完美=稳定。不完美=不稳定。

但他的方案——不是完美——是共治。

系统没有"共治"这个概念。

他需要让系统理解——就像赵灵儿在教系统理解"人"一样——他需要教系统理解"参与"。

他站起来。

"我去找林昭。"

"她怎么了？"

"她有自制终端机。我需要她在我的方案提交之前——帮我做一件事。"

赵灵儿没有问什么事。她站起来，把外套脱下来——他的外套——叠好放在沙发上。

"我跟你一起去。"

陈渡看着她。"你应该睡觉。"

"我睡了十一个小时了。"赵灵儿说。"再睡——我就分不清是睡觉还是系统节能模式了。"

她说这句话的时候语气很轻——但陈渡听懂了。她在怕。怕自己睡过去之后——就醒不来了。

"走吧。"

两个人骑一辆电动车。

林昭开门的时候，看到陈渡身后的赵灵儿，愣了一下——她没见过赵灵儿本人。

"进来吧。"

三个人站在林昭那间堆满电路板的小屋里。陈渡把自己写的三行字递给她。林昭看完，沉默了很久。

"你这是在——把管理员权限分出去。"

"不是分出去——是开放输入。"

"结果是一样的。你不再是唯一写规则的人——你是唯一审核规则的人。"

"对。"

林昭看着那三行字。"系统不会理解'审核'是什么意思。系统只理解'执行'和'不执行'。你提交'审核'——系统会把它翻译成'延迟执行'——然后积压。"

"那就需要有人每个月坐下来——审批一次。"

"谁？"

"管理员。"

"你。"

"我。"

"你愿意每个月坐在防空洞里——审批几百条来自陌生人的规则修改建议——一直坐到——坐到什么？"

陈渡没有回答。

赵灵儿替他回答了:"坐到有人接替他。"

林昭看了赵灵儿一眼。然后她低下头，看着那三行字。

"你知道你需要什么吗？"

陈渡摇头。

"你需要一条系统底层的钩子——把'审核'这个人类动作——翻译成系统能理解的'定期执行'。你的方案——系统不会拒绝——但系统会用它的方式执行——而它的方式——是等到积压到一定数量——一次性批准或拒绝。"

"那就不是审核——是垃圾桶。"

"对。所以你需要一条底层钩子——让系统每个月自动生成一份待审核列表——推送到管理员终端——并且在这之前——不执行任何未经审核的修改。"

陈渡看着林昭。"你能写这条钩子？"

"能。"林昭说。"但代价——是你的纹路。"

陈渡低头看了一眼自己的胸口。

赵灵儿伸手——握住了陈渡的手腕。

"我来。"

陈渡看着她。

"我来写这条钩子。"赵灵儿说。"我不是管理员——但我写的规则不需要终端机。钩子不需要管理员权限——它只需要系统底层的访问通道。我有。"

"代价是纹路。"

"我知道。"

"你的纹路已经到肩膀了——再写一条——"

"再写一条——就过了肩膀。"赵灵儿说。"但还没到心脏。"

她松开陈渡的手。

"我写了十一条规则了。前五条是保护你的。后面的是在帮系统。这一条——是帮你的方案落地的。"

她看着陈渡的眼睛。

"你让我跟你来——不就是为了这个？"

陈渡没有否认。

林昭站在两个人中间。她看了看陈渡——又看了看赵灵儿。她转身走到自制终端机前，打开了一个界面。

"写吧。"

赵灵儿走过去。她没有碰键盘。她只是闭上眼睛。

屋子里安静了几秒。

然后赵灵儿睁开眼睛——她右手臂上——纹路从肩头往前延伸了一小段。不到两厘米。但颜色很深——几乎和系统底层的蓝一样深。

"写完了。"

陈渡看着那一段新的纹路。他没有说话。

林昭在终端机前操作了几下，然后抬头。

"钩子生效了。系统已经记录——从今天开始——每月自动生成规则审核列表。"

陈渡站在原地。

他的方案——写完了。三行字。一条钩子。赵灵儿的一段纹路。

现在——他只需要提交给系统。

但他没有动。

因为他知道——提交之后——就没有回头路了。

（第184章 完）
