# 第192章 准备

当天晚上——陈渡趁赵灵儿睡着的时候——去了防空洞。

他没有叫醒她。他留了一张字条——"我去买早饭。很快回来。"——压在茶几上的遥控器下面。他写的时候笔尖在纸上停了一下——因为他知道这不是真话——但他还是写了。

他去了防空洞——但不是去提交确认。他去查了一件事。

管理员终端可以查询所有编译者的规则数量和类型——包括他自己的。他打开了系统数据面板——输入了自己的编号。

屏幕显示:

```
管理员 #02 | 陈渡
规则数量: 10条(含1条元规则)
纹路位置: 锁骨下大约三厘米
纹路进度: 约65%
```

他盯着那行数字——"65%"。十厘米到心脏。如果消耗他——转化量足够。系统会稳定的——格式化永久取消——方案继续运行——只是管理员换了人。

但系统没有备用管理员。他编译之后——管理员位置会空出来——系统会在所有编译者中自动选择下一个——可能是苏墨——可能是林昭——可能是任何一个。

他不确定那个人会不会维持他的方案。

他在管理员终端上又查到了另一个数据——关于编译者#48。

```
编译者 #48 | 赵灵儿
规则数量: 12条(含1条底层钩子)
纹路位置: 肩部
纹路进度: 约40%
```

十二条。比他自己还多。如果消耗她——转化量足够——甚至还有多余。系统会提升到更高的稳定水平。

他关掉了屏幕。

他不可能消耗她。

在黑暗中——他做了一个决定。不是消耗自己——也不是消耗她——是找到沈拓说的那个"没有完全被编译的人"。如果那个人真的存在——如果她可以在系统里存在而不被消耗——那也许——有另一条路。

但他只有64小时。他不知道那个人在哪里——不知道她是谁——甚至不确定沈拓说的是不是真的。

他站起来。椅子腿在地面上刮了一下——发出一声短促的摩擦声——在空旷的地下空间里回荡了一下就消失了。

他走出防空洞。

凌晨的街道很安静。路灯把行道树的影子拉得很长——风吹过来——树叶沙沙地响——影子在地上晃。他走到电动车旁边——没有立刻跨上去。他站了一会儿——凌晨两点的城市——没有人的街道——偶尔有一辆出租车开过——轮胎在柏油路面上发出低沉的声音。

他骑上车——不是回家——是去图书馆。

凌晨三点的图书馆——门锁着。铁栅栏拉在玻璃门后面——里面的灯全灭了——只有走廊尽头的应急灯亮着惨绿色的光。他知道沈铮住在里面——图书馆五楼有一间小屋子——是管理员值班室——沈铮把它当卧室用了。

他敲门——铁栅栏震得哐哐响。敲了好一阵——里面才传来脚步声——慢——但不重。

沈铮出现在玻璃门后面。他披着一件旧外套——里面的衬衫扣子扣错了一颗——明显是刚被叫醒。他看到陈渡的脸——什么也没问——掏出钥匙——打开了铁栅栏。

门吱呀一声开了。

"你又来了。"

"那个人——你还没告诉我——她是谁。"

沈铮站在门廊里。应急灯的光从侧面照过来——在他的脸上投下明暗交错的线条。他的表情不是犹豫——是衡量——像在判断——告诉他之后——会发生什么。

"她是我嫂子。"

陈渡愣住了。

"沈拓的妻子。她也是一名编译者——#00。沈拓被编译之前——花了很长时间——想办法让她不被完全编译。他成功了——但代价是他自己提前被编译。"

"她在哪里？"

"在系统最底层。不是以编译者的身份——是以一段规则的形式。沈拓把她写成了一条规则——一条永远不会被执行的规则——只是存在。"

沈铮看着他。门口的风灌进来——夏天的夜风——带着柏油路面和白天的余温。

"你是第一个知道这件事的人——除了我。"

陈渡站在门口。凌晨的风吹过来——吹在他脸上——但他没有觉得凉。

"如果我把她——从规则还原成编译者——"

"你会消耗她。"沈铮说。"她现在的存在形式——就是一条规则。如果你解除她的规则形态——她就变回编译者——然后被消耗。"

"那——"

"没有别的办法了。"沈铮说。"我哥试了三十二年。他找到的唯一的出路——是把她变成一条不会被执行、不会被删除的规则——永远存在系统里。不是活着——但也不是消失。"

陈渡站在原地。

沈拓用自己被编译的代价——把他的妻子变成了一条规则。不是为了让她活过来——是为了让她不消失。

他也想保护赵灵儿。

但他不想把赵灵儿变成一条规则——永远困在系统里——不能说话——不能回应——只是存在。

他转身。

"你去哪里？"

"回家。"

他骑上车。凌晨的街道——路灯一盏一盏往后退——风声在耳边——他的影子在路灯下被拉长——缩短——又拉长。

他做了一个决定。

不是消耗自己——不是消耗她——也不是把她变成规则。

他要在剩下的时间里——找到第四条路。

（第192章 完）
