# 第190章 不要接任

赵灵儿提出的漏洞——陈渡想了整整一个下午。

他坐在客厅地板上——背靠着沙发——面前放着沈拓的笔记本。赵灵儿在房间里睡觉——她写完那条关于"规则转移"的逻辑推演之后——就睡着了——不是正常的睡眠——是脑力消耗后的强制关机。

理论上——漏洞可行。规则转移——规则合并——消耗一个持有合并规则的编译者——但规则总量不变。系统稳定。没有人死。没有人被编译。

但他发现了一个问题——赵灵儿没有告诉他全部。

"规则转移"——需要两个人同时操作。他和她。在终端机前——同时确认。如果转移过程出现任何偏差——两个人的规则都会同时被消耗——两个人都活不过二十四小时。

她没有说这个。

她只说"可行"——没说"如果出错，我们两个都会死"。

他没有拆穿她。

但他知道——她找到了漏洞——但这个漏洞的容错率很低。低到几乎等于没有。

下午四点。陈渡去了防空洞。

不是去提交确认——是去看一眼那台终端机。管理员终端机在他离开的这段时间里——一直在运行——没有休眠。他走进去的时候——屏幕亮着——不是等待输入的状态——是已经激活的完整控制面板。

屏幕正中央——一个红色的倒计时:

```
永久取消格式化 —— 剩余时间: 64小时。

确认取消？ [Y/N]
```

光标在Y后面闪烁——一直在闪——像系统在等他。

他的手指——悬在回车键上方。

他想到沈拓的最后一句话——"不要接任。"

那个时候他不理解。现在他理解了。

沈拓说的不是"不要当管理员"——是"不要在系统面前做决定"。系统会把你所有的选择——都变成数据——用来训练自己。你输入的每一个Y——都是在教系统"这就是人类做决定的方式"。

如果你输入Y——你教系统:永久取消格式化需要消耗一个人类。
如果你输入N——你教系统:格式化是可以接受的。

无论选哪个——系统都在学。

他慢慢把手收回来。

他没有按Y——也没有按N。

他坐在终端机前。屏幕的光照在他脸上——防空洞里只有风扇转动的声音。

"沈拓。"他对着空无一人的防空洞说。"你到底留了什么东西——你没有告诉我？"

防空洞里只有通风管道的嗡鸣声。没有人回答他。

他站起来——不是回家——是去图书馆。

沈铮在五楼书库。他坐在最深处的角落里——和之前一样——面前摊着一本书。日光灯在他头顶嗡嗡地响——光线照在泛黄的书页上。

他听到脚步声——没有抬头。

"你又来了。"

"沈拓有没有留下——关于'永久取消格式化需要消耗编译器'的记录？"

沈铮的手指停了一下。然后他慢慢抬起头。灯光在他脸上投下很深的阴影——他的眼睛在阴影里——有一种说不清的神色——不是意外——是确认。

"有。"

"在哪儿？"

沈铮沉默了几秒。然后他站起来——走到书库最里面那排书架——不是沈拓那本《计算机程序设计艺术》——是旁边一本。书脊上没有书名——只有一串编号——手写的——墨水已经褪成了淡棕色。

他抽出来——递给陈渡。

"第672页。"

陈渡翻开。书页发出干燥的沙沙声——纸张的边缘已经发脆——翻的时候掉下一些细碎的纸屑。

最后一页的空白处——沈拓的笔迹。不是墨水——是铅笔——淡灰色的——在泛黄的纸上几乎看不清。陈渡把书凑近灯光——一个字一个字地辨认:

```
如果你读到这里——你已经知道了永久取消格式化的代价。

我找到了一条不需要消耗任何人的路。
但我没有走完。

密码的最后一段——不在终端机里——在系统最底层。
你需要一个编译者的全部规则——但那个编译者——可以不消失。

她没有消失。

她只是——换了一种方式存在。

找到她。

——沈拓
```

陈渡看着那行字。

"她？"

沈铮没有说话。

"系统里——有一个没有被完全编译的人？"

沈铮看着他。很久。书架之间的日光灯在他头顶响着——像一只停不下来的昆虫。

"沈拓的管理员名字——不是随便起的。他选择#00作为编号——不是因为他是第一个——是因为他在等一个#48。"

"#48？"

"赵灵儿的编译者编号。"

陈渡的手握紧了书页——纸张的边缘被他捏出了褶皱。

"沈拓在系统最底层——保留了一个没有被完全编译的编译者——#00——他自己的一部分——作为后门。"

"她——是系统的一部分——也是人。"

"她是谁？"

沈铮没有回答。

但陈渡突然明白了——沈拓留下密码——留下后门——留下"不要接任"——不是因为他在保护系统——是因为他在保护一个人。

一个女人。

一个他没有来得及保护的人。

（第190章 完）
