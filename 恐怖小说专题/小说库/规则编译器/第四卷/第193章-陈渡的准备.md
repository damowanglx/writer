# 第193章 陈渡的准备

第二天一早。倒计时:四十八小时。

陈渡醒来的时候——赵灵儿已经出门了。她留了一张字条:"我去城西帮人改规则。中午回来。"

他坐在床边。把字条读了两遍。她的字——写得不大——笔画有一点圆——像她这个人。

然后他做了一件事——他打开管理员终端——不是防空洞那台——是手机上的远程界面。他打开了一个他之前从没打开过的菜单——"管理员权限——继承与转移设置"。

他在里面写了一行指令:

```
管理员 #02 日志:
如果我被编译——系统自动将赵灵儿的编译者状态清除——恢复为普通人。
此操作优先于所有规则。
```

他设置了触发条件:当他被系统标记为"已编译"时——此指令自动执行。

然后他关闭了手机。

他换了一件干净的衣服——白色T恤——她说过他穿白色好看。不是去见谁——是觉得如果要走——不要穿着昨天的衣服走。

他站在镜子前面——看着镜子里的自己。二十四岁。锁骨下面——黑色的纹路像树枝一样分叉——延伸——停在心脏上方大约三厘米的位置。他伸手——隔着T恤——按了一下那个位置。皮肤下面是肋骨——肋骨下面是心脏。纹路还有大约三厘米的路要走。

他放下手——从镜子前走开。

他坐在客厅里。窗帘拉开了一半。阳光从缝隙里照进来——在木地板上投下一道斜斜的光线——光里有细小的灰尘在浮动。

他在想——怎么跟赵灵儿说。

他可以在防空洞里直接执行——不告诉她——等她发现的时候——他已经不在了。但那是沈拓的方式——他不想像沈拓一样——留下一个永远解不开的谜。

他可以选择告诉她——然后让她看着自己走。

他坐在那里——想了很久。阳光从地板上慢慢移动——从他的脚边移到了茶几下面。

中午的时候——赵灵儿推门进来了。

她手里拎着一个塑料袋——里面装着两个饭盒。她换了鞋——看到陈渡坐在客厅里——换了一身干净的衣服——沙发旁边的茶几上放着一杯没动过的水。

她站在门口——没有往里走。

"你决定了。"

不是问句。

陈渡看着她。

"我决定了。"

赵灵儿没有问"你决定什么了"。她走到沙发前——把塑料袋放在茶几上——坐下。她没有打开饭盒——只是坐在那里——两只手放在膝盖上。

"你打算怎么做？"

"去防空洞。输入确认。消耗我自己的规则。"

赵灵儿沉默了几秒。

"你想过告诉我吗？"

"想过。但我不知道——说出来之后——我还能不能走。"

赵灵儿低着头。她的手指在膝盖上轻轻交握——指尖有一点白——握得太用力了。

"你什么时候走？"

"今天晚上。"

赵灵儿点头。她站起来——走到厨房——倒了两杯水。一杯放在陈渡面前——一杯自己握着。

她喝了一口。水在她喉咙里——咽下去的声音很轻。

"那我晚上去送你。"

陈渡看着她。

她说的不是"我晚上去阻止你"——是"我晚上去送你"。

"你不阻止我？"

赵灵儿摇了摇头。

"阻止不了。你也不会听。"

她端着水杯——走到窗前。

"你知道吗——我想到你会走这条路。从你第一次在防空洞拿到管理员权限的那天——我就想到了。"

"那你——"

"我一直在准备。"赵灵儿说。"准备你走的那天——我不哭。"

她转过身。窗外的光从她背后照进来——她的脸在阴影里。

"但我没准备好。"

她握着水杯的手——有一点点抖。

"我没准备好——你走了之后——我怎么办。"

陈渡站起来。他走到她面前——看到她握着水杯的手——指节发白。

然后他伸手——握住了她端着水杯的手。

"你不会没有准备的。因为我给你留了一条路。"

赵灵儿看着他。

"你的编译者状态——会在我被编译之后——自动清除。你会变回普通人。没有纹路。没有规则。没有系统。"

赵灵儿没有说话。

"你不会记得任何关于规则的事——因为那些记忆是和纹路绑定的。你不会记得我——"

"我不会不记得你。"

她的声音很轻——但很肯定。

"编译者状态清除——清的是纹路和系统连接——不清除记忆。"

陈渡愣住了。

"林昭查过。系统底层有记录:编译者状态清除后——意识中关于规则的知识会消失——但关于人的记忆——保留。"

她看着他。

"所以你走了之后——我还会记得你。我会记得你每天早上喝一杯温水——记得你骑电动车的时候左转弯会减速——记得你睡觉的时候眉头总是皱着的——记得你那天在阳台上说'我不会让你走到那一步'。"

她握着水杯的手——已经不抖了。

"我会记得所有的事。只是——不记得规则。"

陈渡站在那里。

他没有想到——她连这个都查过了。

她不是没有准备。

她是把所有准备都做好了——只等他做决定。

（第193章 完）
